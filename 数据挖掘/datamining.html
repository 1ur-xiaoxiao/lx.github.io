Chapter 02

线性回归

Linear Regression

数学分析 高等代数 概率论 数理统计 前置知识 ↑

机器学习的"Hello World"。假设目标变量 $y$ 服从以 $Xw$ 为均值的正态分布，极大似然估计等价于最小二乘。岭回归和 Lasso 分别引入 L2/L1 正则化，从贝叶斯视角对应高斯先验和拉普拉斯先验。

数学推导

1. 最小二乘解析解（正规方程）

损失函数：$L(w) = \|y - Xw\|^2 = (y-Xw)^T(y-Xw)$

对 $w$ 求梯度：$\frac{\partial L}{\partial w} = -2X^T(y - Xw)$

令梯度为零：$X^TX \, w = X^Ty$，若 $X^TX$ 可逆，则：

$$w^* = (X^TX)^{-1}X^Ty$$

2. 岭回归（L2 正则化）

损失：$L(w) = \|y-Xw\|^2 + \lambda\|w\|^2$

解析解：$w^* = (X^TX + \lambda I)^{-1}X^Ty$

贝叶斯解释：等价于先验 $w \sim N(0, \tau^2 I)$，其中 $\lambda \propto 1/\tau^2$。

3. Lasso（L1 正则化）与软阈值算子

损失 $L(w) = \|y-Xw\|^2 + \lambda\|w\|_1$ 无解析解，用坐标下降法求解。第 $j$ 个系数的更新使用软阈值规则：

$$S(z, \gamma) = \text{sign}(z) \cdot \max(|z| - \gamma, 0)$$
与数学系课程的联系：
  • 高等代数：$X^TX$ 的正定性、矩阵求逆、伪逆 $\Rightarrow$ 正规方程的可解性
  • 数学分析：多元函数求极值、梯度为零的必要条件
  • 概率论：正态分布假设 $y|X \sim N(Xw, \sigma^2 I)$，极大似然估计
  • 数理统计：偏差-方差权衡——$\lambda$ 越大，偏差↑方差↓

Python 实现

正规方程求解

def fit(self, X, y):
    if self.fit_intercept:
        X = np.column_stack([np.ones(X.shape[0]), X])
    XtX = X.T @ X
    Xty = X.T @ y
    # 用 solve 而非求逆，数值更稳定
    try:
        theta = np.linalg.solve(XtX, Xty)
    except np.linalg.LinAlgError:
        theta = np.linalg.pinv(XtX) @ Xty

Lasso 软阈值函数

@staticmethod
def _soft_threshold(z, gamma):
    """S(z, γ) = sign(z) · max(|z| - γ, 0)"""
    if z > gamma:   return z - gamma
    elif z < -gamma: return z + gamma
    else:            return 0.0

关键观察

  • 岭回归系数较小但整体非零（L2 只收缩不归零）
  • Lasso 将部分系数压缩到恰好为零（L1 实现特征选择）

Chapter 03

K 近邻

K-Nearest Neighbors

数学分析 高等代数 概率论 前置知识 ↑

最直观的非参数方法——"近朱者赤"。不做任何分布假设，基于距离度量进行多数投票。KD 树通过递归空间划分将搜索复杂度从 $O(n)$ 降至 $O(\log n)$。

数学推导

1. 闵可夫斯基距离

$$d(x, x') = \left(\sum_{j=1}^{d} |x_j - x'_j|^p\right)^{1/p}$$

$p=2$ 为欧氏距离，$p=1$ 为曼哈顿距离。这是 $L^p$ 范数的直接应用。

2. 最近邻误差率上界

设 $P^*$ 为贝叶斯最优错误率，最近邻分类器的错误率 $R$ 满足：

$$P^* \leq R \leq 2P^*(1-P^*)$$

即最近邻的错误率不超过贝叶斯最优的两倍。当 $P^*$ 很小时，$R \approx 2P^*$。

3. KD 树划分

递归沿坐标轴取中位数划分空间，搜索时利用三角不等式剪枝。复杂度 $O(n\log n)$ 建树，$O(\log n)$ 查询。

与数学系课程的联系：
  • 数学分析：度量空间公理、$L^p$ 范数、三角不等式
  • 高等代数：向量空间、内积与距离的关系
  • 概率论：条件概率、贝叶斯最优分类器、错误率分析

Python 实现

KNN 多数投票预测

def _predict_one(self, x):
    # 闵可夫斯基距离: d(x,x') = (Σ|x_j-x'_j|^p)^{1/p}
    distances = np.sum(np.abs(self.X_train_ - x) ** self.p, axis=1) ** (1/self.p)
    knn_indices = np.argsort(distances)[:self.n_neighbors]
    knn_labels = self.y_train_[knn_indices]
    return Counter(knn_labels).most_common(1)[0][0]

可视化与交互

调节 K 值观察决策边界变化，鼠标悬停查看近邻

关键观察

  • KNN 是惰性学习——训练只存储数据，所有计算在预测时进行
  • K=1 时对噪声极度敏感（高方差），但训练误差为 0
  • K 过大会把远处不相关的样本也纳入投票（高偏差）
  • 高维空间中距离趋同（维度灾难），KD 树退化为暴力搜索
Chapter 04

朴素贝叶斯

Naive Bayes

概率论 数理统计 前置知识 ↑

以贝叶斯定理为基石，"朴素"地假设特征条件独立。看似简单，却在文本分类等领域大放异彩——生成模型的经典代表。

数学推导

1. 贝叶斯定理

$$P(C_k|x) = \frac{P(x|C_k) \cdot P(C_k)}{P(x)}$$

2. 条件独立假设

"朴素"假设：给定类别后各特征相互独立：

$$P(x|C_k) = \prod_{i=1}^{d} P(x_i|C_k)$$

3. 高斯朴素贝叶斯的对数后验

假设 $P(x_i|c) = N(\mu_{c,i}, \sigma^2_{c,i})$，取对数：

$$\log P(c|x) \propto \log P(c) + \sum_i \left[-\frac{1}{2}\log(2\pi\sigma^2) - \frac{(x_i-\mu)^2}{2\sigma^2}\right]$$

4. 拉普拉斯平滑

$$P(x_i|c) = \frac{\text{count}(x_i, c) + \alpha}{\sum_i \text{count}(x_i, c) + \alpha \cdot n}$$

$\alpha=1$ 即拉普拉斯平滑，防止零概率导致整个后验为零。

与数学系课程的联系：
  • 概率论：贝叶斯公式、条件概率、先验与后验、全概率公式
  • 数理统计：极大似然估计 $\hat{\mu} = \bar{x}$，$\hat{\sigma}^2 = \frac{1}{n}\sum(x_i-\bar{x})^2$

Python 实现

高斯朴素贝叶斯——参数估计

def fit(self, X, y):
    for i, c in enumerate(self.classes_):
        X_c = X[y == c]
        self.priors_[i] = len(X_c) / len(X)
        self.means_[i] = np.mean(X_c, axis=0)
        self.vars_[i] = np.var(X_c, axis=0) + 1e-9  # 防零方差

点击画布任意位置查看该点的后验概率

关键观察

  • 条件独立假设在现实中几乎不成立，但朴素贝叶斯仍表现良好
  • 用对数计算而非连乘是为了数值稳定（log 将乘法转为加法）
  • 朴素贝叶斯是生成模型：先学 $P(x|y)$ 和 $P(y)$，再推 $P(y|x)$
Chapter 05

逻辑回归

Logistic Regression

数学分析 高等代数 概率论 数理统计 前置知识 ↑

名字带"回归"，本质是分类。通过 Sigmoid 将线性输出映射到概率空间，用极大似然估计求解。损失函数是凸的，保证梯度下降找到全局最优。

数学推导

1. Sigmoid 函数

$$\sigma(z) = \frac{1}{1+e^{-z}}, \quad \sigma'(z) = \sigma(z)(1-\sigma(z))$$

2. 交叉熵损失（伯努利 MLE）

假设 $y|x \sim \text{Bernoulli}(\sigma(w^Tx))$，负对数似然：

$$J(w) = -\frac{1}{n}\sum_i \left[y_i\log\sigma(w^Tx_i) + (1-y_i)\log(1-\sigma(w^Tx_i))\right]$$

3. 梯度与 Hessian

$$\nabla J = \frac{1}{n}X^T(\sigma(Xw) - y)$$ $$H = \frac{1}{n}X^T \text{diag}(p(1-p)) X \succeq 0$$

$H$ 半正定 $\Rightarrow$ $J(w)$ 是凸函数 $\Rightarrow$ 任意局部最优即全局最优。

4. 牛顿法更新

$$w \leftarrow w - H^{-1}\nabla J$$

二次收敛速度，但每步需计算并求逆 Hessian 矩阵。

与数学系课程的联系：
  • 数学分析：凸函数判定、Taylor 展开、Hessian 矩阵、极值条件
  • 高等代数：正定/半正定矩阵、矩阵求逆
  • 概率论：伯努利分布、指数族、典型连接函数
  • 数理统计：极大似然估计、Fisher 信息量 $I(w) = E[H]$

Python 实现

梯度下降与牛顿法

grad = X_aug.T @ (p - y) / n_samples
if self.method == 'gradient_descent':
    update = self.lr * grad
elif self.method == 'newton':
    D = p * (1 - p)
    X_weighted = X_aug * np.sqrt(D[:, np.newaxis])
    H = X_weighted.T @ X_weighted / n_samples
    update = np.linalg.solve(H, grad)
self.w_ -= update

可视化与交互

调节 Sigmoid 斜率，对比 GD 与 Newton 收敛速度

关键观察

  • 牛顿法通常 5-10 次迭代就收敛（二次收敛速度）
  • 梯度下降需要更多迭代，且对学习率敏感
  • 交叉熵 + Sigmoid 的组合保证了凸性（全局最优存在）
  • 逻辑回归是最大熵模型在二分类时的特例
Chapter 06

最大熵

Maximum Entropy

数学分析 概率论 前置知识 ↑

在满足已知约束的前提下，选择熵最大的分布——"不做多余假设"的哲学。通过拉格朗日对偶推导出指数族形式，与逻辑回归殊途同归。

数学推导

1. 信息熵

$$H(P) = -\sum_{i} p_i \log p_i$$

均匀分布时熵最大（$H = \log n$），退化分布时熵最小（$H = 0$）。

2. 约束优化问题

$$\max_{P} H(P) = -\sum_{x,y} \tilde{P}(x)P(y|x)\log P(y|x)$$ $$\text{s.t.} \quad \sum_y P(y|x) = 1, \quad E_P[f_i] = E_{\tilde{P}}[f_i]$$

3. 拉格朗日对偶 → 指数族形式

引入拉格朗日乘子 $w_i$，求解得最优解为：

$$P_w(y|x) = \frac{1}{Z_w(x)} \exp\left(\sum_i w_i f_i(x,y)\right)$$

其中 $Z_w(x) = \sum_y \exp(\sum_i w_i f_i(x,y))$ 是配分函数。

4. 梯度 = 模型期望 - 经验期望

$$\frac{\partial L}{\partial w_k} = \frac{1}{n}\sum_i \left[P(y=k|x_i) - I(y_i=k)\right] \cdot x_i$$
与数学系课程的联系：
  • 数学分析：约束优化、拉格朗日乘子法、KKT 条件
  • 概率论：信息熵、KL 散度 $D_{KL}(P\|Q) = \sum P\log\frac{P}{Q} \geq 0$

Python 实现

熵的计算

def entropy(p):
    """H(P) = -Σ p_i log p_i"""
    p = np.asarray(p)
    p = p[p > 0]  # 0·log(0) = 0（取极限）
    return -np.sum(p * np.log(p))

可视化与交互

拖动滑块选择 p 值，观察二值分布的熵变化

关键观察

  • 最大熵 = "不说不知道的事"——在约束下选最均匀的分布
  • 对偶函数是凹函数，可用任意凸优化方法求解
  • 最大熵模型的多分类形式等价于 softmax 回归
  • 梯度 = 模型期望 - 经验期望，直观反映约束满足程度
Chapter 07

决策树与随机森林

Decision Trees & Random Forests

概率论 数理统计 前置知识 ↑

用"分而治之"的策略递归划分特征空间。从 ID3、C4.5 到 CART，每次划分让子节点纯度最高。随机森林通过 Bagging + 随机特征选择显著降低方差。

数学推导

1. 信息熵

$$H(D) = -\sum_{k=1}^{K} p_k \log_2 p_k$$

2. 信息增益（ID3）

$$g(D, A) = H(D) - \sum_v \frac{|D^v|}{|D|} H(D^v)$$

选增益最大的特征。偏向取值多的特征。

3. 信息增益比（C4.5）

$$g_R(D, A) = \frac{g(D, A)}{H_A(D)}, \quad H_A(D) = -\sum_v \frac{|D^v|}{|D|}\log_2\frac{|D^v|}{|D|}$$

4. 基尼指数（CART）

$$\text{Gini}(D) = 1 - \sum_{k=1}^{K} p_k^2$$

统计直觉：随机抽两个样本，它们类别不一致的概率。

5. 随机森林方差上界

$$\text{Var} \leq \rho\sigma^2 + \frac{(1-\rho)\sigma^2}{B}$$

$\rho$ 为树间相关系数，$B$ 为树的数量。随机特征选择降低 $\rho$。

与数学系课程的联系：
  • 概率论：信息熵的凸性、$\log$ 函数性质
  • 数理统计：偏差-方差分解——单树低偏差高方差，森林低偏差低方差

Python 实现

节点不纯度计算

def _node_impurity(self, y):
    _, counts = np.unique(y, return_counts=True)
    probs = counts / counts.sum()
    if self.criterion in ('entropy', 'gain_ratio'):
        return -np.sum(probs * np.log2(probs + 1e-15))
    elif self.criterion == 'gini':
        return 1 - np.sum(probs ** 2)

  • 基尼指数计算更快（无需 log），实践中效果与熵相近
  • 随机森林通过 Bagging + 随机特征，显著降低方差
Chapter 08

感知机

Perceptron

高等代数 数学分析 前置知识 ↑

神经网络的"老祖宗"。寻找超平面将数据线性可分，用误分类驱动的 SGD 更新参数。Novikoff 定理保证在线性可分条件下有限步收敛。

数学推导

1. 模型形式

$$f(x) = \text{sign}(w^Tx + b)$$

2. 损失函数（误分类驱动）

$$L(w, b) = -\sum_{\text{误分类}} y_i(w^Tx_i + b)$$

几何意义：误分类点到超平面距离之和的符号修正版。

3. SGD 更新规则

对每个误分类样本 $(x_i, y_i)$：

$$w \leftarrow w + \eta \cdot y_i \cdot x_i, \quad b \leftarrow b + \eta \cdot y_i$$

为什么有效？若 $y_i=+1$ 被误分为 $-1$，则 $w^Tx_i+b < 0$，更新后 $w^Tx_i$ 增大 $\eta\|x_i\|^2 > 0$。

4. Novikoff 收敛定理

若数据线性可分，$\|x_i\| \leq R$，存在 $\gamma > 0$ 使 $y_i(w^{*T}x_i+b^*) \geq \gamma$，则误分类次数 $\leq (R/\gamma)^2$。

与数学系课程的联系：
  • 高等代数：超平面 $w^Tx+b=0$、内积、线性可分性
  • 数学分析：SGD 收敛性分析、序列极限

Python 实现

感知机训练循环

for epoch in range(self.max_iter):
    n_errors = 0
    for i in range(n_samples):
        if y[i] * (np.dot(self.w_, X[i]) + self.b_) <= 0:
            self.w_ += self.lr * y[i] * X[i]
            self.b_ += self.lr * y[i]
            n_errors += 1
    if n_errors == 0:
        break  # 所有样本正确分类

可视化与交互

点击"训练一步"观察超平面如何调整

关键观察

  • 感知机只在线性可分数据上收敛（Novikoff 定理保证）
  • 解不唯一——初始值和样本顺序不同，解也不同
  • 感知机是神经网络的基础，多层感知机 → 深度学习
Chapter 09

支持向量机

Support Vector Machine

高等代数 数学分析 前置知识 ↑

追求最大间隔的优雅分类器。从硬间隔到软间隔，从核技巧到 SMO 算法——几何直觉与凸优化的完美结合。解只由支持向量决定，天然稀疏。

数学推导

1. 硬间隔原始问题

$$\min_{w,b} \frac{1}{2}\|w\|^2 \quad \text{s.t.} \quad y_i(w^Tx_i + b) \geq 1, \forall i$$

间隔 $= 2/\|w\|$，最大化间隔 $\Leftrightarrow$ 最小化 $\|w\|^2/2$。

2. 拉格朗日对偶

引入乘子 $\alpha_i \geq 0$，由 KKT 条件 $w = \sum_i \alpha_i y_i x_i$，对偶问题：

$$\max_\alpha \sum_i \alpha_i - \frac{1}{2}\sum_{i,j}\alpha_i\alpha_j y_iy_j x_i^Tx_j$$ $$\text{s.t.} \quad \sum_i \alpha_i y_i = 0, \quad \alpha_i \geq 0$$

3. 软间隔

引入松弛变量 $\xi_i \geq 0$，惩罚参数 $C$：

$$\min \frac{1}{2}\|w\|^2 + C\sum_i \xi_i \quad \text{s.t.} \quad y_i(w^Tx_i+b) \geq 1-\xi_i$$

对偶形式仅约束变为 $0 \leq \alpha_i \leq C$。

4. 核技巧

将 $x_i^Tx_j$ 替换为核函数 $K(x_i, x_j) = \phi(x_i)^T\phi(x_j)$：

线性核：$K = x^Tx'$，RBF 核：$K(x,x') = \exp(-\gamma\|x-x'\|^2)$

与数学系课程的联系：
  • 高等代数：二次型、正定矩阵、核矩阵的半正定性
  • 数学分析：拉格朗日对偶、KKT 条件、凸优化、强对偶定理

Python 实现

SMO 核心——更新两个乘子

eta = self._K[i,i] + self._K[j,j] - 2*self._K[i,j]
if eta <= 0: continue
self.alpha_[j] += y[j] * (E_i - E_j) / eta
self.alpha_[j] = np.clip(self.alpha_[j], L, H)
self.alpha_[i] += y[i]*y[j]*(alpha_j_old - self.alpha_[j])

  • SVM 只依赖支持向量——其他样本对模型没有影响
  • C 越小 → 间隔越大 → 允许更多误分类 → 偏差↑方差↓
  • 核技巧隐式地做特征映射，无需显式计算高维特征
  • SMO 每次只优化 2 个 $\alpha$，把大 QP 问题分解为可解析求解的子问题
Chapter 10

EM 算法

Expectation-Maximization

数学分析 概率论 数理统计 前置知识 ↑

处理含隐变量模型的利器。E 步求期望，M 步求极大——迭代逼近最大似然估计。单调性保证每次迭代似然不会下降。

数学推导

1. 不完全数据似然

$$\ell(\theta) = \log P(X|\theta) = \log \sum_z P(X, Z|\theta)$$

直接优化困难（log 内有求和）。EM 将其转化为迭代优化 Q 函数。

2. E 步：构造 Q 函数

$$Q(\theta|\theta^{(t)}) = E_{Z|X,\theta^{(t)}}[\log P(X,Z|\theta)]$$

3. M 步：最大化 Q 函数

$$\theta^{(t+1)} = \arg\max_\theta Q(\theta|\theta^{(t)})$$

4. 单调性（Jensen 不等式）

$$\ell(\theta^{(t+1)}) \geq \ell(\theta^{(t)})$$

5. GMM 的 E 步 / M 步

E 步（责任值）：$\gamma_{ik} = \frac{\pi_k \cdot N(x_i|\mu_k,\Sigma_k)}{\sum_j \pi_j \cdot N(x_i|\mu_j,\Sigma_j)}$

M 步：$\mu_k = \frac{\sum_i \gamma_{ik} x_i}{\sum_i \gamma_{ik}}$，$\pi_k = \frac{\sum_i \gamma_{ik}}{n}$

与数学系课程的联系：
  • 数学分析：Jensen 不等式 $f(E[X]) \leq E[f(X)]$（$f$ 凸）、迭代优化收敛
  • 概率论：条件期望、混合分布、全概率公式
  • 数理统计：含隐变量的极大似然估计、Fisher 信息

Python 实现

E 步——计算责任值

def _e_step(self, X):
    responsibilities = np.zeros((n_samples, self.n_components))
    for k in range(self.n_components):
        rv = multivariate_normal(mean=self.means_[k], cov=self.covars_[k])
        responsibilities[:, k] = rv.pdf(X)
    responsibilities *= self.weights_
    responsibilities /= responsibilities.sum(axis=1, keepdims=True)
    return responsibilities

  • EM 只能保证局部最优，结果依赖初始化
  • 广义用于 GMM、HMM、缺失数据、因子分析等
Chapter 11

提升方法

Boosting

数学分析 概率论 数理统计 前置知识 ↑

三个臭皮匠，顶个诸葛亮。从 AdaBoost 到 GBDT 再到 XGBoost，理解"加法模型 + 前向分步 + 指数损失"的统一框架。

数学推导

1. 指数损失与前向分步

$$L = \sum_i \exp(-y_i f(x_i)), \quad f(x) = \sum_t \alpha_t G_t(x)$$

2. AdaBoost 权重公式

$$\alpha_t = \frac{1}{2}\ln\frac{1-\epsilon_t}{\epsilon_t}$$

样本权重更新：$D_{t+1}(i) \propto D_t(i) \cdot \exp(-\alpha_t y_i G_t(x_i))$

3. GBDT：函数空间的梯度下降

第 $t$ 轮拟合负梯度（对平方损失即残差）：

$$r_i = -\frac{\partial L}{\partial f}\bigg|_{f=f_{t-1}} = y_i - f_{t-1}(x_i)$$

更新：$f_t(x) = f_{t-1}(x) + \eta \cdot h_t(x)$

4. XGBoost：二阶 Taylor 展开

$$\text{Obj}_t \approx \sum_i \left[g_i h_t(x_i) + \frac{1}{2}h_i h_t(x_i)^2\right] + \gamma T + \frac{1}{2}\lambda\sum_j w_j^2$$

其中 $g_i = \partial L/\partial f_{t-1}$，$h_i = \partial^2 L/\partial f_{t-1}^2$。

与数学系课程的联系：
  • 数学分析：Taylor 展开、泛函梯度下降、加法模型的收敛性
  • 概率论：指数族分布、指数损失的统计解释
  • 数理统计：集成学习理论、偏差-方差——Boosting 降低偏差

Python 实现

AdaBoost 权重更新

error = np.sum(sample_weights * incorrect) / np.sum(sample_weights)
alpha = 0.5 * np.log((1 - error) / max(error, 1e-10))
sample_weights *= np.exp(-alpha * y * y_pred)
sample_weights /= np.sum(sample_weights)

GBDT 拟合残差

residuals = y - current_pred  # 负梯度 = 残差
tree = DecisionTreeRegressor(max_depth=self.max_depth)
tree.fit(X, residuals)
current_pred += self.learning_rate * tree.predict(X)

关键观察

  • AdaBoost 通过样本重加权让后续分类器关注困难样本
  • GBDT = 函数空间中的梯度下降，每步拟合负梯度

核心统计思想

贯穿整门课程的几条主线，帮助你从更高的视角俯瞰机器学习全景。

🎯 极大似然估计 (MLE)

逻辑回归、EM、GMM……绝大多数模型的参数求解都统一到"找到使观测数据概率最大的参数"。

⚖️ 偏差-方差权衡

正则化、剪枝、Boosting 迭代次数——本质上都在做同一个权衡。

📈 正则化与先验

L2 正则化 = 高斯先验，L1 = 拉普拉斯先验。频率学派与贝叶斯学派殊途同归。

🔍 对偶性

SVM 对偶、最大熵对偶——不仅带来计算便利，更揭示核方法和支持向量的本质。

📊 生成 vs 判别

朴素贝叶斯建模 $P(X|Y)$，逻辑回归直接建模 $P(Y|X)$。两种范式各有优劣。

🎲 集成学习哲学

Bagging 降低方差，Boosting 降低偏差。"群体智慧"在机器学习中的力量。
