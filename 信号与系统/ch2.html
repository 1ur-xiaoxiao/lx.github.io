第二章 · 连续时间系统的时域分析

核心问题：给定输入 $x(t)$ 和系统的微分方程描述，如何求出输出 $y(t)$？解法有两类——经典法（齐次解 + 特解）和卷积法（零输入 + 零状态）。后者的分解思想是现代信号处理的基础。

本章目录

  1. 微分方程描述与经典解法
  2. 零输入响应与零状态响应
  3. 冲激响应 $h(t)$
  4. 卷积积分

2.1 微分方程描述与经典解法

定义 · 连续LTI系统的微分方程

$n$ 阶 LTI 系统的一般微分方程为：

$$\sum_{k=0}^{n} a_k \frac{\dd^k y(t)}{\dd t^k} = \sum_{k=0}^{m} b_k \frac{\dd^k x(t)}{\dd t^k}$$

其中 $a_k, b_k$ 为常系数（时不变性保证）。通常 $n \geq m$（物理可实现系统）。

方程解读

左边是输出的各阶导数的线性组合，右边是输入的各阶导数的线性组合。系数的常值性由"时不变"保证，线性组合的形式由"线性"保证。

对于电路系统，微分方程可由基尔霍夫定律和元件 V-I 关系（$v_L = L\frac{\dd i}{\dd t}$、$i_C = C\frac{\dd v}{\dd t}$）导出。

经典解法：齐次解 + 特解

经典解的构成

全解 = 齐次解（自由响应）+ 特解（受迫响应）

  • 齐次解 $y_h(t)$：令输入为零（$x(t) \equiv 0$），解齐次方程。形式为 $\sum_{i=1}^n C_i e^{\lambda_i t}$，其中 $\lambda_i$ 是特征根。
  • 特解 $y_p(t)$：根据输入 $x(t)$ 的函数形式设定特解形式，代入原方程确定系数。
  • 待定系数：$C_i$ 由初始条件 $y(0^+), y'(0^+), \ldots$ 确定。
特征方程与特征根

对 $n$ 阶系统 $\sum_{k=0}^{n} a_k y^{(k)}(t) = x(t)$（为简便设右端只有 $x(t)$），特征方程为：

$$a_n \lambda^n + a_{n-1} \lambda^{n-1} + \cdots + a_1 \lambda + a_0 = 0$$

$n$ 个特征根 $\lambda_1, \lambda_2, \ldots, \lambda_n$ 决定了系统的固有响应模式。

▶ 例题 · 经典法求解

题目：系统的微分方程为 $y''(t) + 5y'(t) + 6y(t) = e^{-t}$，初始条件 $y(0) = 0$，$y'(0) = 1$，求全响应。

解：

(1) 齐次解：特征方程 $\lambda^2 + 5\lambda + 6 = 0$，$\lambda_1 = -2$，$\lambda_2 = -3$。齐次解 $y_h(t) = C_1 e^{-2t} + C_2 e^{-3t}$。

(2) 特解：设 $y_p(t) = Ae^{-t}$，代入：$Ae^{-t} - 5Ae^{-t} + 6Ae^{-t} = e^{-t}$，$2A = 1$，$A = \frac{1}{2}$。特解 $y_p(t) = \frac{1}{2}e^{-t}$。

(3) 全解：$y(t) = C_1 e^{-2t} + C_2 e^{-3t} + \frac{1}{2}e^{-t}$。

(4) 用初始条件定 $C_1, C_2$：

$y(0) = C_1 + C_2 + \frac{1}{2} = 0 \Rightarrow C_1 + C_2 = -\frac{1}{2}$

$y'(t) = -2C_1 e^{-2t} - 3C_2 e^{-3t} - \frac{1}{2}e^{-t}$，$y'(0) = -2C_1 - 3C_2 - \frac{1}{2} = 1$

解得 $C_1 = 1$，$C_2 = -\frac{3}{2}$。

全解：$y(t) = e^{-2t} - \frac{3}{2}e^{-3t} + \frac{1}{2}e^{-t}$。

经典法的局限性

经典法虽然直接，但有两个缺陷：(1) 输入改变时需要重新设定特解形式并重新计算；(2) 不能自然地分离由初始状态引起的响应和由输入引起的响应。这引出了下节更具洞察力的分解方法。

2.2 零输入响应与零状态响应

两种响应的分解

将全响应 $y(t)$ 分解为两部分之和：

$$y(t) = \underbrace{y_{zi}(t)}_{\text{零输入响应}} + \underbrace{y_{zs}(t)}_{\text{零状态响应}}$$
  • 零输入响应 $y_{zi}(t)$：输入 $x(t) \equiv 0$，仅由系统初始状态（初始储能）产生的响应。满足齐次方程。
  • 零状态响应 $y_{zs}(t)$：初始状态为零，仅由输入 $x(t)$ 产生的响应。

这种分解是线性系统叠加原理的直接结果。

与经典法分解的区别

注意：$y_{zi}$ ≠ 经典法的齐次解，$y_{zs}$ ≠ 特解。

  • 齐次解/特解是按方程结构分的（齐次方程 vs 非齐次方程）
  • 零输入/零状态是按物理因果分的（初始储能 vs 外部激励）
  • $y_{zs}$ 中通常既包含齐次解分量（由输入激发的固有模式），也包含特解分量

零状态响应的齐次分量部分常称为受迫响应的固有分量——输入将系统"踢"起来后，系统以固有模式振荡的部分。

$0^-$ 与 $0^+$ 的跳变

初始状态通常给在 $t = 0^-$（激励接入之前）。当输入包含 $\deltafn(t)$ 或其导数时，系统状态可能在 $t = 0$ 处发生跳变。此时需用冲激函数匹配法从 $y(0^-)$ 求 $y(0^+)$，再用 $y(0^+)$ 确定零输入响应中的待定系数。

关键规则：如果微分方程右端含有 $\deltafn(t)$ 的 $k$ 阶导数，则最高阶导数项 $y^{(n)}(t)$ 将包含 $\deltafn(t)$ 的 $k$ 阶导数，$y^{(n-1)}(t)$ 将包含 $k-1$ 阶导数……$y^{(n-k-1)}(t)$ 将包含跳变。

2.3 冲激响应 $h(t)$

定义 · 冲激响应

冲激响应 $h(t)$是系统在初始状态为零的条件下，输入为单位冲激函数 $\deltafn(t)$ 时的零状态响应：

$$h(t) = \mathcal{T}\{\deltafn(t)\} \quad \text{（初始状态为零）}$$
为什么 $h(t)$ 如此重要？

$h(t)$ 是 LTI 系统的"身份证"——知道了 $h(t)$，就完全知道了系统对任意输入的零状态响应（通过卷积）。原因：任意信号 $x(t)$ 可以表示为无穷多个加权冲激的积分（$\deltafn(t)$ 的筛分性质），而线性+时不变性保证系统对每个加权冲激的响应都是加权时移的 $h(t)$，总响应就是这些响应的叠加——即卷积积分。

冲激响应的求解

方法：齐次解 + 冲激函数匹配

对于 $n$ 阶系统，当输入为 $\deltafn(t)$ 时：

  1. $t > 0$ 时 $\deltafn(t) = 0$，故 $h(t)$ 在 $t > 0$ 时满足齐次方程，形式为 $\sum_{i=1}^n C_i e^{\lambda_i t}\u(t)$。
  2. $t = 0$ 处 $\deltafn(t)$ 引起初始条件的跳变——需用冲激函数匹配法确定 $h(0^+), h'(0^+), \ldots, h^{(n-1)}(0^+)$。
  3. 用这些 $0^+$ 条件确定 $C_i$，得到 $h(t)$。
例题 · 求一阶系统的冲激响应

题目：一阶系统 $y'(t) + ay(t) = x(t)$，求 $h(t)$。

解：令 $x(t) = \deltafn(t)$，初始状态为零。

$t > 0$ 时 $\deltafn(t) = 0$，$h(t)$ 满足 $h'(t) + ah(t) = 0$，故 $h(t) = Ce^{-at}$。

对 $y'(t) + ay(t) = \deltafn(t)$ 在 $[0^-, 0^+]$ 上积分：

$\int_{0^-}^{0^+} h'(t)\dd t + a\int_{0^-}^{0^+} h(t)\dd t = \int_{0^-}^{0^+} \deltafn(t)\dd t = 1$

左边第一项 $= h(0^+) - h(0^-) = h(0^+)$（因为 $h(0^-) = 0$，初始零状态）

左边第二项 $\approx 0$（$h(t)$ 在无穷小区间上的积分趋于 0，只要 $h$ 没有 $\deltafn$ 分量）

故 $h(0^+) = 1$，$C = 1$。冲激响应：$h(t) = e^{-at}\u(t)$。

阶跃响应 $g(t)$

定义 · 阶跃响应

系统在零状态下对单位阶跃信号 $\u(t)$ 的响应：$g(t) = \mathcal{T}\{\u(t)\}$。

由 LTI 系统的微积分性质：

$$h(t) = \frac{\dd}{\dd t}g(t), \qquad g(t) = \int_{-\infty}^t h(\tau) \dd\tau$$

2.4 卷积积分

定义 · 卷积积分

两函数 $x(t)$ 与 $h(t)$ 的卷积积分定义为：

$$y(t) = x(t) * h(t) = \int_{-\infty}^{\infty} x(\tau)\,h(t - \tau) \dd\tau$$

对因果系统（$h(t) = 0, \forall t < 0$）和因果输入（$x(t) = 0, \forall t < 0$）：

$$y(t) = \int_{0}^{t} x(\tau)\,h(t - \tau) \dd\tau$$
卷积的物理直觉

理解卷积的关键在于：$h(t - \tau)$ 是系统对发生在时刻 $\tau$ 的单位冲激的响应，在时刻 $t$ 的取值。$x(\tau)\dd\tau$ 是输入在时刻 $\tau$ 的"强度"。将所有这些加权响应叠加（积分）起来，就得到时刻 $t$ 的总输出。

卷积积分的三个步骤（图解法）：

  1. 反褶：将 $h(\tau)$ 反褶为 $h(-\tau)$
  2. 平移：将 $h(-\tau)$ 平移 $t$ 得到 $h(t - \tau)$
  3. 相乘积分：计算 $x(\tau) \cdot h(t - \tau)$ 的积分面积

卷积积分的关键性质

卷积的代数性质
性质表达式
交换律$x(t) * h(t) = h(t) * x(t)$
分配律$x(t) * [h_1(t) + h_2(t)] = x(t) * h_1(t) + x(t) * h_2(t)$
结合律$[x(t) * h_1(t)] * h_2(t) = x(t) * [h_1(t) * h_2(t)]$

结合律的物理含义：两个系统级联后的冲激响应等于各自冲激响应的卷积，且级联顺序可交换。

卷积的微积分性质

设 $y(t) = x(t) * h(t)$，则：

$$\frac{\dd}{\dd t}[x(t) * h(t)] = \frac{\dd x(t)}{\dd t} * h(t) = x(t) * \frac{\dd h(t)}{\dd t}$$
$$\int_{-\infty}^t [x(\tau) * h(\tau)] \dd\tau = \left[\int_{-\infty}^t x(\tau)\dd\tau\right] * h(t) = x(t) * \left[\int_{-\infty}^t h(\tau)\dd\tau\right]$$

重要推论：$x(t) * \deltafn(t) = x(t)$；$x(t) * \deltafn(t-t_0) = x(t-t_0)$；$x(t) * \u(t) = \int_{-\infty}^t x(\tau)\dd\tau$。

卷积的时移性质

若 $y(t) = x(t) * h(t)$，则：

$$x(t - t_1) * h(t - t_2) = y(t - t_1 - t_2)$$
与 $\deltafn(t)$ 的卷积

$\deltafn(t)$ 是卷积的单位元：

$$x(t) * \deltafn(t) = x(t)$$

一般形式：$x(t) * \deltafn^{(n)}(t - t_0) = x^{(n)}(t - t_0)$（$\deltafn$ 的 $n$ 阶导数与信号卷积 = 信号的 $n$ 阶导数并延时）。

▶ 例题 · 图解法求卷积

题目：$x(t) = \u(t) - \u(t-2)$（宽度为 2 的矩形脉冲），$h(t) = e^{-t}\u(t)$。求 $y(t) = x(t) * h(t)$。

解：用图解法分段计算。

$t < 0$：两函数没有重叠，$y(t) = 0$。

$0 \leq t < 2$：$y(t) = \int_0^t 1 \cdot e^{-(t-\tau)} \dd\tau = e^{-t} \int_0^t e^{\tau} \dd\tau = e^{-t}(e^t - 1) = 1 - e^{-t}$。

$t \geq 2$：$y(t) = \int_0^2 1 \cdot e^{-(t-\tau)} \dd\tau = e^{-t}(e^2 - 1) = (e^2 - 1)e^{-t}$。

物理含义：矩形脉冲通过 RC 低通电路——输出先以指数形式上升，脉冲结束后再以指数形式衰减。

本章总结：时域分析的核心逻辑

连续时间系统时域分析的核心链条是：

微分方程 → 零输入/零状态分解 → 冲激响应 $h(t)$ → 卷积积分 → 任意输入下的零状态响应

这个链条在离散时间系统中将被完整复制：差分方程 → 零输入/零状态分解 → 单位样值响应 $h[n]$ → 卷积和。掌握好本章的思维框架，第六章将事半功倍。
