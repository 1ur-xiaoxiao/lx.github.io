第四章 · 拉普拉斯变换与 s 域分析

傅里叶变换的推广：通过引入衰减因子 $e^{-\sigma t}$，将傅里叶变换的适用范围从"绝对可积"大幅扩展。拉普拉斯变换不仅是求解微分方程的利器，其零极点概念更为系统分析提供了深刻的几何直觉。

4.1 从傅里叶到拉普拉斯

傅里叶变换的困难

傅里叶变换要求信号绝对可积：$\int_{-\infty}^{\infty} |f(t)| \dd t < \infty$。但很多常见信号不满足这个条件：

  • $\u(t)$：阶跃信号（不衰减，积分发散）
  • $e^{\alpha t}\u(t)$，$\alpha > 0$：指数增长信号
  • $t^n\u(t)$：幂函数增长信号

这些信号在工程中频繁出现（系统响应、斜坡输入等），不能做傅里叶变换给分析带来不便。

解决方案：强制衰减

技巧：给 $f(t)$ 乘上一个指数衰减因子 $e^{-\sigma t}$（$\sigma$ 为实数），使 $f(t)e^{-\sigma t}$ 变得绝对可积，再对其做傅里叶变换：

$$\LL\{f(t)\} = \int_{-\infty}^{\infty} f(t)e^{-\sigma t} e^{-j\omega t} \dd t = \int_{-\infty}^{\infty} f(t) e^{-(\sigma + j\omega)t} \dd t$$

令 $s = \sigma + j\omega$，即得拉普拉斯变换。$s$ 是复频率：实部 $\sigma$ 控制衰减/增长，虚部 $\omega$ 控制振荡频率。

4.2 拉普拉斯变换的定义与收敛域

定义 · 双边拉普拉斯变换
$$F(s) = \LL\{f(t)\} = \int_{-\infty}^{\infty} f(t) \, e^{-st} \dd t$$

其中 $s = \sigma + j\omega \in \CC$。使得积分收敛的 $s$ 的取值范围称为收敛域（ROC: Region of Convergence）。

定义 · 单边拉普拉斯变换

实际中更常用的是单边拉普拉斯变换（$t$ 从 $0^-$ 开始）：

$$F(s) = \int_{0^-}^{\infty} f(t) \, e^{-st} \dd t$$

从 $0^-$ 开始可以捕捉 $t=0$ 处的冲激函数。本课程主要讨论单边拉普拉斯变换。

收敛域（ROC）：拉普拉斯变换的"身份证"

拉普拉斯变换与傅里叶变换的一个关键区别是：$F(s)$ 的表达式必须伴随收敛域才有完整意义。同一个 $F(s)$ 代数表达式，不同的收敛域对应不同的 $f(t)$。

收敛域的一般形式：对右边信号（$t < t_0$ 时为零），ROC 为 $\Re(s) > \sigma_0$（右半平面）；对左边信号（$t > t_0$ 时为零），ROC 为 $\Re(s) < \sigma_0$（左半平面）；对双边信号，ROC 为带状区域 $\sigma_1 < \Re(s) < \sigma_2$。

ROC 的性质
  1. ROC 在 s 平面中由平行于 $j\omega$ 轴的直线界定（带状区域）。
  2. ROC 内不能包含任何极点（极点是使 $F(s) \to \infty$ 的点）。
  3. 若 $f(t)$ 是有限持续时间的（时限信号），则 ROC 是整个 s 平面（可能除去 $s=0$ 或 $|s|=\infty$）。
  4. 若 $f(t)$ 是右边信号，ROC 是某个 $\sigma_0$ 以右的半平面。
单边拉普拉斯变换的简化

对于因果信号（$t < 0$ 时 $f(t)=0$），单边拉普拉斯变换的 ROC 总是 $\Re(s) > \sigma_0$ 的形式（某个 σ 以右的半平面）。这使得在单边变换的范围内，"表达式中最大的极点实部以右"就自动是 ROC。所以在不引起歧义时，可以省略 ROC 的显式声明。

与傅里叶变换的对比

当 ROC 包含 $j\omega$ 轴时，令 $s = j\omega$ 即得傅里叶变换：$F(\omega) = F(s)|_{s=j\omega}$。

例如：$\u(t)$ 的拉普拉斯变换是 $\frac{1}{s}$（$\Re(s) > 0$），其傅里叶变换是 $\pi\deltafn(\omega) + \frac{1}{j\omega}$——当 $\sigma=0$ 时 $s=j\omega$，但 $j\omega$ 轴恰好是 ROC 的边界（不是严格内部），所以不能简单代入。

微分性质：s 域分析的"杀手锏"

时域微分性质是拉普拉斯变换解决微分方程的关键：它将微分方程转化为代数方程。

例如：$y''(t) + 5y'(t) + 6y(t) = x(t)$，零初始状态下，变换得 $(s^2 + 5s + 6)Y(s) = X(s)$，即 $Y(s) = \frac{X(s)}{s^2 + 5s + 6}$。解微分方程变成了简单的代数运算和逆变换。

初值定理与终值定理

初值定理：$f(0^+) = \lim_{s \to \infty} sF(s)$（若极限存在）。用于从 $F(s)$ 直接求出信号的初始值，无需做逆变换。

终值定理：$\lim_{t \to \infty} f(t) = \lim_{s \to 0} sF(s)$，前提是 $sF(s)$ 的所有极点具有负实部（即系统稳定，$f(\infty)$ 存在且有限）。

注意：终值定理不能用于不稳定系统（例如极点位于虚轴或右半平面时 $f(\infty)$ 不存在或发散）。

4.5 拉普拉斯逆变换

定义 · 逆拉普拉斯变换
$$f(t) = \LL^{-1}\{F(s)\} = \frac{1}{2\pi j} \int_{\sigma - j\infty}^{\sigma + j\infty} F(s) \, e^{st} \dd s$$

这是复变函数中的围道积分。实际中绝大多数情况使用部分分式展开法，配合查表完成逆变换。

部分分式展开法

大部分 $F(s)$ 是有理分式：$F(s) = \frac{N(s)}{D(s)} = \frac{b_m s^m + \cdots + b_1 s + b_0}{s^n + a_{n-1}s^{n-1} + \cdots + a_1 s + a_0}$（$m < n$，真分式）。

三种基本情形的展开

情况1：单实极点。$D(s) = (s-p_1)(s-p_2)\cdots(s-p_n)$，$p_i$ 互异：

$$F(s) = \sum_{i=1}^n \frac{A_i}{s - p_i}, \quad A_i = (s-p_i)F(s)\big|_{s=p_i}$$ $$\LL^{-1}\left\{\frac{A_i}{s-p_i}\right\} = A_i e^{p_i t}\u(t)$$

情况2：重极点。$D(s)$ 含 $(s-p_1)^k$：

$$F(s) = \frac{A_{11}}{s-p_1} + \frac{A_{12}}{(s-p_1)^2} + \cdots + \frac{A_{1k}}{(s-p_1)^k} + \cdots$$ $$A_{1j} = \frac{1}{(k-j)!} \frac{\dd^{k-j}}{\dd s^{k-j}}\left[(s-p_1)^k F(s)\right]\Big|_{s=p_1}$$ $$\LL^{-1}\left\{\frac{A}{(s-p)^k}\right\} = \frac{A}{(k-1)!} t^{k-1} e^{pt}\u(t)$$

情况3：共轭复极点。将共轭对 $p = \sigma \pm j\omega$ 合并为二次因式 $\frac{As + B}{(s-\sigma)^2 + \omega^2}$，对应 $e^{\sigma t}(C_1\cos\omega t + C_2\sin\omega t)\u(t)$。

▶ 例题 · 部分分式展开

题目：求 $F(s) = \frac{s+3}{s^2 + 3s + 2}$ 的逆变换。

解：

分母分解：$s^2 + 3s + 2 = (s+1)(s+2)$。

展开：$\frac{s+3}{(s+1)(s+2)} = \frac{A}{s+1} + \frac{B}{s+2}$。

$A = (s+1)F(s)|_{s=-1} = \frac{-1+3}{-1+2} = \frac{2}{1} = 2$。

$B = (s+2)F(s)|_{s=-2} = \frac{-2+3}{-2+1} = \frac{1}{-1} = -1$。

$F(s) = \frac{2}{s+1} - \frac{1}{s+2}$，逆变换：$f(t) = (2e^{-t} - e^{-2t})\u(t)$。

4.6 s 域系统分析

定义 · 系统函数 $H(s)$

系统函数（传输函数）定义为冲激响应的拉普拉斯变换：

$$H(s) = \LL\{h(t)\} = \frac{Y_{zs}(s)}{X(s)}$$

其中 $Y_{zs}(s)$ 是零状态响应的拉普拉斯变换，$X(s)$ 是输入的拉普拉斯变换。

$H(s)$ 与 $H(\omega)$ 的关系

$H(\omega)$（频率响应）= $H(s)|_{s=j\omega}$，条件是 ROC 包含 $j\omega$ 轴。

$H(s)$ 是 $H(\omega)$ 的解析延拓：将角频率 $\omega$ 从实数推广到复数 $s = \sigma + j\omega$，获得更丰富的系统信息（如稳定性、暂态行为）。

零极点图

定义 · 零点与极点
  • 零点：使 $H(s) = 0$ 的 $s$ 值（用 ○ 表示）
  • 极点：使 $H(s) \to \infty$ 的 $s$ 值（用 × 表示）

对于有理系统函数 $H(s) = \frac{N(s)}{D(s)}$，零点 = 分子根，极点 = 分母根（即特征根）。

零极点与系统行为的对应
极点位置$h(t)$ 行为稳定性
左半平面（$\Re(p) < 0$）指数衰减稳定
$j\omega$ 轴上的单极点等幅振荡或常数临界（边界）稳定
$j\omega$ 轴上的重极点幅度增长的振荡不稳定
右半平面（$\Re(p) > 0$）指数增长不稳定
稳定性判据

一个因果 LTI 系统是BIBO 稳定的当且仅当 $H(s)$ 的所有极点都位于 s 左半平面（$\Re(p_i) < 0$），即 ROC 包含 $j\omega$ 轴。

直觉：极点实部为负 → 冲激响应的每个模式 $e^{p_i t}$ 都指数衰减 → 冲激响应绝对可积 → BIBO 稳定。反之，若有极点实部为正 → 冲激响应中至少有一个模式指数增长 → 有界输入可能产生无界输出。

本章总结：s 域分析的核心优势
  1. 微分方程 → 代数方程：拉普拉斯变换将时域的微积分运算转换为 s 域的代数运算。
  2. 零极点 → 系统行为的几何直观：极点位置直接反映冲激响应的模式和稳定性。
  3. 卷积 → 乘法：$Y(s) = H(s)X(s)$ 比 $y(t) = h(t) * x(t)$ 简便得多。
  4. 推广傅里叶变换：可以处理指数增长信号，收敛域的概念使分析更严谨。
