8.1 信号的矢量空间分析

核心思想：把信号看作向量

将信号视为函数空间中的向量，把线性代数中的内积、正交、投影等概念推广到连续时间函数。这一视角统一了傅里叶分析、相关分析、匹配滤波等诸多主题——它们本质上都是在做"向量的内积"。

信号的内积

在区间 $[t_1, t_2]$ 上，两个实信号 $f_1(t)$ 和 $f_2(t)$ 的内积定义为：

$$\langle f_1, f_2 \rangle = \int_{t_1}^{t_2} f_1(t) f_2(t) \dd t$$

若 $\langle f_1, f_2 \rangle = 0$，称两信号正交。

关键理解：傅里叶级数中的系数计算 $a_n = \frac{2}{T}\int_T f(t)\cos(n\omega_0 t)\dd t$，本质上就是 $f(t)$ 与基函数 $\cos(n\omega_0 t)$ 的内积。傅里叶分析就是在做"向正交基上的投影"。

定义 · 相关函数

两个能量信号的互相关函数：

$$R_{12}(\tau) = \int_{-\infty}^{\infty} f_1(t) f_2(t - \tau) \dd t$$

自相关函数：

$$R(\tau) = \int_{-\infty}^{\infty} f(t) f(t - \tau) \dd t$$

互相关与卷积的关系：$R_{12}(\tau) = f_1(\tau) * f_2(-\tau)$。

物理含义：自相关衡量信号与其自身时移版本的"相似度"。$R(0) = E$（信号能量）；$R(\tau) \leq R(0)$（零时延处自相关最大）。

相关与卷积的关系

卷积：$f_1(t) * f_2(t) = \int f_1(\tau)f_2(t-\tau)\dd\tau$（反褶 + 平移 + 积分）

相关：$R_{12}(\tau) = \int f_1(t)f_2(t-\tau)\dd t$（不反褶 + 平移 + 积分）

唯一的区别是是否反褶。当信号为偶函数时二者等价。

Wiener-Khinchin 定理简版

能量信号的自相关函数与能量谱密度互为傅里叶变换对：

$$\FF\{R(\tau)\} = |F(\omega)|^2 = \mathcal{E}(\omega)$$

8.2 离散傅里叶变换（DFT）

为什么需要 DFT？

计算机只能处理有限长的离散数据。DFT 是唯一可以在计算机上精确计算的傅里叶变换形式。它与连续傅里叶变换的关系链条是：

连续 FT → 时域抽样 → DTFT → 频域抽样（时域周期延拓）→ DFT

DFT 本质上是离散傅里叶级数（DFS）取主值区间得到的。

定义 · $N$ 点 DFT

对长度为 $N$ 的序列 $x[n]$（$n = 0, 1, \ldots, N-1$）：

$$X[k] = \sum_{n=0}^{N-1} x[n] \, e^{-j\frac{2\pi}{N}kn}, \quad k = 0, 1, \ldots, N-1$$

逆变换（IDFT）：

$$x[n] = \frac{1}{N} \sum_{k=0}^{N-1} X[k] \, e^{j\frac{2\pi}{N}kn}, \quad n = 0, 1, \ldots, N-1$$

记 $W_N = e^{-j2\pi/N}$（旋转因子），DFT 和 IDFT 可简洁地表示为 $X = W \cdot x$ 和 $x = \frac{1}{N}W^* \cdot X$。

DFT 的隐含周期性

$X[k]$ 和 $x[n]$ 在定义区间外都是隐含周期延拓的：$X[k+N] = X[k]$，$x[n+N] = x[n]$。这是 DFT 来自 DFS 的先天特性，许多 DFT 性质（如循环卷积）都源于此。

快速傅里叶变换（FFT）的思想

直接计算 $N$ 点 DFT 需要 $O(N^2)$ 次复数乘法。Cooley-Tukey FFT 算法利用旋转因子 $W_N$ 的周期性和对称性，将计算分解为两个 $N/2$ 点的 DFT，递归下去得到 $O(N\log N)$ 的复杂度。这是 20 世纪最重要的算法之一，使实时频谱分析成为可能。

8.3 模拟与数字滤波器概要

模拟滤波器

从理想到可实现

理想滤波器（矩形幅频）物理不可实现。实际滤波器设计的目标是用有理传递函数（物理可实现）来逼近理想幅度特性。三大经典逼近方式：

类型特点适用场景
Butterworth通带最平坦、无波纹；过渡带较宽对通带平坦度要求高的场合
Chebyshev I 型通带有等波纹、过渡带比 BW 窄对截止特性要求高、允许通带波动
Chebyshev II 型 / 椭圆通带和阻带均有波纹、过渡带最窄对截止陡度要求最高的场合

这些滤波器通过频带变换（低通→高通/带通/带阻）可以产生各种类型的滤波器。

数字滤波器

IIR 与 FIR：数字滤波器的两大流派
IIR（无限冲激响应）FIR（有限冲激响应）
差分方程$y[n] = \sum b_k x[n-k] - \sum a_k y[n-k]$
（有反馈，含 $y[n-k]$ 项）		$y[n] = \sum b_k x[n-k]$
（无反馈，纯前馈）
极点有极点（可能不稳定）无极点（除 $z=0$ 外，始终稳定）
相位通常非线性可做到严格线性相位
效率低阶即可实现陡峭截止需要较高阶数

8.4 反馈系统

反馈系统的基本结构

将系统输出的一部分（或全部）反馈回输入端，与输入信号叠加后再进入前向通路。基本方程为：

$$Y(s) = H_1(s)[X(s) - H_2(s)Y(s)]$$

其中 $H_1(s)$ 是前向通路传递函数，$H_2(s)$ 是反馈通路传递函数。

闭环传递函数：

$$H(s) = \frac{Y(s)}{X(s)} = \frac{H_1(s)}{1 + H_1(s)H_2(s)}$$
反馈的好处与代价

好处：提高稳定性（对参数变化不敏感）、改善频率响应、降低输出阻抗（对负载不敏感）。几乎所有高性能放大器和控制系统都使用负反馈。

代价：增益降低（负反馈以牺牲增益换取稳定性）、可能产生自激振荡（当回路增益 $H_1(s)H_2(s)$ 在某些频率处满足 Barkhausen 条件：$|H_1 H_2| \geq 1$ 且 $\angle (H_1 H_2) = 180^\circ$）。

稳定性判据：通过考察 $1 + H_1(s)H_2(s)$ 的零点分布（即特征方程的根），判断闭环系统是否稳定。Nyquist 判据和根轨迹法是主要工具。

8.5 状态变量分析

状态变量法的动机

传统的输入-输出描述（微分方程、传递函数）只刻画外部行为，不反映系统内部状态。状态变量法引入一组内部变量（状态变量），用一阶微分方程组来描述系统——这特别适合多输入多输出（MIMO）系统，也是现代控制理论的基础。

状态方程与输出方程

对连续时间系统：

$$\begin{aligned} \dot{\mathbf{x}}(t) &= \mathbf{A}\mathbf{x}(t) + \mathbf{B}\mathbf{u}(t) \\ \mathbf{y}(t) &= \mathbf{C}\mathbf{x}(t) + \mathbf{D}\mathbf{u}(t) \end{aligned}$$

其中 $\mathbf{x}(t)$ 是状态向量，$\mathbf{u}(t)$ 是输入向量，$\mathbf{y}(t)$ 是输出向量。

$\mathbf{A}$：系统矩阵（决定固有动态特性，其特征值 = 系统极点）。$\mathbf{B}$：输入矩阵。$\mathbf{C}$：输出矩阵。$\mathbf{D}$：直通矩阵。

与传递函数的关系

系统函数与状态空间描述的关系：$H(s) = \mathbf{C}(s\mathbf{I} - \mathbf{A})^{-1}\mathbf{B} + \mathbf{D}$。

$H(s)$ 的极点 = $\mathbf{A}$ 的特征值。状态变量描述不是唯一的——同一系统的不同状态变量选择对应不同的 $(\mathbf{A}, \mathbf{B}, \mathbf{C}, \mathbf{D})$ 实现（相似变换）。

对离散系统，状态方程变为 $\mathbf{x}[n+1] = \mathbf{A}\mathbf{x}[n] + \mathbf{B}\mathbf{u}[n]$，其余不变。
