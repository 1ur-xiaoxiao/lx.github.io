第三章 · 傅里叶变换

信号分析的"显微镜"：傅里叶变换将时域信号映射到频域，揭示信号的频率组成。这一思想的深刻性怎么强调都不过分——它不仅是信号与系统的核心，更是整个信息科学的基础。

本章目录

  1. 傅里叶级数：周期信号的频域表示

3.1 傅里叶级数：周期信号的频域表示

核心直觉：从"看到"到"听出来"

傅里叶的核心思想很简单：任何周期信号都可以分解为无穷多个正弦波（或复指数）的加权和。就像管弦乐团中，你听到的是所有乐器的合奏，但傅里叶分析可以告诉你每种乐器各自在演奏什么。

数学上：一个周期为 $T$ 的信号 $f(t)$，可以写成 $\sum_{n} c_n e^{jn\omega_0 t}$，其中 $\omega_0 = 2\pi/T$ 是基波角频率，$c_n$ 是各次谐波的复振幅（频谱）。

三角形式的傅里叶级数

定义 · 三角傅里叶级数

设 $f(t)$ 是周期为 $T$、角频率 $\omega_0 = 2\pi/T$ 的周期信号，满足 Dirichlet 条件，则：

$$f(t) = \frac{a_0}{2} + \sum_{n=1}^{\infty} [a_n \cos(n\omega_0 t) + b_n \sin(n\omega_0 t)]$$

其中傅里叶系数为：

$$a_n = \frac{2}{T} \int_{t_0}^{t_0+T} f(t) \cos(n\omega_0 t) \dd t, \quad n = 0, 1, 2, \ldots$$ $$b_n = \frac{2}{T} \int_{t_0}^{t_0+T} f(t) \sin(n\omega_0 t) \dd t, \quad n = 1, 2, 3, \ldots$$
$a_n$ 和 $b_n$ 的物理含义
  • $a_0/2$：信号的直流分量（平均值）
  • $a_n$：第 $n$ 次谐波的余弦分量幅度（偶对称分量）
  • $b_n$：第 $n$ 次谐波的正弦分量幅度（奇对称分量）
  • 第 $n$ 次谐波的幅度：$A_n = \sqrt{a_n^2 + b_n^2}$，相位：$\varphi_n = -\arctan(b_n/a_n)$
奇偶信号的系数简化
  • $f(t)$ 为偶函数：$b_n = 0$（只有余弦项）；$a_n = \frac{4}{T}\int_0^{T/2} f(t)\cos(n\omega_0 t)\dd t$
  • $f(t)$ 为奇函数：$a_n = 0$（只有正弦项）；$b_n = \frac{4}{T}\int_0^{T/2} f(t)\sin(n\omega_0 t)\dd t$
  • $f(t)$ 为奇谐函数（$f(t+T/2) = -f(t)$）：只有奇次谐波（$n=1,3,5,\ldots$）
  • $f(t)$ 为偶谐函数（$f(t+T/2) = f(t)$）：只有偶次谐波（$n=0,2,4,\ldots$）

指数形式的傅里叶级数

定义 · 指数傅里叶级数

利用欧拉公式 $e^{jn\omega_0 t} = \cos(n\omega_0 t) + j\sin(n\omega_0 t)$，将三角形式合成为更紧凑的指数形式：

$$f(t) = \sum_{n=-\infty}^{\infty} F_n \, e^{jn\omega_0 t}$$

其中复傅里叶系数：

$$F_n = \frac{1}{T} \int_{t_0}^{t_0+T} f(t) \, e^{-jn\omega_0 t} \dd t$$

$F_n$ 与三角系数的关系：$F_0 = a_0/2$；$F_n = \frac{1}{2}(a_n - jb_n)$（$n>0$）；$F_{-n} = F_n^*$。

为什么指数形式更优美

指数形式把问题统一为复指数函数 $e^{jn\omega_0 t}$ 的线性组合。这正是第一章提到的核心观点：$e^{st}$ 是 LTI 系统的特征函数。指数形式的傅里叶级数自然地与系统分析对接——每个频率分量 $F_n e^{jn\omega_0 t}$ 通过系统后，只需乘以 $H(jn\omega_0)$。

Dirichlet 条件

定理 · Dirichlet 充分条件

若周期信号 $f(t)$ 在一个周期内满足：

  1. 绝对可积：$\int_T |f(t)|\dd t < \infty$
  2. 有限个极大值和极小值
  3. 有限个第一类间断点

则其傅里叶级数收敛。在连续点收敛于 $f(t)$，在间断点收敛于 $\frac{f(t^+) + f(t^-)}{2}$。

工程中绝大多数信号都满足 Dirichlet 条件，所以我们可以放心地使用傅里叶分析方法。

Gibbs 现象

Gibbs 现象：有限项近似的过冲

用有限项傅里叶级数逼近含跳变的信号时，在跳变点附近会出现约 9% 的过冲，且不随项数增加而消失（只是振荡变窄、更靠近跳变点）。这是傅里叶级数非一致收敛的表现。

Gibbs 现象告诉我们：时域的"尖锐"需要频域的"广谱"来描述——一个跳变信号的频谱衰减得很慢（$\sim 1/n$）。这也解释了为什么数字系统中方波信号包含丰富的高频分量，容易产生电磁干扰。

3.2 傅里叶变换的定义

从级数到变换：极限过渡的直觉

周期信号 → 非周期信号的关键步骤：令周期 $T \to \infty$。

周期变大 → 基频 $\omega_0 = 2\pi/T \to 0$ → 离散的谱线间距趋于零 → 离散频谱变为连续频谱。同时，傅里叶系数 $F_n$ 趋于零（能量分散到无限宽的频带上），但 $T \cdot F_n$ 趋于有限值——这就是频谱密度函数 $F(\omega)$。

定义 · 傅里叶变换与逆变换

傅里叶正变换：

$$F(\omega) = \FF\{f(t)\} = \int_{-\infty}^{\infty} f(t) \, e^{-j\omega t} \dd t$$

傅里叶逆变换：

$$f(t) = \FF^{-1}\{F(\omega)\} = \frac{1}{2\pi} \int_{-\infty}^{\infty} F(\omega) \, e^{j\omega t} \dd\omega$$

$F(\omega)$ 称为 $f(t)$ 的频谱密度函数，简称频谱。一般而言 $F(\omega)$ 是复函数：$F(\omega) = |F(\omega)| e^{j\varphi(\omega)}$，其中 $|F(\omega)|$ 是幅度谱，$\varphi(\omega)$ 是相位谱。

关于 $F(\omega)$ vs $F(j\omega)$ 的记号

本书使用 $F(\omega)$ 表示傅里叶变换。在拉普拉斯变换的语境下，傅里叶变换可视为 $s = j\omega$ 时的特例：$F(\omega) = F(s)|_{s=j\omega}$。此时也写作 $F(j\omega)$。两种记号本质一致。

傅里叶变换存在的充分条件

若 $f(t)$ 满足：

  1. 绝对可积：$\int_{-\infty}^{\infty} |f(t)| \dd t < \infty$
  2. 在任意有限区间内满足 Dirichlet 条件

则其傅里叶变换存在。但这个条件是充分但不必要的——如 $\sgn(t)$ 不绝对可积，但其傅里叶变换（在广义函数意义下）存在。

3.3 典型信号的傅里叶变换对

以下变换对是必须记住的"基本功"。考试中的复杂题目通常通过性质的组合，从这些基本的变换对出发来求解。

最重要的对偶关系

时域的宽度与频域的宽度成反比：窄脉冲 → 宽频谱；宽脉冲 → 窄频谱。

时域的跳变与频域的衰减速度相关：跳变越剧烈，频谱衰减越慢（高频分量越丰富）。连续可导次数越高，频谱衰减越快（$\sim 1/\omega^{k+1}$，$k$ 为可导次数）。

这是测不准原理在信号处理中的体现——信号的持续时间与其带宽的乘积有一个下限。

最重要的性质：卷积定理

时域卷积 ↔ 频域相乘：$y(t) = x(t) * h(t) \;\Longleftrightarrow\; Y(\omega) = X(\omega) \cdot H(\omega)$

这一性质是整个变换域分析方法的基石。它将第二章中繁琐的卷积积分转化为频域中的简单乘法。求出 $Y(\omega)$ 后再通过逆变换得到 $y(t)$，这通常比直接计算卷积积分简便得多。

$H(\omega)$ 称为系统的频率响应（或传输函数）：$H(\omega) = \FF\{h(t)\}$。

对称性：最容易被忽视的解题利器

若 $f(t) \leftrightarrow F(\omega)$，则 $F(t) \leftrightarrow 2\pi f(-\omega)$。

这意味着：如果你知道 $\rect(t) \leftrightarrow \sinc(\omega/2)$，那么你立刻就知道 $\sinc(t/2) \leftrightarrow 2\pi\rect(\omega)$。对称性可以将已知的变换对翻倍，极大地扩展你的"解题武器库"。

▶ 例题 · 利用性质求傅里叶变换

题目：求 $f(t) = e^{-2(t-1)}\u(t-1)$ 的傅里叶变换。

解：先求出基本变换对：$e^{-2t}\u(t) \leftrightarrow \frac{1}{2 + j\omega}$。

利用时移性质：$f(t-t_0) \leftrightarrow F(\omega)e^{-j\omega t_0}$。

故 $f(t) = e^{-2(t-1)}\u(t-1) \leftrightarrow \frac{1}{2 + j\omega} \cdot e^{-j\omega \cdot 1} = \frac{e^{-j\omega}}{2 + j\omega}$。

这就是性质的力量——不需要从头计算积分，直接从基本变换对出发，利用性质即可得到结果。

3.5 能量谱与功率谱

定义 · 能量谱密度

对于能量信号 $f(t)$，其能量谱密度 $\mathcal{E}(\omega)$ 定义为：

$$\mathcal{E}(\omega) = |F(\omega)|^2$$

能量谱密度是实偶函数（当 $f(t)$ 为实信号时），反映了信号能量在各频率上的分布。

Parseval 能量定理

信号的总能量在时域和频域中相等：

$$E = \int_{-\infty}^{\infty} |f(t)|^2 \dd t = \frac{1}{2\pi} \int_{-\infty}^{\infty} |F(\omega)|^2 \dd\omega = \frac{1}{\pi} \int_{0}^{\infty} |F(\omega)|^2 \dd\omega$$

最后的等号利用了 $|F(\omega)|^2$ 为偶函数的性质（当 $f(t)$ 为实信号时）。

定义 · 功率谱密度（功率信号）

对于功率信号（如周期信号），功率谱密度定义为：

$$S(\omega) = \lim_{T \to \infty} \frac{1}{T} |F_T(\omega)|^2$$

其中 $F_T(\omega)$ 是信号截断到 $[-T/2, T/2]$ 后的傅里叶变换。

对于周期为 $T_0$ 的周期信号：$S(\omega) = 2\pi \sum_{n=-\infty}^{\infty} |F_n|^2 \deltafn(\omega - n\omega_0)$。

本章总结：傅里叶分析的三层递进
  1. 傅里叶级数：周期信号的"DNA 解码"——将周期信号分解为谐波分量，得到离散频谱。
  2. 傅里叶变换：非周期信号的"频谱显微镜"——将时域函数映射为频谱密度函数，得到连续频谱。
  3. 卷积定理 + 频率响应：LTI 系统分析的"高速公路"——时域卷积变为频域乘法，$H(\omega)$ 完整刻画系统的频率选择特性。
