第七章 · Z 变换与 z 域分析

离散时间的"拉普拉斯变换"：Z 变换之于差分方程，正如拉普拉斯变换之于微分方程。它把卷积和变为多项式乘法，把差分方程变为代数方程，把稳定性分析变为极点在单位圆内外的几何判断。

7.1 Z 变换的定义与收敛域

定义 · Z 变换

序列 $x[n]$ 的双边 Z 变换定义为：

$$X(z) = \ZT\{x[n]\} = \sum_{n=-\infty}^{\infty} x[n] \, z^{-n}$$

其中 $z = re^{j\Omega}$ 是复变量。使级数收敛的 $z$ 的取值范围称为收敛域（ROC）。

单边 Z 变换

对因果序列（$n < 0$ 时 $x[n] = 0$），单边 Z 变换为：

$$X(z) = \sum_{n=0}^{\infty} x[n] \, z^{-n}$$
从拉普拉斯变换到 Z 变换

Z 变换可以从拉普拉斯变换推导出来：对连续信号 $x(t)$ 做理想抽样得 $x_s(t) = \sum_n x(nT)\delta(t-nT)$，其拉普拉斯变换为 $\sum_n x(nT)e^{-snT}$。令 $z = e^{sT}$，即得 Z 变换。

关键映射：$z = e^{sT}$。s 平面的虚轴（$s=j\omega$）映射到 z 平面的单位圆（$|z|=1$）。s 左半平面映射到 z 平面单位圆内部。这解释了为什么离散系统稳定性由"极点是否在单位圆内"来决定。

ROC：Z 变换的"身份证"

与拉普拉斯变换完全类似：$X(z)$ 的代数表达式必须伴随 ROC 才有完整意义。同一个 $X(z)$ 表达式，不同 ROC 对应不同序列。

ROC 的形式：

  • 右边序列：ROC 是 $|z| > r_0$（某个圆的外部）
  • 左边序列：ROC 是 $|z| < r_0$（某个圆的内部）
  • 双边序列：ROC 是圆环 $r_1 < |z| < r_2$
  • 有限长序列：ROC 是整个 z 平面（可能除去 $z=0$ 或 $|z|=\infty$）

$z^{-1}$ 形式的表达

Z 变换常写成 $z^{-1}$ 的有理分式形式，这在系统实现（如数字滤波器的直接形式结构）中更为方便。例如 $u[n] \leftrightarrow \frac{1}{1-z^{-1}}$（$|z|>1$）。两种形式是等价的，注意 ROC 的一致性即可。

7.3 Z 变换的性质

Z 变换性质一览
性质时域z 域
线性$a_1 x_1[n] + a_2 x_2[n]$$a_1 X_1(z) + a_2 X_2(z)$（ROC 至少为交集）
时移$x[n-n_0]u[n-n_0]$$z^{-n_0} X(z)$

最重要的性质：时移

时移性质 $x[n-1] \leftrightarrow z^{-1}X(z)$ 是 Z 变换解决差分方程的核心。$z^{-1}$ 就是离散时间系统中的"单位延迟算子"——这正是数字滤波器实现的基础。每一个 $z^{-1}$ 对应一个寄存器/存储单元。

7.4 逆 Z 变换

方法一 · 幂级数展开法（长除法）

将 $X(z)$ 展开为 $z^{-1}$ 的幂级数（在收敛域内），系数序列即为 $x[n]$。

对右边序列（$|z| > r$），按 $z^{-1}$ 的升幂排列做长除。对左边序列（$|z| < r$），按 $z$ 的升幂排列做长除。

此法适合求前几项值，不适合求封闭表达式。

方法二 · 部分分式展开法（最常用）

与拉普拉斯逆变换几乎完全对偶。将 $X(z)$ 写成以 $z$ 为变量的有理分式，做部分分式展开后查表：

单极点：$\frac{A}{z-p} \to$ 对应项。常用形式：

$$\frac{X(z)}{z} = \sum_i \frac{A_i}{z - p_i}$$

乘回 $z$：$X(z) = \sum_i \frac{A_i z}{z - p_i}$，每项逆变换 $A_i p_i^n u[n]$（对右边序列）。

重极点：$\frac{z}{(z-p)^k}$ 对应 $\frac{1}{(k-1)!} \frac{(n+k-2)!}{(n-1)!} p^n u[n]$ 等形式。

▶ 例题 · 部分分式展开法

题目：求 $X(z) = \frac{z}{(z-2)(z-3)}$（$|z| > 3$，右边序列）的逆变换。

解：

$\frac{X(z)}{z} = \frac{1}{(z-2)(z-3)} = \frac{A}{z-2} + \frac{B}{z-3}$

$A = \frac{1}{z-3}|_{z=2} = \frac{1}{-1} = -1$，$B = \frac{1}{z-2}|_{z=3} = 1$

$X(z) = \frac{-z}{z-2} + \frac{z}{z-3}$

$x[n] = -2^n u[n] + 3^n u[n] = (3^n - 2^n)u[n]$。

7.5 z 域系统分析

定义 · 系统函数 $H(z)$
$$H(z) = \ZT\{h[n]\} = \frac{Y(z)}{X(z)}$$

$H(z)$ 是单位样值响应的 Z 变换。对于由差分方程 $\sum_{k=0}^N a_k y[n-k] = \sum_{k=0}^M b_k x[n-k]$ 描述的系统：

$$H(z) = \frac{\sum_{k=0}^M b_k z^{-k}}{\sum_{k=0}^N a_k z^{-k}}$$
离散系统的稳定性判据

因果 LTI 离散系统 BIBO 稳定 $\Longleftrightarrow$ $H(z)$ 的所有极点都在单位圆内（$|p_i| < 1$）。

等价表述：ROC 包含单位圆。

类比：连续系统"所有极点在左半平面" ↔ 离散系统"所有极点在单位圆内"。这来自映射 $z = e^{sT}$：s 左半平面 → $|z| = |e^{(\sigma+j\omega)T}| = e^{\sigma T} < 1$。

频率响应（DTFT）

若 ROC 包含单位圆，则系统频率响应为 $H(e^{j\Omega}) = H(z)|_{z=e^{j\Omega}}$。

$H(e^{j\Omega})$ 是以 $2\pi$ 为周期的周期函数——这是离散时间系统独有的特性。离散时间信号的频谱总是周期性的（抽样 → 频域周期延拓）。
