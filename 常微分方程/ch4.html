第四章 · 高阶线性微分方程

4.1 高阶线性方程的一般理论

定义 · $n$ 阶线性方程
$$y^{(n)} + p_1(x) y^{(n-1)} + \cdots + p_n(x) y = f(x)$$

当 $f(x) \equiv 0$ 时称为齐次的，否则称为非齐次的。

叠加原理

设 $y_1, y_2, \ldots, y_k$ 是齐次方程的解，则它们的任意线性组合

$$C_1 y_1 + C_2 y_2 + \cdots + C_k y_k$$

也是齐次方程的解。

齐次方程通解结构

$n$ 阶齐次方程有 $n$ 个线性无关的解 $y_1, y_2, \ldots, y_n$，通解为：

$$y = C_1 y_1 + C_2 y_2 + \cdots + C_n y_n$$

线性无关性可用 Wronsky 行列式判断：

$$W(x) = \begin{vmatrix} y_1 & y_2 & \cdots & y_n \\ y_1' & y_2' & \cdots & y_n' \\ \vdots & \vdots & \ddots & \vdots \\ y_1^{(n-1)} & y_2^{(n-1)} & \cdots & y_n^{(n-1)} \end{vmatrix} \neq 0$$
非齐次方程通解结构

非齐次方程的通解 = 对应齐次方程的通解 + 非齐次方程的一个特解：

$$y = C_1 y_1 + \cdots + C_n y_n + y^*$$

4.2 常系数齐次方程

特征方程法

对 $n$ 阶常系数齐次方程

$$y^{(n)} + a_1 y^{(n-1)} + \cdots + a_n y = 0$$

令 $y = e^{\lambda x}$，代入得特征方程：

$$\lambda^n + a_1 \lambda^{n-1} + \cdots + a_n = 0$$
二阶常系数齐次方程的通解

方程 $y'' + py' + qy = 0$，特征方程 $\lambda^2 + p\lambda + q = 0$，判别式 $\Delta = p^2 - 4q$：

特征根情况通解
两个不等实根 $\lambda_1 \neq \lambda_2$$y = C_1 e^{\lambda_1 x} + C_2 e^{\lambda_2 x}$
重根 $\lambda_1 = \lambda_2 = \lambda$$y = (C_1 + C_2 x) e^{\lambda x}$
共轭复根 $\alpha \pm \beta i$$y = e^{\alpha x}(C_1 \cos\beta x + C_2 \sin\beta x)$
例1 · 特征根法

题目：求解 $y'' - 3y' + 2y = 0$。

解：特征方程 $\lambda^2 - 3\lambda + 2 = 0$，解得 $\lambda_1 = 1, \lambda_2 = 2$。

通解：$y = C_1 e^x + C_2 e^{2x}$

例2 · 重根情况

题目：求解 $y'' - 4y' + 4y = 0$。

解：特征方程 $\lambda^2 - 4\lambda + 4 = 0$，$\lambda = 2$（二重根）。

通解：$y = (C_1 + C_2 x) e^{2x}$

例3 · 复根情况

题目：求解 $y'' + y = 0$（简谐振动）。

解：特征方程 $\lambda^2 + 1 = 0$，$\lambda = \pm i$。

通解：$y = C_1 \cos x + C_2 \sin x$

4.3 常系数非齐次方程

待定系数法

对 $y'' + py' + qy = f(x)$，当 $f(x)$ 为以下形式时，特解可用待定系数法：

类型 1：$f(x) = P_m(x) e^{\alpha x}$

设 $y^* = x^k Q_m(x) e^{\alpha x}$，其中 $Q_m(x)$ 是 $m$ 次多项式，$k$ 是 $\alpha$ 作为特征根的重数。

类型 2：$f(x) = e^{\alpha x}[P_m(x)\cos\beta x + P_n(x)\sin\beta x]$

设 $y^* = x^k e^{\alpha x}[R_l(x)\cos\beta x + S_l(x)\sin\beta x]$，其中 $l = \max(m, n)$。

▶ 例题 · 待定系数法

题目：求解 $y'' - 3y' + 2y = e^{3x}$。

解：齐次通解 $y = C_1 e^x + C_2 e^{2x}$（由例1）。

$f(x) = e^{3x}$，$\alpha = 3$ 不是特征根（$k = 0$），设 $y^* = Ae^{3x}$。

$y^{*'} = 3Ae^{3x}$，$y^{*''} = 9Ae^{3x}$

代入：$9A - 9A + 2A = 1$，得 $A = \dfrac{1}{2}$。

通解：$y = C_1 e^x + C_2 e^{2x} + \dfrac{1}{2}e^{3x}$

▶ 例题 · 重根时的待定系数法

题目：求解 $y'' - 2y' + y = xe^x$。

解：特征方程 $\lambda^2 - 2\lambda + 1 = 0$，$\lambda = 1$（二重根）。

齐次通解 $y = (C_1 + C_2 x)e^x$。

$f(x) = xe^x$，$\alpha = 1$ 是二重特征根（$k = 2$），设 $y^* = x^2(Ax + B)e^x$。

代入方程比较系数，解得 $A = \dfrac{1}{6}$，$B = 0$。

通解：$y = (C_1 + C_2 x)e^x + \dfrac{x^3}{6}e^x$

4.4 常数变易法

常数变易法

当 $f(x)$ 不适合待定系数法时，可用常数变易法。设齐次方程的基本解组为 $y_1, \ldots, y_n$，设

$$y^* = u_1(x) y_1 + u_2(x) y_2 + \cdots + u_n(x) y_n$$

其中 $u_i'(x)$ 满足方程组：

$$\begin{cases} u_1' y_1 + u_2' y_2 + \cdots + u_n' y_n = 0 \\ u_1' y_1' + u_2' y_2' + \cdots + u_n' y_n' = 0 \\ \cdots \\ u_1' y_1^{(n-1)} + u_2' y_2^{(n-1)} + \cdots + u_n' y_n^{(n-1)} = f(x) \end{cases}$$
常数变易法 vs 待定系数法
待定系数法常数变易法
适用范围$f(x)$ 为多项式、指数、三角函数的组合任意 $f(x)$
计算量较小较大
通用性有限通用

4.5 降阶法

已知一个特解时的降阶

设 $y_1$ 是 $n$ 阶齐次方程的一个非零特解，令 $y = y_1 u$，代入方程可将阶数降低一阶。

二阶方程的降阶

设 $y_1$ 是 $y'' + P(x)y' + Q(x)y = 0$ 的特解，令 $y = y_1 u$，则

$$u'' + \left(\frac{2y_1'}{y_1} + P\right) u' = 0$$

这是关于 $u'$ 的一阶方程，可分离变量求解。

▶ 例题 · 降阶法

题目：已知 $y_1 = x$ 是 $x^2 y'' - xy' + y = 0$ 的解，求通解。

解：令 $y = xu$，则 $y' = xu' + u$，$y'' = xu'' + 2u'$。

代入：$x^2(xu'' + 2u') - x(xu' + u) + xu = 0$

$x^3 u'' + 2x^2 u' - x^2 u' - xu + xu = 0$，即 $x^3 u'' + x^2 u' = 0$

$xu'' + u' = 0$，令 $v = u'$：$xv' + v = 0$

分离变量：$\dfrac{v'}{v} = -\dfrac{1}{x}$，$\ln|v| = -\ln|x|$，$v = \dfrac{1}{x}$

$u = \ln|x|$，所以 $y_2 = x\ln|x|$。

通解：$y = C_1 x + C_2 x\ln|x|$

4.6 综合例题

本章核心要点
  1. 齐次方程通解 = $n$ 个线性无关解的线性组合
  2. 非齐次方程通解 = 齐次通解 + 非齐次特解
  3. 常系数方程用特征根法，根据根的类型写通解
  4. 非齐次特解用待定系数法（特殊情况）或常数变易法（通用）
  5. 已知一个特解可用降阶法
▶ 例题 · Euler 方程

题目：求解 $x^2 y'' + 3xy' + y = 0$（Euler 方程）。

解：令 $x = e^t$（$x > 0$），则 $t = \ln x$。

$y' = \dfrac{1}{x}\dfrac{\dd y}{\dd t}$，$y'' = \dfrac{1}{x^2}\left(\dfrac{\dd^2 y}{\dd t^2} - \dfrac{\dd y}{\dd t}\right)$

代入：$\dfrac{\dd^2 y}{\dd t^2} - \dfrac{\dd y}{\dd t} + 3\dfrac{\dd y}{\dd t} + y = 0$

$\dfrac{\dd^2 y}{\dd t^2} + 2\dfrac{\dd y}{\dd t} + y = 0$

特征方程 $\lambda^2 + 2\lambda + 1 = 0$，$\lambda = -1$（重根）。

$y = (C_1 + C_2 t)e^{-t} = (C_1 + C_2 \ln x) \cdot \dfrac{1}{x}$
