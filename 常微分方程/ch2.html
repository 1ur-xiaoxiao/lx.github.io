第二章 · 一阶微分方程的初等解法

2.1 可分离变量方程

定义 · 可分离变量方程

形如

$$\frac{\dd y}{\dd x} = f(x) g(y)$$

的一阶方程称为可分离变量方程。

求解方法

当 $g(y) \neq 0$ 时，分离变量：

$$\frac{\dd y}{g(y)} = f(x) \dd x$$

两边积分：

$$\int \frac{\dd y}{g(y)} = \int f(x) \dd x + C$$
例1 · 求解可分离变量方程

题目：求解 $y' = \dfrac{x}{y}$。

解：分离变量：$y \dd y = x \dd x$

两边积分：$\dfrac{y^2}{2} = \dfrac{x^2}{2} + C$

通解：$y^2 - x^2 = 2C$，即 $y^2 - x^2 = C'$（双曲线族）。

注意丢失的解

分离变量时除以 $g(y)$，可能丢失使 $g(y) = 0$ 的常数解。求解后需要检验。

例如 $y' = y^2$：分离变量得 $-\dfrac{1}{y} = x + C$，即 $y = -\dfrac{1}{x+C}$。但 $y = 0$ 也是解（使 $g(y) = y^2 = 0$），它不包含在通解中。

2.2 一阶线性方程

定义 · 一阶线性方程

形如

$$y' + P(x) y = Q(x)$$

的方程称为一阶线性方程。当 $Q(x) \equiv 0$ 时称为齐次的，否则称为非齐次的。

常数变易法与通解公式

先求齐次方程 $y' + P(x)y = 0$ 的通解：$y = Ce^{-\int P(x)\dd x}$。

将常数 $C$ 替换为函数 $C(x)$，代入非齐次方程，解得：

$$y = e^{-\int P(x)\dd x} \left[\int Q(x) e^{\int P(x)\dd x} \dd x + C\right]$$

这就是一阶线性方程的通解公式。

例2 · 求解一阶线性方程

题目：求解 $y' + \dfrac{y}{x} = x^2$。

解：$P(x) = \dfrac{1}{x}$，$Q(x) = x^2$。

$\int P(x)\dd x = \ln|x|$，$e^{\int P\dd x} = |x|$（取 $x > 0$ 时为 $x$）。

$y = \dfrac{1}{x}\left[\int x^2 \cdot x \dd x + C\right] = \dfrac{1}{x}\left[\dfrac{x^4}{4} + C\right] = \dfrac{x^3}{4} + \dfrac{C}{x}$

常数变易法的思想

常数变易法是线性微分方程理论的核心方法。其思想是：先求齐次方程的通解，再把通解中的常数"变易"为函数，代入非齐次方程确定这个函数。

这个方法不仅适用于一阶方程，也是高阶线性方程和线性方程组求解的基础（见第四、五章）。

2.3 Bernoulli 方程

定义 · Bernoulli 方程

形如

$$y' + P(x) y = Q(x) y^n \quad (n \neq 0, 1)$$

的方程称为 Bernoulli 方程。

求解方法

令 $z = y^{1-n}$，则 $z' = (1-n) y^{-n} y'$。

原方程化为关于 $z$ 的一阶线性方程：

$$z' + (1-n) P(x) z = (1-n) Q(x)$$

用通解公式求出 $z$，再回代 $y = z^{\frac{1}{1-n}}$。

▶ 例题 · Bernoulli 方程

题目：求解 $y' + \dfrac{y}{x} = y^2 \ln x$。

解：这是 Bernoulli 方程，$n = 2$。令 $z = y^{-1}$，则 $z' = -y^{-2} y'$。

原方程两边除以 $y^2$：$y^{-2} y' + \dfrac{y^{-1}}{x} = \ln x$

即 $-z' + \dfrac{z}{x} = \ln x$，或 $z' - \dfrac{z}{x} = -\ln x$。

这是一阶线性方程，$P(x) = -\dfrac{1}{x}$，$Q(x) = -\ln x$。

$e^{\int P\dd x} = e^{-\ln x} = \dfrac{1}{x}$

$z = x\left[\int -\ln x \cdot \dfrac{1}{x} \dd x + C\right] = x\left[-\dfrac{(\ln x)^2}{2} + C\right]$

回代 $y = \dfrac{1}{z} = \dfrac{1}{Cx - \frac{x(\ln x)^2}{2}}$

2.4 恰当方程

定义 · 恰当方程（全微分方程）

若存在函数 $u(x, y)$ 使得

$$\dd u = M(x, y) \dd x + N(x, y) \dd y$$

则方程 $M(x, y) \dd x + N(x, y) \dd y = 0$ 称为恰当方程（或全微分方程），其通解为 $u(x, y) = C$。

恰当方程的判别条件

设 $M, N$ 在单连通区域 $D$ 上有连续一阶偏导数，则

$$M \dd x + N \dd y = 0 \text{ 是恰当方程} \iff \frac{\partial M}{\partial y} = \frac{\partial N}{\partial x}$$
恰当方程的求解

若 $\dfrac{\partial M}{\partial y} = \dfrac{\partial N}{\partial x}$，则：

$$u(x, y) = \int_{x_0}^{x} M(t, y_0) \dd t + \int_{y_0}^{y} N(x, t) \dd t = C$$

或更实用的方法：由 $\dfrac{\partial u}{\partial x} = M$ 积分得 $u = \int M \dd x + \varphi(y)$，再由 $\dfrac{\partial u}{\partial y} = N$ 确定 $\varphi(y)$。

▶ 例题 · 恰当方程

题目：求解 $(2xy + 3) \dd x + (x^2 + 4y) \dd y = 0$。

解：$M = 2xy + 3$，$N = x^2 + 4y$。

$\dfrac{\partial M}{\partial y} = 2x$，$\dfrac{\partial N}{\partial x} = 2x$，相等，是恰当方程。

由 $\dfrac{\partial u}{\partial x} = 2xy + 3$，积分得 $u = x^2 y + 3x + \varphi(y)$。

由 $\dfrac{\partial u}{\partial y} = x^2 + \varphi'(y) = x^2 + 4y$，得 $\varphi'(y) = 4y$，$\varphi(y) = 2y^2$。

通解：$x^2 y + 3x + 2y^2 = C$

2.5 积分因子

定义 · 积分因子

若存在函数 $\mu(x, y) \neq 0$ 使得

$$\mu M \dd x + \mu N \dd y = 0$$

是恰当方程，则 $\mu(x, y)$ 称为原方程的积分因子。

积分因子的存在性

理论上，只要 $M \dd x + N \dd y = 0$ 有解，就存在积分因子。但实际上找到积分因子往往很困难。

以下是一些特殊情况下的积分因子：

特殊情况的积分因子
条件积分因子
$\frac{1}{N}\left(\frac{\partial M}{\partial y} - \frac{\partial N}{\partial x}\right) = f(x)$$\mu(x) = e^{\int f(x)\dd x}$
$\frac{1}{M}\left(\frac{\partial N}{\partial x} - \frac{\partial M}{\partial y}\right) = g(y)$$\mu(y) = e^{\int g(y)\dd y}$
$M \dd x + N \dd y = 0$ 是齐次方程$\mu = \frac{1}{xM + yN}$
▶ 例题 · 用积分因子求解

题目：求解 $y \dd x - x \dd y = 0$。

解：$M = y$，$N = -x$，$\dfrac{\partial M}{\partial y} = 1 \neq -1 = \dfrac{\partial N}{\partial x}$，不是恰当方程。

$\dfrac{1}{N}\left(\dfrac{\partial M}{\partial y} - \dfrac{\partial N}{\partial x}\right) = \dfrac{1}{-x}(1-(-1)) = -\dfrac{2}{x}$，只含 $x$。

积分因子 $\mu = e^{\int -\frac{2}{x}\dd x} = e^{-2\ln x} = \dfrac{1}{x^2}$。

乘以 $\mu$：$\dfrac{y}{x^2}\dd x - \dfrac{1}{x}\dd y = 0$，这是恰当方程。

通解：$\dfrac{y}{x} = C$，即 $y = Cx$。

2.6 综合例题

一阶方程求解策略
  1. 先看是否能分离变量
  2. 若 $y' + P(x)y = Q(x)$，用通解公式
  3. 若 $y' + P(x)y = Q(x)y^n$，用 Bernoulli 代换
  4. 若 $M\dd x + N\dd y = 0$ 且 $\dfrac{\partial M}{\partial y} = \dfrac{\partial N}{\partial x}$，是恰当方程
  5. 否则尝试找积分因子
  6. 以上都不行，考虑变量代换或其他技巧
▶ 例题 · 齐次方程

题目：求解 $y' = \dfrac{y}{x} + \tan\dfrac{y}{x}$。

解：这是齐次方程。令 $u = \dfrac{y}{x}$，则 $y = ux$，$y' = u'x + u$。

代入：$u'x + u = u + \tan u$，即 $u'x = \tan u$。

分离变量：$\dfrac{\dd u}{\tan u} = \dfrac{\dd x}{x}$，即 $\cot u \dd u = \dfrac{\dd x}{x}$

积分：$\ln|\sin u| = \ln|x| + C_1$，即 $\sin u = Cx$

回代：$\sin\dfrac{y}{x} = Cx$，通解为 $\sin\dfrac{y}{x} = Cx$。

▶ 例题 · Riccati 方程（化归思想）

题目：已知 $y_1 = x$ 是方程 $y' = y^2 - \dfrac{y}{x} + \dfrac{1}{x^2}$ 的一个特解，求通解。

解：令 $y = y_1 + z = x + z$，代入方程。

$y' = 1 + z'$，$y^2 = x^2 + 2xz + z^2$

$1 + z' = x^2 + 2xz + z^2 - \dfrac{x+z}{x} + \dfrac{1}{x^2}$

$= x^2 + 2xz + z^2 - 1 - \dfrac{z}{x} + \dfrac{1}{x^2}$

利用 $y_1 = x$ 是特解，化简得 $z' = \left(2x - \dfrac{1}{x}\right)z + z^2$。

这是关于 $z$ 的 Bernoulli 方程（$n = 2$），令 $w = z^{-1}$ 可化为线性方程求解。
