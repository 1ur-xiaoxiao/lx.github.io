3.1 Picard 逐次逼近法

考虑初值问题 $y' = f(x, y)$，$y(x_0) = y_0$。Picard 方法的基本思想是将微分方程转化为积分方程，然后用迭代法逐步逼近解。

积分方程形式

初值问题 $y' = f(x, y)$，$y(x_0) = y_0$ 等价于积分方程

$$y(x) = y_0 + \int_{x_0}^{x} f(t, y(t)) \dd t$$
Picard 逐次逼近序列

定义函数序列 $\{\varphi_n(x)\}$：

$$\varphi_0(x) = y_0$$ $$\varphi_n(x) = y_0 + \int_{x_0}^{x} f(t, \varphi_{n-1}(t)) \dd t, \quad n = 1, 2, 3, \ldots$$

这个序列称为 Picard 逐次逼近序列。

例1 · Picard 逼近计算

题目：用 Picard 方法求 $y' = y$，$y(0) = 1$ 的逐次逼近。

解：$\varphi_0(x) = 1$

$\varphi_1(x) = 1 + \int_0^x 1 \dd t = 1 + x$

$\varphi_2(x) = 1 + \int_0^x (1+t) \dd t = 1 + x + \dfrac{x^2}{2}$

$\varphi_3(x) = 1 + x + \dfrac{x^2}{2} + \dfrac{x^3}{6}$

一般地，$\varphi_n(x) = \displaystyle\sum_{k=0}^{n} \frac{x^k}{k!}$，收敛到 $e^x$（精确解）。

3.2 解的存在唯一性定理

Picard 存在唯一性定理

设 $f(x, y)$ 在矩形区域

$$R: |x - x_0| \leq a, \quad |y - y_0| \leq b$$

上满足：

  1. $f(x, y)$ 在 $R$ 上连续（保证解存在）
  2. $f(x, y)$ 关于 $y$ 满足 Lipschitz 条件：$\exists\, L > 0$，使得
$$|f(x, y_1) - f(x, y_2)| \leq L |y_1 - y_2|, \quad \forall (x, y_1), (x, y_2) \in R$$

则初值问题 $y' = f(x, y)$，$y(x_0) = y_0$ 在区间 $|x - x_0| \leq h$ 上存在唯一解，其中 $h = \min\left(a, \dfrac{b}{M}\right)$，$M = \max_R |f(x, y)|$。

Lipschitz 条件

Lipschitz 条件比"连续"强，但比"有界偏导数"弱。实际应用中，最常用的判别方法是：

若 $\dfrac{\partial f}{\partial y}$ 在 $R$ 上连续（从而有界），则 $f$ 关于 $y$ 满足 Lipschitz 条件。

Lipschitz 条件保证了唯一性。仅有连续性只能保证存在性（Peano 定理）。

▶ 例题 · 验证 Lipschitz 条件

题目：验证 $f(x, y) = x^2 + y^2$ 在 $|x| \leq 1, |y| \leq 1$ 上关于 $y$ 满足 Lipschitz 条件。

解：$\dfrac{\partial f}{\partial y} = 2y$，在 $|y| \leq 1$ 上 $|2y| \leq 2$。

由中值定理：$|f(x, y_1) - f(x, y_2)| = |2\xi| \cdot |y_1 - y_2| \leq 2|y_1 - y_2|$

所以 Lipschitz 常数 $L = 2$。

3.3 Peano 存在定理

Peano 存在定理

设 $f(x, y)$ 在矩形区域 $R$ 上连续，则初值问题 $y' = f(x, y)$，$y(x_0) = y_0$ 在 $|x - x_0| \leq h$ 上至少存在一个解。

（不要求 Lipschitz 条件，但不能保证唯一性。）

Peano vs Picard
Peano 定理Picard 定理
条件$f$ 连续$f$ 连续 + Lipschitz
结论解存在（至少一个）解存在且唯一
证明方法紧性论证逐次逼近（构造性）

Picard 定理更强（条件更强，结论也更强），但 Peano 定理的条件更弱，适用范围更广。

▶ 不满足 Lipschitz 条件的例子

题目：$y' = \sqrt{|y|}$，$y(0) = 0$。讨论解的存在唯一性。

解：$f(x, y) = \sqrt{|y|}$ 连续，所以 Peano 定理保证解存在。

但 $\dfrac{\partial f}{\partial y} = \dfrac{\operatorname{sgn}(y)}{2\sqrt{|y|}}$，当 $y \to 0$ 时趋于无穷，不满足 Lipschitz 条件。

实际上，$y = 0$ 和 $y = \dfrac{x^2}{4}$（$x \geq 0$）都是解，不唯一！

3.4 解的延拓

解的延拓定理

设 $f(x, y)$ 在区域 $D$ 上连续且关于 $y$ 满足局部 Lipschitz 条件，则初值问题的解可以延拓到区域 $D$ 的边界。

具体地，解 $y = \varphi(x)$ 的最大存在区间 $(a, b)$ 满足：当 $x \to a^+$ 或 $x \to b^-$ 时，$(x, \varphi(x))$ 趋向 $D$ 的边界，或 $|\varphi(x)| \to \infty$。

延拓定理的意义

局部存在唯一性定理给出的解可能只在一个小区间上存在。延拓定理告诉我们：只要 $f$ 的性质允许，解可以一直延伸到"边界"。

例如 $y' = y^2$，$y(0) = 1$ 的解 $y = \dfrac{1}{1-x}$ 在 $x \to 1^-$ 时趋于无穷，所以最大存在区间是 $(-\infty, 1)$。

3.5 解对初值的连续依赖性

解对初值的连续依赖性

设 $f(x, y)$ 在区域 $D$ 上连续且关于 $y$ 满足局部 Lipschitz 条件，$\varphi(x, x_0, y_0)$ 是初值问题的解。则 $\varphi$ 关于初值 $(x_0, y_0)$ 连续。

直观理解：初值的微小变化只导致解的微小变化。

物理意义

这个定理保证了微分方程模型的稳定性——如果初始条件的测量有小误差，预测结果也只有小误差。否则微分方程就无法用于实际预测。

3.6 综合例题

本章核心要点
  1. Picard 逐次逼近法将微分方程转化为积分方程的迭代
  2. 存在唯一性需要两个条件：连续性（存在性）+ Lipschitz 条件（唯一性）
  3. Lipschitz 条件通常通过 $\partial f / \partial y$ 有界来验证
  4. Peano 定理只要连续性，但不能保证唯一
  5. 解可以延拓到区域边界或无穷
▶ 例题 · 存在唯一性的应用

题目：讨论 $y' = y^{1/3}$，$y(0) = 0$ 的解的存在唯一性。

解：$f(x, y) = y^{1/3}$ 在 $(0, 0)$ 附近连续，Peano 定理保证解存在。

$\dfrac{\partial f}{\partial y} = \dfrac{1}{3} y^{-2/3}$，在 $y = 0$ 处无界，不满足 Lipschitz 条件。

实际上 $y = 0$ 和 $y = \left(\dfrac{2x}{3}\right)^{3/2}$（$x \geq 0$）都是过原点的解，不唯一。
