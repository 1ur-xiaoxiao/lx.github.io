第六章 · 常微分方程的幂级数解法

当初等方法失效时：用幂级数求解微分方程，Frobenius 方法与正则奇点理论。

本章目录

  1. 幂级数解法的基本思想

6.1 幂级数解法的基本思想

许多重要的微分方程（如 Bessel 方程、Legendre 方程）无法用初等函数表示解。幂级数解法提供了构造解的系统方法。

基本思路

对二阶线性方程 $y'' + p(x)y' + q(x)y = 0$，假设解具有幂级数形式

$$y = \sum_{n=0}^{\infty} a_n (x - x_0)^n$$

代入方程，比较系数得到递推关系，从而确定系数 $a_n$。

6.2 常点邻域的幂级数解

定义 · 常点与奇点

对 $y'' + p(x)y' + q(x)y = 0$，若 $p(x)$ 和 $q(x)$ 在 $x_0$ 的某邻域内解析（可展开为收敛幂级数），则 $x_0$ 称为方程的常点，否则称为奇点。

常点邻域的幂级数解

若 $x_0$ 是方程的常点，则方程在 $x_0$ 的某个邻域内存在两个线性无关的幂级数解。

▶ 例题 · 常点处的幂级数解

题目：求 $y'' - xy = 0$ 在 $x = 0$ 附近的幂级数解。

解：$p(x) = 0$，$q(x) = -x$，$x = 0$ 是常点。设 $y = \displaystyle\sum_{n=0}^{\infty} a_n x^n$。

$y'' = \displaystyle\sum_{n=2}^{\infty} n(n-1) a_n x^{n-2} = \sum_{n=0}^{\infty} (n+2)(n+1) a_{n+2} x^n$

$xy = \displaystyle\sum_{n=0}^{\infty} a_n x^{n+1} = \sum_{n=1}^{\infty} a_{n-1} x^n$

代入 $y'' - xy = 0$，比较 $x^n$ 的系数：

$n = 0$：$2 \cdot 1 \cdot a_2 = 0$，$a_2 = 0$

$n \geq 1$：$(n+2)(n+1) a_{n+2} - a_{n-1} = 0$，即 $a_{n+2} = \dfrac{a_{n-1}}{(n+2)(n+1)}$

取 $a_0 = 1, a_1 = 0$：$a_2 = 0$，$a_3 = \dfrac{a_0}{3 \cdot 2} = \dfrac{1}{6}$，$a_4 = a_5 = 0$，$a_6 = \dfrac{a_3}{6 \cdot 5} = \dfrac{1}{180}$，……

$y_1 = 1 + \dfrac{x^3}{6} + \dfrac{x^6}{180} + \cdots$

取 $a_0 = 0, a_1 = 1$：$a_2 = 0$，$a_4 = \dfrac{a_1}{4 \cdot 3} = \dfrac{1}{12}$，$a_5 = 0$，$a_7 = \dfrac{a_4}{7 \cdot 6} = \dfrac{1}{504}$，……

$y_2 = x + \dfrac{x^4}{12} + \dfrac{x^7}{504} + \cdots$

6.3 正则奇点与 Frobenius 方法

定义 · 正则奇点

设 $x_0$ 是 $y'' + p(x)y' + q(x)y = 0$ 的奇点。若

$$(x - x_0) p(x) \quad \text{和} \quad (x - x_0)^2 q(x)$$

在 $x_0$ 处解析，则 $x_0$ 称为正则奇点，否则为非正则奇点。

Frobenius 方法

在正则奇点 $x_0$ 处，设解具有形式：

$$y = (x - x_0)^r \sum_{n=0}^{\infty} a_n (x - x_0)^n, \quad a_0 \neq 0$$

代入方程得到指标方程（indicial equation），确定 $r$ 的值，再由递推关系确定 $a_n$。

Frobenius 定理

设 $x_0$ 是正则奇点，指标方程的两个根为 $r_1, r_2$（$r_1 \geq r_2$）。

  1. $r_1 - r_2 \notin \ZZ$：两个线性无关的 Frobenius 解
  2. $r_1 = r_2$：一个 Frobenius 解，第二个解含对数项
  3. $r_1 - r_2 \in \ZZ^+$：一个 Frobenius 解，第二个解可能含对数项

6.4 Bessel 方程

Bessel 方程
$$x^2 y'' + xy' + (x^2 - \nu^2) y = 0$$

$x = 0$ 是正则奇点。化为标准形式：$y'' + \dfrac{1}{x}y' + \left(1 - \dfrac{\nu^2}{x^2}\right)y = 0$。

Bessel 函数

指标方程：$r(r-1) + r - \nu^2 = 0$，即 $r^2 = \nu^2$，$r = \pm\nu$。

当 $\nu \notin \ZZ$ 时，两个线性无关解为 Bessel 函数：

$$J_\nu(x) = \sum_{k=0}^{\infty} \frac{(-1)^k}{k! \Gamma(k+\nu+1)} \left(\frac{x}{2}\right)^{2k+\nu}$$ $$J_{-\nu}(x) = \sum_{k=0}^{\infty} \frac{(-1)^k}{k! \Gamma(k-\nu+1)} \left(\frac{x}{2}\right)^{2k-\nu}$$
Bessel 方程的物理背景

Bessel 方程在物理学中无处不在：圆柱坐标下的热传导、波动问题都会导出 Bessel 方程。$J_\nu(x)$ 描述了圆对称系统的振动模式。

6.5 Legendre 方程

Legendre 方程
$$(1 - x^2) y'' - 2xy' + n(n+1) y = 0$$

$x = 0$ 是常点，$x = \pm 1$ 是正则奇点。

Legendre 多项式

当 $n$ 为非负整数时，方程有一个 $n$ 次多项式解——Legendre 多项式：

$$P_n(x) = \frac{1}{2^n n!} \frac{\dd^n}{\dd x^n}(x^2 - 1)^n$$

（Rodrigues 公式）

前几个：$P_0(x) = 1$，$P_1(x) = x$，$P_2(x) = \dfrac{3x^2 - 1}{2}$，$P_3(x) = \dfrac{5x^3 - 3x}{2}$

Legendre 方程的物理背景

Legendre 方程在球坐标下的 Laplace 方程分离变量时出现。$P_n(x)$ 是球谐函数的基础，在电磁学、量子力学中有广泛应用。

6.6 综合例题

本章核心要点
  1. 常点处的解可以展开为 Taylor 级数
  2. 正则奇点处用 Frobenius 方法，解含 $(x-x_0)^r$ 因子
  3. 指标方程决定 $r$ 的值，递推关系决定系数
  4. Bessel 方程和 Legendre 方程是最重要的特殊函数方程
▶ 例题 · Euler 方程的 Frobenius 解法

题目：用 Frobenius 方法求 $x^2 y'' + xy' - y = 0$ 在 $x = 0$ 附近的解。

解：$p(x) = \dfrac{1}{x}$，$q(x) = -\dfrac{1}{x^2}$，$x = 0$ 是正则奇点。

设 $y = x^r \displaystyle\sum_{n=0}^{\infty} a_n x^n$，代入得指标方程：

$r(r-1) + r - 1 = 0$，即 $r^2 - 1 = 0$，$r = \pm 1$。

$r_1 - r_2 = 2 \in \ZZ$，但代入后发现 $a_n = 0$（$n \geq 1$），得 $y_1 = x$。

第二个解：$y_2 = \dfrac{1}{x}$（$r = -1$），不含对数项。

通解：$y = C_1 x + C_2 \dfrac{1}{x}$
