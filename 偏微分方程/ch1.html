1.1 物理推导：弦如何振动？

本章动机

拨动一根吉他弦，它为什么振动？振动按什么规律传播？为什么手指按在不同的品位上会发出不同的音高？本章从物理出发，逐步建立波动方程的数学理论。学完本章后，你将能够：

  • 从物理定律出发推导波动方程
  • 用 d'Alembert 公式直接写出无限长弦上任意初始扰动的解
  • 用分离变量法求解有界弦的振动（含非齐次方程与边界条件）
  • 用齐次化原理和特征函数展开法处理非齐次问题
  • 理解能量守恒如何保证解的唯一性

物理模型与基本假设

考虑一根长度为 $l$ 的柔软均匀细弦，两端固定，在平衡位置附近做微小横振动。设 $u(x,t)$ 表示弦上坐标为 $x$ 的点在时刻 $t$ 的横向位移。

直觉建立

想象你站在桥上，看着下方一条被拉紧的水平缆绳。你踢了它一脚——扰动以一个确定的速度沿缆绳传播。这就是波。关键问题是：位移 $u$ 如何依赖于位置 $x$ 和时间 $t$？

基本假设
  1. 弦是完全柔软的（无抗弯刚度），张力沿切线方向；
  2. 振动是微小的：$|\pd u/\pd x| \ll 1$，从而张力的水平分量近似为常数 $T_0$；
  3. 弦是均匀的，线密度 $\rho$ 为常数；
  4. 只考虑横向振动，忽略重力和阻尼。

从牛顿定律到偏微分方程

取弦上微元 $[x, x+\Delta x]$：

  • 在 $x$ 处，张力 $T_1$ 与水平方向夹角为 $\alpha_1$；
  • 在 $x+\Delta x$ 处，张力 $T_2$ 与水平方向夹角为 $\alpha_2$。

由水平方向力平衡：$T_2 \cos\alpha_2 = T_1 \cos\alpha_1 = T_0$（常数）。

由竖直方向 Newton 第二定律：

$$T_2 \sin\alpha_2 - T_1 \sin\alpha_1 = \rho \Delta x \cdot \frac{\pd^2 u}{\pd t^2}.$$

在小振动假设下 $\tan\alpha \approx \sin\alpha \approx \pd u/\pd x$，代入得

$$T_0\left[\frac{\pd u}{\pd x}\bigg|_{x+\Delta x} - \frac{\pd u}{\pd x}\bigg|_{x}\right] = \rho \Delta x \cdot \frac{\pd^2 u}{\pd t^2}.$$

两边除以 $\Delta x$ 并令 $\Delta x \to 0$：

一维波动方程
$$\boxed{\frac{\pd^2 u}{\pd t^2} = a^2 \frac{\pd^2 u}{\pd x^2}}, \quad a = \sqrt{\frac{T_0}{\rho}}.$$

其中 $a$ 为波速。张力越大、弦越轻，波速越快。

物理直觉：为什么 $u_{tt} \propto u_{xx}$？

$u_{xx}$ 是弦的弯曲程度（曲率的线性近似）。弦弯曲得越厉害，张力的竖直分量差越大，产生的加速度 $u_{tt}$ 就越大。所以 $u_{tt}$ 正比于 $u_{xx}$——这就是波动方程的核心物理：弯曲驱动加速度。

定解条件

初始条件与边界条件

初始条件（两个！）：因为方程含二阶时间导数，需要两个初始条件：

$$u(x,0) = \varphi(x), \quad u_t(x,0) = \psi(x), \quad 0 \le x \le l.$$

$\varphi(x)$ 为初始位移（弦被拨成什么形状），$\psi(x)$ 为初始速度（弦被多快地释放）。

边界条件（两端固定）：$u(0,t) = 0, \quad u(l,t) = 0, \quad t \ge 0.$

例题 1.1.1

一根长 $l$ 的弦，初始位移 $\varphi(x) = \sin(\pi x/l)$，初始速度 $\psi(x)=0$，两端固定。写出定解问题。

▶ 详细解答

$$\begin{cases} u_{tt} = a^2 u_{xx}, & 0 < x < l,\ t > 0, \\ u(0,t) = u(l,t) = 0, & t \ge 0, \\ u(x,0) = \sin\frac{\pi x}{l},\; u_t(x,0) = 0, & 0 \le x \le l. \end{cases}$$

物理直觉：初始形状恰好是一个正弦半波——这是弦的"基频模态"。振动时应保持正弦形态做简谐振荡。后面用分离变量法可验证：$u(x,t) = \sin\frac{\pi x}{l}\cos\frac{\pi a t}{l}$。

1.2 一维初值问题：d'Alembert 公式

考虑无限长弦上的波动（无边界，只有初始条件）——Cauchy问题：

$$\begin{cases} u_{tt} = a^2 u_{xx}, & x \in \RR,\ t > 0, \\ u(x,0) = \varphi(x),\; u_t(x,0) = \psi(x), & x \in \RR. \end{cases}$$
核心思想：化PDE为两个ODE

如果定义新变量 $\xi = x + at$，$\eta = x - at$，波动方程 $u_{tt} = a^2 u_{xx}$ 奇迹般地变成

$$\frac{\pd^2 u}{\pd\xi\,\pd\eta} = 0.$$

这就好办了——对 $\xi$ 积分一次，对 $\eta$ 再积分一次，通解就出来了！

推导过程

引入特征变换 $\xi = x + at$，$\eta = x - at$。由链式法则：

$$\frac{\pd}{\pd x} = \frac{\pd}{\pd\xi} + \frac{\pd}{\pd\eta}, \quad \frac{\pd}{\pd t} = a\left(\frac{\pd}{\pd\xi} - \frac{\pd}{\pd\eta}\right).$$

代入 $u_{tt} = a^2 u_{xx}$ 得 $\frac{\pd^2 u}{\pd\xi\,\pd\eta} = 0$。

逐次积分得通解：$u(x,t) = f(x+at) + g(x-at)$，其中 $f, g$ 为任意二次可微函数。

$f(x+at)$ 是左行波（以速度 $a$ 向 $-x$ 方向传播），$g(x-at)$ 是右行波（以速度 $a$ 向 $+x$ 方向传播）。任何初始扰动都会分解为左右两个行波！

代入初始条件确定 $f, g$：

$$f(x) + g(x) = \varphi(x), \quad a f'(x) - a g'(x) = \psi(x).$$

第二式积分得 $f(x) - g(x) = \frac{1}{a}\int_0^x \psi(s)\,\dd s + C$。联立解得：

d'Alembert 公式（1747）
$$\boxed{u(x,t) = \frac{1}{2}\big[\varphi(x+at) + \varphi(x-at)\big] + \frac{1}{2a}\int_{x-at}^{x+at}\psi(s)\,\dd s}.$$

若 $\varphi \in C^2(\RR)$，$\psi \in C^1(\RR)$，此公式给出初值问题的唯一经典解。

公式的物理含义

  • 第一项：初始位移 $\varphi$ 分解为两个"半高"的波，各以速度 $a$ 向左右传播；
  • 第二项：初始速度 $\psi$ 在区间 $[x-at, x+at]$ 上的累积效应。
例题 1.2.1

求解 $u_{tt} = u_{xx}$，$u(x,0) = e^{-x^2}$，$u_t(x,0) = 0$。

▶ 详细解答

$a=1$，$\psi=0$，由 d'Alembert 公式直接得：

$$u(x,t) = \frac{1}{2}\big[e^{-(x+t)^2} + e^{-(x-t)^2}\big].$$

物理图像：初始的钟形"鼓包"分裂为两个高度减半的钟形波，分别以速度 $1$ 向左右传播，形状保持不变。

例题 1.2.2

求解 $u_{tt} = 4u_{xx}$，$u(x,0)=0$，$u_t(x,0) = \begin{cases} 1, & |x| \le 1, \\ 0, & |x| > 1. \end{cases}$

▶ 详细解答

$a=2$，$\varphi=0$。由 d'Alembert 公式：$u(x,t) = \frac{1}{4}\int_{x-2t}^{x+2t}\psi(s)\,\dd s$。

在 $(x,t)$ 平面上分段讨论积分区间 $[x-2t, x+2t]$ 与 $[-1,1]$ 的交集。当 $t$ 很小时 $[x-2t, x+2t] \subset [-1,1]$，积分值为 $4t$，故 $u = t$。当 $t$ 增大时需分区域讨论，得到一个向两侧扩散的矩形脉冲。

1.3 依赖区间、决定区域与影响区域

d'Alembert 公式揭示了一个深刻的几何结构：信息以有限速度 $a$ 沿特征线传播。

特征线与三个核心概念

特征线：$x - at = \text{const}$ 和 $x + at = \text{const}$，斜率为 $\pm 1/a$。

依赖区间：点 $(x_0, t_0)$ 处的解 $u(x_0, t_0)$ 仅依赖于初始数据在区间 $[x_0 - at_0,\; x_0 + at_0]$ 上的值。

决定区域：过 $(x_0, 0)$ 向上作两条特征线，围成的三角形区域——该区域内任一点的值完全由 $x_0$ 处的初始数据决定。

影响区域：点 $(x_0, 0)$ 发出的两条特征线之间的区域（向右上方展开的锥形）——$(x_0, 0)$ 处初始数据的改变会影响到这个区域内的所有点。

物理直觉

站在 $(x_0, 0)$ 处跺一脚产生的扰动，以速度 $a$ 向两侧传播。在时刻 $t$，这个扰动最远到达 $x_0 \pm at$。在 $(x,t)$ 平面上，这个传播过程由两条特征线 $x - x_0 = \pm at$ 界定。

如果你在 $x_0$ 跺脚，位于 $(x_1, t_1)$ 的观察者能感受到你的跺脚，当且仅当 $|x_1 - x_0| \le a t_1$——即观察者在你的影响区域内。

有限传播速度——波动方程区别于热传导方程的根本特征

波动方程的信号以有限速度 $a$ 传播。而热传导方程中，一个点的温度改变会瞬间（理论上）影响到任意远处的温度。这一根本区别将在第二章中深入讨论。

1.4 半无界问题：延拓法（反射法）

现在考虑半无界区域 $x \ge 0$ 上的波动——即弦有一端固定在 $x=0$ 处。

Dirichlet 条件（固定端点）

Dirichlet 半无界问题
$$\begin{cases} u_{tt} = a^2 u_{xx}, & x > 0,\ t > 0, \\ u(x,0) = \varphi(x),\; u_t(x,0) = \psi(x), & x \ge 0, \\ u(0,t) = 0, & t \ge 0. \end{cases}$$
核心技巧：奇延拓——"想象弦的另一半"

把初始数据 $\varphi, \psi$ 奇延拓到 $x < 0$（即令 $\tilde\varphi(-x) = -\varphi(x)$），则 $x=0$ 自动是节点（$u=0$），满足Dirichlet条件。然后在全空间上用 d'Alembert 公式，再取 $x \ge 0$ 的部分即可。

物理图像：一个波到达固定端点后，会以相反的相位反射回来——就像你甩动绳子的一端，波在固定端"弹回来"时上下颠倒了。

奇延拓：$\tilde{\varphi}(x) = \begin{cases} \varphi(x), & x \ge 0, \\ -\varphi(-x), & x < 0, \end{cases} \quad \tilde{\psi}(x) = \begin{cases} \psi(x), & x \ge 0, \\ -\psi(-x), & x < 0. \end{cases}$

当 $x - at \ge 0$（尚未到达边界）：直接用 d'Alembert 公式；

当 $x - at < 0$（已反射）：

$$u(x,t) = \frac{1}{2}[\varphi(x+at) - \varphi(at-x)] + \frac{1}{2a}\left[\int_0^{x+at}\psi(s)\,\dd s - \int_0^{at-x}\psi(s)\,\dd s\right].$$
例题 1.4.1（固定端反射 · 奇延拓）

求解 $u_{tt} = u_{xx}$（$x>0$），$u(x,0) = 0$，$u_t(x,0) = 1$，$u(0,t) = 0$。

▶ 详细解答

奇延拓：$\tilde\psi(x) = \operatorname{sgn}(x)$（$x>0$ 时 $\tilde\psi=1$，$x<0$ 时 $\tilde\psi=-1$）。

当 $x \ge t$（波前未到边界）：$u(x,t) = \frac{1}{2}\int_{x-t}^{x+t} 1\,\dd s = t$。

当 $0 < x < t$（已反射）：$u(x,t) = \frac{1}{2}[\int_0^{x+t}1\,\dd s - \int_0^{x-t}(-1)\,\dd s] = x$。

综合：$u(x,t) = \min(x, t)$。在波前 $x=t$ 到达之前位移等于 $x$，到达之后位移等于 $t$（波前沿 $x=t$ 以速度 1 向右推进）。

Neumann 条件（自由端点）

Neumann 半无界问题
$$\begin{cases} u_{tt} = a^2 u_{xx}, & x > 0,\ t > 0, \\ u(x,0) = \varphi(x),\; u_t(x,0) = \psi(x), & x \ge 0, \\ u_x(0,t) = 0, & t \ge 0. \end{cases}$$
核心技巧：偶延拓——"照镜子，不翻转"

把初始数据 $\varphi, \psi$ 偶延拓到 $x < 0$（即令 $\tilde\varphi(-x) = \varphi(x)$，不加负号），则 $x=0$ 处导数自动为零（偶函数在原点导数为零），满足 Neumann 条件。然后在全空间上用 d'Alembert 公式，再取 $x \ge 0$ 的部分即可。

物理图像：波到达自由端点后，以相同相位反射回来——端点像套在光滑圆环上一样随波上下晃动，波峰反射回来仍是波峰。

偶延拓：$\tilde{\varphi}(x) = \begin{cases} \varphi(x), & x \ge 0, \\ \varphi(-x), & x < 0, \end{cases} \quad \tilde{\psi}(x) = \begin{cases} \psi(x), & x \ge 0, \\ \psi(-x), & x < 0. \end{cases}$

当 $x - at \ge 0$（尚未到达边界）：直接用 d'Alembert 公式；

当 $x - at < 0$（已反射），将 d'Alembert 积分区间 $[x-at, x+at]$ 跨过原点拆为负半轴和正半轴两段：

$$\int_{x-at}^{x+at} \tilde{\psi}(s)\,\dd s = \int_{x-at}^{0} \tilde{\psi}(s)\,\dd s + \int_{0}^{x+at} \tilde{\psi}(s)\,\dd s.$$

负半轴上 $\tilde{\psi}(s) = \psi(-s)$（注意前面没有负号，这是与奇延拓的关键区别）。令 $t'=-s$：

$$\int_{x-at}^{0} \psi(-s)\,\dd s = -\int_{at-x}^{0} \psi(t')\,\dd t' = \int_{0}^{at-x} \psi(t')\,\dd t'.$$

故反射后的积分中，两项之间是加号（$+$），与奇延拓的减号正好相反：

$$u(x,t) = \frac{1}{2}[\varphi(x+at) + \varphi(at-x)] + \frac{1}{2a}\left[\int_0^{x+at}\psi(s)\,\dd s + \int_0^{at-x}\psi(s)\,\dd s\right].$$

例题 1.4.2（自由端反射 · 偶延拓）

求解 $u_{tt} = u_{xx}$（$x>0$），$u(x,0) = 0$，$u_t(x,0) = 1$，$u_x(0,t) = 0$。

▶ 详细解答

偶延拓：$\tilde\psi(x) = 1$（对一切 $x$，包括 $x<0$ 也取 $1$，不加负号——常数函数的偶延拓就是它自身）。

当 $x \ge t$（波前未到边界）：$u(x,t) = \frac{1}{2}\int_{x-t}^{x+t} 1\,\dd s = t$。

当 $0 < x < t$（已反射，$x-t<0$，积分区间跨过原点）：

$$\begin{aligned} u(x,t) &= \frac{1}{2}\left[\int_{x-t}^{0} 1\,\dd s + \int_0^{x+t} 1\,\dd s\right] \\ &= \frac{1}{2}\Big[(0 - (x-t)) + (x+t)\Big] = t. \end{aligned}$$

综合：$u(x,t) = t$（对所有 $x>0$）。

与 例题 1.4.1 对比：同样的初速度 $\psi=1$，固定端条件下解为 $\min(x,t)$（波被反相弹回），而自由端条件下解恒为 $t$——边界不产生反作用力，整根半无界弦以匀速向上运动，端点随波一起移动。

验证边界条件：$u(0,t) = t \neq 0$（端点在动！），但对 $x$ 求偏导得 $u_x(0,t) \equiv 0$，满足 Neumann 条件。这就是自由端"端点跟着波一起上下晃动"的画面。

齐次化原理（Duhamel 原理）—— 非齐次波动方程的 Cauchy 问题

d'Alembert 公式解决了齐次波动方程 $u_{tt}=a^2u_{xx}$ 的 Cauchy 问题。但在实际中，弦上可能作用着外力（如重力、风力等），导致方程变为非齐次的：

$$u_{tt} = a^2 u_{xx} + f(x,t).$$

$f(x,t)$ 称为源项，表示 $t$ 时刻在 $x$ 处单位质量所受的外力。本节介绍处理这类问题的核心方法——齐次化原理（又称 Duhamel 原理），它将非齐次问题转化为一系列齐次问题的叠加。更一般的 Duhamel 原理在第四章 §4.5中统一论述。

问题陈述

非齐次波动方程的 Cauchy 问题
$$\begin{cases} u_{tt} = a^2 u_{xx} + f(x,t), & x \in \RR,\ t > 0, \\ u(x,0) = \varphi(x),\; u_t(x,0) = \psi(x), & x \in \RR. \end{cases}$$

由叠加原理，解可写成 $u = u_h + u_p$，其中：

  • $u_h$ 是齐次方程（$f \equiv 0$）满足初始条件 $(\varphi, \psi)$ 的解——直接由 d'Alembert 公式给出。
  • $u_p$ 是非齐次方程满足零初始条件的解——这正是齐次化原理要解决的部分。

因此，只需集中精力求解零初始条件下的非齐次问题：

$$\begin{cases} u_{tt} = a^2 u_{xx} + f(x,t), & x \in \RR,\ t > 0, \\ u(x,0) = 0,\; u_t(x,0) = 0, & x \in \RR. \end{cases}$$

核心思想：把外力看作"一连串瞬时冲量"

物理直觉

想象弦上的外力 $f(x,t)$ 是由无穷多个"瞬时敲击"组成的：在时刻 $\tau$，弦在位置 $x$ 处被敲了一下，获得一个瞬时速度增量 $f(x,\tau)\,\dd\tau$（冲量 = 力 × 时间微元）。之后的每一瞬间弦都自由振动（没有外力了），但敲击产生的扰动会传播和叠加。

因此，总效果 = 将 $\tau=0$ 到 $\tau=t$ 之间所有时刻的敲击所引起的自由振动全部叠加起来。

关键辅助问题

设 $\tau \ge 0$ 为固定时刻。考虑下面这个辅助齐次问题：

辅助问题 $v(x,t;\tau)$
$$\begin{cases} v_{tt} = a^2 v_{xx}, & x \in \RR,\ t > \tau, \\ v(x,\tau;\tau) = 0,\; v_t(x,\tau;\tau) = f(x,\tau), & x \in \RR. \end{cases}$$

这描述的是：在时刻 $\tau$，弦获得了一个瞬时速度分布 $f(x,\tau)$（初始位移为零），之后自由振动。由 d'Alembert 公式（将时间原点平移到 $\tau$，即令 $t' = t-\tau$）：

$$v(x,t;\tau) = \frac{1}{2a}\int_{x-a(t-\tau)}^{x+a(t-\tau)} f(s,\tau)\,\dd s.$$

对 $v$ 的公式的理解：从 $\tau$ 时刻开始到 $t$ 时刻，扰动的传播距离为 $a(t-\tau)$。$f(s,\tau)$ 只在以 $x$ 为中心、半径 $a(t-\tau)$ 的区间内影响 $v(x,t;\tau)$——这正是波动有限传播速度 $a$ 的体现。

齐次化原理 → 叠加所有瞬时冲量

齐次化原理断言：原问题（零初始条件）的解是辅助问题解关于 $\tau$ 的叠加（积分）：

$$u_p(x,t) = \int_0^t v(x,t;\tau)\,\dd\tau.$$

这个结论的数学依据是：对 $u_p(x,t) = \int_0^t v(x,t;\tau)\,\dd\tau$ 求导（使用含参积分求导的 Leibniz 法则），可以验证它确实满足非齐次方程和零初始条件（验证过程见下方）。

代入 $v$ 的 d'Alembert 表达式，得到最终公式：

非齐次波动方程 Cauchy 问题的解（零初始条件）
$$u_p(x,t) = \frac{1}{2a}\int_0^t\int_{x-a(t-\tau)}^{x+a(t-\tau)} f(s,\tau)\,\dd s\,\dd\tau.$$

几何意义：积分区域是 $(s,\tau)$ 平面上的特征三角形（以 $(x,t)$ 为顶点的向后特征锥截出的区域）。

验证：$u_p$ 确实满足非齐次方程

由含参积分求导的 Leibniz 公式：

$$\begin{aligned} \frac{\pd u_p}{\pd t} &= v(x,t;t) + \int_0^t \frac{\pd v}{\pd t}(x,t;\tau)\,\dd\tau \\ &= 0 + \int_0^t v_t(x,t;\tau)\,\dd\tau \quad (\text{因为 } v(x,t;t)=0), \\ \frac{\pd^2 u_p}{\pd t^2} &= v_t(x,t;t) + \int_0^t v_{tt}(x,t;\tau)\,\dd\tau \\ &= f(x,t) + \int_0^t a^2 v_{xx}(x,t;\tau)\,\dd\tau \quad (\text{用 } v_t(x,t;t)=f(x,t),\; v_{tt}=a^2v_{xx}), \\ &= f(x,t) + a^2 \frac{\pd^2 u_p}{\pd x^2}. \end{aligned}$$

故 $u_{tt} = a^2 u_{xx} + f$。初始条件 $u_p(x,0)=0$（积分区间长度为零）、$\frac{\pd u_p}{\pd t}(x,0)=0$ 显然成立。$\square$

完整解（齐次化原理 + d'Alembert 叠加）

将齐次解（d'Alembert）和非齐次特解（Duhamel 积分）叠加，得到最一般的 Cauchy 问题的解：

$$u(x,t) = \underbrace{\frac{1}{2}[\varphi(x+at) + \varphi(x-at)] + \frac{1}{2a}\int_{x-at}^{x+at}\psi(s)\,\dd s}_{\text{d'Alembert（齐次部分）}} + \underbrace{\frac{1}{2a}\int_0^t\int_{x-a(t-\tau)}^{x+a(t-\tau)} f(s,\tau)\,\dd s\,\dd\tau}_{\text{Duhamel 积分（非齐次部分）}}.$$

这就是一维波动方程 Cauchy 问题的完全解公式。从此非齐次波动方程在无界区域上的求解有了标准流程。

与 Duhamel 原理的一般形式的关系

这里的推导是 Duhamel 原理在波动方程中的特例。更一般地，若 $L$ 是线性算子，$u(x,t)$ 满足 $Lu = f$ 和齐次定解条件，则 $u = \int_0^t v(\cdot,t;\tau)\,\dd\tau$，其中 $v$ 满足 $Lv = 0$ 且 $v|_{t=\tau}=0$，$\pd_t v|_{t=\tau} = f(\cdot,\tau)$（对波动方程）或 $v|_{t=\tau} = f(\cdot,\tau)$（对热传导方程）。系统性的讨论参见第四章 §4.5（Duhamel 原理）。

例题 1.4.3（非齐次波动方程）

求解 $u_{tt} = u_{xx} + xt$（$x \in \RR, t>0$），$u(x,0) = 0$，$u_t(x,0) = 0$。

▶ 详细解答

$a=1$，$f(x,t)=xt$。零初始条件，直接用 Duhamel 积分：

$$u(x,t) = \frac{1}{2}\int_0^t\int_{x-(t-\tau)}^{x+(t-\tau)} s\tau\,\dd s\,\dd\tau.$$

先对 $s$ 积分（$\tau$ 固定）：

$$\int_{x-(t-\tau)}^{x+(t-\tau)} s\tau\,\dd s = \tau\left[\frac{s^2}{2}\right]_{x-(t-\tau)}^{x+(t-\tau)} = \frac{\tau}{2}\Big[(x+(t-\tau))^2 - (x-(t-\tau))^2\Big].$$

利用 $(A+B)^2-(A-B)^2 = 4AB$（取 $A=x$, $B=t-\tau$）：

$$= \frac{\tau}{2} \cdot 4x(t-\tau) = 2x\tau(t-\tau).$$

再对 $\tau$ 积分：

$$u(x,t) = \frac{1}{2}\int_0^t 2x\tau(t-\tau)\,\dd\tau = x\int_0^t (t\tau - \tau^2)\,\dd\tau = x\left[t\cdot\frac{t^2}{2} - \frac{t^3}{3}\right] = x\left(\frac{t^3}{2} - \frac{t^3}{3}\right) = \frac{xt^3}{6}.$$

验证：$u(x,0)=0$，$u_t(x,0)=0$。$u_{tt} = xt$，$u_{xx} = 0$，$u_{tt} - u_{xx} = xt = f(x,t)$。✓

例题 1.4.4（非齐次 + 非零初始条件，叠加求解）

求解 $u_{tt} = u_{xx} + \cos x$（$x \in \RR, t>0$），$u(x,0) = \sin x$，$u_t(x,0) = 0$。

▶ 详细解答

方法：叠加原理（齐次部分 + 非齐次特解）。

齐次部分 $u_h$：$u_{tt}=u_{xx}$，$u(x,0)=\sin x$，$u_t(x,0)=0$。由 d'Alembert：

$$u_h(x,t) = \frac{1}{2}[\sin(x+t) + \sin(x-t)] = \sin x\cos t.$$

非齐次特解 $u_p$：$u_{tt}=u_{xx}+\cos x$，零初始条件。$f(x,t)=\cos x$ 与 $t$ 无关。

用 Duhamel 积分（$a=1$）：

$$\begin{aligned} u_p(x,t) &= \frac{1}{2}\int_0^t\int_{x-(t-\tau)}^{x+(t-\tau)} \cos s\,\dd s\,\dd\tau \\ &= \frac{1}{2}\int_0^t [\sin(x+(t-\tau)) - \sin(x-(t-\tau))]\,\dd\tau \\ &= \frac{1}{2}\int_0^t 2\cos x\sin(t-\tau)\,\dd\tau \quad (\sin(A+B)-\sin(A-B)=2\cos A\sin B) \\ &= \cos x\int_0^t \sin(t-\tau)\,\dd\tau = \cos x\Big[\cos(t-\tau)\Big]_{\tau=0}^{\tau=t} \\ &= \cos x\,(\cos 0 - \cos t) = \cos x\,(1 - \cos t). \end{aligned}$$

完整解：$u(x,t) = u_h + u_p = \sin x\cos t + \cos x\,(1-\cos t)$。

验证：$u(x,0)=\sin x$。$u_t(x,0)=0$。$u_{tt}-u_{xx} = (-\sin x\cos t - \cos x\cos t) - (-\sin x\cos t + \cos x(1-\cos t))$。重新整理：$u_{tt} = -\sin x\cos t + \cos x\cos t$，$u_{xx} = -\sin x\cos t - \cos x(1-\cos t)$，差 $= \cos x(\cos t + 1 - \cos t) = \cos x = f$。✓

1.5 有界弦与分离变量法

当弦的长度有限且两端固定时，波在边界处不断反射，d'Alembert 公式不再直接适用。这时我们需要分离变量法——这是求解有界区域上 PDE 最基础也最重要的方法，也是 Fourier 分析在 PDE 中最经典的应用。

核心思想：把复杂运动分解为"纯音"的叠加

一根两端固定的弦，它的振动不是任意的——只有特定频率、特定形状的振动才能稳定存在。这些稳定的振动模式称为本征模态（固有模态）。分离变量法的核心思路是：

  1. 假设解可以写成 $u(x,t) = X(x) \cdot T(t)$（空间形状 × 时间演化）；
  2. 代入方程，把 PDE 拆成两个关于 $X$ 和 $T$ 的常微分方程（ODE）；
  3. 先解 $X$ 的 ODE（含边界条件），得到所有可能的"空间形状"（本征函数）；
  4. 再对每个空间形状，解 $T$ 的 ODE 得到时间演化；
  5. 最后用叠加原理把所有模态组合起来，用初始条件确定组合系数。

这正是 Fourier 分析的物理起源——弦的音色由各谐波的相对强度（Fourier 系数）决定。（关于 Fourier 级数的完整讲解，见附录 A 节 · 傅里叶级数。）

问题陈述：两端固定弦的自由振动

定解问题
$$\begin{cases} u_{tt} = a^2 u_{xx}, & 0 < x < l,\ t > 0, \\ u(0,t) = u(l,t) = 0, & t \ge 0, \\ u(x,0) = \varphi(x),\; u_t(x,0) = \psi(x), & 0 \le x \le l. \end{cases}$$

其中 $a = \sqrt{T/\rho}$ 是波速（$T$ 为张力，$\rho$ 为线密度）。边界条件 $u(0,t)=u(l,t)=0$ 表示两端固定。

第一步：分离变量——将 PDE 拆为两个 ODE

设解的形式为乘积：

$$u(x,t) = X(x)\,T(t),$$

其中 $X$ 只依赖于 $x$，$T$ 只依赖于 $t$。代入波动方程 $u_{tt} = a^2 u_{xx}$：

$$X(x)T''(t) = a^2 X''(x)T(t).$$

两边同除以 $a^2 X(x)T(t)$（假设 $X \neq 0, T \neq 0$）：

$$\frac{T''(t)}{a^2 T(t)} = \frac{X''(x)}{X(x)}.$$

关键观察：等式左边只依赖于 $t$，右边只依赖于 $x$。要使它们对所有 $(x,t)$ 都相等，唯一的可能是两边都等于同一个常数。记这个常数为 $-\lambda$（取负号是为了后面得到振荡解，$\lambda$ 本身预期为正）：

$$\frac{T''(t)}{a^2 T(t)} = \frac{X''(x)}{X(x)} = -\lambda.$$

于是我们得到了两个独立的常微分方程：

分离后的 ODE 系统
$$\begin{aligned} X''(x) + \lambda X(x) &= 0, \quad 0 < x < l, \\ T''(t) + a^2\lambda\, T(t) &= 0, \quad t > 0. \end{aligned}$$

边界条件 $u(0,t)=X(0)T(t)=0$ 和 $u(l,t)=X(l)T(t)=0$ 对所有 $t$ 成立，而 $T(t) \not\equiv 0$（否则解恒为零），故必须有：

$$X(0) = 0,\quad X(l) = 0.$$

至此，我们得到了 $X(x)$ 的一个齐次边界值问题——也称为本征值问题。

第二步：求解 $X(x)$ 的本征值问题——完整分类讨论

现在求解常微分方程边值问题：

$$X''(x) + \lambda X(x) = 0,\quad X(0)=0,\; X(l)=0.$$

这是一个二阶线性常系数齐次 ODE。其解法是写特征方程 $r^2 + \lambda = 0$，根据 $\lambda$ 的符号分为三种情况。唯一有非零解的情况才对应物理上可以存在的振动模态。

情况一：$\lambda < 0$（设 $\lambda = -\mu^2$，$\mu > 0$）

特征方程：$r^2 - \mu^2 = 0$，根 $r = \pm\mu$（两个不相等的实根）。

通解：$X(x) = C_1 e^{\mu x} + C_2 e^{-\mu x}$，或等价地 $X(x) = A\cosh\mu x + B\sinh\mu x$。

施加边界条件：

$$\begin{aligned} X(0) &= C_1 + C_2 = 0 \;\Longrightarrow\; C_2 = -C_1, \\ X(l) &= C_1(e^{\mu l} - e^{-\mu l}) = 2C_1\sinh\mu l = 0. \end{aligned}$$

由于 $\mu > 0$，$\sinh\mu l > 0$，故 $C_1 = 0$，进而 $C_2 = 0$。只有平凡解 $X(x) \equiv 0$。

情况二：$\lambda = 0$

此时方程为 $X''(x) = 0$。积分两次得通解：$X(x) = Ax + B$。

施加边界条件：

$$\begin{aligned} X(0) &= B = 0, \\ X(l) &= A l = 0 \;\Longrightarrow\; A = 0. \end{aligned}$$

只有平凡解 $X(x) \equiv 0$。

情况三：$\lambda > 0$（设 $\lambda = \omega^2$，$\omega > 0$）  ⭐ 唯一有非平凡解的情况

特征方程：$r^2 + \omega^2 = 0$，根 $r = \pm i\omega$（一对共轭复根）。

由 ODE 理论（见附录 A.3 节中关于正弦/余弦函数的讨论），复根 $r = \pm i\omega$ 对应的通解为：

$$X(x) = C_1\cos\omega x + C_2\sin\omega x.$$

（推导依据：Euler 公式 $e^{\pm i\omega x} = \cos\omega x \pm i\sin\omega x$，取实线性组合即得上述形式。）

施加第一个边界条件 $X(0) = 0$：

$$X(0) = C_1\cos 0 + C_2\sin 0 = C_1 = 0.$$

故 $X(x) = C_2\sin\omega x$。施加第二个边界条件 $X(l) = 0$：

$$X(l) = C_2\sin\omega l = 0.$$

如果 $C_2 = 0$，则又回到平凡解。为了得到非平凡解，必须有：

$$\sin\omega l = 0 \;\Longrightarrow\; \omega l = n\pi,\quad n = 1, 2, 3, \ldots$$

（$n=0$ 给出 $\omega=0$ 即 $\lambda=0$，对应平凡解；$n$ 取负整数给出相同的函数只差一个符号，归入正的 $n$。）

于是我们得到可数无穷多个满足边界条件的非平凡解：

$$\omega_n = \frac{n\pi}{l},\quad n = 1, 2, 3, \ldots$$
本征值与本征函数
$$\lambda_n = \left(\frac{n\pi}{l}\right)^2,\qquad X_n(x) = \sin\frac{n\pi x}{l},\quad n = 1, 2, 3, \ldots$$

$\lambda_n$ 称为本征值（eigenvalue），$X_n(x)$ 称为本征函数（eigenfunction）。数学上这属于 Sturm-Liouville 理论的特例（详见第四章 §4.7）。

物理意义：$\omega_n = n\pi a/l$ 是弦的固有角频率。$n=1$ 对应基频（fundamental frequency），$n \ge 2$ 对应泛音（overtone / harmonic）。基频决定音高，泛音的相对强度决定音色。

第三步：求解 $T(t)$ 的方程——每个模态的时间演化

对每个本征值 $\lambda_n = (n\pi/l)^2$，$T_n(t)$ 满足二阶线性常系数 ODE：

$$T_n''(t) + \frac{n^2\pi^2 a^2}{l^2}\,T_n(t) = 0.$$

记 $\omega_n = n\pi a/l$（第 $n$ 个固有角频率），方程写作：

$$T_n''(t) + \omega_n^2\,T_n(t) = 0.$$
解法：特征方程法

这是 简谐振动方程（simple harmonic oscillator）——常数系数二阶线性 ODE。

Step 1：写出特征方程 $r^2 + \omega_n^2 = 0$。

Step 2：求特征根 $r = \pm i\omega_n$（一对纯虚根）。

Step 3：由 ODE 标准结论——当特征根为 $\alpha \pm i\beta$ 时，通解为 $e^{\alpha t}(C_1\cos\beta t + C_2\sin\beta t)$。此处 $\alpha=0, \beta=\omega_n$，故：

$$T_n(t) = A_n\cos\omega_n t + B_n\sin\omega_n t,$$

其中 $A_n, B_n$ 为任意常数。等价地，也可写成 $T_n(t) = R_n\cos(\omega_n t - \delta_n)$（振幅-相位形式），其中 $R_n = \sqrt{A_n^2 + B_n^2}$，$\tan\delta_n = B_n/A_n$。

这里用到的 ODE 知识：二阶线性常系数齐次方程 $y'' + py' + qy = 0$ 的解法——写特征方程 $r^2 + pr + q = 0$，根据根的判别式 $\Delta = p^2-4q$ 区分三种情况（两实根 / 重根 / 复根）写通解。详见任何常微分方程教材中"线性常系数方程"章节。

为什么 $T_n$ 不衰减？

注意到 $T_n(t)$ 是纯振荡的（振幅不变）——这说明理想弦模型中没有阻尼。如果考虑空气阻力或内摩擦，方程中会出现 $u_t$ 项（一阶导数），OED 特征根会有负实部，导致振幅随时间指数衰减（见第四章 §4.8 阻尼波动方程）。

第四步：叠加原理——组合所有模态

由线性齐次方程的叠加原理，任意有限个解的线性组合仍是解。推广到无穷级数（在适当的收敛条件下），我们把所有本征模态叠加起来：

$$u(x,t) = \sum_{n=1}^{\infty} X_n(x)T_n(t) = \sum_{n=1}^{\infty} \left(A_n \cos\frac{n\pi a t}{l} + B_n \sin\frac{n\pi a t}{l}\right) \sin\frac{n\pi x}{l}.$$

其中系数 $A_n, B_n$ 尚未确定——它们将由初始条件 $\varphi(x), \psi(x)$ 来唯一确定。

从数学上看，这恰好是把解 $u(x,t)$ 在每个固定时刻 $t$ 展开成了关于 $\{\sin(n\pi x/l)\}_{n=1}^\infty$ 的Fourier 正弦级数。关于三角级数展开的系统理论，见附录 A.2–A.4 节。

第五步：由初始条件确定 Fourier 系数

现在利用初始条件来确定所有的 $A_n$ 和 $B_n$。

由 $u(x,0) = \varphi(x)$：令 $t=0$，$\cos 0 = 1$，$\sin 0 = 0$：

$$\varphi(x) = u(x,0) = \sum_{n=1}^{\infty} A_n \sin\frac{n\pi x}{l}. \tag{★}$$

由 $u_t(x,0) = \psi(x)$：先对 $u$ 关于 $t$ 求偏导：

$$u_t(x,t) = \sum_{n=1}^{\infty} \frac{n\pi a}{l}\left(-A_n \sin\frac{n\pi a t}{l} + B_n \cos\frac{n\pi a t}{l}\right) \sin\frac{n\pi x}{l}.$$

令 $t=0$：

$$\psi(x) = u_t(x,0) = \sum_{n=1}^{\infty} \frac{n\pi a}{l}B_n \sin\frac{n\pi x}{l}. \tag{★★}$$

至此，问题转化为：如何把已知函数 $\varphi(x)$ 和 $\psi(x)$ 展开为正弦级数？如果 $\varphi$ 和 $\psi$ 恰好已经是若干 $\sin(n\pi x/l)$ 的线性组合，则系数可以直接读出（如例题 1.5.1）。一般情况下，需要用到 Fourier 系数的积分公式，其推导依赖于正弦函数的正交性。

正弦函数的正交性——Fourier 系数公式的核心

在区间 $[0, l]$ 上，考察两个不同频率的正弦函数的乘积积分：

$$\begin{aligned} \int_0^l \sin\frac{m\pi x}{l}\sin\frac{n\pi x}{l}\,\dd x &= \frac{1}{2}\int_0^l \left[\cos\frac{(m-n)\pi x}{l} - \cos\frac{(m+n)\pi x}{l}\right]\dd x \quad\text{（积化和差）}\\ &= \frac{1}{2}\left[\frac{l}{(m-n)\pi}\sin\frac{(m-n)\pi x}{l} - \frac{l}{(m+n)\pi}\sin\frac{(m+n)\pi x}{l}\right]_0^l. \end{aligned}$$

当 $m \neq n$ 时：两个正弦项在 $x=0$ 和 $x=l$ 处均为零（因为 $\sin(k\pi)=0$），故积分 $=0$。

当 $m = n$ 时：$\cos(0) = 1$，$\cos\frac{2n\pi x}{l}$ 在一个周期上的积分为零：

$$\int_0^l \sin^2\frac{n\pi x}{l}\,\dd x = \frac{1}{2}\int_0^l \left(1 - \cos\frac{2n\pi x}{l}\right)\dd x = \frac{1}{2}\left[x - \frac{l}{2n\pi}\sin\frac{2n\pi x}{l}\right]_0^l = \frac{l}{2}.$$

综上，正交关系为：

$$\int_0^l \sin\frac{m\pi x}{l}\sin\frac{n\pi x}{l}\,\dd x = \begin{cases} 0, & m \neq n, \\[4pt] \dfrac{l}{2}, & m = n. \end{cases}$$

这可以紧凑地写成 $\displaystyle\int_0^l \sin\frac{m\pi x}{l}\sin\frac{n\pi x}{l}\,\dd x = \frac{l}{2}\,\delta_{mn}$，其中 $\delta_{mn}$ 是 Kronecker delta（$m=n$ 时为 1，否则为 0）。关于正交性的更多讨论，见附录 A.3 节（三角函数的正交性）。

利用正交性，从 (★) 式求 $A_n$：将 (★) 两边同乘以 $\sin\frac{m\pi x}{l}$，在 $[0,l]$ 上积分：

$$\begin{aligned} \int_0^l \varphi(x)\sin\frac{m\pi x}{l}\,\dd x &= \int_0^l \left[\sum_{n=1}^{\infty} A_n \sin\frac{n\pi x}{l}\right]\sin\frac{m\pi x}{l}\,\dd x \\ &= \sum_{n=1}^{\infty} A_n \int_0^l \sin\frac{n\pi x}{l}\sin\frac{m\pi x}{l}\,\dd x \\ &= A_m \cdot \frac{l}{2} \quad\text{（所有 } n \neq m \text{ 的项积分为零）}. \end{aligned}$$

（这里假设了级数可以逐项积分——对于连续或分段连续的 $\varphi$，这是成立的。）

同理，从 (★★) 式求 $B_n$：将 (★★) 两边同乘以 $\sin\frac{m\pi x}{l}$ 并积分：

$$\int_0^l \psi(x)\sin\frac{m\pi x}{l}\,\dd x = \frac{m\pi a}{l}B_m \cdot \frac{l}{2} = \frac{m\pi a}{2}B_m.$$
Fourier 系数公式（分离变量法的最终答案）
$$A_n = \frac{2}{l}\int_0^l \varphi(x)\sin\frac{n\pi x}{l}\,\dd x, \quad n = 1, 2, 3, \ldots$$ $$B_n = \frac{2}{n\pi a}\int_0^l \psi(x)\sin\frac{n\pi x}{l}\,\dd x, \quad n = 1, 2, 3, \ldots$$

这就是Fourier 正弦级数的系数公式。$A_n$ 由初始位移 $\varphi(x)$ 决定，$B_n$ 由初始速度 $\psi(x)$ 决定。完整解为：

$$u(x,t) = \sum_{n=1}^{\infty} \left(A_n \cos\frac{n\pi a t}{l} + B_n \sin\frac{n\pi a t}{l}\right) \sin\frac{n\pi x}{l}.$$
关于 Fourier 系数的计算与收敛性

计算 $A_n, B_n$ 时需要计算定积分 $\int_0^l \varphi(x)\sin\frac{n\pi x}{l}\dd x$。常用的技巧包括分部积分和三角恒等式。如果你对 Fourier 系数的计算还不熟练，强烈建议先看附录 A.5–A.6 节的完整计算例题。

关于级数的收敛性：如果 $\varphi \in C^2$ 且满足相容性条件 $\varphi(0)=\varphi(l)=0$，则 Fourier 级数一致收敛到 $\varphi$，解的级数表示也是古典解。更一般的收敛条件见附录 A.7 节。

分离变量法揭示：弦的振动是各种"纯音"（正弦本征模态）的叠加。钢琴、小提琴的音色差异正是由于不同谐波（泛音）的相对强度不同——即 Fourier 系数 $A_n, B_n$ 的分布不同。这就是 Fourier 分析的物理起源！如果你对 Fourier 级数的计算和正交性还不太熟悉，可以参考附录 A 节 · 傅里叶级数的系统讲解。

例题 1.5.1（初始条件已是本征函数的组合——系数直接读出）

求解 $u_{tt} = u_{xx}$（$0 < x < \pi$），$u(0,t)=u(\pi,t)=0$，$u(x,0)=\sin x + \frac{1}{2}\sin 3x$，$u_t(x,0)=0$。

▶ 详细解答

识别参数：$a=1$，$l=\pi$。本征函数 $X_n(x) = \sin nx$（因为 $\sin(n\pi x/\pi) = \sin nx$），本征值 $\lambda_n = n^2$，固有频率 $\omega_n = n$。

初始位移：$\varphi(x) = \sin x + \frac{1}{2}\sin 3x$。与 Fourier 正弦级数 $\sum A_n\sin nx$ 对比，直接读出：

$$A_1 = 1,\quad A_3 = \frac{1}{2},\quad \text{其余 } A_n = 0.$$

（不需要计算积分！因为 $\varphi$ 已经展开成了 $\sin nx$ 的线性组合。这正是 Fourier 级数的"坐标表示"——$\varphi$ 在本征函数基底下的坐标就是 $(0,1,0,\frac{1}{2},0,0,\ldots)$。）

初始速度：$\psi(x) = 0$，故所有 $B_n = 0$。

叠加得解：

$$u(x,t) = \sin x \cos t + \frac{1}{2}\sin 3x \cos 3t.$$

分析：解由基频模态（$n=1$，角频率 $\omega_1=1$）和第三泛音（$n=3$，角频率 $\omega_3=3$）叠加而成。注意 $n=3$ 的振动频率是 $n=1$ 的三倍——这正是音乐中"纯五度+八度"的音程关系。$n=2$ 的模态在本例中未被激发。

例题 1.5.2（需要计算 Fourier 积分求系数）

求解 $u_{tt} = u_{xx}$（$0 < x < \pi$），$u(0,t)=u(\pi,t)=0$，$u(x,0)=0$，$u_t(x,0)=x$。

▶ 详细解答

$a=1$，$l=\pi$。$\varphi(x)=0$ 故所有 $A_n=0$。需要计算 $B_n$：

$$B_n = \frac{2}{n\pi}\int_0^\pi x\sin nx\,\dd x.$$

计算积分：用分部积分法，令 $u=x, \dd v=\sin nx\,\dd x$，则 $\dd u=\dd x, v=-\frac{1}{n}\cos nx$：

$$\begin{aligned} \int_0^\pi x\sin nx\,\dd x &= \left[-\frac{x}{n}\cos nx\right]_0^\pi + \frac{1}{n}\int_0^\pi \cos nx\,\dd x \\ &= -\frac{\pi}{n}\cos n\pi - 0 + \frac{1}{n}\left[\frac{\sin nx}{n}\right]_0^\pi \\ &= -\frac{\pi}{n}(-1)^n + 0 = \frac{\pi(-1)^{n+1}}{n}. \end{aligned}$$

因此：

$$B_n = \frac{2}{n\pi} \cdot \frac{\pi(-1)^{n+1}}{n} = \frac{2(-1)^{n+1}}{n^2}.$$

叠加得解：

$$u(x,t) = \sum_{n=1}^{\infty} \frac{2(-1)^{n+1}}{n^2} \sin(nt) \sin(nx).$$

由于系数按 $1/n^2$ 衰减，级数收敛很快。$n=1$ 模态 $2\sin t\sin x$ 占主导地位。

补充：非齐次方程与非齐次边界条件

前面的分离变量法只处理了齐次方程、齐次边界条件的情况。现在考虑更一般的问题：

两类非齐次性
  1. 非齐次方程（有源项）：$u_{tt} = a^2 u_{xx} + f(x,t)$
  2. 非齐次边界条件：如 $u(0,t) = \mu(t),\; u(l,t) = \nu(t)$

通常两种非齐次性同时存在。处理策略是"先转化边界，再展开方程"。

第一步：非齐次边界条件 → 齐次化

设边界条件为 $u(0,t) = \mu(t)$，$u(l,t) = \nu(t)$。寻找一个辅助函数 $w(x,t)$（称为边界提升函数），使：

$$w(0,t) = \mu(t), \quad w(l,t) = \nu(t).$$

最简选择——线性插值：

$$w(x,t) = \mu(t) + \frac{x}{l}\bigl[\nu(t) - \mu(t)\bigr] = \left(1 - \frac{x}{l}\right)\mu(t) + \frac{x}{l}\nu(t).$$

令 $v(x,t) = u(x,t) - w(x,t)$，则 $v$ 满足齐次边界条件 $v(0,t) = v(l,t) = 0$。但 PDE 会引入额外的非齐次项：

$$v_{tt} = a^2 v_{xx} + \underbrace{\bigl[f(x,t) - w_{tt} + a^2 w_{xx}\bigr]}_{\tilde{f}(x,t)}.$$
注意

边界提升函数 $w(x,t)$ 的选择不唯一——只要边界条件满足即可。选择简单的形式（如线性插值）能减少后续计算量。对于 $w_{xx}=0$ 的线性 $w$，修正源项简化为 $\tilde{f} = f - w_{tt}$。

第二步：特征函数展开法（处理非齐次方程）

现在问题化为齐次边界条件 + 非齐次方程：

$$\begin{cases} v_{tt} = a^2 v_{xx} + \tilde{f}(x,t), & 0 < x < l,\; t > 0,\\[2pt] v(0,t) = v(l,t) = 0, \\[2pt] v(x,0) = \tilde{\varphi}(x), \quad v_t(x,0) = \tilde{\psi}(x). \end{cases}$$
核心思想：特征函数展开

将 $v(x,t)$ 和 $\tilde{f}(x,t)$ 都按齐次边界条件对应的本征函数系展开。对于 Dirichlet 边界条件 $v(0,t)=v(l,t)=0$，本征函数系为 $\{\sin\frac{n\pi x}{l}\}_{n=1}^\infty$：

$$v(x,t) = \sum_{n=1}^{\infty} T_n(t) \sin\frac{n\pi x}{l}, \qquad \tilde{f}(x,t) = \sum_{n=1}^{\infty} f_n(t) \sin\frac{n\pi x}{l}.$$

其中 $f_n(t)$ 由 Fourier 正弦系数公式给出（将 $t$ 视为参数）：

$$f_n(t) = \frac{2}{l}\int_0^l \tilde{f}(x,t)\,\sin\frac{n\pi x}{l}\,\dd x.$$

代入 PDE：将级数代入 $v_{tt} = a^2 v_{xx} + \tilde{f}$，利用 $\frac{\dd^2}{\dd x^2}\sin\frac{n\pi x}{l} = -(\frac{n\pi}{l})^2\sin\frac{n\pi x}{l}$：

$$\sum_{n=1}^{\infty} T_n''(t) \sin\frac{n\pi x}{l} = \sum_{n=1}^{\infty} \left[-a^2\left(\frac{n\pi}{l}\right)^2 T_n(t) + f_n(t)\right] \sin\frac{n\pi x}{l}.$$

比较 $\sin\frac{n\pi x}{l}$ 的系数（由本征函数的正交性，等式两边各 Fourier 系数必须相等），得到一族解耦的常微分方程：

特征函数展开法 · ODE 系统
$$\boxed{T_n''(t) + \omega_n^2 T_n(t) = f_n(t), \qquad \omega_n = \frac{n\pi a}{l}}$$

初始条件由 $v(x,0)$ 和 $v_t(x,0)$ 的 Fourier 展开给出：

$$T_n(0) = \tilde{A}_n = \frac{2}{l}\int_0^l \tilde{\varphi}(x)\,\sin\frac{n\pi x}{l}\,\dd x,$$ $$T_n'(0) = \tilde{B}_n = \frac{2}{l}\int_0^l \tilde{\psi}(x)\,\sin\frac{n\pi x}{l}\,\dd x.$$

这是关于 $T_n(t)$ 的常系数二阶非齐次线性 ODE。由常微分方程理论，其通解为齐次通解 + 非齐次特解。齐次通解为 $T_n^{(h)}(t) = C_1\cos\omega_n t + C_2\sin\omega_n t$。非齐次特解可由常数变易法（或 Duhamel 积分）得到：

$$T_n^{(p)}(t) = \frac{1}{\omega_n}\int_0^t f_n(\tau)\,\sin\bigl(\omega_n(t-\tau)\bigr)\,\dd\tau.$$

匹配初始条件后，得到完整的系数函数：

特征函数展开法 · 完整解
$$T_n(t) = \tilde{A}_n\cos(\omega_n t) + \frac{\tilde{B}_n}{\omega_n}\sin(\omega_n t) + \frac{1}{\omega_n}\int_0^t f_n(\tau)\,\sin\bigl(\omega_n(t-\tau)\bigr)\,\dd\tau.$$

最终解为 $u(x,t) = w(x,t) + \sum_{n=1}^{\infty} T_n(t)\sin\frac{n\pi x}{l}$.

物理解释

前两项是自由振动（初始位移和初始速度激发的本征模态振荡），第三项 Duhamel 积分是受迫振动——外力 $f(x,t)$ 在时刻 $\tau$ 产生的瞬时冲量，经过时间 $(t-\tau)$ 后对第 $n$ 号模态的累积贡献。整个解 = 边界提升 + 自由振动 + 受迫振动，体现了叠加原理的层层递进。

例题 1.5.3（非齐次方程 + 齐次边界条件）

求解 $u_{tt} = u_{xx} + \sin x$（$0 < x < \pi$），$u(0,t)=u(\pi,t)=0$，$u(x,0)=0$，$u_t(x,0)=0$。

▶ 详细解答

分析：边界条件已是齐次的，无需第 1 步。$a=1$，$l=\pi$，本征函数系 $\{\sin nx\}_{n=1}^\infty$，$\omega_n = n$。

展开源项：$f(x,t) = \sin x$ 恰好已经是本征函数！比较 $\sin x = 1\cdot\sin(1x) + 0\cdot\sin(2x) + \cdots$，直接读出：

$$f_1(t) = 1, \qquad f_n(t) = 0 \;\; (n \ge 2).$$

写 ODE 系统：零初始条件意味着 $T_n(0)=T_n'(0)=0$。

  • $n = 1$：$T_1'' + T_1 = 1$，$T_1(0)=T_1'(0)=0$。
  • $n \ge 2$：$T_n'' + n^2 T_n = 0$，$T_n(0)=T_n'(0)=0$ ⇒ $T_n(t) \equiv 0$。

解 $T_1$ 的 ODE：齐次通解 $T_1^{(h)} = C_1\cos t + C_2\sin t$，特解设为常数 $T_1^{(p)} = A$，代入得 $A=1$。故 $T_1(t) = C_1\cos t + C_2\sin t + 1$。由 $T_1(0)=0$ 得 $C_1+1=0$ ⇒ $C_1=-1$。由 $T_1'(0)=0$ 得 $C_2=0$。因此：

$$T_1(t) = 1 - \cos t.$$

最终解：

$$u(x,t) = (1 - \cos t)\sin x.$$

物理解释：外力 $\sin x$ 恰好匹配基频本征函数的空间形状，因此只激发 $n=1$ 模态。解在平衡位置 $u=\sin x$ 附近以频率 $\omega_1=1$ 做简谐振荡——这好比以 $\sin x$ 形状的力持续推动弦的基频模态，使其在平衡点附近来回摆动。

例题 1.5.4（非齐次边界条件）

求解 $u_{tt} = u_{xx}$（$0 < x < 1$），$u(0,t)=0$，$u(1,t)=\sin t$，$u(x,0)=0$，$u_t(x,0)=0$。

▶ 详细解答

第 1 步 · 边界齐次化：$l=1$，$\mu(t)=0$，$\nu(t)=\sin t$。构造线性提升函数：

$$w(x,t) = (1-x)\cdot 0 + x\cdot\sin t = x\sin t.$$

令 $v = u - w$，则 $v(0,t)=v(1,t)=0$（齐次）。计算 $w_{tt} = -x\sin t$，$w_{xx}=0$。新方程：

$$v_{tt} = v_{xx} - w_{tt} = v_{xx} + x\sin t.$$

新初始条件：$v(x,0) = u(x,0) - w(x,0) = 0 - x\cdot 0 = 0$，$v_t(x,0) = u_t(x,0) - w_t(x,0) = 0 - x\cos 0 = -x$.

第 2 步：本征函数系 $\{\sin n\pi x\}_{n=1}^\infty$（因为 $l=1$），$\omega_n = n\pi$。

第 3 步 · 展开源项：$\tilde{f}(x,t) = x\sin t$。

$$f_n(t) = 2\int_0^1 x\sin t \cdot \sin(n\pi x)\,\dd x = 2\sin t\int_0^1 x\sin(n\pi x)\,\dd x.$$

利用例题 1.5.2 的积分结果 $\int_0^1 x\sin(n\pi x)\dd x = \frac{(-1)^{n+1}}{n\pi}$：

$$f_n(t) = \frac{2(-1)^{n+1}}{n\pi}\sin t.$$

展开初始条件：$\tilde{\varphi}(x)=0$ ⇒ $\tilde{A}_n=0$。$\tilde{\psi}(x) = -x$ ⇒

$$\tilde{B}_n = 2\int_0^1 (-x)\sin(n\pi x)\,\dd x = -\frac{2(-1)^{n+1}}{n\pi}.$$

第 4 步 · 解 ODE：$T_n'' + (n\pi)^2 T_n = \frac{2(-1)^{n+1}}{n\pi}\sin t$，$T_n(0)=0$，$T_n'(0)=\tilde{B}_n$。

齐次通解：$T_n^{(h)} = C_1\cos(n\pi t) + C_2\sin(n\pi t)$。非齐次特解：右端 $\sin t$ 频率为 1，当 $n\pi \neq 1$（即 $n \neq 1/\pi$，对所有正整数 $n$ 均成立，因 $\pi \notin \mathbb{Q}$ 的倒数不是整数），无共振。设特解 $T_n^{(p)} = K\sin t$，代入得：

$$(-K + (n\pi)^2 K)\sin t = \frac{2(-1)^{n+1}}{n\pi}\sin t \;\Longrightarrow\; K = \frac{2(-1)^{n+1}}{n\pi[(n\pi)^2-1]}.$$

匹配初始条件：$T_n(0)=C_1=0$。$T_n'(t) = C_2 n\pi\cos(n\pi t) + K\cos t$，$T_n'(0) = C_2 n\pi + K = \tilde{B}_n = -\frac{2(-1)^{n+1}}{n\pi}$。因此：

$$C_2 = \frac{1}{n\pi}\left(-\frac{2(-1)^{n+1}}{n\pi} - K\right) = -\frac{2(-1)^{n+1}}{n\pi}\cdot\frac{1}{n\pi}\left[1 + \frac{1}{(n\pi)^2-1}\right].$$

第 5 步：$u(x,t) = x\sin t + \sum_{n=1}^\infty T_n(t)\sin(n\pi x)$。

此例展示了非齐次边界条件处理的完整流程。实际计算中，级数通常只取前几项近似。

1.6 高维波动方程

$n$ 维波动方程：$u_{tt} = a^2 \lap u = a^2 \sum_{i=1}^n \frac{\pd^2 u}{\pd x_i^2}.$

三维——Kirchhoff-Poisson 公式

三维初值问题的解

对 $u_{tt} = a^2 \lap u$（$x \in \RR^3$），$u|_{t=0}=\varphi$，$u_t|_{t=0}=\psi$：

$$u(x,t) = \frac{\pd}{\pd t}\big[t \cdot M_{at}[\varphi](x)\big] + t \cdot M_{at}[\psi](x),$$

其中 $M_{r}[\varphi](x) = \frac{1}{4\pi r^2}\int_{|\xi|=r}\varphi(x+\xi)\,\dd S_\xi$ 是 $\varphi$ 在以 $x$ 为球心、$r$ 为半径的球面上的平均值。

推导思路：球面平均法

定义球面平均 $M(r,t) = \frac{1}{4\pi r^2}\int_{|\xi|=r}u(x+\xi,t)\,\dd S$。可证 $rM$ 满足一维波动方程 $(rM)_{tt} = a^2(rM)_{rr}$。用 d'Alembert 公式求解 $rM$，再令 $r \to 0$ 即得 Poisson 公式。这就是"降维"的力量——高维问题化为一维！

二维波动方程与Poisson公式

二维初值问题的Poisson公式

对于 $u_{tt} = a^2(u_{xx}+u_{yy})$（$\RR^2$ 上的Cauchy问题），$u|_{t=0}=\varphi$，$u_t|_{t=0}=\psi$，其解为：

$$u(x,y,t) = \frac{1}{2\pi a}\frac{\pd}{\pd t}\iint_{B_{at}(x,y)}\frac{\varphi(\xi,\eta)}{\sqrt{a^2t^2-(\xi-x)^2-(\eta-y)^2}}\,\dd\xi\dd\eta + \frac{1}{2\pi a}\iint_{B_{at}(x,y)}\frac{\psi(\xi,\eta)}{\sqrt{a^2t^2-(\xi-x)^2-(\eta-y)^2}}\,\dd\xi\dd\eta.$$

其中 $B_{at}(x,y)$ 是以 $(x,y)$ 为圆心、$at$ 为半径的圆盘。这称为Poisson公式（二维）。

二维波动方程的依赖区域

在二维波动方程中，依赖区域是以观测点 $(x_0, y_0, t_0)$ 为顶点的向后锥体在 $t=0$ 平面截出的圆盘：

$$D = \{(x,y) \mid (x-x_0)^2 + (y-y_0)^2 \le (a t_0)^2\}.$$

即位于 $(x_0, y_0)$ 处、时刻 $t_0$ 的解值仅依赖于初始时刻 $t=0$ 时以 $(x_0, y_0)$ 为圆心、$a t_0$ 为半径的圆盘内的数据。

这与三维情况的球面依赖（只有距离恰好为 $a t_0$ 的点）有本质区别！

Huygens 原理（三维 vs 二维）

Huygens 原理

三维空间：点 $(x_0, t_0)$ 处的解仅依赖于球面 $|\xi| = at_0$ 上的初始数据（不是球体内部！）。扰动过后不留余波——信号传播是"干净"的。

二维空间：解依赖于整个圆盘 $|\xi| \le at_0$ 上的初始数据。二维波传播存在"余波"（wake），Huygens 原理不成立。

物理含义：在三维世界中，你听到的声音是清晰的——声波经过后不会留下回响；如果世界是二维的，声音会一直"嗡嗡"响。

例题 1.6.1（三维Poisson公式）

三维波动方程，$a=1$，$\varphi=0$，$\psi(x)=1$（$|x|<1$），$\psi=0$（$|x|\ge 1$）。求 $u(0,t)$。

▶ 详细解答

$u(0,t) = t \cdot M_t[\psi](0) = \frac{1}{4\pi t}\int_{|\xi|=t}\psi(\xi)\,\dd S_\xi$。

$t < 1$：球面完全在 $|\xi|<1$ 内，$\psi=1$，积分 $= 4\pi t^2$，故 $u(0,t) = t$。

$t > 1$：球面完全在 $|\xi|\ge 1$ 外，$\psi=0$，故 $u(0,t) = 0$。

体现了 Huygens 原理：扰动到达原点后立刻离开，不留余波。

例题 1.6.2（考试重点：二维依赖区域）

二维波动方程 $u_{tt} = u_{xx} + u_{yy}$，已知 $a=1$。求点 $(x,y,t) = (1, 2, 6)$ 处的 $u$ 值依赖于初始平面上哪个区域的数据？写出该区域的解析表达式。

▶ 详细解答

$a=1, t_0=6$，所以依赖区域是以 $(1, 2)$ 为圆心、$a t_0 = 6$ 为半径的圆盘。

该区域为：

$$\{(x,y) \in \RR^2 \mid (x-1)^2 + (y-2)^2 \le 36\}.$$

注意这是圆盘（不等式是 $\le$），不是圆周。这是因为在二维波动方程中，初始数据对整个圆盘内的贡献都会影响到观测点的值（降维法得出）。因此正确答案是 $(x-1)^2+(y-2)^2 \le 36$。

如果换为三维波动方程 $u_{tt} = u_{xx}+u_{yy}+u_{zz}$，依赖区域将是球面 $(x-1)^2+(y-2)^2+(z-z_0)^2 = 36$。

例题 1.6.3（考试重点：三维Poisson公式求具体值）

求解三维波动方程Cauchy问题：

$$\begin{cases} u_{tt} - \lap u = 0, & (x,y,z) \in \RR^3,\ t > 0, \\ u(x,y,z,0) = 0, & \\ u_t(x,y,z,0) = x^2. \end{cases}$$

求 $u(0,0,0,t)$ 的表达式。

▶ 详细解答

$a=1$，$\varphi=0$，$\psi(x,y,z)=x^2$。由Poisson公式（三维Kirchhoff公式）：

$$u(0,t) = t \cdot M_t[\psi](0) = \frac{t}{4\pi t^2} \int_{|\xi|=t} \psi(\xi) \,\dd S_\xi = \frac{1}{4\pi t} \int_{|\xi|=t} \xi_1^2 \,\dd S_\xi.$$

需要计算 $\int_{|\xi|=t} \xi_1^2 \,\dd S_\xi$。由对称性，在球面 $|\xi|=t$ 上：

$$\int_{|\xi|=t} \xi_1^2 \,\dd S = \int_{|\xi|=t} \xi_2^2 \,\dd S = \int_{|\xi|=t} \xi_3^2 \,\dd S.$$

三者之和为：$\int_{|\xi|=t} (\xi_1^2 + \xi_2^2 + \xi_3^2) \,\dd S = \int_{|\xi|=t} t^2 \,\dd S = t^2 \cdot 4\pi t^2 = 4\pi t^4$。

因此每份各占三分之一：$\displaystyle\int_{|\xi|=t} \xi_1^2 \,\dd S = \frac{4\pi t^4}{3}$。

代入得：

$$u(0,t) = \frac{1}{4\pi t} \cdot \frac{4\pi t^4}{3} = \frac{t^3}{3}.$$

验证：$u_t(0,t) = t^2$，$t \to 0^+$ 时 $u_t(0,0) = 0$（与 $\psi(0)=0$ 一致）；$u(0,t) = t^3/3$ 确实是三维波动方程的解。

注意这里用到了球面积分中的对称性技巧——这是考试中经常出现的题型。

1.7 能量积分与唯一性

能量积分

对于波动方程 $u_{tt} = a^2 u_{xx}$，定义能量积分：

$$E(t) = \frac{1}{2}\int_{-\infty}^{+\infty}\big[u_t^2(x,t) + a^2 u_x^2(x,t)\big]\,\dd x.$$

第一项是动能（$\frac{1}{2}mv^2$ 的连续版本：$v = u_t$），第二项是势能（弦被拉伸储存的弹性势能，正比于 $u_x^2$）。

能量守恒定理

若 $u$ 是波动方程的解且在无穷远处足够快地趋于零，则 $E(t) = E(0) = \text{const}.$

证明

$$\frac{\dd E}{\dd t} = \int_{-\infty}^{+\infty}(u_t u_{tt} + a^2 u_x u_{xt})\,\dd x.$$

代入 $u_{tt} = a^2 u_{xx}$：

$$\frac{\dd E}{\dd t} = a^2\int_{-\infty}^{+\infty}(u_t u_{xx} + u_x u_{xt})\,\dd x = a^2\int_{-\infty}^{+\infty}\frac{\pd}{\pd x}(u_t u_x)\,\dd x = 0.$$

能量方法推出唯一性

唯一性定理

波动方程初值问题的经典解是唯一的。

证明

设 $u_1, u_2$ 是两个解，$w = u_1 - u_2$。则 $w$ 满足齐次方程 $w_{tt} = a^2 w_{xx}$ 和零初始条件。$w$ 的能量 $E(0) = 0$（初始位移和速度均为零），由能量守恒 $E(t) \equiv 0$，故 $w_t \equiv 0$，$w_x \equiv 0$，从而 $w \equiv 0$，即 $u_1 = u_2$。

本章总结：波动方程的知识结构

$$\begin{array}{c} \text{物理推导（弦振动）} \\ \downarrow \\ \text{d'Alembert公式（无限弦）} \quad \longleftrightarrow \quad \text{分离变量法（有界弦）} \\ \downarrow \qquad \qquad \qquad \qquad \qquad \downarrow \\ \text{特征线·依赖/影响区域} \qquad \qquad \text{本征值·本征函数·Fourier展开} \\ \downarrow \qquad \qquad \qquad \qquad \qquad \downarrow \\ \text{延拓法（半无界）} \qquad \qquad \qquad \text{叠加原理} \\ \downarrow \\ \text{高维波动（Poisson公式·Huygens原理）} \\ \downarrow \\ \text{能量积分 → 唯一性、稳定性} \end{array}$$

核心方法：特征线法（d'Alembert公式）、分离变量法、延拓法、能量方法。

物理直觉：波 = 两个行波的叠加，以有限速度传播，能量守恒。

习题 1.1 d'Alembert公式应用 基础 考试热点

用 d'Alembert 公式求解以下 Cauchy 问题：

$$\begin{cases} u_{tt} = 9 u_{xx}, & x \in \RR,\ t > 0, \\ u(x,0) = \sin x, & x \in \RR, \\ u_t(x,0) = 0, & x \in \RR. \end{cases}$$

▶ 查看答案

$a = 3$，$\varphi(x) = \sin x$，$\psi(x) = 0$。由 d'Alembert 公式：

$$u(x,t) = \frac{1}{2}[\sin(x+3t) + \sin(x-3t)] = \sin x \cos 3t.$$

验证：$u_{tt} = -9\sin x \cos 3t$，$u_{xx} = -\sin x \cos 3t$，$u_{tt} = 9u_{xx}$ 成立。$u(x,0)=\sin x$，$u_t(x,0)=0$。

习题 1.2 半无界问题——延拓法 基础

求解半无界弦上的波动方程：

$$\begin{cases} u_{tt} = 4 u_{xx}, & x > 0,\ t > 0, \\ u(x,0) = 0,\; u_t(x,0) = 1, & x \ge 0, \\ u(0,t) = 0, & t \ge 0. \end{cases}$$

（提示：$u(0,t)=0$ 为 Dirichlet 边界条件，应用奇延拓。）

▶ 查看答案

$a=2$。奇延拓 $\tilde\psi(x) = \operatorname{sgn}(x)$（$x>0$ 时为 $1$，$x<0$ 时为 $-1$）。

当 $x \ge 2t$（波前未到边界）：$u(x,t) = \frac{1}{4}\int_{x-2t}^{x+2t} 1\,\dd s = t$。

当 $0 \le x \lt 2t$（已反射）：$u(x,t) = \frac{1}{4}\big[\int_0^{x+2t} 1\,\dd s - \int_0^{x-2t} (-1)\,\dd s\big] = \frac{x}{2}$。

综合：$u(x,t) = \min(t, x/2)$。

习题 1.3 依赖区间与影响区域 应用

考虑波动方程 $u_{tt} = u_{xx}$（$a=1$）。

(1) 点 $(x,t) = (3, 2)$ 处的解值依赖于初始线上哪个区间的数据？

(2) 初始点 $(x,0) = (1, 0)$ 处的一个扰动，在 $t=4$ 时影响到 $x$ 轴上的哪些点？

(3) 在 $(x,t)$ 平面上画出示意图，标出依赖区间、决定区域和影响区域。

▶ 查看答案

(1) 依赖区间：$[x - at, x + at] = [3-2, 3+2] = [1, 5]$。点 $(3,2)$ 的解仅依赖于初始数据在区间 $[1, 5]$ 上的值。

(2) 影响区域：过 $(1, 0)$ 作两条特征线 $x - 1 = \pm t$。在 $t=4$ 时，影响区域覆盖 $x \in [1-4, 1+4] = [-3, 5]$。

(3) 示意图要点：依赖区间是 $x$ 轴上的线段，决定区域是过该线段两端点向上作的三角形，影响区域是过初始点向上作的锥形。

习题 1.4 分离变量法——有界弦振动 应用 考试热点

用分离变量法求解以下有界弦的振动问题：

$$\begin{cases} u_{tt} = u_{xx}, & 0 < x < 2,\ t > 0, \\ u(0,t) = u(2,t) = 0, & t \ge 0, \\ u(x,0) = x(2-x),\; u_t(x,0) = 0, & 0 \le x \le 2. \end{cases}$$

写出解的前三项（$n=1,2,3$）的显式表达式。

▶ 查看答案

$a=1$，$l=2$。本征函数 $X_n(x) = \sin(n\pi x/2)$，$\lambda_n = (n\pi/2)^2$。

$\psi(x)=0$，所以 $B_n = 0$。只需计算 $A_n$：

$$A_n = \frac{2}{2}\int_0^2 x(2-x)\sin\frac{n\pi x}{2}\,\dd x = \int_0^2 (2x - x^2)\sin\frac{n\pi x}{2}\,\dd x.$$

分部积分得：$A_n = \frac{16}{n^3\pi^3}[1 - (-1)^n]$。

$n$ 为偶数时 $A_n = 0$；$n$ 为奇数时 $A_n = \frac{32}{n^3\pi^3}$。

前三项（$n=1,3,5$）：

$$u(x,t) = \frac{32}{\pi^3}\left(\sin\frac{\pi x}{2}\cos\frac{\pi t}{2} + \frac{1}{27}\sin\frac{3\pi x}{2}\cos\frac{3\pi t}{2} + \frac{1}{125}\sin\frac{5\pi x}{2}\cos\frac{5\pi t}{2} + \cdots\right).$$

注意系数以 $1/n^3$ 衰减，级数收敛很快。

习题 1.5 能量积分与唯一性 证明 考试热点

对一维波动方程 $u_{tt} = a^2 u_{xx}$（$x \in \RR$）：

(1) 写出能量积分 $E(t)$ 的定义，并证明 $E(t)$ 守恒（即 $E'(t) = 0$）。

(2) 利用能量守恒证明：若两个解具有相同的初始条件，则它们在所有时刻相等（唯一性）。

▶ 查看答案

(1) 定义 $E(t) = \frac{1}{2}\int_{-\infty}^{+\infty}[u_t^2(x,t) + a^2 u_x^2(x,t)]\,\dd x$。

求导：$E'(t) = \int_{-\infty}^{+\infty}(u_t u_{tt} + a^2 u_x u_{xt})\,\dd x$。代入 $u_{tt} = a^2 u_{xx}$：

$E'(t) = a^2\int_{-\infty}^{+\infty}(u_t u_{xx} + u_x u_{xt})\,\dd x = a^2\int_{-\infty}^{+\infty}\frac{\pd}{\pd x}(u_t u_x)\,\dd x = a^2[u_t u_x]_{-\infty}^{+\infty} = 0$（假设解在无穷远处足够快地衰减）。

(2) 设 $u_1, u_2$ 为两个解，$w = u_1 - u_2$ 满足齐次方程和零初始条件。$E_w(0) = 0$（因为 $w(x,0)=0, w_t(x,0)=0$），由能量守恒 $E_w(t) \equiv 0$，故 $w_t \equiv 0, w_x \equiv 0$。又 $w(x,0)=0$，故 $w \equiv 0$，即 $u_1 = u_2$。
