A. 傅里叶级数——从正弦波叠加说起

A.1 核心直觉：任何周期函数 = 正弦波的"菜谱"

直觉建立：正弦波是"基本食材"

想象你在听一首歌。这首歌可以分解为：低音鼓的沉闷低频 + 人声的中频 + 镲片的高频嘶嘶声。每一个频率分量都是一种纯音（正弦波）。把所有这些纯音按照适当的强度（振幅）叠加在一起，就还原出了你听到的那首歌。

傅里叶的惊人洞察是：任何一个周期函数，都可以写成一系列正弦波和余弦波的叠加。每个正弦/余弦波的频率是基频的整数倍，振幅则由函数的"形状"决定。

把这个想法写成数学公式，就是傅里叶级数。

A.2 三角级数的定义

定义 A.1（三角级数）

设 $f(x)$ 是周期为 $2\pi$ 的函数。它的傅里叶级数为：

$$f(x) \sim \frac{a_0}{2} + \sum_{n=1}^{\infty} \big(a_n \cos nx + b_n \sin nx\big),$$

其中系数 $a_n, b_n$ 称为傅里叶系数，由以下积分给出：

$$a_n = \frac{1}{\pi}\int_{-\pi}^{\pi} f(x)\cos nx\,\dd x, \quad n = 0, 1, 2, \ldots$$ $$b_n = \frac{1}{\pi}\int_{-\pi}^{\pi} f(x)\sin nx\,\dd x, \quad n = 1, 2, 3, \ldots$$
记号说明

注意 $a_0$ 是 $\cos(0\cdot x) = 1$ 对应的系数。$a_0/2$ 就是函数在一个周期上的平均值：

$$\frac{a_0}{2} = \frac{1}{2\pi}\int_{-\pi}^{\pi} f(x)\,\dd x.$$

$a_0$ 写成 $a_0/2$ 只是为了让 $a_n$ 的通项公式在 $n=0$ 时也成立（代入 $n=0$ 即得 $a_0 = \frac{1}{\pi}\int_{-\pi}^{\pi} f(x)\,\dd x$）。

A.3 为什么刚好是这些系数？——正交性

傅里叶系数公式不是凭空猜出来的，它来源于三角函数系的一个关键性质——正交性。

三角函数系的正交性

在区间 $[-\pi, \pi]$ 上，集合 $\{1, \cos x, \sin x, \cos 2x, \sin 2x, \ldots\}$ 中任意两个不同函数的内积为零：

$$\int_{-\pi}^{\pi} \cos mx\cos nx\,\dd x = \begin{cases} 0, & m \neq n, \\ \pi, & m = n \neq 0, \\ 2\pi, & m = n = 0. \end{cases}$$ $$\int_{-\pi}^{\pi} \sin mx\sin nx\,\dd x = \begin{cases} 0, & m \neq n, \\ \pi, & m = n \neq 0. \end{cases}$$ $$\int_{-\pi}^{\pi} \cos mx\sin nx\,\dd x = 0, \quad \forall m, n.$$
类比理解：正交 = "互不干扰"

在三维空间中，向量 $\vec{e}_1 = (1,0,0), \vec{e}_2 = (0,1,0), \vec{e}_3 = (0,0,1)$ 两两正交。任何一个向量 $\vec{v}$ 都可以唯一地表示为 $\vec{v} = c_1\vec{e}_1 + c_2\vec{e}_2 + c_3\vec{e}_3$，其中系数由内积给出：$c_i = \vec{v}\cdot\vec{e}_i$。

三角函数系做的事情完全一样——在一个函数空间中提供了一组"正交坐标轴"。傅里叶系数 $a_n, b_n$ 就是函数 $f$ 在这些"坐标轴"上的投影（内积），其计算方式与向量投影完全类似：

$$a_n = \frac{\langle f, \cos nx \rangle}{\langle \cos nx, \cos nx \rangle} = \frac{\int_{-\pi}^{\pi} f(x)\cos nx\,\dd x}{\int_{-\pi}^{\pi} \cos^2 nx\,\dd x}.$$

分母对于 $n \ge 1$ 为 $\pi$，对于 $n=0$ 为 $2\pi$。

正交性验证（以 $\cos mx\cos nx$ 为例）

利用积化和差公式 $\cos mx\cos nx = \frac{1}{2}\big[\cos(m+n)x + \cos(m-n)x\big]$：

若 $m \neq n$：

$$\int_{-\pi}^{\pi} \cos mx\cos nx\,\dd x = \frac{1}{2}\int_{-\pi}^{\pi} \big[\cos(m+n)x + \cos(m-n)x\big]\dd x = 0.$$

（因为 $\cos kx$ 在完整周期上的积分为零，$k \neq 0$。）

若 $m = n \neq 0$：$\cos^2 nx = \frac{1+\cos 2nx}{2}$，积分得 $\frac{1}{2} \cdot 2\pi = \pi$。

A.4 周期不是 $2\pi$ 怎么办？——一般周期的 Fourier 级数

上面的公式假设函数的周期为 $2\pi$。对于一般周期 $2l$ 的函数 $f(x)$，只需要做一个简单的缩放：

周期 $2l$ 的 Fourier 级数

设 $f(x)$ 周期为 $2l$，在 $[-l, l]$ 上可积。其 Fourier 级数为：

$$f(x) \sim \frac{a_0}{2} + \sum_{n=1}^{\infty} \left(a_n \cos\frac{n\pi x}{l} + b_n \sin\frac{n\pi x}{l}\right),$$

其中：

$$a_n = \frac{1}{l}\int_{-l}^{l} f(x)\cos\frac{n\pi x}{l}\,\dd x, \quad n = 0,1,2,\ldots$$ $$b_n = \frac{1}{l}\int_{-l}^{l} f(x)\sin\frac{n\pi x}{l}\,\dd x, \quad n = 1,2,3,\ldots$$
例题 A.1

设 $f(x) = x$，$x \in (-\pi, \pi)$，周期为 $2\pi$。求其 Fourier 级数。

▶ 详细解答

$f(x) = x$ 是奇函数，因此所有余弦系数 $a_n = 0$。

正弦系数（分部积分）：

$$b_n = \frac{1}{\pi}\int_{-\pi}^{\pi} x\sin nx\,\dd x = \frac{2}{\pi}\int_{0}^{\pi} x\sin nx\,\dd x$$ $$= \frac{2}{\pi}\left[-\frac{x\cos nx}{n}\bigg|_0^{\pi} + \frac{1}{n}\int_0^{\pi}\cos nx\,\dd x\right] = \frac{2}{\pi}\left[-\frac{\pi\cos n\pi}{n} + 0\right]$$ $$= -\frac{2}{n}\cos n\pi = \frac{2(-1)^{n+1}}{n}.$$

因此：

$$x \sim 2\left(\frac{\sin x}{1} - \frac{\sin 2x}{2} + \frac{\sin 3x}{3} - \frac{\sin 4x}{4} + \cdots\right) = 2\sum_{n=1}^{\infty}\frac{(-1)^{n+1}}{n}\sin nx.$$

注意在 $x = \pm\pi$ 处，级数收敛于 $(f(\pi^+) + f(\pi^-))/2 = (\pi + (-\pi))/2 = 0$，与 $f(\pm\pi)$ 的实际值不相等——这正是 Dirichlet 收敛定理要解释的现象。

A.5 什么时候级数收敛到原函数？——Dirichlet 收敛定理

Dirichlet 收敛定理

设 $f(x)$ 以 $2\pi$ 为周期，在 $[-\pi, \pi]$ 上分段光滑（即 $f$ 和 $f'$ 都只有有限个第一类间断点），则 $f$ 的 Fourier 级数在每一点 $x$ 收敛于

$$\frac{f(x^+) + f(x^-)}{2},$$

即该点左右极限的算术平均。特别地，在 $f$ 的连续点处，级数收敛于 $f(x)$ 本身。

常见考试陷阱：间断点处的收敛值

考试中经常考察 Fourier 级数在间断点处的和。记住：级数不会"选边站"，而是取左右极限的平均值。例如 $f(x) = x$ 的 Fourier 级数在 $x = \pi$ 处收敛于 $0$（而非 $\pi$ 或 $-\pi$）。

在 PDE 的分离变量法中，初始条件 $\varphi(x)$ 必须满足 Dirichlet 条件才能保证 Fourier 展开逐点收敛——这被编入了定解问题的相容性条件中。

A.6 奇偶延拓——正弦级数与余弦级数

在 PDE 的有界区域问题中（如一端固定、一端绝热的杆），我们常常只需要正弦级数或只需要余弦级数。这时就要用到延拓技术。

奇延拓 → 正弦级数（Dirichlet 边界条件 $u=0$）

若 $f(x)$ 只在 $[0, l]$ 上有定义，将其奇延拓到 $[-l, l]$：

$$\tilde{f}(x) = \begin{cases} f(x), & 0 \le x \le l, \\ -f(-x), & -l \le x \lt 0. \end{cases}$$

则 $\tilde{f}$ 是奇函数，$a_n = 0$，展开为正弦级数：

$$f(x) \sim \sum_{n=1}^{\infty} b_n \sin\frac{n\pi x}{l}, \quad b_n = \frac{2}{l}\int_0^l f(x)\sin\frac{n\pi x}{l}\,\dd x.$$
偶延拓 → 余弦级数（Neumann 边界条件 $u_x=0$）

将 $f(x)$ 偶延拓到 $[-l, l]$：

$$\tilde{f}(x) = \begin{cases} f(x), & 0 \le x \le l, \\ f(-x), & -l \le x \lt 0. \end{cases}$$

则 $\tilde{f}$ 是偶函数，$b_n = 0$，展开为余弦级数：

$$f(x) \sim \frac{a_0}{2} + \sum_{n=1}^{\infty} a_n \cos\frac{n\pi x}{l}, \quad a_n = \frac{2}{l}\int_0^l f(x)\cos\frac{n\pi x}{l}\,\dd x.$$
与 PDE 的深刻联系

你会发现：分离变量法中边界条件的不同类型（Dirichlet / Neumann）恰好对应着 Fourier 展开中不同的函数系（正弦 / 余弦）。这不是巧合——边界条件决定了本征函数系，而本征函数系的正交性和完备性（由 Sturm-Liouville 理论保证）使得任意初始条件都可以在该函数系下展开。这正是第四章要深入探讨的内容。

例题 A.2

设 $f(x) = x$，$x \in [0, \pi]$。分别求其正弦级数和余弦级数展开。

▶ 详细解答

正弦级数（奇延拓到 $[-\pi, \pi]$）：$l = \pi$。

$$b_n = \frac{2}{\pi}\int_0^{\pi} x\sin nx\,\dd x = -\frac{2}{n}\cos n\pi = \frac{2(-1)^{n+1}}{n}.$$ $$x \sim 2\sum_{n=1}^{\infty}\frac{(-1)^{n+1}}{n}\sin nx, \quad x \in [0,\pi).$$

在 $x = \pi$ 处级数和为 $0$（奇延拓后的间断点）。

余弦级数（偶延拓到 $[-\pi, \pi]$）：

$$a_0 = \frac{2}{\pi}\int_0^{\pi} x\,\dd x = \pi.$$ $$a_n = \frac{2}{\pi}\int_0^{\pi} x\cos nx\,\dd x = \frac{2}{n^2\pi}\big[(-1)^n - 1\big], \quad n \ge 1.$$

当 $n$ 为偶数时 $a_n = 0$；$n$ 为奇数时 $a_n = -\frac{4}{n^2\pi}$。

$$x \sim \frac{\pi}{2} - \frac{4}{\pi}\sum_{k=0}^{\infty}\frac{\cos(2k+1)x}{(2k+1)^2}, \quad x \in [0,\pi].$$

余弦级数在 $x=\pi$ 处收敛于 $f(\pi) = \pi$（偶延拓后连续）。注意余弦级数收敛更快（系数 $O(1/n^2)$ vs 正弦的 $O(1/n)$）。

A.7 Parseval 等式——能量守恒

Parseval 等式

若 $f$ 在 $[-\pi, \pi]$ 上平方可积，则：

$$\frac{1}{\pi}\int_{-\pi}^{\pi} f^2(x)\,\dd x = \frac{a_0^2}{2} + \sum_{n=1}^{\infty}(a_n^2 + b_n^2).$$

Parseval 等式表达的是：信号的总能量 = 各频率分量能量的总和。在 PDE 中，这是能量积分的最简形式——各 Fourier 模态的能量相互独立，总能量是各模态能量的代数和。

B. 从级数到积分——周期趋于无穷时的极限

B.1 核心问题：非周期函数怎么办？

直觉建立：把周期"撑到无穷大"

傅里叶级数要求函数是周期的。但很多实际问题中，函数定义在整个实轴上且不是周期的——比如热传导方程 Cauchy 问题的初始温度分布 $\varphi(x)$。

解决思路：把定义在全空间 $\RR$ 上的函数看作是周期趋于无穷大的周期函数的极限。当周期 $2l \to \infty$ 时，相邻频率之间的间隔 $\Delta\omega = \pi/l \to 0$——离散的频率谱变成了连续的频率谱。级数中的求和 $\sum$ 变成了积分 $\int$。

B.2 Fourier 积分定理

Fourier 积分（实数形式）

设 $f(x)$ 在 $\RR$ 上分段光滑且绝对可积（$\int_{-\infty}^{+\infty} |f(x)|\,\dd x \lt \infty$），则

$$f(x) = \int_0^{\infty} \big[A(\omega)\cos\omega x + B(\omega)\sin\omega x\big]\,\dd\omega,$$

其中：

$$A(\omega) = \frac{1}{\pi}\int_{-\infty}^{+\infty} f(t)\cos\omega t\,\dd t, \quad B(\omega) = \frac{1}{\pi}\int_{-\infty}^{+\infty} f(t)\sin\omega t\,\dd t.$$
与级数的精确类比

$$\begin{array}{c|c} \text{Fourier 级数（周期 $2l$）} & \text{Fourier 积分（全空间 $\RR$）} \\ \hline \displaystyle f(x) = \frac{a_0}{2} + \sum_{n=1}^{\infty}(a_n\cos\frac{n\pi x}{l} + b_n\sin\frac{n\pi x}{l}) & \displaystyle f(x) = \int_0^{\infty}[A(\omega)\cos\omega x + B(\omega)\sin\omega x]\,\dd\omega \\ \displaystyle a_n = \frac{1}{l}\int_{-l}^{l} f(t)\cos\frac{n\pi t}{l}\,\dd t & \displaystyle A(\omega) = \frac{1}{\pi}\int_{-\infty}^{+\infty} f(t)\cos\omega t\,\dd t \\ \text{离散频率：} \omega_n = n\pi/l & \text{连续频率：} \omega \in (0, +\infty) \end{array}$$

注意 $\frac{1}{\pi}$ vs $\frac{1}{l}$

级数系数中的分母是 $1/l$，而积分中的分母是 $1/\pi$。这是因为在过渡 $l \to \infty$ 时，因子 $\Delta\omega = \pi/l$ 被吸收到了积分中。具体推导如下（见下一小节）。

B.3 从级数到积分的推导概要

将周期 $2l$ 的 Fourier 级数用复数形式写出（见 C.1 节），令 $\xi_n = n\pi/l$，$\Delta\xi = \pi/l$：

$$f(x) = \sum_{n=-\infty}^{\infty} \frac{1}{2l}\left(\int_{-l}^{l} f(t)e^{-i\xi_n t}\dd t\right) e^{i\xi_n x}.$$

令 $l \to \infty$，则 $\Delta\xi \to 0$，求和变为积分：

$$f(x) = \frac{1}{2\pi}\int_{-\infty}^{+\infty} \left(\int_{-\infty}^{+\infty} f(t)e^{-i\xi t}\dd t\right) e^{i\xi x}\dd\xi.$$

括号中的表达式正是 Fourier 变换 $\hat{f}(\xi)$——这就是 Fourier 积分定理的复数形式，也是 Fourier 变换和逆变换的来源。

C. 傅里叶变换——完整讲解

C.1 用复数写——一个更优雅的形式

为什么用复数？

由欧拉公式 $e^{i\theta} = \cos\theta + i\sin\theta$，正弦和余弦可以统一写成指数形式。这使得 Fourier 分析的公式变得极其对称和简洁，也使得计算（尤其是求导和卷积）变得异常方便。

在 PDE 中，复数形式的 Fourier 变换是主流——简化了从 $u_{xx}$ 到 $-\xi^2\hat{u}$ 的转换，使求解过程清爽利落。

定义 C.1（Fourier 变换与逆变换）

设 $f: \RR \to \CC$ 满足适当的可积条件。其Fourier 变换为：

$$\boxed{\hat{f}(\xi) = \mathcal{F}[f](\xi) = \int_{-\infty}^{+\infty} f(x)\,e^{-i\xi x}\,\dd x}, \quad \xi \in \RR.$$

Fourier 逆变换（从频率域回到空间域/时间域）：

$$\boxed{f(x) = \mathcal{F}^{-1}[\hat{f}](x) = \frac{1}{2\pi}\int_{-\infty}^{+\infty} \hat{f}(\xi)\,e^{i\xi x}\,\dd\xi}.$$

记号 $\hat{f}$ 或 $\mathcal{F}[f]$ 都常用。在 PDE 中 $\xi$ 称为频率变量（或波数），$x$ 为空间变量。

物理中常见的另一种约定

物理和工程中常见的定义把 $2\pi$ 放在指数里：

$$\hat{f}(k) = \int_{-\infty}^{+\infty} f(x)e^{-2\pi i k x}\,\dd x, \quad f(x) = \int_{-\infty}^{+\infty} \hat{f}(k)e^{2\pi i k x}\,\dd k.$$

两种约定本质相同，本教程采用数学中更常见的第一种（$\frac{1}{2\pi}$ 在逆变换中）。考试中注意看清题目用的是哪种约定！

C.2 Fourier 变换的核心性质

以下性质是 Fourier 变换在 PDE 中发挥威力的原因。标 ☆ 的是 PDE 求解中最常用的三条。

Fourier 变换性质速查表
性质时域 $f(x)$频域 $\hat{f}(\xi)$在 PDE 中的用途
☆ 微分 $f'(x)$ $i\xi\,\hat{f}(\xi)$ 把对 $x$ 的偏导变成乘以 $i\xi$——PDE 降为 ODE
☆ 高阶微分 $f^{(n)}(x)$ $(i\xi)^n\hat{f}(\xi)$ $\mathcal{F}[u_{xx}] = -\xi^2\hat{u}$（热方程、波方程核心）
线性 $af(x) + bg(x)$ $a\hat{f}(\xi) + b\hat{g}(\xi)$ 叠加原理在频率域中依然成立
平移（空间） $f(x - a)$ $e^{-ia\xi}\,\hat{f}(\xi)$ 平移在频域体现为相位变化
平移（频率） $e^{i\omega x}f(x)$ $\hat{f}(\xi - \omega)$ 调制 ⇔ 频移（通信原理基础）
伸缩 $f(ax)$，$a \neq 0$ $\frac{1}{|a|}\hat{f}(\xi/a)$ 压缩时域 ⇔ 展宽频域（不确定性原理）
☆ 卷积 $(f * g)(x)$ $\hat{f}(\xi)\,\hat{g}(\xi)$ 热核卷积 → 乘积的逆变换（热传导方程解的核心）
Parseval / Plancherel $\int |f|^2\dd x$ $\frac{1}{2\pi}\int |\hat{f}|^2\dd\xi$ 能量在时域和频域中的守恒
微分性质的证明

设 $f$ 在 $\RR$ 上连续可微且当 $|x| \to \infty$ 时 $f(x) \to 0$（PDE 中通常满足）：

$$\mathcal{F}[f'](\xi) = \int_{-\infty}^{+\infty} f'(x)e^{-i\xi x}\dd x$$

分部积分（$u = e^{-i\xi x}$，$\dd v = f'(x)\dd x$）：

$$= \Big[f(x)e^{-i\xi x}\Big]_{-\infty}^{+\infty} - \int_{-\infty}^{+\infty} f(x)(-i\xi)e^{-i\xi x}\dd x = 0 + i\xi\int_{-\infty}^{+\infty} f(x)e^{-i\xi x}\dd x = i\xi\,\hat{f}(\xi).$$

重复应用即得 $\mathcal{F}[f^{(n)}](\xi) = (i\xi)^n\hat{f}(\xi)$。这一性质是将 PDE 转换为 ODE 的数学基础。

符号函数 $\operatorname{sgn}(x)$ $\displaystyle\frac{2}{i\xi}$ ch2 半无界问题的奇延拓
例题 C.1（关键！Gauss 函数的 Fourier 变换）

验证 $\displaystyle\mathcal{F}[e^{-ax^2}](\xi) = \sqrt{\frac{\pi}{a}}\,e^{-\xi^2/(4a)}$（$a \gt 0$）。这个结果是热传导方程热核推导的数学核心。

▶ 详细解答

$$\mathcal{F}[e^{-ax^2}](\xi) = \int_{-\infty}^{+\infty} e^{-ax^2}e^{-i\xi x}\dd x = \int_{-\infty}^{+\infty} e^{-a(x^2 + \frac{i\xi}{a}x)}\dd x.$$

配方法：$x^2 + \frac{i\xi}{a}x = \left(x + \frac{i\xi}{2a}\right)^2 + \frac{\xi^2}{4a^2}$。

$$\mathcal{F}[e^{-ax^2}](\xi) = e^{-\xi^2/(4a)}\int_{-\infty}^{+\infty} e^{-a(x + i\xi/(2a))^2}\dd x.$$

作复平移 $z = x + i\xi/(2a)$，沿实轴积分经围道积分可证（或直接用"复平移不影响 Gauss 积分的值"这一事实）：

$$\int_{-\infty}^{+\infty} e^{-a(x + i\xi/(2a))^2}\dd x = \int_{-\infty}^{+\infty} e^{-ax^2}\dd x = \sqrt{\frac{\pi}{a}}.$$

因此 $\mathcal{F}[e^{-ax^2}](\xi) = \sqrt{\frac{\pi}{a}}\,e^{-\xi^2/(4a)}$。注意：Gauss 函数的 Fourier 变换仍然是 Gauss 函数！这是 Gauss 函数独有的性质。

在热传导方程中的应用：取 $a = 1/(4t)$，则 $\mathcal{F}[e^{-x^2/(4t)}] = \sqrt{4\pi t}\,e^{-\xi^2 t}$，进而 $\mathcal{F}^{-1}[e^{-\xi^2 t}] = \frac{1}{\sqrt{4\pi t}}e^{-x^2/(4t)}$——这就是热核。

C.4 卷积定理——PDE 求解的"瑞士军刀"

定义 C.2（卷积）

两个函数 $f, g$ 在 $\RR$ 上的卷积定义为：

$$(f * g)(x) = \int_{-\infty}^{+\infty} f(x - y)\,g(y)\,\dd y = \int_{-\infty}^{+\infty} f(y)\,g(x - y)\,\dd y.$$
卷积定理
$$\mathcal{F}[f * g](\xi) = \hat{f}(\xi)\,\hat{g}(\xi),$$ $$\mathcal{F}^{-1}[\hat{f}\,\hat{g}](x) = (f * g)(x).$$

即：卷积的 Fourier 变换 = 各自 Fourier 变换的乘积。反之，乘积的逆变换 = 各自逆变换的卷积。

在 PDE 中卷积定理的妙用

回顾第二章热传导方程的求解过程：

  1. 方程经 Fourier 变换后解得 $\hat{u}(\xi,t) = \hat{\varphi}(\xi) \cdot e^{-a^2\xi^2 t}$
  2. $e^{-a^2\xi^2 t}$ 的逆变换 $\mathcal{F}^{-1}[e^{-a^2\xi^2 t}] = K(x,t)$（热核）
  3. 由卷积定理：$u(x,t) = \varphi(x) * K(x,t)$

这正是"基本解的卷积给出一般解"这一现代 PDE 理论核心思想的雏形。第六章的广义函数理论将把这一思想系统化。

E. 两周速成学习计划（目标期末 85+）

计划设计理念

这个计划基于以下原则：

  • 80/20 法则：20% 的核心内容覆盖 80% 的考试分数。优先掌握三类经典方程的求解方法和定性性质，而非面面俱到地学完所有细节。
  • 先具体、后抽象：先通过大量例题掌握怎么做（计算），再理解为什么（理论）。这与谷超豪教材的编排逻辑一致。
  • 重复暴露：Fourier 级数、分离变量法、Green 函数等核心工具会在不同章节反复出现——每次遇到都是一次复习和深化。
  • 输出驱动：精读一章后合上书默写关键公式和解题步骤，然后再对照原文。这比反复阅读有效 10 倍。
  • 目标导向：85 分 ≠ 满分。允许在一些纯理论证明（如极值原理的严格 $\varepsilon$ 论证）上只掌握思路而不默写细节，把时间花在会考计算的地方。

考试策略速递

85+ 考试策略
  1. 先做计算题（通常占 60–70 分）：分离变量法、Fourier 变换法、d'Alembert 公式、Green 函数法。这些是确定性最强的分数——方法对就能拿满。

  3. 最后做证明题（通常占 15–20 分）：极值原理推论、能量积分唯一性。即使不能完整写出严格证明，也要写出核心思路（如"引入辅助函数 $v = u - \varepsilon t$"）——这在很多老师那里能拿到一半以上的分数。
  4. 时间分配：若有 3 小时，计算题分配 1 小时 40 分钟，概念题 20 分钟，证明题 40 分钟，最后 20 分钟检查。
  5. 检查重点：边界条件是否满足？初始条件是否满足？系数的积分上下限和因子 $2/l$ 是否正确？正弦还是余弦？
  6. 写清楚步骤：PDE 考试步骤分很重要。即使最终答案有误，清晰的解题思路（分离变量 → 本征值问题 → 叠加 → 定系数）也能拿到大部分分数。
最常见的 5 个考场失分点
  1. 混淆边界条件对应的本征函数：$u=0$ → 正弦 $\sin$；$u_x=0$ → 余弦 $\cos$。零点条件 → 正弦，极值点条件 → 余弦。
  2. Fourier 系数公式忘因子 $2/l$：正交性积分 $\int_0^l \sin^2(n\pi x/l)\dd x = l/2$，所以系数 $C_n = \frac{2}{l}\int_0^l f(x)\sin(n\pi x/l)\dd x$，别忘了前面的 $2/l$。
  3. Fourier 逆变换忘 $\frac{1}{2\pi}$：$\mathcal{F}^{-1}[\hat{f}](x) = \frac{1}{2\pi}\int_{-\infty}^{+\infty}\hat{f}(\xi)e^{i\xi x}\dd\xi$。
  4. 分离变量时常数符号取反：设 $\frac{X''}{X} = \frac{T'}{a^2T} = -\lambda$（取负号是为了得到振荡型空间解 $X'' + \lambda X = 0$）。如果取成 $+\lambda$，后续全都反了。
  5. 延拓方向搞反：Dirichlet 条件（$u=0$）→ 奇延拓（$-$ 号）；Neumann 条件（$u_x=0$）→ 偶延拓（$+$ 号）。口诀："定温奇，绝热偶"。
最后的话

11 天的速成计划强度很高，但它是有节奏的：前5天建立计算能力（会做题），后6天建立理论视野（懂为什么）+ 查漏补缺。如果你严格按照这个计划执行，85 分是完全现实的目标。

最重要的是：不要只是"看"——要"做"。PDE 是推导出来的，不是读出来的。每一道例题，先遮住答案自己推一遍。等你合上书也能独立写出 d'Alembert 公式的推导、热核的求解过程、分离变量法的五步流程时——你就不只是为考试准备了，你是真正学会了偏微分方程。
