2.1 物理推导：热量如何扩散？

本章动机

把一杯热水放在桌上，它为什么会慢慢变凉？一根铁棍一端在火里烧，另一端为什么也热起来？这些现象背后是同一个物理过程——热传导（扩散）。本章从零开始，建立热传导方程的数学理论。学完本章后，你将能够：

  • 从Fourier热传导定律和能量守恒推导热传导方程
  • 用Fourier变换法求解全空间上的初值问题
  • 用分离变量法求解有界杆上的热传导
  • 用延拓法处理半无界问题
  • 理解极值原理及其深刻推论——热传导的不可逆性和解的唯一性
  • 领会热传导方程最神奇的性质：无论初始温度多粗糙，瞬间变得光滑

Fourier热传导定律

热传导的物理基础由法国数学物理学家Joseph Fourier在1807年提出。他的核心洞察是：热流密度正比于温度梯度，方向从高温指向低温。

Fourier热传导定律

设 $u(x,t)$ 表示物体在位置 $x$、时刻 $t$ 的温度。热流密度向量 $\vec{q}$（单位时间通过单位面积的热量）满足：

$$\vec{q} = -k\,\grad u,$$

其中 $k > 0$ 为热传导系数（thermal conductivity），单位是 $\mathrm{W/(m\cdot K)}$。负号表示热量从高温处流向低温处——这是热力学第二定律的体现。

直觉建立：温度梯度决定热流

想象一根金属棒，左端插入沸水（$100^\circ\mathrm{C}$），右端插入冰水（$0^\circ\mathrm{C}$）。温度沿棒的分布是不均匀的——存在温度梯度。热量就会从高温端流向低温端，而且温度变化越陡峭，热流越强。

如果温度是均匀的（$\grad u = 0$），就没有热流——这就是热平衡状态。Fourier定律说的就是这个简单道理：不均匀产生流动，梯度驱动扩散。

能量守恒推导热传导方程

设物体占据区域 $\Omega \subset \RR^3$，其密度为 $\rho$，比热容为 $c$。在物体内部任取一个子区域 $D \subset \Omega$。

能量守恒原理告诉我们：$D$ 内热量的增加率 = 通过边界 $\partial D$ 流入的热量 + $D$ 内热源产生的热量。

用数学语言表达：

$$\frac{\partial}{\partial t}\int_D c\rho u\,\dd V = -\oint_{\partial D}\vec{q}\cdot\vec{n}\,\dd S + \int_D F\,\dd V,$$

其中 $F(x,t)$ 为热源强度（单位体积单位时间产生的热量），$\vec{n}$ 为单位外法向量。右边第一项前面有负号，是因为 $\vec{q}\cdot\vec{n}$ 表示流出的热量，而我们需要的是流入的热量。

代入Fourier定律 $\vec{q} = -k\grad u$，并由Gauss散度定理：

$$-\oint_{\partial D}(-k\grad u)\cdot\vec{n}\,\dd S = \int_D \div(k\grad u)\,\dd V.$$

因此：

$$\int_D c\rho\frac{\partial u}{\partial t}\,\dd V = \int_D \big[\div(k\grad u) + F\big]\,\dd V.$$

由于 $D$ 是任意选取的子区域，被积函数必须相等：

$$c\rho\frac{\partial u}{\partial t} = \div(k\grad u) + F.$$

当介质均匀（$k$ 为常数）且无内部热源（$F = 0$）时，我们得到了经典的热传导方程：

热传导方程（扩散方程）
$$\boxed{\frac{\partial u}{\partial t} = a^2 \lap u}, \quad a^2 = \frac{k}{c\rho}.$$

在一维情形：$\displaystyle u_t = a^2 u_{xx}$。

在三维情形：$\displaystyle u_t = a^2(u_{xx} + u_{yy} + u_{zz})$。

其中 $a^2$ 称为热扩散率（thermal diffusivity），单位 $\mathrm{m^2/s}$。$a^2$ 越大，热量扩散越快。

物理直觉：为什么 $u_t \propto u_{xx}$？

$u_{xx}$ 衡量的是温度的曲率——即温度分布"弯曲"的程度。如果温度分布是线性的（$u_{xx}=0$），各处流入和流出的热量相等，温度不随时间变化——这是稳态。如果温度分布是"凸"的（$u_{xx} \lt 0$，中间高两边低），热量从中部向两侧扩散，中部温度降低——$u_t \lt 0$。反之，温度分布是"凹"的（$u_{xx} \gt 0$），热量向中部汇聚，中部温度升高——$u_t \gt 0$。

所以热传导方程 $u_t = a^2 u_{xx}$ 说的是：温度曲率驱动温度变化率。曲率越大，温度变化越快——最终驱使温度趋于均匀（$u_{xx} \to 0$）。

定解条件

与波动方程（含二阶时间导数，需要两个初始条件）不同，热传导方程只含一阶时间导数，因此只需要一个初始条件。

初始条件与三种边界条件

初始条件：$u(x,0) = \varphi(x)$，给出初始时刻的温度分布。

边界条件（三种典型类型）：

  1. Dirichlet条件（第一类）——给定边界温度： $$u|_{\partial\Omega} = g(x,t).$$ 例如：杆的端点保持在固定温度（恒温槽）。
  2. Neumann条件（第二类）——给定边界热流密度： $$\frac{\partial u}{\partial n}\bigg|_{\partial\Omega} = g(x,t).$$ 特别地，$g=0$ 表示绝热边界（无热量通过边界）。
  3. Robin条件（第三类）——对流换热（Newton冷却定律）： $$\left(\frac{\partial u}{\partial n} + \sigma u\right)\bigg|_{\partial\Omega} = g(x,t).$$ 描述了边界与周围环境的热交换，$\sigma$ 为换热系数。

与波动方程的本质区别

抛物型 vs 双曲型：两类方程的根本差异

热传导方程 $u_t = a^2 u_{xx}$ 是抛物型方程，波动方程 $u_{tt} = a^2 u_{xx}$ 是双曲型方程。仅一字之差，性质天差地别：

  • 时间导数阶数不同：热传导方程只需一个初始条件（初始温度），波动方程需要两个（初始位移和初始速度），因为方程含 $u_{tt}$ ——二阶时间导数"记住"了初始的"惯性"；
  • 传播速度不同：热传导方程有无穷传播速度——一个点的温度改变瞬间（理论上）影响整个空间；波动方程有有限传播速度 $a$——信息沿特征线传播，$|x| > at$ 的区域尚未收到扰动；
  • 可逆性不同：热传导方程不可逆——时间反演 $t \to -t$ 后方程变为 $u_t = -a^2 u_{xx}$（反向热方程），这是一个不适定问题（ill-posed）；波动方程在 $t \to -t$ 下不变（因为 $u_{tt}$ 是二阶导数），物理过程可逆。
例题 2.1.1

一根长为 $l$ 的均匀细杆，侧面绝热，左端温度恒为 $0^\circ\mathrm{C}$，右端绝热（无热流通过）。杆的初始温度分布为 $u(x,0) = x(l-x)$。写出完整的定解问题。

▶ 详细解答

$$\begin{cases} u_t = a^2 u_{xx}, & 0 \lt x \lt l,\ t \gt 0, \\[4pt] u(0,t) = 0, \quad u_x(l,t) = 0, & t \gt 0, \\[4pt] u(x,0) = x(l-x), & 0 \le x \le l. \end{cases}$$

注意：左端是Dirichlet条件（恒温），右端是Neumann条件（绝热，$u_x=0$ 表示无热流通过）。这是一个混合边值问题。

例题 2.1.2

判断在三维各向同性均匀介质中（无热源），稳态（$u_t = 0$）下的温度分布满足什么方程。

▶ 详细解答

当温度不再随时间变化（$u_t = 0$），热传导方程退化为：

$$\lap u = 0, \quad \text{即} \quad \frac{\partial^2 u}{\partial x^2} + \frac{\partial^2 u}{\partial y^2} + \frac{\partial^2 u}{\partial z^2} = 0.$$

这就是Laplace方程——椭圆型方程的典型代表。它描述了热平衡（稳态）下的温度分布，也描述了静电势、不可压缩流体的速度势等。我们将在第三章详细研究它。

2.2 初值问题的求解：Fourier变换法

考虑全空间 $\RR$ 上的Cauchy问题（纯初值问题，无边界）：

$$\begin{cases} u_t = a^2 u_{xx}, & x \in \RR,\ t \gt 0, \\ u(x,0) = \varphi(x), & x \in \RR. \end{cases}$$
为什么用Fourier变换？

热传导方程在空间方向是二阶导数，时间方向是一阶导数。Fourier变换的妙处在于：它把对 $x$ 的求导 $\partial_x$ 变成乘以 $i\xi$——从而把偏微分方程变成常微分方程！一旦变成ODE，解起来就简单多了。

这就像用对数把乘法变成加法一样——Fourier变换把"求导"这个难操作的事情变成了"乘法"这个简单的事情。

Fourier变换简述（从零介绍）

如需更深入的理解

以下是对 Fourier 变换的扼要回顾，足以完成热传导方程的求解。如果你希望更系统地学习 Fourier 变换的定义来源（从级数到积分的过渡）、完整的性质推导、以及更多计算例题，请参阅附录 B 节（从级数到积分）和C 节（傅里叶变换完整讲解）。

如果你已经熟悉 Fourier 变换的基本操作，可以跳过这个提示，继续往下阅读。

对于定义在 $\RR$ 上的函数 $f(x)$（满足适当的可积条件），其Fourier变换定义为：

$$\hat{f}(\xi) = \mathcal{F}[f](\xi) = \int_{-\infty}^{+\infty} f(x)\,e^{-i\xi x}\,\dd x, \quad \xi \in \RR.$$

Fourier逆变换（从频率域回到空间域）：

$$f(x) = \mathcal{F}^{-1}[\hat{f}](x) = \frac{1}{2\pi}\int_{-\infty}^{+\infty} \hat{f}(\xi)\,e^{i\xi x}\,\dd\xi.$$

Fourier变换的核心性质——把求导变成乘法：

$$\mathcal{F}[f'(x)](\xi) = i\xi\,\hat{f}(\xi), \quad \mathcal{F}[f''(x)](\xi) = (i\xi)^2\hat{f}(\xi) = -\xi^2\hat{f}(\xi).$$

另一个关键性质是卷积定理：

$$\mathcal{F}[f * g] = \hat{f} \cdot \hat{g}, \quad \text{其中} \quad (f * g)(x) = \int_{-\infty}^{+\infty} f(x-y)g(y)\,\dd y.$$

用Fourier变换求解热传导方程

对热传导方程 $u_t = a^2 u_{xx}$ 两边关于 $x$ 取Fourier变换。设 $\hat{u}(\xi,t) = \mathcal{F}[u(\cdot,t)](\xi)$：

  • 左边：$\mathcal{F}[u_t] = \frac{\dd\hat{u}}{\dd t}$（Fourier变换不涉及 $t$，求导可与变换交换）；
  • 右边：$\mathcal{F}[a^2 u_{xx}] = a^2 (-\xi^2)\hat{u} = -a^2\xi^2\hat{u}$。

于是偏微分方程化为关于 $t$ 的常微分方程：

$$\frac{\dd\hat{u}}{\dd t} = -a^2\xi^2\,\hat{u}.$$

这是一个最简单的一阶ODE！以 $\xi$ 为参数，解为：

$$\hat{u}(\xi,t) = \hat{u}(\xi,0)\,e^{-a^2\xi^2 t} = \hat{\varphi}(\xi)\,e^{-a^2\xi^2 t}.$$

现在需要求逆Fourier变换。$e^{-a^2\xi^2 t}$ 的逆Fourier变换是一个经典的Gauss积分：

$$\mathcal{F}^{-1}\big[e^{-a^2\xi^2 t}\big](x) = \frac{1}{2a\sqrt{\pi t}}\exp\!\left(-\frac{x^2}{4a^2 t}\right).$$

由卷积定理，乘积的逆变换等于逆变换的卷积：

$$u(x,t) = \varphi(x) * \mathcal{F}^{-1}\big[e^{-a^2\xi^2 t}\big](x) = \int_{-\infty}^{+\infty} \varphi(y) \cdot \frac{1}{2a\sqrt{\pi t}}\exp\!\left(-\frac{(x-y)^2}{4a^2 t}\right)\dd y.$$
热传导方程Cauchy问题的解
$$u(x,t) = \int_{-\infty}^{+\infty} K(x-y,t)\,\varphi(y)\,\dd y,$$

其中热核（基本解 / Gauss核）为：

$$\boxed{K(x,t) = \frac{1}{2a\sqrt{\pi t}}\exp\!\left(-\frac{x^2}{4a^2 t}\right)}, \quad t \gt 0.$$
热核的物理含义：热量的"点扩散"

热核 $K(x,t)$ 是初始条件为 $\varphi(x) = \delta(x)$（Dirac delta函数——在 $x=0$ 处集中了单位热量）时的解。它描述了一个点热源如何随时间扩散：初始时刻热量集中在一点，随后以Gauss分布的形式逐渐扩散开来。

任意初始温度分布的解，就是这些"点扩散"的加权叠加（卷积）——初始温度 $\varphi(y)$ 在每一点 $y$ 的热量独立地按热核的规律扩散，总效果就是它们叠加在一起。

热核的五个基本性质

热核的基本性质
  1. 正性：$K(x,t) \gt 0$，$\forall x \in \RR$，$t \gt 0$。热量总是正的，没有"负温度"扩散。
  2. 归一性：$\displaystyle\int_{-\infty}^{+\infty} K(x,t)\,\dd x = 1$，$\forall t \gt 0$。总热量守恒——无论怎么扩散，热量总量不变。
  3. 逼近初始数据：$\displaystyle\lim_{t \to 0^+} \int_{-\infty}^{+\infty} K(x-y,t)\,\varphi(y)\,\dd y = \varphi(x)$（在 $\varphi$ 的连续点处）。当 $t \to 0^+$ 时热核趋向于 $\delta(x)$。
  4. 光滑性：对每个 $t \gt 0$，$K(x,t)$ 是 $x$ 的 $C^\infty$ 函数。Gauss函数的指数衰减保证了一切阶导数存在。
  5. 满足方程：$K_t = a^2 K_{xx}$，$\forall x \in \RR$，$t \gt 0$。热核本身也是热传导方程的解。
性质2的验证（归一性）

令 $z = \dfrac{x}{2a\sqrt{t}}$，则 $\dd x = 2a\sqrt{t}\,\dd z$：

$$\int_{-\infty}^{+\infty} K(x,t)\,\dd x = \frac{1}{2a\sqrt{\pi t}}\int_{-\infty}^{+\infty} e^{-x^2/(4a^2 t)}\,\dd x = \frac{1}{\sqrt{\pi}}\int_{-\infty}^{+\infty} e^{-z^2}\,\dd z = 1.$$

这里用到了经典Gauss积分：$\displaystyle\int_{-\infty}^{+\infty} e^{-z^2}\dd z = \sqrt{\pi}$。这个积分的计算方法是平方后转化为极坐标下的二重积分。

例题 2.2.1

求解 $u_t = u_{xx}$（取 $a=1$），初始温度 $\varphi(x) = e^{-|x|}$。讨论解的渐近行为。

▶ 详细解答

由热核公式（$a=1$）：

$$u(x,t) = \frac{1}{2\sqrt{\pi t}}\int_{-\infty}^{+\infty} e^{-|y|}\,e^{-(x-y)^2/(4t)}\,\dd y.$$

将积分拆分为 $y \lt 0$ 和 $y \gt 0$ 两部分，分别配方化为误差函数。更直接地，利用Fourier变换：

$$\hat{\varphi}(\xi) = \int_{-\infty}^{+\infty} e^{-|x|}e^{-i\xi x}\dd x = \frac{2}{1+\xi^2},$$ $$\hat{u}(\xi,t) = \frac{2}{1+\xi^2}\,e^{-\xi^2 t}.$$

当 $t \to 0^+$：$u(x,t) \to e^{-|x|}$（恢复初始温度）。

当 $t \to +\infty$：由热核性质，$u(x,t) \to 0$（无限空间上热量扩散到无穷远处，各处温度趋于零）。更精确地，当 $t$ 很大时，$\displaystyle u(0,t) \sim \frac{1}{\sqrt{\pi t}}$。

物理解释：初始集中在原点附近的"尖峰"热量，随时间不断向远处扩散。由于全空间无限大，热量"稀释"到整个空间最终导致每一点温度都趋于零。

例题 2.2.2

求解 $u_t = u_{xx}$，初始温度 $\varphi(x) = \begin{cases} 1, & |x| \le 1, \\ 0, & |x| \gt 1. \end{cases}$ 求 $u(0,t)$。

▶ 详细解答

在 $x=0$ 处：

$$u(0,t) = \frac{1}{2\sqrt{\pi t}}\int_{-1}^{1} e^{-y^2/(4t)}\,\dd y.$$

令 $z = \dfrac{y}{2\sqrt{t}}$，$\dd y = 2\sqrt{t}\,\dd z$：

$$u(0,t) = \frac{1}{2\sqrt{\pi t}} \cdot 2\sqrt{t}\int_{-1/(2\sqrt{t})}^{1/(2\sqrt{t})} e^{-z^2}\dd z = \frac{1}{\sqrt{\pi}}\int_{-1/(2\sqrt{t})}^{1/(2\sqrt{t})} e^{-z^2}\dd z.$$

引入误差函数 $\displaystyle\operatorname{erf}(s) = \frac{2}{\sqrt{\pi}}\int_0^s e^{-z^2}\dd z$（满足 $\operatorname{erf}(0)=0$，$\operatorname{erf}(+\infty)=1$）：

$$u(0,t) = \operatorname{erf}\!\left(\frac{1}{2\sqrt{t}}\right).$$

验证极限行为：

  • $t \to 0^+$：$\operatorname{erf}(+\infty) = 1$，与 $\varphi(0) = 1$ 一致；
  • $t \to +\infty$：$\operatorname{erf}(0^+) = 0$，温度趋于零。

物理图像：初始矩形温度脉冲的尖锐边缘立即被"模糊化"——热传导的平滑效应在起作用。

考试标准流程（五步法框架）

考试中的"完整求解"要求

热传导方程Cauchy问题是考试中的经典大题。以下是标准五步法——注意前两步（Fourier变换化PDE为ODE、求解ODE）在上文已详细讲解，这里提炼为紧凑的答题模板：

五步法速查
  1. Fourier变换：对 $u_t = a^2 u_{xx}$ 关于 $x$ 取变换 $\mathcal{F}$，得 $\frac{\dd\hat{u}}{\dd t} = -a^2\xi^2\hat{u}$；
  2. 解ODE：$\hat{u}(\xi,t) = \hat{\varphi}(\xi)\,e^{-a^2\xi^2 t}$；
  3. 逆变换：$u(x,t) = \mathcal{F}^{-1}[\hat{\varphi}(\xi)e^{-a^2\xi^2 t}]$；
  4. 计算热核：$\mathcal{F}^{-1}[e^{-a^2\xi^2 t}] = K(x,t) = \frac{1}{2a\sqrt{\pi t}}e^{-x^2/(4a^2 t)}$；
  5. 卷积写出解：$u(x,t) = \int_{-\infty}^{+\infty} K(x-y,t)\,\varphi(y)\,\dd y$。
考试注释要点
  1. 需说明 $\varphi$ 满足 Fourier 变换条件（如 $\varphi \in L^1(\RR)$ 或有界连续）；
  2. 卷积定理的使用要注明条件（$\hat{\varphi}$ 与 $e^{-a^2\xi^2 t}$ 的乘积在 $L^1$ 中）；
  3. 若考试给出具体 $\varphi(x)$，要显式计算出 $u(x,t)$（通常用到误差函数 $\operatorname{erf}$）；
  4. 最后往往还需要讨论解的渐近行为（见下文）。
例题 2.2.3（考试完整大题示例 — 原 §2.7 例题合并）

用 Fourier 变换法求解热传导方程 Cauchy 问题：

$$u_t = u_{xx} \quad (x \in \RR, t > 0), \qquad u(x,0) = e^{-x^2}.$$

并分析解的渐近性态。

▶ 详细解答

解：$a=1$，$\varphi(x)=e^{-x^2}$。

第一步，对 $\varphi$ 做 Fourier 变换：$\hat{\varphi}(\xi) = \int_{-\infty}^{+\infty} e^{-x^2} e^{-i\xi x}\dd x = \sqrt{\pi}\,e^{-\xi^2/4}$（经典 Gauss 积分）。

第二步，$\hat{u}(\xi,t) = \hat{\varphi}(\xi) e^{-\xi^2 t} = \sqrt{\pi}\,e^{-\xi^2/4} e^{-\xi^2 t} = \sqrt{\pi}\,e^{-\xi^2(t+1/4)}$。

第三步，逆变换：$u(x,t) = \frac{1}{2\pi}\int_{-\infty}^{+\infty} \sqrt{\pi}\,e^{-\xi^2(t+1/4)} e^{i\xi x}\dd\xi$。

配方：$-\xi^2(t+1/4) + i\xi x = -(t+1/4)\left(\xi - \frac{ix}{2(t+1/4)}\right)^2 - \frac{x^2}{4(t+1/4)}$。

$$\begin{aligned} u(x,t) &= \frac{\sqrt{\pi}}{2\pi} \cdot \sqrt{\frac{\pi}{t+1/4}} \cdot \exp\!\left(-\frac{x^2}{4(t+1/4)}\right) \\ &= \frac{1}{\sqrt{4t+1}} \exp\!\left(-\frac{x^2}{4t+1}\right). \end{aligned}$$

验证：$t=0$ 时 $u(x,0) = e^{-x^2}$，满足初始条件。直接求导可验证满足 $u_t = u_{xx}$。

渐近分析：

  • 固定 $x$，$t \to +\infty$ 时：$u(x,t) \sim \frac{1}{\sqrt{4t}} e^{-x^2/(4t)} \sim \frac{1}{2\sqrt{t}} \to 0$。温度趋于零的速率为 $O(t^{-1/2})$。
  • 固定 $t$，$x \to \pm\infty$ 时：$u(x,t) \sim \frac{1}{\sqrt{4t+1}} e^{-x^2/(4t+1)} \to 0$，Gauss 型指数衰减。
  • 总热量守恒：$\int_{-\infty}^{+\infty} u(x,t)\dd x = \sqrt{\pi}$（与 $t$ 无关）。

全空间解的渐近行为（原 §2.8 内容并入）

为什么关心渐近行为？

在物理和工程中，我们往往不关心 $t=0.01$ 时的精确温度，而更关心：经过足够长的时间后，系统最终如何？趋近速度有多快？

全空间解的渐近衰减

对于 $u_t = a^2 u_{xx}$（$x \in \RR$），初始条件 $u(x,0)=\varphi(x) \in L^1(\RR) \cap L^\infty(\RR)$。则：

  1. 逐点衰减：对每个固定的 $x$，$\displaystyle \lim_{t \to +\infty} u(x,t) = 0$。
  2. 衰减速率：$\displaystyle |u(x,t)| \le \frac{\|\varphi\|_{L^1}}{2a\sqrt{\pi t}}$。特别地，$\max_x |u(x,t)|$ 以 $O(t^{-1/2})$ 速率衰减。
  3. 总热量守恒：$\displaystyle \int_{-\infty}^{+\infty} u(x,t)\,\dd x = \int_{-\infty}^{+\infty} \varphi(x)\,\dd x$（对任意 $t > 0$ 为常数）。
  4. 渐近 Profile：当 $t \to +\infty$ 时，解的形态越来越接近热核本身的形状： $$u(x,t) \sim \frac{M}{2a\sqrt{\pi t}} \exp\!\left(-\frac{x^2}{4a^2 t}\right), \quad M = \int_{-\infty}^{+\infty} \varphi(x)\,\dd x.$$ 即无论初始分布是什么形状，经过足够长的时间后，解都趋于一个中心化的 Gauss 分布——这是中心极限定理在 PDE 层面的体现。
逐点衰减估计的证明

由热核公式：

$$|u(x,t)| = \left|\int_{-\infty}^{+\infty} \frac{1}{2a\sqrt{\pi t}} e^{-(x-y)^2/(4a^2 t)} \varphi(y)\,\dd y\right| \le \frac{1}{2a\sqrt{\pi t}} \int_{-\infty}^{+\infty} |\varphi(y)|\,\dd y = \frac{\|\varphi\|_{L^1}}{2a\sqrt{\pi t}}.$$

因为 $e^{-(x-y)^2/(4a^2 t)} \le 1$，直接将指数因子放大为 1 即得此简洁而有用的估计。

例题 2.2.4（渐近性态分析 — 原 §2.8 例题合并）

求解热传导方程初值问题：

$$u_t = u_{xx} \quad (x \in \RR, t > 0), \qquad u(x,0) = \begin{cases} 1, & |x| \le 1, \\ 0, & |x| > 1. \end{cases}$$

(1) $u(0,t)$ 的衰减速率是多少？ (2) 解的渐近 Profile 是什么形状？

▶ 详细解答

$a=1$，$\varphi$ 的 $L^1$ 范数 $M = \int_{-1}^1 1\,\dd x = 2$（总热量为 2）。

(1) 衰减速率：由一般估计：

$$|u(x,t)| \le \frac{\|\varphi\|_{L^1}}{2\sqrt{\pi t}} = \frac{1}{\sqrt{\pi t}}.$$

因此 $|u(0,t)|$ 以 $O(t^{-1/2})$ 的代数速率衰减。更精确地，在例题 2.2.2 中已求得 $u(0,t) = \operatorname{erf}(1/(2\sqrt{t}))$。当 $t$ 很大时，$\operatorname{erf}(z) \sim 2z/\sqrt{\pi}$（$z \ll 1$），故 $u(0,t) \sim \frac{1}{\sqrt{\pi t}}$。

(2) 渐近 Profile：由渐近 Profile 定理，当 $t \to +\infty$ 时：

$$u(x,t) \sim \frac{M}{2\sqrt{\pi t}} e^{-x^2/(4t)} = \frac{1}{\sqrt{\pi t}} e^{-x^2/(4t)}.$$

即无论初始是矩形脉冲还是其他有界支集函数，经过足够长的时间后，温度分布都趋近于宽度 $O(\sqrt{t})$、高度 $O(t^{-1/2})$ 的 Gauss 钟形曲线——热核的多次卷积等价于中心极限定理中独立随机变量之和的极限分布。

2.3 有界杆与分离变量法

当杆的长度有限时，边界条件的影响不可忽略，Fourier变换法不再直接适用。这时我们需要分离变量法——它也是求解有界区域上PDE最基础也最重要的方法之一。

核心思想：时间与空间解耦

猜测解可以写成"仅依赖于空间的函数 $\times$ 仅依赖于时间的函数"：$u(x,t) = X(x) \cdot T(t)$。代入方程后，含 $X$ 的项和含 $T$ 的项会分到等号两边——这意味着它们必须等于同一个常数。这样就得到了两个ODE，每个都比原来的PDE简单得多。

这就像把一个大问题拆成两个小问题：找一个"空间形状"和一个"时间衰减规律"，它们的乘积就是热传导的模式。

Dirichlet边值问题

考虑两端恒温为零的杆上的热传导：

$$\begin{cases} u_t = a^2 u_{xx}, & 0 \lt x \lt l,\ t \gt 0, \\ u(0,t) = u(l,t) = 0, & t \gt 0, \\ u(x,0) = \varphi(x), & 0 \le x \le l. \end{cases}$$

第一步：分离变量。设 $u(x,t) = X(x)T(t)$，代入方程：

$$X(x)T'(t) = a^2 X''(x)T(t) \implies \frac{T'(t)}{a^2 T(t)} = \frac{X''(x)}{X(x)} = -\lambda.$$

左边只依赖于 $t$，右边只依赖于 $x$，故两者必等于同一个常数，记作 $-\lambda$（取负号是为了后面得到振荡型的空间解）。

第二步：求解 $X$ 的本征值问题。

$$X''(x) + \lambda X(x) = 0, \quad X(0) = X(l) = 0.$$

分情况讨论：

  • 若 $\lambda \lt 0$，令 $\lambda = -\mu^2$（$\mu \gt 0$）。通解 $X(x) = C_1 e^{\mu x} + C_2 e^{-\mu x}$。由 $X(0)=0$ 得 $C_1+C_2=0$，再由 $X(l)=0$ 得 $C_1(e^{\mu l}-e^{-\mu l})=0$，推得 $C_1=C_2=0$——只有零解。
  • 若 $\lambda = 0$，$X(x) = C_1 + C_2 x$，边界条件给出 $C_1=0$，$C_2 l=0$——也只有零解。
  • 若 $\lambda \gt 0$，令 $\lambda = \omega^2$（$\omega \gt 0$）。通解 $X(x) = C_1\cos\omega x + C_2\sin\omega x$。由 $X(0)=0$ 得 $C_1=0$；由 $X(l)=0$ 得 $\sin\omega l = 0$，即 $\omega l = n\pi$，$n = 1, 2, 3, \ldots$
本征值与本征函数（Dirichlet边值）
$$\lambda_n = \left(\frac{n\pi}{l}\right)^2, \quad X_n(x) = \sin\frac{n\pi x}{l}, \quad n = 1, 2, 3, \ldots$$

$\lambda_n$ 就是杆的"热模态"的本征值，$X_n(x)$ 是相应的本征函数。$n=1$ 对应基模态（温度分布是半个正弦波），$n=2$ 对应第二模态（一个完整的正弦波），以此类推。

第三步：求解 $T$ 的方程。对第 $n$ 个模态：

$$T_n'(t) + a^2\lambda_n T_n(t) = 0 \implies T_n(t) = C_n e^{-a^2\lambda_n t} = C_n e^{-a^2(n\pi/l)^2 t}.$$

注意：$T_n(t)$ 以指数衰减，而且衰减速度正比于 $n^2$——高频模态衰减得非常快！

第四步：叠加。把所有可能模态的贡献加起来（线性方程的叠加原理）：

$$u(x,t) = \sum_{n=1}^{\infty} C_n e^{-a^2(n\pi/l)^2 t}\,\sin\frac{n\pi x}{l}.$$

第五步：由初始条件确定系数。令 $t=0$：

$$\varphi(x) = u(x,0) = \sum_{n=1}^{\infty} C_n \sin\frac{n\pi x}{l}.$$

这是 $\varphi$ 在 $[0,l]$ 上的Fourier正弦级数展开。利用正弦函数的正交性 $\int_0^l \sin\frac{m\pi x}{l}\sin\frac{n\pi x}{l}\,\dd x = \frac{l}{2}\delta_{mn}$：

$$C_n = \frac{2}{l}\int_0^l \varphi(x)\,\sin\frac{n\pi x}{l}\,\dd x, \quad n = 1, 2, 3, \ldots$$
Dirichlet有界杆热传导问题的完整解
$$u(x,t) = \sum_{n=1}^{\infty} C_n e^{-a^2(n\pi/l)^2 t}\,\sin\frac{n\pi x}{l}, \quad C_n = \frac{2}{l}\int_0^l \varphi(x)\sin\frac{n\pi x}{l}\,\dd x.$$

Neumann边值问题

若两端绝热（$u_x(0,t) = u_x(l,t) = 0$），本征值问题变为：

$$X''(x) + \lambda X(x) = 0, \quad X'(0) = X'(l) = 0.$$

类似分析可得：

$$\lambda_n = \left(\frac{n\pi}{l}\right)^2, \quad X_n(x) = \cos\frac{n\pi x}{l}, \quad n = 0, 1, 2, \ldots$$

注意 $n=0$ 给出 $\lambda_0 = 0$，$X_0(x) = 1$（常数温度分布——这是绝热杆的稳态解），$T_0(t) = C_0$（不衰减）。解为：

$$u(x,t) = C_0 + \sum_{n=1}^{\infty} C_n e^{-a^2(n\pi/l)^2 t}\,\cos\frac{n\pi x}{l},$$

其中 $C_0 = \frac{1}{l}\int_0^l \varphi(x)\,\dd x$（初始平均温度），$C_n = \frac{2}{l}\int_0^l \varphi(x)\cos\frac{n\pi x}{l}\,\dd x$（$n \ge 1$）。

物理直觉：高频模态快速死亡

热传导方程分离变量解的核心特征：第 $n$ 个模态的时间因子是 $e^{-a^2(n\pi/l)^2 t}$。衰减指数正比于 $n^2$——

  • $n=1$（基频）的衰减时间尺度为 $\tau_1 = l^2/(a^2\pi^2)$；
  • $n=2$（第二泛音）的衰减时间尺度为 $\tau_2 = l^2/(4a^2\pi^2) = \tau_1/4$——快四倍；
  • $n=10$ 的衰减时间尺度为 $\tau_{10} = \tau_1/100$——快一百倍！

这意味着：复杂精细的温度花样（高频成分）转瞬即逝，很快只剩下最平滑的基频模态。经过足够长的时间后，温度分布几乎全是基频的形态。这就是热传导的"平滑效应"——复杂结构迅速消亡，简单结构长期存留。

与弦振动形成鲜明对比：弦振动中各模态的振幅永不衰减（理想无阻尼），复杂的花样可以永久存在。这是抛物型与双曲型方程的又一根本性差异。

例题 2.3.1

求解 $u_t = u_{xx}$（$0 \lt x \lt \pi$），$u(0,t)=u(\pi,t)=0$，$u(x,0) = \sin x + \frac{1}{2}\sin 3x$。

▶ 详细解答

$a=1$，$l=\pi$。初始条件已经是Fourier正弦级数的形式：$C_1 = 1$，$C_3 = \frac{1}{2}$，其余 $C_n = 0$。

直接写出解：

$$u(x,t) = e^{-t}\sin x + \frac{1}{2}e^{-9t}\sin 3x.$$

注意：$n=3$ 的模态衰减速度为 $e^{-9t}$，是 $n=1$ 模态（$e^{-t}$）的9倍。当 $t$ 稍大时（比如 $t = 0.5$），第一项约为 $0.606\sin x$，第二项仅为约 $0.0055\sin 3x$——高频成分已经几乎消失。

当 $t \to +\infty$：$u(x,t) \to 0$（因为两端恒温为零，热量最终全部流出杆）。

例题 2.3.2

求解 $u_t = u_{xx}$（$0 \lt x \lt 1$），$u_x(0,t)=u_x(1,t)=0$（绝热杆），$u(x,0) = x$。求 $t \to +\infty$ 时解的极限。

▶ 详细解答

$a=1$，$l=1$。Neumann边界条件，本征函数为 $\cos n\pi x$（$n=0,1,2,\ldots$）。

初始平均温度：$C_0 = \int_0^1 x\,\dd x = \frac{1}{2}$。

$n \ge 1$ 时的系数（分部积分）：

$$C_n = 2\int_0^1 x\cos(n\pi x)\,\dd x = 2\left[\frac{x\sin(n\pi x)}{n\pi}\bigg|_0^1 - \frac{1}{n\pi}\int_0^1\sin(n\pi x)\,\dd x\right] = \frac{2}{n^2\pi^2}\big[(-1)^n - 1\big].$$

当 $n$ 为偶数时 $C_n = 0$；当 $n$ 为奇数时 $C_n = -\frac{4}{n^2\pi^2}$。

解为：

$$u(x,t) = \frac{1}{2} - \frac{4}{\pi^2}\sum_{k=0}^{\infty}\frac{e^{-(2k+1)^2\pi^2 t}}{(2k+1)^2}\cos\big((2k+1)\pi x\big).$$

当 $t \to +\infty$：所有含 $e^{-n^2\pi^2 t}$ 的项都衰减为零，只剩下 $C_0 = 1/2$。

$$\lim_{t\to+\infty} u(x,t) = \frac{1}{2}.$$

物理意义：绝热杆中总热量守恒。初始温度从 $0$（$x=0$）线性增长到 $1$（$x=1$），平均值为 $1/2$。经过足够长的时间后，热量均匀分布，整根杆的温度趋于平均值 $1/2$。这就是热平衡。

有界杆解的渐近行为（原 §2.8 内容并入）

有界杆解的大时间行为

对于 $u_t = a^2 u_{xx}$（$0 < x < l$），其解为 $u(x,t) = \sum_{n=1}^\infty C_n e^{-a^2(n\pi/l)^2 t} X_n(x)$。

  1. Dirichlet 边值（$u(0,t) = u(l,t) = 0$）：当 $t \to +\infty$ 时，所有模态都指数衰减到零。主导项为 $n=1$ 模态：$u(x,t) \sim C_1 e^{-a^2(\pi/l)^2 t} \sin\frac{\pi x}{l}$。衰减速率：指数衰减，特征时间尺度 $\tau = l^2/(a^2\pi^2)$。
  2. Neumann 边值（$u_x(0,t) = u_x(l,t) = 0$）：当 $t \to +\infty$ 时，解趋于常数——初始平均温度： $$\lim_{t \to +\infty} u(x,t) = \frac{1}{l}\int_0^l \varphi(x)\,\dd x = C_0.$$ 趋近常数的速率为 $O(e^{-a^2(\pi/l)^2 t})$（由 $n=1$ 模态决定）。
直觉对比：全空间 vs 有界杆
  • 全空间（$\RR$）：热量可以扩散到无穷远处，没有边界将热量"困住"。温度以代数速率 $O(t^{-1/2})$ 衰减到零——缓慢但持续下降。
  • 有界杆（Dirichlet）：两端恒温为零，热量持续从两端流出。温度以指数速率 $O(e^{-\lambda_1 t})$ 衰减——非常快，因为有边界帮助"散热"。
  • 有界杆（Neumann / 绝热）：两端无热流通过，总热量被"困住"在杆内。温度最终趋于一个非零常数——平均值，永不衰减到零。趋近速率同样为指数。

Robin边值问题（第三类边界条件）

在实际传热问题中，杆的端点通常不是理想绝热或恒温，而是通过对流与环境交换热量。Newton 冷却定律指出：端点的热流密度正比于端点温度与环境温度的差。当环境温度为零时，边界条件为：

$$\begin{cases} u_x(0,t) - h\,u(0,t) = 0, & h \ge 0,\\[2pt] u_x(l,t) + H\,u(l,t) = 0, & H \ge 0. \end{cases}$$

参数 $h, H$ 称为换热系数（或 Biot 数）。$h=0$ 退化为 Neumann（绝热），$h \to +\infty$ 退化为 Dirichlet（恒温）。Robin 条件是这两类理想化条件的插值。

分离变量后，空间部分 $X(x)$ 满足的 Sturm-Liouville 问题为：

$$X''(x) + \lambda X(x) = 0, \quad X'(0) - hX(0) = 0, \quad X'(l) + HX(l) = 0.$$

此处 $\lambda \le 0$ 的情况经过边界条件检验只给出零解（证明略），故只需考虑 $\lambda = \omega^2 \gt 0$。通解为：

$$X(x) = A\cos\omega x + B\sin\omega x.$$

由 $X'(0) - hX(0) = 0$：$B\omega - hA = 0$，故 $B = \frac{h}{\omega}A$。取 $A = \omega$（任意常数因子），得：

$$X(x) = \omega\cos\omega x + h\sin\omega x.$$

由 $x = l$ 处的边界条件 $X'(l) + HX(l) = 0$ 给出特征值方程：

Robin边值的本征值方程
$$\boxed{\tan(\omega l) = \frac{\omega(h + H)}{\omega^2 - hH}}$$

此超越方程的根 $\{\omega_n\}_{n=1}^\infty$ 给出本征值 $\lambda_n = \omega_n^2$，对应的本征函数为 $X_n(x) = \omega_n\cos\omega_n x + h\sin\omega_n x$。

本征值的定性分析
  • 当 $h$ 和 $H$ 都很小（接近绝热）：$\tan(\omega l) \approx 0$，故 $\omega_n \approx \frac{(n-1)\pi}{l}$，$n = 1, 2, \ldots$——接近 Neumann 情况。
  • 当 $h$ 和 $H$ 都很大（接近恒温）：$\tan(\omega l) \to \infty$，故 $\omega_n \approx \frac{(n-\frac{1}{2})\pi}{l}$——接近 Dirichlet 情况（$n\pi/l$ 偏移了 $\pi/(2l)$）。
  • 一般情况：本征值落在 Neumann 和 Dirichlet 之间，需要通过数值（或图解法）求解超越方程。

本征函数系 $\{X_n(x)\}$ 在 $[0,l]$ 上仍满足正交性（这是 Sturm-Liouville 理论的一般结论），但不再是简单的正弦或余弦。正交权为 1（标准 Sturm-Liouville 形式）：

$$\int_0^l X_m(x)X_n(x)\,\dd x = 0 \quad (m \neq n).$$

范数 $\|X_n\|^2 = \int_0^l X_n^2(x)\,\dd x$ 可通过积分公式计算（此处从略）。Fourier 系数由下式给出：

$$C_n = \frac{1}{\|X_n\|^2}\int_0^l \varphi(x)X_n(x)\,\dd x.$$

时间部分同前：$T_n(t) = e^{-a^2\omega_n^2 t}$。最终解为：

$$u(x,t) = \sum_{n=1}^{\infty} C_n e^{-a^2\omega_n^2 t}\,X_n(x).$$
例题 2.3.3（Robin边值条件下解的渐近行为）

考虑 $u_t = u_{xx}$（$0 < x < 1$），Robin 边界条件 $u_x(0,t) - u(0,t) = 0$，$u_x(1,t) + u(1,t) = 0$。不具体求解，分析 $t \to +\infty$ 时解的渐近行为。

▶ 详细解答

$h = H = 1$，本征值方程为 $\tan\omega = \frac{2\omega}{\omega^2 - 1}$。

求最小本征值 $\omega_1$：用图解法。$y = \tan\omega$ 与 $y = 2\omega/(\omega^2-1)$ 的交点。$\omega=0$ 时右端为 0；$\omega$ 从 0 增大时右端单调递增直至 $\omega=1$ 处的渐近线。$\tan\omega$ 从 0 开始增长。两条曲线在 $(0, \pi/2)$ 内有唯一交点，数值约为 $\omega_1 \approx 1.307$。

对应的衰减指数为 $e^{-\omega_1^2 t} \approx e^{-1.708 t}$。

对比：

  • 若为 Dirichlet（恒温零度）：$\omega_1 = \pi \approx 3.142$，衰减 $e^{-\pi^2 t} \approx e^{-9.870 t}$。
  • 若为 Neumann（绝热）：$\omega_1 = 0$，温度趋于非零常数。
  • Robin（对流散热）：$e^{-1.708 t}$——衰减速度介于 Neumann（不衰减）和 Dirichlet（快速衰减）之间。换热系数越小（越接近绝热），衰减越慢；系数越大（越接近恒温），衰减越快。这是符合物理直觉的。

当 $t \to +\infty$：$u(x,t) \sim C_1 e^{-1.708 t}(\cos 1.307x + \frac{1}{1.307}\sin 1.307x) \to 0$——对流散热导致杆最终冷却到环境温度（零度），但比两端直接接触零度热源的 Dirichlet 情况慢得多。

2.4 半无界问题：延拓法

现在考虑半无界区域 $x \gt 0$ 上的热传导。物理场景是：一根从 $x=0$ 向右无限延伸的杆，左端有某种边界条件。

核心技巧：延拓——"想象杆的另一半"

半无界问题的诀窍是：把定义域从 $x \ge 0$ 扩展到整个 $\RR$，使得边界条件在 $x=0$ 处自动满足。这需要根据边界条件的类型，把初始数据对称地（或反对称地）延拓到负半轴。然后就可以用全空间的Fourier变换法（即热核卷积公式）来解了。

物理直觉：奇延拓相当于在 $x \lt 0$ 处"镜像"了一个温度分布，但符号相反——$x=0$ 处正负相消，温度自动为零。偶延拓相当于在负半轴放了一个完全相同的温度分布——$x=0$ 处左右对称，热流自动为零。

Dirichlet条件：奇延拓

Dirichlet半无界热传导问题
$$\begin{cases} u_t = a^2 u_{xx}, & x \gt 0,\ t \gt 0, \\ u(x,0) = \varphi(x), & x \ge 0, \\ u(0,t) = 0, & t \gt 0. \end{cases}$$

将 $\varphi(x)$ 奇延拓到全空间：

$$\tilde{\varphi}(x) = \begin{cases} \varphi(x), & x \ge 0, \\ -\varphi(-x), & x \lt 0. \end{cases}$$

因为 $\tilde{\varphi}$ 是奇函数，全空间解在 $x=0$ 处自动为零。代入热核公式并利用奇函数的性质化简：

Dirichlet半无界问题的解
$$u(x,t) = \frac{1}{2a\sqrt{\pi t}}\int_0^{+\infty} \varphi(y)\left[e^{-\frac{(x-y)^2}{4a^2 t}} - e^{-\frac{(x+y)^2}{4a^2 t}}\right]\dd y.$$

括号中第一项是"直接传播"（热量从 $y$ 直接传到 $x$），第二项是"镜像反射"（热量从镜像点 $-y$ 传到 $x$，带负号，代表边界 $x=0$ 的吸收效应）。

Neumann条件：偶延拓

若边界条件为 $u_x(0,t) = 0$（绝热边界），则对 $\varphi$ 做偶延拓：

$$\tilde{\varphi}(x) = \begin{cases} \varphi(x), & x \ge 0, \\ \varphi(-x), & x \lt 0. \end{cases}$$

偶延拓保证 $u_x(0,t) = 0$（偶函数在原点导数为零）。解为：

$$u(x,t) = \frac{1}{2a\sqrt{\pi t}}\int_0^{+\infty} \varphi(y)\left[e^{-\frac{(x-y)^2}{4a^2 t}} + e^{-\frac{(x+y)^2}{4a^2 t}}\right]\dd y.$$

注意括号中第二项是加号——镜像点对 $x$ 的热贡献与原点相同（绝热边界像一面镜子）。

常见错误：混淆延拓方向

许多初学者会把延拓方向搞反：Dirichlet条件对应奇延拓（符号相反），Neumann条件对应偶延拓（符号相同）。记忆口诀："定温奇，绝热偶"。物理上，定温边界 $u=0$ 要求镜像源是"冷源"（负号），以抵消热量保持边界为零度；绝热边界 $u_x=0$ 要求镜像源是"热源"（正号），以对称分布保证无净热流通过边界。

例题 2.4.1

求解 $u_t = u_{xx}$（$x \gt 0$），$u(x,0) = 1$，$u(0,t) = 0$。

▶ 详细解答

$a=1$，奇延拓 $\tilde{\varphi}(x) = \operatorname{sgn}(x)$（$x \gt 0$ 时为 $1$，$x \lt 0$ 时为 $-1$）。

$$u(x,t) = \frac{1}{2\sqrt{\pi t}}\int_0^{+\infty} \left[e^{-\frac{(x-y)^2}{4t}} - e^{-\frac{(x+y)^2}{4t}}\right]\dd y.$$

第一项令 $z = (y-x)/(2\sqrt{t})$，第二项令 $z = (y+x)/(2\sqrt{t})$：

$$u(x,t) = \frac{1}{\sqrt{\pi}}\left[\int_{-x/(2\sqrt{t})}^{+\infty} e^{-z^2}\dd z - \int_{x/(2\sqrt{t})}^{+\infty} e^{-z^2}\dd z\right].$$

由误差函数的对称性：$\int_{a}^{+\infty} e^{-z^2}\dd z = \frac{\sqrt{\pi}}{2}\big(1 - \operatorname{erf}(a)\big)$，且 $\operatorname{erf}(-a) = -\operatorname{erf}(a)$：

$$u(x,t) = \frac{1}{\sqrt{\pi}}\int_{-x/(2\sqrt{t})}^{x/(2\sqrt{t})} e^{-z^2}\dd z = \operatorname{erf}\!\left(\frac{x}{2\sqrt{t}}\right).$$

验证：$u(0,t) = \operatorname{erf}(0) = 0$（边界条件满足）。当 $t \to 0^+$（$x \gt 0$）：$\operatorname{erf}(+\infty) = 1$（初始条件满足）。当 $t \to +\infty$：$\operatorname{erf}(0^+) = 0$（边界冷却效应最终使整根杆降温到零）。

这个解揭示了热传导的一个关键特征：相似变量 $x/(2\sqrt{t})$ 自然地出现——温度只依赖于 $x/\sqrt{t}$ 这个组合。这意味着在 $(x,t)$ 平面上，温度沿抛物线 $x \propto \sqrt{t}$ 是常数。

例题 2.4.2

求解 $u_t = u_{xx}$（$x \gt 0$），$u(x,0) = 0$（初始温度为零），$u_x(0,t) = -q$（$q \gt 0$ 为常数，即左端有恒定热流输入）。

▶ 详细解答

这是一个非齐次Neumann边界条件问题。技巧是令 $v(x,t) = u(x,t) + qx$，则：

  • $v_t = u_t = u_{xx} = v_{xx}$（$v$ 满足同样的热传导方程）；
  • $v_x(0,t) = u_x(0,t) + q = 0$（变为齐次Neumann条件）；
  • $v(x,0) = qx$（新的初始条件）。

对 $v$ 用偶延拓法求解。$\tilde{\varphi}(x) = q|x|$，代入偶延拓解公式：

$$v(x,t) = \frac{q}{2\sqrt{\pi t}}\int_0^{+\infty} y\left[e^{-(x-y)^2/(4t)} + e^{-(x+y)^2/(4t)}\right]\dd y.$$

故 $u(x,t) = v(x,t) - qx$。通过配方和误差函数可算出具体表达式。当 $t$ 很大时：

$$u(x,t) \approx \frac{2q\sqrt{t}}{\sqrt{\pi}}e^{-x^2/(4t)} - qx + \cdots$$

物理意义：热量从 $x=0$ 处不断输入（恒定热流 $q$），向右方扩散。在 $x=0$ 处温度随时间以 $u(0,t) \approx 2q\sqrt{t}/\sqrt{\pi}$ 的速度上升。

2.5 极值原理

极值原理是热传导方程最深刻的定性性质之一。它不只是数学上的优雅定理——它从根本上揭示了热传导的不可逆性和热力学第二定律的数学表达。

直觉建立：温度峰值只能来自过去

想象一根铁杆正在冷却。在某一时刻 $t \gt 0$，杆上会不会突然出现一个比初始温度更高、也比两端温度更高的"热点"？直觉告诉我们：不会。热量只能从高温流向低温，不可能自发集中。杆上任意点在任意时刻的温度，不可能超过初始时刻和边界上的最高温度。

这就是极值原理的物理直觉——温度的最大值和最小值由"过去"（初始条件）和"边界"决定，不会在内部自发产生新的极值。

弱极值原理

抛物边界

考虑时空区域 $\bar{D}_T = [0,l] \times [0,T]$。其抛物边界 $\Gamma$ 是指：

$$\Gamma = \{(x,0) : 0 \le x \le l\} \cup \{(0,t) : 0 \le t \le T\} \cup \{(l,t) : 0 \le t \le T\}.$$

即初始时刻的线段加左右两侧的竖直线段——不包括顶部 $t = T$（这是未来时刻，不是边界）。

弱极值原理

设 $u(x,t)$ 在 $\bar{D}_T$ 上连续，在 $D_T = (0,l) \times (0,T]$ 内满足热传导方程 $u_t = a^2 u_{xx}$。则 $u$ 在 $\bar{D}_T$ 上的最大值和最小值都在抛物边界 $\Gamma$ 上取到。即：

$$\max_{(x,t)\in\bar{D}_T} u(x,t) = \max_{(x,t)\in\Gamma} u(x,t), \quad \min_{(x,t)\in\bar{D}_T} u(x,t) = \min_{(x,t)\in\Gamma} u(x,t).$$
证明思路（辅助函数法）

只证最大值部分（最小值对 $-u$ 应用同样推理）。设 $M = \max_\Gamma u$。

假设存在内部点 $(x_0, t_0) \in D_T$（$t_0 \gt 0$）使 $u(x_0, t_0) \gt M$。我们希望推出矛盾。

关键技巧：引入辅助函数 $v(x,t) = u(x,t) - \varepsilon t$，其中 $\varepsilon \gt 0$ 是一个很小的正数。则 $v$ 满足：

$$v_t - a^2 v_{xx} = u_t - a^2 u_{xx} - \varepsilon = -\varepsilon \lt 0.$$

若 $v$ 在内部某点 $(x_1, t_1) \in D_T$ 达到最大值，则在最大值点处有 $v_t \ge 0$（若 $t_1 \lt T$ 则 $v_t = 0$；若 $t_1 = T$ 也可能 $v_t \ge 0$）且 $v_{xx} \le 0$。从而：

$$v_t - a^2 v_{xx} \ge 0.$$

但这与 $v_t - a^2 v_{xx} = -\varepsilon \lt 0$ 矛盾！因此 $v$ 的最大值必在 $\Gamma$ 上取到。

于是对任意 $(x,t) \in \bar{D}_T$：

$$u(x,t) = v(x,t) + \varepsilon t \le \max_\Gamma v + \varepsilon T \le \max_\Gamma u + \varepsilon T.$$

令 $\varepsilon \to 0^+$，即得 $u(x,t) \le \max_\Gamma u$。$\square$

极值原理的直接推论

推论1：初边值问题解的唯一性

热传导方程初边值问题的解，如果存在，则唯一。

唯一性证明

设 $u_1, u_2$ 是同一问题的两个解，令 $w = u_1 - u_2$。则 $w$ 满足：

$$w_t = a^2 w_{xx}, \quad w(x,0) = 0, \quad w(0,t) = w(l,t) = 0.$$

由极值原理，$w$ 的最大值和最小值都在抛物边界 $\Gamma$ 上取到。在 $\Gamma$ 上 $w = 0$，故 $w \equiv 0$，即 $u_1 = u_2$。$\square$

推论2：最大模估计（解的稳定性）

若 $|u(x,0)| \le M_0$，$|u(0,t)| \le M_1$，$|u(l,t)| \le M_2$，则：

$$|u(x,t)| \le \max(M_0, M_1, M_2), \quad \forall (x,t) \in \bar{D}_T.$$

这意味着解连续依赖于初边值数据——输入数据的微小变化不会导致解的巨大变化。这是Hadamard意义下适定性（well-posedness）的重要组成部分。

极值原理的深刻含义：热传导的不可逆性

极值原理不仅仅是唯一性的工具——它揭示了热传导本质上的不可逆性：

  • 温度单调趋于均匀：随着时间推移，最大温度只减不增，最小温度只增不减——温度分布不断"变平"，最终趋于均匀。
  • 时间反演的不适定性：如果尝试逆转时间 $t \to -t$，方程变为 $u_t = -a^2 u_{xx}$（反向热方程）。对其应用极值原理会发现：解对初始数据极其敏感，微小的测量误差在反向演化中会被指数放大。这是一个经典的不适定问题（ill-posed problem）。
  • 热力学第二定律的数学回响：极值原理是热力学第二定律（孤立系统的熵只增不减）在PDE层面的精确表达——热量的自发扩散是不可逆的，不可能在不消耗外功的情况下让热量从低温自发流向高温。

非齐次热传导方程的极值原理

非齐次热传导方程

考虑含源项的热传导方程：

$$u_t = a^2 u_{xx} + f(x,t), \quad (x,t) \in D_T = (0,l) \times (0,T].$$

方程是非齐次的——$f(x,t)$ 代表内部热源（$f>0$ 表示有热量输入，$f<0$ 表示有热量输出）。

非齐次热方程的弱极值原理

设 $u \in C(\bar{D}_T) \cap C^2(D_T)$ 满足 $u_t = a^2 u_{xx} + f(x,t)$。则：

  • 若 $f(x,t) \le 0$ 在 $D_T$ 内恒成立，则 $u$ 在 $\bar{D}_T$ 上的最大值在抛物边界 $\Gamma$ 上取到；
  • 若 $f(x,t) \ge 0$ 在 $D_T$ 内恒成立，则 $u$ 在 $\bar{D}_T$ 上的最小值在抛物边界 $\Gamma$ 上取到。

换句话说：$f \le 0$ 时最大值原理成立，$f \ge 0$ 时最小值原理成立。

证明思路

只证 $f \le 0$ 时最大值原理的情况。

设 $v(x,t) = u(x,t) - \varepsilon t$，$\varepsilon > 0$。则：

$$v_t - a^2 v_{xx} = (u_t - a^2 u_{xx}) - \varepsilon = f(x,t) - \varepsilon \le -\varepsilon < 0.$$

若 $v$ 在内点 $(x_1, t_1) \in D_T$ 取最大值，则 $v_t \ge 0$，$v_{xx} \le 0$，从而 $v_t - a^2 v_{xx} \ge 0$。这与 $v_t - a^2 v_{xx} < 0$ 矛盾。

因此 $v$ 的最大值在 $\Gamma$ 上取到。令 $\varepsilon \to 0^+$，得 $u$ 的最大值也在 $\Gamma$ 上取到。

若 $f \le 0$ 不满足（比如 $f > 0$），则辅助函数 $v_t - a^2 v_{xx} = f - \varepsilon$ 的符号不确定，上述论证失效——这就是为什么极值原理的成立依赖于 $f$ 的符号条件。

例题 2.5.3（考试重点：非齐次方程极值原理条件）

填空题：对于非齐次热传导方程 $u_t = a^2 u_{xx} + f(x,t)$，若源项 $f$ 满足______，则最大值原理成立；若源项 $f$ 满足______，则最小值原理成立。

▶ 详细解答

答案：$f(x,t) \le 0$（最大值原理）；$f(x,t) \ge 0$（最小值原理）。

物理理解：

  • $f \le 0$ 表示内部有"冷源"（吸热），热量被持续抽走，温度不会自发升高——因此最大值只能来自初始或边界；
  • $f \ge 0$ 表示内部有"热源"（放热），热量被持续注入，温度不会自发降低——因此最小值只能来自初始或边界。

如果 $f$ 的符号不满足上述条件（如 $f$ 可正可负），则不能直接用极值原理推出解的先验界。

例题 2.5.1

设 $u$ 满足 $u_t = u_{xx}$（$0 \lt x \lt \pi$，$t \gt 0$），$u(0,t) = u(\pi,t) = 0$，$u(x,0) = \sin x + \frac{1}{2}\sin 2x$。求 $\displaystyle\max_{x\in[0,\pi],\,t\ge 0} u(x,t)$ 以及取到最大值的位置。

▶ 详细解答

由极值原理，最大值在抛物边界 $\Gamma$（即 $t=0$ 或 $x=0$ 或 $x=\pi$）上取到。在 $x=0$ 和 $x=\pi$ 处 $u=0$，故最大值必在 $t=0$ 时取到。

求 $f(x) = \sin x + \frac{1}{2}\sin 2x = \sin x + \sin x\cos x = \sin x(1 + \cos x)$ 在 $[0,\pi]$ 上的最大值。

$f'(x) = \cos x + \cos 2x = \cos x + (2\cos^2 x - 1) = 2\cos^2 x + \cos x - 1 = 0$。

解得 $\cos x = \frac{1}{2}$（$x = \pi/3$）或 $\cos x = -1$（$x = \pi$，但 $f(\pi) = 0$）。

$f(\pi/3) = \sin\frac{\pi}{3} + \frac{1}{2}\sin\frac{2\pi}{3} = \frac{\sqrt{3}}{2} + \frac{1}{2}\cdot\frac{\sqrt{3}}{2} = \frac{3\sqrt{3}}{4}$。

端点：$f(0) = 0$，$f(\pi) = 0$。

$$\max u = \frac{3\sqrt{3}}{4} \approx 1.299,$$

在 $(x,t) = (\pi/3,\,0)$ 处取到。由极值原理，对任意 $t \gt 0$ 都有 $u(x,t) \le 3\sqrt{3}/4$。

事实上用分离变量法可求得 $u(x,t) = e^{-t}\sin x + \frac{1}{2}e^{-4t}\sin 2x$，确实处处不大于 $3\sqrt{3}/4$，且随 $t$ 增大严格递减。

例题 2.5.2

利用极值原理证明：若 $u$ 满足 $u_t = u_{xx}$（$0 \lt x \lt l$，$t \gt 0$），$u(x,0) = 0$，$u(0,t) = 0$，$|u(l,t)| \le M$，则 $|u(x,t)| \le M$ 对所有 $(x,t)$ 成立。

▶ 详细解答

在抛物边界 $\Gamma$ 上：

  • $t=0$：$u(x,0) = 0$，$|u| = 0 \le M$；
  • $x=0$：$u(0,t) = 0$，$|u| = 0 \le M$；
  • $x=l$：$|u(l,t)| \le M$。

因此在 $\Gamma$ 上 $|u| \le M$。由极值原理（分别对 $u$ 和 $-u$ 应用），在 $\bar{D}_T$ 上也有 $|u| \le M$。$\square$

这个结论的物理含义是：如果杆初始温度为零，左端保持零度，右端温度不超过 $M$，则整根杆的温度永远不会超过 $M$——热量不会"放大"，只会"传导"。

2.6 解的正则性：即时光滑化

这是热传导方程最"神奇"的性质之一：无论初始温度分布多么粗糙（哪怕有尖角、跳跃、甚至处处不可微），只要 $t \gt 0$，解就立即变成无穷次可微的。热传导像一台"光滑化机器"，瞬间抹平一切不规则。

物理直觉：扩散抹平一切

把一滴墨水滴入清水中。初始时刻墨水集中在一小块区域，边界清晰（甚至尖锐）。但仅仅一瞬间后，墨水开始扩散，边界变得模糊——再也没有清晰的边缘了。随着时间的推移，墨水分布变得越来越平滑。

这就是热传导（扩散）的"即时光滑化"效应在物理世界中的体现。数学上，这种效应由热核的Gauss型光滑性保证——Gauss函数是"最光滑"的函数之一（无穷次可微且实解析），热核卷积将这种光滑性"传递"给了任意初始数据。

无穷次可微性定理

解的光滑性定理

设 $\varphi(x)$ 在 $\RR$ 上有界连续，$u(x,t) = \int_{-\infty}^{+\infty} K(x-y,t)\,\varphi(y)\,\dd y$ 是热传导方程Cauchy问题的解。则：

  1. 对每个 $t \gt 0$，$u(x,t)$ 是 $x$ 的 $C^\infty$ 函数（任意阶导数存在且连续）；
  2. $u(x,t)$ 在区域 $\RR \times (0,+\infty)$ 上是 $C^\infty$ 的（对 $x$ 和 $t$ 的任意阶偏导数都存在且连续）；
  3. 进一步，在适当条件下（如 $\varphi$ 有界且满足一定的增长条件），$u(x,t)$ 对 $x$ 是实解析的——在每一点附近可以展开为收敛的幂级数。
证明思路

核心在于热核 $K(x-y,t) = \frac{1}{2a\sqrt{\pi t}}e^{-(x-y)^2/(4a^2 t)}$ 的Gauss型结构。对任意偏导数 $\partial_x^m\partial_t^n K(x-y,t)$：

  • 每求一次导会多出一个关于 $(x-y)$ 的多项式因子；
  • 但 $e^{-(x-y)^2/(4a^2 t)}$ 的指数衰减远远快于任何多项式的增长；
  • 因此 $\partial_x^m\partial_t^n K(x-y,t)$ 关于 $y$ 仍然是指数衰减的，在 $\RR$ 上绝对可积。

于是由控制收敛定理，可以在积分号下求导：

$$\partial_x^m\partial_t^n u(x,t) = \int_{-\infty}^{+\infty} \partial_x^m\partial_t^n K(x-y,t)\,\varphi(y)\,\dd y.$$

由于 $|\varphi|$ 有界，右边的积分对任意 $m,n$ 一致收敛，得到的表达式连续。因此 $u \in C^\infty(\RR \times (0,+\infty))$。$\square$

与波动方程的鲜明对比：光滑性的根本分歧

这是抛物型与双曲型方程最深刻的差异之一：

  • 波动方程：解的光滑性与初始数据完全相同。如果初始位移 $\varphi \in C^2$，则 $u \in C^2$——但不更光滑。如果 $\varphi$ 仅连续而不可微，$u$ 也仅连续。波动方程忠实地传播奇异性——初始数据中的任何"粗糙"特征会沿着特征线传播而不被抹平（想想d'Alembert公式：$u(x,t) = [\varphi(x+t) + \varphi(x-t)]/2$，解的光滑性显然不会超过 $\varphi$ 的光滑性）。
  • 热传导方程：无论初始数据多么粗糙（哪怕只是有界可测），$t \gt 0$ 时解立即 $C^\infty$。有限传播速度 vs 无穷传播速度 导致了这一根本分歧：波的信号以有限速度沿特征线传播，保留了初始的"形状信息"；热的信号以无穷速度扩散（Gauss核的拖尾延伸至无穷远处），在传播过程中所有尖锐特征都被"平均化"了。
例题 2.6.1

设初始温度 $\varphi(x) = |x|$（在 $x=0$ 处不可微）。验证 $t \gt 0$ 时热传导方程的解 $u(x,t)$ 在 $x=0$ 处可微，并求 $u_x(0,t)$。

▶ 详细解答

取 $a=1$，由热核公式：

$$u(x,t) = \frac{1}{2\sqrt{\pi t}}\int_{-\infty}^{+\infty} |y|\,e^{-(x-y)^2/(4t)}\,\dd y.$$

在积分号下对 $x$ 求导：

$$u_x(x,t) = \frac{1}{2\sqrt{\pi t}}\int_{-\infty}^{+\infty} |y| \cdot \frac{-(x-y)}{2t}\,e^{-(x-y)^2/(4t)}\,\dd y.$$

在 $x=0$ 处：

$$u_x(0,t) = \frac{1}{4t\sqrt{\pi t}}\int_{-\infty}^{+\infty} y\,|y|\,e^{-y^2/(4t)}\,\dd y.$$

被积函数 $y|y|e^{-y^2/(4t)}$ 是奇函数（因为 $y|y|$ 是奇函数），在整个实轴上的积分为零。故：

$$u_x(0,t) = 0.$$

初始函数 $\varphi(x) = |x|$ 在 $x=0$ 处有一个"尖点"（左导数为 $-1$，右导数为 $+1$），是不可微的。但 $t \gt 0$ 时解在 $x=0$ 处变得光滑可微——热核的卷积瞬间"磨平"了尖点。

可以继续求高阶导数，全部存在——验证了即时光滑化定理。

例题 2.6.2

讨论热传导方程解的解析性：证明 $u(x,t) = e^{-t}\cos x$ 关于 $x$ 是实解析的。

▶ 详细解答

首先验证 $u_t = -e^{-t}\cos x = u_{xx}$——$u$ 确实是热传导方程的解。

关于 $x$ 展开 $\cos x$：

$$\cos x = \sum_{n=0}^{\infty} \frac{(-1)^n x^{2n}}{(2n)!}, \quad \forall x \in \RR.$$

该幂级数的收敛半径为 $+\infty$（由比值判别法：$\lim_{n\to\infty} |a_{n+1}/a_n| = \lim_{n\to\infty} (2n+1)(2n+2)^{-1}|x|^2 = 0$）。因此 $u(x,t) = e^{-t}\cos x$ 在每一点 $x \in \RR$ 附近都可以展开为收敛的幂级数——它是实解析的。

事实上，热传导方程任意 $t \gt 0$ 时的解都至少是解析的（如果初始数据满足适当的条件）。这是热传导方程独有的"解析化"（analytic smoothing）效应。

可视化 · 热传导方程初值问题的解

观察热传导方程 $u_t = u_{xx}$ 的解随时间的演化。蓝色虚线为初始温度分布 $\varphi(x)$，红色实线为当前时刻的解 $u(x,t)$。注意温度峰值的衰减和分布的展宽——这是扩散的本质特征。

$u(x,t) = \displaystyle\int_{-\infty}^{+\infty} \frac{1}{2\sqrt{\pi t}} e^{-(x-\xi)^2/(4t)} \varphi(\xi)\,\dd\xi$（热核卷积）

本章总结：热传导方程的知识结构

$$\begin{array}{c} \text{物理推导（Fourier定律 + 能量守恒）} \\ \downarrow \\ u_t = a^2 u_{xx} \quad \text{（抛物型方程）} \\ \downarrow \\ \begin{array}{ccc} \text{Fourier变换法} & \quad & \text{分离变量法} \\ \text{（全空间Cauchy问题）} & & \text{（有界杆 $[0,l]$）} \\ \downarrow & & \downarrow \\ \text{热核} \; K(x,t) = \frac{e^{-x^2/(4a^2 t)}}{2a\sqrt{\pi t}} & & \text{本征展开} \; \sum C_n e^{-a^2\lambda_n t}X_n(x) \\ \downarrow & & \downarrow \\ \text{渐近分析：全空间代数衰减 } O(t^{-1/2}) & & \text{渐近分析：有界杆指数衰减 } O(e^{-\lambda_1 t}) \end{array} \\ \downarrow \\ \text{延拓法（半无界问题）} \quad \longleftrightarrow \quad \text{极值原理} \\ \downarrow \qquad \qquad \qquad \qquad \qquad \downarrow \\ \text{奇延拓(Dirichlet) / 偶延拓(Neumann)} \qquad \text{唯一性·稳定性·不可逆性} \\ \downarrow \\ \text{解的正则性：即时光滑化（$C^\infty$，解析性）} \end{array}$$

核心方法：Fourier变换法、分离变量法、延拓法（反射法）、极值原理（含非齐次情况）、渐近分析方法。

核心概念：热核（基本解）、本征值与本征函数、抛物边界、误差函数。

物理直觉：热量从高温流向低温，温度曲率驱动温度变化率。初始的尖锐特征瞬间被扩散抹平，温度分布单调趋于均匀。这一不可逆过程是热力学第二定律的数学体现。

与波动方程的根本区别：

  • 抛物型 vs 双曲型：一阶时间导数 vs 二阶时间导数；
  • 无穷传播速度 vs 有限传播速度：热量瞬间影响全空间 vs 波沿特征线以有限速度传播；
  • 即时光滑化 vs 忠实传播奇性：初始粗糙被瞬间抹平 vs 解的奇性沿特征线传播；
  • 不可逆 vs 可逆：方程在时间反演下改变类型 vs 方程在时间反演下不变。

习题 2.1 · Fourier变换法求解Cauchy问题 基础 考试热点

用Fourier变换法求解以下热传导方程Cauchy问题（$a=1$）：

$$u_t = u_{xx} \quad (x \in \RR,\ t > 0), \qquad u(x,0) = e^{-x^2/2}.$$

给出解的显式表达式，并验证当 $t \to 0^+$ 时解趋于初始条件。

▶ 参考答案

对 $\varphi(x)=e^{-x^2/2}$ 做 Fourier 变换：

$$\hat{\varphi}(\xi) = \int_{-\infty}^{+\infty} e^{-x^2/2} e^{-i\xi x}\dd x = \sqrt{2\pi}\,e^{-\xi^2/2}.$$

则 $\hat{u}(\xi,t) = \sqrt{2\pi}\,e^{-\xi^2/2} e^{-\xi^2 t} = \sqrt{2\pi}\,e^{-\xi^2(t+1/2)}$。

逆变换：$u(x,t) = \frac{1}{2\pi} \int_{-\infty}^{+\infty} \sqrt{2\pi}\,e^{-\xi^2(t+1/2)} e^{i\xi x}\dd\xi$。

配方后得：

$$u(x,t) = \frac{1}{\sqrt{2t+1}} \exp\!\left(-\frac{x^2}{2(2t+1)}\right).$$

验证：$t=0$ 时 $u(x,0) = e^{-x^2/2} = \varphi(x)$。直接求导可验证 $u_t = u_{xx}$。

习题 2.2 · 分离变量法求解有界杆热传导 基础 考试热点

求解以下Dirichlet有界杆热传导问题（$a=1$，$l=\pi$）：

$$\begin{cases} u_t = u_{xx}, & 0 < x < \pi,\ t > 0, \\ u(0,t) = u(\pi,t) = 0, & t > 0, \\ u(x,0) = x(\pi - x), & 0 \le x \le \pi. \end{cases}$$

给出解的Fourier级数表达式，并求 $t \to +\infty$ 时解的渐近主项。

▶ 参考答案

分离变量：$u(x,t) = \sum_{n=1}^{\infty} C_n e^{-n^2 t} \sin(nx)$。

计算系数：

$$C_n = \frac{2}{\pi}\int_0^\pi x(\pi-x)\sin(nx)\,\dd x.$$

两次分部积分可得：

$$C_n = \begin{cases} \dfrac{8}{\pi n^3}, & n \text{ 为奇数}, \\ 0, & n \text{ 为偶数}. \end{cases}$$

因此解为：

$$u(x,t) = \frac{8}{\pi}\sum_{k=0}^{\infty} \frac{e^{-(2k+1)^2 t}}{(2k+1)^3}\sin\big((2k+1)x\big).$$

当 $t \to +\infty$，主导项为 $n=1$：

$$u(x,t) \sim \frac{8}{\pi} e^{-t} \sin x.$$

衰减率为指数衰减 $O(e^{-t})$。

习题 2.3 · 半无界问题与延拓法 基础

求解半无界Dirichlet热传导问题（$a=1$）：

$$\begin{cases} u_t = u_{xx}, & x > 0,\ t > 0, \\ u(0,t) = 0, & t > 0, \\ u(x,0) = e^{-x}, & x \ge 0. \end{cases}$$

用奇延拓法求解，并将解用误差函数表示。

▶ 参考答案

奇延拓：$\tilde{\varphi}(x) = e^{-|x|} \cdot \operatorname{sgn}(x)$。

由Dirichlet半无界解公式：

$$u(x,t) = \frac{1}{2\sqrt{\pi t}}\int_0^{+\infty} e^{-y}\left[e^{-\frac{(x-y)^2}{4t}} - e^{-\frac{(x+y)^2}{4t}}\right]\dd y.$$

配方后分离积分：

$$\begin{aligned} u(x,t) &= \frac{1}{2\sqrt{\pi t}} e^{t-x} \int_0^{\infty} e^{-\frac{(y-x+2t)^2}{4t}}\dd y - \frac{1}{2\sqrt{\pi t}} e^{t+x} \int_0^{\infty} e^{-\frac{(y+x+2t)^2}{4t}}\dd y \\ &= \frac{1}{2} e^{t-x}\!\left[1 - \operatorname{erf}\!\left(\frac{2t-x}{2\sqrt{t}}\right)\right] - \frac{1}{2} e^{t+x}\!\left[1 - \operatorname{erf}\!\left(\frac{2t+x}{2\sqrt{t}}\right)\right]. \end{aligned}$$

验证：$x=0$ 时两项抵消，$u(0,t)=0$；$t \to 0^+$ 时可恢复 $u(x,0)=e^{-x}$。

习题 2.4 · 极值原理与唯一性 证明 考试热点

(1) 写出并证明热传导方程初边值问题的弱极值原理（辅助函数法）。

(2) 利用极值原理证明热传导方程初边值问题解的唯一性。

(3) 若源项 $f(x,t) \le 0$，证明最大值原理仍成立；若 $f(x,t) \ge 0$，证明最小值原理成立。

▶ 参考答案

(1) 弱极值原理（齐次情形）：

设 $u \in C(\bar{D}_T) \cap C^2(D_T)$ 满足 $u_t = a^2 u_{xx}$。令抛物边界 $\Gamma = \{(x,0): 0 \le x \le l\} \cup \{(0,t): 0 \le t \le T\} \cup \{(l,t): 0 \le t \le T\}$。

证明：设 $M = \max_\Gamma u$。反设存在内点使 $u > M$。引入辅助函数 $v = u - \varepsilon t$（$\varepsilon > 0$），则 $v_t - a^2 v_{xx} = -\varepsilon < 0$。若 $v$ 在内点取极大，则 $v_t \ge 0$，$v_{xx} \le 0$，从而 $v_t - a^2 v_{xx} \ge 0$，矛盾。故 $v$ 的最大值在 $\Gamma$ 上，令 $\varepsilon \to 0^+$ 即得 $u \le M$。

(2) 唯一性：设 $u_1, u_2$ 为两个解，令 $w = u_1 - u_2$。则 $w$ 满足齐次方程和零初边值条件。由极值原理，$w$ 的最大值和最小值都在抛物边界上取到——均为零，故 $w \equiv 0$，即 $u_1 = u_2$。

(3) 非齐次情形：证 $f \le 0$ 时最大值原理。令 $v = u - \varepsilon t$，则 $v_t - a^2 v_{xx} = f - \varepsilon \le -\varepsilon < 0$，后续与 (1) 相同。$f \ge 0$ 时最小值原理对 $-u$ 应用同理可得。

习题 2.5 · 渐近性态与衰减速率 综合

考虑热传导方程Cauchy问题（$a=1$）：

$$u_t = u_{xx}\ (x \in \RR,\ t > 0), \qquad u(x,0) = \begin{cases} \cos x, & |x| \le \pi/2, \\ 0, & |x| > \pi/2. \end{cases}$$

(1) 求总热量 $M = \int_{-\infty}^{+\infty} \varphi(x)\,\dd x$。

(2) 利用渐近Profile定理，写出 $t \to +\infty$ 时解的渐近表达式（首项）。

(3) 比较全空间衰减速率 $O(t^{-1/2})$ 与有界杆Dirichlet问题的指数衰减速率 $O(e^{-\lambda_1 t})$，解释为何边界条件对衰减速率有关键影响。

▶ 参考答案

(1) 总热量：

$$M = \int_{-\pi/2}^{\pi/2} \cos x\,\dd x = \sin x\Big|_{-\pi/2}^{\pi/2} = 2.$$

(2) 渐近Profile：当 $t \to +\infty$，

$$u(x,t) \sim \frac{M}{2\sqrt{\pi t}} e^{-x^2/(4t)} = \frac{1}{\sqrt{\pi t}} e^{-x^2/(4t)}.$$

无论初始分布如何复杂，经足够长时间后都趋于中心化Gauss分布。

(3) 边界影响分析：

  • 全空间（$\RR$）无边界：热量只能通过向无穷远处扩散来"稀释"，扩散速度为 $O(\sqrt{t})$，温度以代数速率 $O(t^{-1/2})$ 衰减——边界不存在，热量无处"逃逸"，只能无穷稀释。
  • 有界杆Dirichlet（$u=0$ 在边界）：边界像"吸热器"，持续从系统中抽走热量。两端恒温为零等同于强制冷却，热量被边界高效带走，导致指数衰减 $O(e^{-\lambda_1 t})$——比全空间快得多。
  • 有界杆Neumann（绝热边界）：边界不通热流，总热量守恒，温度趋于非零常数。趋近速率为指数 $O(e^{-\lambda_1 t})$，但极限值不为零。

关键洞察：代数和指数的分野——无穷域上的衰减是代数级的（"稀释"过程），有界域上的衰减是指数级的（"冷却"过程）。在物理和工程中，指数衰减通常意味着系统在特征时间 $\tau = 1/\lambda_1$ 后基本达到稳态。
