4.1 为什么要分类？

从具体到抽象

前四章我们分别深入学习了波动方程、热传导方程和Laplace方程。尽管它们都是二阶线性PDE，但它们的数学行为截然不同：

  • 波动方程有有限传播速度，热传导方程有无穷传播速度
  • 波动方程不光滑化初始数据，热传导方程即时光滑化
  • Laplace方程的解在边界取到极值

现在的问题是：这些差异的根源是什么？ 如何不经过求解就直接从方程的形式判断其行为？

答案隐藏在方程主部系数之中——本章的核心发现是：二阶项系数矩阵的判别式符号决定了方程的一切。

4.2 二阶线性方程的一般形式

定义 4.1（二阶线性PDE的一般形式）

两个自变量 $x, y$ 的二阶线性偏微分方程的一般形式为

$$a_{11}u_{xx} + 2a_{12}u_{xy} + a_{22}u_{yy} + b_1 u_x + b_2 u_y + cu = f,$$

其中 $a_{11}, a_{12}, a_{22}, b_1, b_2, c, f$ 均为 $(x, y)$ 的已知函数。

二次偏导数项 $a_{11}u_{xx} + 2a_{12}u_{xy} + a_{22}u_{yy}$ 称为方程的主部（principal part）。

定义 4.2（系数矩阵与判别式）

方程主部的系数矩阵为 $A = \begin{pmatrix} a_{11} & a_{12} \\ a_{12} & a_{22} \end{pmatrix}.$

定义判别式 $\Delta = a_{12}^2 - a_{11}a_{22} = -\det A.$

注意符号约定

若方程写为 $a u_{xx} + 2b u_{xy} + c u_{yy} + \cdots = 0$，则 $\Delta = b^2 - ac$。但若写为 $a u_{xx} + b u_{xy} + c u_{yy} + \cdots = 0$（无因子2），则 $\Delta = (b/2)^2 - ac$。务必看清交叉项的系数约定！

4.3 判别式与方程分类

定义 4.3（方程的分类）

在点 $(x_0, y_0)$ 处，根据判别式 $\Delta = a_{12}^2 - a_{11}a_{22}$：

  • $\Delta > 0$：双曲型（hyperbolic）——"波动家族"
  • $\Delta = 0$：抛物型（parabolic）——"热传导家族"
  • $\Delta < 0$：椭圆型（elliptic）——"Laplace家族"
特征值视角

分类也可通过系数矩阵 $A$ 的特征值判定：

  • $A$ 有两个异号实特征值 → 双曲型
  • $A$ 有一个零特征值（秩为1）→ 抛物型
  • $A$ 有两个同号实特征值（正定或负定）→ 椭圆型

这解释了为什么叫"双曲"、"抛物"、"椭圆"——它们对应二次型 $a_{11}x^2 + 2a_{12}xy + a_{22}y^2$ 的曲线类型！

例题 4.1

判断方程 $u_{xx} + 2u_{xy} + 5u_{yy} = 0$ 的类型，并求系数矩阵的特征值。

▶ 详细解答

$a_{11}=1, a_{12}=1, a_{22}=5$。$\Delta = 1 - 5 = -4 < 0$，方程为椭圆型。

$A = \begin{pmatrix} 1 & 1 \\ 1 & 5 \end{pmatrix}$，特征值 $\lambda = 3 \pm \sqrt{5} > 0$，两个同号正特征值，确认为椭圆型。

例题 4.2

判断 Tricomi 方程 $y u_{xx} + u_{yy} = 0$ 在各区域的类型。

▶ 详细解答

$a_{11}=y, a_{12}=0, a_{22}=1$。$\Delta = 0 - y \cdot 1 = -y$。

$y > 0$：$\Delta < 0$，椭圆型；$y < 0$：$\Delta > 0$，双曲型；$y = 0$：$\Delta = 0$，抛物型。

这是典型的混合型方程，在跨音速空气动力学中有重要应用。

4.4 化标准型

通过自变量变换，可将方程的主部化为标准型，揭示方程的本质结构。

特征方程

特征方程

对二阶线性PDE，其特征方程为

$$a_{11}\left(\frac{\dd y}{\dd x}\right)^2 - 2a_{12}\frac{\dd y}{\dd x} + a_{22} = 0.$$

解此ODE得到的积分曲线族就是特征曲线族。

三类标准型

双曲型标准型（$\Delta > 0$）

特征方程有两个不同实根 → 两族实特征曲线。取 $\xi, \eta$ 为两族特征曲线的不变量：

$$u_{\xi\eta} = \Phi(\xi,\eta,u,u_\xi,u_\eta) \quad \text{（第一标准型）}.$$

再令 $\alpha = \frac{\xi+\eta}{2}, \beta = \frac{\xi-\eta}{2}$ → $u_{\alpha\alpha} - u_{\beta\beta} = \tilde\Phi$（第二标准型）。

抛物型标准型（$\Delta = 0$）

特征方程有重根 → 一族特征曲线。取 $\xi$ 为特征不变量，$\eta$ 为与 $\xi$ 独立的任意函数：

$$u_{\eta\eta} = \Phi(\xi,\eta,u,u_\xi,u_\eta).$$
椭圆型标准型（$\Delta < 0$）

特征方程有共轭复根。取 $\xi, \eta$ 为复特征不变量的实部和虚部：

$$u_{\xi\xi} + u_{\eta\eta} = \Phi(\xi,\eta,u,u_\xi,u_\eta).$$
化标准型的步骤总结
  1. 写出方程主部的系数 $a_{11}, a_{12}, a_{22}$
  2. 求特征方程 $a_{11}\lambda^2 - 2a_{12}\lambda + a_{22} = 0$（其中 $\lambda = \dd y/\dd x$）
  3. 根据根的类型确定方程类型
  4. 解特征方程求特征曲线族，取适当的 $\xi, \eta$
  5. 利用链式法则计算变换后的偏导数，代入原方程得标准型
例题 4.3

将 $u_{xx} + 4u_{xy} + 3u_{yy} = 0$ 化为标准型。

▶ 详细解答

$a_{11}=1, a_{12}=2, a_{22}=3$。$\Delta = 4-3 = 1 > 0$，双曲型。

特征方程 $\lambda^2 - 4\lambda + 3 = 0$ → $\lambda_1=1, \lambda_2=3$。

特征曲线：$\dd y/\dd x = 1$ → $y-x=C_1$；$\dd y/\dd x = 3$ → $y-3x=C_2$。

取 $\xi = y-x, \eta = y-3x$。链式法则计算偏导数，代入后化简得：

$$u_{\xi\eta} = 0.$$

通解：$u = f(y-x) + g(y-3x)$，$f,g$ 为任意函数。

例题 4.4

将 $u_{xx} + 2u_{xy} + 5u_{yy} = 0$ 化为标准型。

▶ 详细解答

$a_{11}=1, a_{12}=1, a_{22}=5$。$\Delta = 1-5 = -4 < 0$，椭圆型。

特征方程 $\lambda^2 - 2\lambda + 5 = 0$ → $\lambda = 1 \pm 2i$。

复特征曲线：$y = (1+2i)x + C$，即 $\zeta = y - (1+2i)x$ 为复不变量。

分离实部虚部：$\zeta = (y-x) - 2ix$ → 取 $\xi = y-x, \eta = 2x$。

链式法则代入，化简得标准型：$u_{\xi\xi} + u_{\eta\eta} = 0$（Laplace方程）。

例题 4.5（考试重点：分类+特征方向+标准型综合题）

考虑二阶偏微分方程：

$$3u_{xx} + 10u_{xy} + 3u_{yy} = 0.$$

(1) 判断该方程的类型；

(2) 求其特征方向和特征曲线族；

(3) 将其化为标准型，并写出通解。

▶ 详细解答

(1) 类型判断：

$a_{11}=3, a_{12}=5, a_{22}=3$。$\Delta = a_{12}^2 - a_{11}a_{22} = 25 - 9 = 16 > 0$。故方程为双曲型。

(2) 特征方向：

特征方程为 $a_{11}\lambda^2 - 2a_{12}\lambda + a_{22} = 0$，即：

$$3\lambda^2 - 10\lambda + 3 = 0.$$

解此二次方程：$\lambda = \frac{10 \pm \sqrt{100 - 36}}{6} = \frac{10 \pm 8}{6}$。

得两个实根：

$$\lambda_1 = 3, \quad \lambda_2 = \frac{1}{3}.$$

这两个根 $\lambda = \frac{\dd y}{\dd x}$ 就是特征方向。特征曲线由下面积分得到：

对 $\lambda_1 = 3$：$\frac{\dd y}{\dd x} = 3 \implies y - 3x = C_1$（第一族特征曲线）；

对 $\lambda_2 = 1/3$：$\frac{\dd y}{\dd x} = \frac{1}{3} \implies y - \frac{x}{3} = C_2$，即 $3y - x = \text{const}$（第二族特征曲线）。

(3) 化标准型：

取新变量 $\xi = y - 3x$，$\eta = y - \frac{x}{3}$（或等价的 $\eta = 3y - x$）。

利用链式法则（更简单的方法是直接使用"取两族特征曲线的不变量为自变量"）：

计算结果为标准双曲型：

$$u_{\xi\eta} = 0.$$

注意此为标准型（$u_{\xi\eta}=0$），而非需要进一步变换。

通解为：$u = f(y - 3x) + g\!\left(y - \frac{x}{3}\right)$，其中 $f, g$ 为任意 $C^2$ 函数。

或用另一族特征曲线：$u = F(y - 3x) + G(3y - x)$，$F, G$ 为任意 $C^2$ 函数。

快速验证：设 $u = f(y-3x) + g(3y-x)$，则

$u_x = -3f' - g', \quad u_y = f' + 3g'$；

$u_{xx} = 9f'' + g'', \quad u_{xy} = -3f'' - 3g'', \quad u_{yy} = f'' + 9g''$。

$3u_{xx} + 10u_{xy} + 3u_{yy} = 3(9f''+g'') + 10(-3f''-3g'') + 3(f''+9g'') = (27-30+3)f'' + (3-30+27)g'' = 0$。验证正确。

考试技巧：双曲型方程，特征方向由 $a_{11}\lambda^2 - 2a_{12}\lambda + a_{22} = 0$ 给出。标准型取特征曲线的积分不变量为 $\xi, \eta$，总能化为 $u_{\xi\eta}=0$（第一标准型），通解就是 $u = f(\xi) + g(\eta)$。

4.5 叠加原理与Duhamel原理

线性叠加原理

定理 4.1（叠加原理）

设 $L$ 为线性微分算子，满足 $L(au + bv) = aLu + bLv$。则：

  1. 若 $u_1, u_2$ 分别是 $Lu = f_1$ 和 $Lu = f_2$ 的解，则 $u_1+u_2$ 是 $Lu = f_1+f_2$ 的解。
  2. 若 $\{u_n\}$ 是 $Lu = f_n$ 的解，级数 $u = \sum c_n u_n$ 一致收敛且可逐项微分，则 $u$ 是 $Lu = \sum c_n f_n$ 的解。

Duhamel 原理（齐次化原理）

定理 4.2（Duhamel 原理）

将非齐次源项 $f(x,t)$ 视为在时刻 $\tau$ 注入的"瞬时激励"。设 $v(x,t;\tau)$ 是在时刻 $\tau$ 以 $f(x,\tau)$ 为初始值的齐次问题解，则非齐次问题的解为这些瞬时响应的叠加：

$$u(x,t) = \int_0^t v(x,t;\tau)\,\dd\tau.$$
例题 4.5

利用叠加原理分解：$u_{tt} = u_{xx} + \sin(2t)$，$u(x,0)=0, u_t(x,0)=\cos x$。

▶ 详细解答

分解为两个子问题：

子问题1（齐次方程 + 非齐次初值）：$u^{(1)}_{tt} = u^{(1)}_{xx}$, $u^{(1)}(x,0)=0, u^{(1)}_t(x,0)=\cos x$。

子问题2（非齐次方程 + 零初值）：$u^{(2)}_{tt} = u^{(2)}_{xx} + \sin(2t)$, $u^{(2)}(x,0)=0, u^{(2)}_t(x,0)=0$。

子问题1用d'Alembert公式：$u^{(1)} = \cos x \sin t$。

子问题2用Duhamel原理或猜解：$u^{(2)} = \frac{t}{2} - \frac{\sin(2t)}{4}$。

总解：$u = \cos x \sin t + \frac{t}{2} - \frac{\sin(2t)}{4}$。

4.6 Sturm-Liouville 理论

前三章中反复出现的分离变量法，其数学基础是 Sturm-Liouville 理论。本节给出其系统化的数学框架。

定义 4.4（正则 Sturm-Liouville 问题）

在区间 $(a,b)$ 上求非平凡解 $X(x)$ 和常数 $\lambda$，使得

$$\frac{\dd}{\dd x}\left[p(x)\frac{\dd X}{\dd x}\right] + [q(x) + \lambda r(x)]X = 0,$$

满足分离边界条件 $\alpha_1 X(a) + \alpha_2 X'(a) = 0$，$\beta_1 X(b) + \beta_2 X'(b) = 0$。

其中 $p(x) > 0$，$r(x) > 0$ 为权函数。

定理 4.3（S-L问题的基本性质）
  1. 本征值：存在无穷多个实本征值 $\lambda_1 \le \lambda_2 \le \cdots$，$\lambda_n \to +\infty$。
  2. 正交性：不同本征值对应的本征函数关于权 $r(x)$ 正交：$\int_a^b r(x)X_m(x)X_n(x)\dd x = 0$（$m \neq n$）。
  3. 完备性：任意 $f \in L^2_r(a,b)$ 可展开为 Fourier 级数 $f = \sum c_n X_n$（关于 Fourier 级数的正交性与完备性，详见附录 A 节）。
S-L理论的统一视角

前三章中遇到的本征值问题都是S-L问题的特例：

  • 弦振动：$p=1, q=0, r=1$ → 正弦/余弦本征函数
  • Bessel方程：$p(r)=r, q(r)=-m^2/r, r(r)=r$ → Bessel函数
  • Legendre方程：$p(x)=1-x^2, q=0, r=1$ → Legendre多项式

4.8 能量估计方法

能量方法的核心思想

能量方法（Energy Method）是PDE理论中最强大的工具之一。其核心思想是：定义一个与解相关的"能量"泛函 $E(t)$，利用PDE推导 $E(t)$ 满足的微分不等式，从而获取解的估计。 能量方法不需要显式求解，却能给出：

  • 唯一性证明（如果两个解能量差为零，它们相等）；
  • 稳定性估计（初始数据的小扰动导致解的小变化）；
  • 渐近行为（能量衰减的速率决定了解趋于平衡的速度）。

波动方程的能量估计

一维波动方程的能量

对于 $u_{tt} = a^2 u_{xx}$（$x \in \RR$），定义能量：

$$E(t) = \frac{1}{2}\int_{-\infty}^{+\infty} \big[u_t^2(x,t) + a^2 u_x^2(x,t)\big]\,\dd x.$$

可以证明 $\frac{\dd E}{\dd t} = 0$，即能量守恒。这直接推出解的唯一性和稳定性。

一般二阶双曲型方程的能量估计

能量不等式（考试重点）

考虑更一般的双曲型方程（可能有低阶项，如阻尼）：

$$u_{tt} = a^2 u_{xx} + b(x,t) u_x + c(x,t) u_t + d(x,t) u + f(x,t).$$

设 $|b|, |c|, |d| \le M$（系数有界）。定义能量 $E(t) = \frac{1}{2}\int (u_t^2 + a^2 u_x^2 + u^2)\,\dd x$。

则可导出能量不等式（Gronwall型）：

$$\boxed{E(t) \le E(0)\,e^{Ct} + C\int_0^t \|f(\cdot,\tau)\|_{L^2}^2\,e^{C(t-\tau)}\,\dd\tau},$$

其中常数 $C$ 仅依赖于 $a$ 和 $M$。

该不等式有两个关键推论：

  1. 若 $f=0$（齐次方程），则 $E(t) \le E(0) e^{Ct}$——能量至多以指数速率增长（不会自发爆炸）；
  2. 若 $E(0)=0$ 且 $f=0$，则 $E(t) \equiv 0$，从而 $u \equiv 0$——推出唯一性。
能量不等式的推导

第一步：计算 $E'(t)$。

$$E'(t) = \int (u_t u_{tt} + a^2 u_x u_{xt} + u u_t)\,\dd x.$$

第二步：代入方程。

$u_{tt} = a^2 u_{xx} + b u_x + c u_t + d u + f$。代入后：

$$E'(t) = \int [u_t(a^2 u_{xx} + b u_x + c u_t + d u + f) + a^2 u_x u_{xt} + u u_t]\,\dd x.$$

第三步：分部积分处理 $u_t u_{xx}$ 项。

$\int u_t u_{xx}\,\dd x = -\int u_{xt} u_x\,\dd x$（假设无穷远处边界项消失），恰好与 $a^2 u_x u_{xt}$ 项相消。

第四步：用 Cauchy 不等式放缩低阶项。

例：$\int b u_x u_t\,\dd x \le M \int |u_x||u_t|\,\dd x \le \frac{M}{2}\int (u_x^2 + u_t^2)\,\dd x \le M' E(t)$。

第五步：得到微分不等式。

$$E'(t) \le C_1 E(t) + C_2\|f(\cdot,t)\|_{L^2}^2.$$

第六步：用 Gronwall 不等式求解。

Gronwall 引理：若 $y'(t) \le \alpha y(t) + \beta(t)$ 且 $y(0) = y_0$，则

$$y(t) \le y_0 e^{\alpha t} + \int_0^t \beta(s) e^{\alpha(t-s)}\,\dd s.$$

取 $y = E$, $\alpha = C_1$, $\beta(t) = C_2\|f\|^2$，即得能量不等式。

例题 4.8.1（考试重点：能量估计法证明唯一性）

考虑含阻尼的波动方程：

$$u_{tt} + \gamma u_t = a^2 u_{xx}, \quad \gamma > 0, \quad x \in \RR.$$

定义能量 $E(t) = \frac{1}{2}\int_{-\infty}^{+\infty} (u_t^2 + a^2 u_x^2)\,\dd x$。

(1) 证明 $E'(t) \le 0$（即能量不增，由于阻尼耗散）；

(2) 由此证明初值问题解的唯一性；

(3) 求 $E(t)$ 满足的精确估计。

▶ 详细解答

(1) 能量衰减：

$$E'(t) = \int_{-\infty}^{+\infty} (u_t u_{tt} + a^2 u_x u_{xt})\,\dd x.$$

代入 $u_{tt} = a^2 u_{xx} - \gamma u_t$：

$$\begin{aligned} E'(t) &= \int_{-\infty}^{+\infty} [u_t(a^2 u_{xx} - \gamma u_t) + a^2 u_x u_{xt}]\,\dd x \\ &= a^2\int_{-\infty}^{+\infty} (u_t u_{xx} + u_x u_{xt})\,\dd x - \gamma\int_{-\infty}^{+\infty} u_t^2\,\dd x. \end{aligned}$$

第一项 $= a^2\int \frac{\pd}{\pd x}(u_t u_x)\,\dd x = 0$（无穷远处边界项消失）。故：

$$E'(t) = -\gamma\int_{-\infty}^{+\infty} u_t^2(x,t)\,\dd x \le 0.$$

特别注意：$E'(t) \le 0$（严格地，只要 $u_t \neq 0$），故能量单调递减。这是物理上阻尼耗散能量的数学表达。

(2) 唯一性：

设 $u_1, u_2$ 为两个解，$w = u_1 - u_2$ 满足齐次方程和零初始条件 $w(x,0)=0, w_t(x,0)=0$。$E_w(0)=0$，由 $E'(t) \le 0$ 得 $E(t) \le E(0) = 0$。但 $E(t) \ge 0$，故 $E(t) \equiv 0$，从而 $w_t \equiv 0, w_x \equiv 0$，$w \equiv \text{const} = 0$（由 $w(x,0)=0$）。即 $u_1 = u_2$。

(3) 精确估计：

由 $E'(t) = -\gamma\int u_t^2\,\dd x \ge -2\gamma E(t)$（因为 $\int u_t^2 \le \int(u_t^2 + a^2 u_x^2) = 2E$）。这给出 $E'(t) \ge -2\gamma E(t)$，即 $E(t) \ge E(0) e^{-2\gamma t}$。

另一方面，由直接估计可进一步得 $E(t) \le E(0) e^{-2\gamma t}$（在适当的边界条件下），故 $E(t) \le E(0) e^{-2\gamma t}$——能量以指数速率衰减。

例题 4.8.2（考试重点：能量不等式的一般形式）

证明：若 $u$ 满足 $u_{tt} + u_t = u_{xx} + u_x + f$，则存在仅依赖于方程系数的常数 $C > 0$，使得能量

$$E(t) = \frac{1}{2}\int_{-\infty}^{+\infty}(u_t^2 + u_x^2 + u^2)\,\dd x$$

满足

$$E(t) \le E(0) e^{Ct} + C\int_0^t e^{C(t-\tau)}\|f(\cdot,\tau)\|_{L^2}^2\,\dd\tau.$$
▶ 详细解答

对 $E(t)$ 求导并代入方程：

$$E'(t) = \int (u_t u_{tt} + u_x u_{xt} + u u_t)\,\dd x.$$

代入 $u_{tt} = u_{xx} + u_x - u_t + f$：

$$\begin{aligned} E'(t) &= \int [u_t(u_{xx} + u_x - u_t + f) + u_x u_{xt} + u u_t]\,\dd x \\ &= \int (u_t u_{xx} + u_x u_{xt})\,\dd x + \int u_t u_x\,\dd x - \int u_t^2\,\dd x + \int u_t f\,\dd x + \int u u_t\,\dd x. \end{aligned}$$

第一项 $= \int \frac{\pd}{\pd x}(u_t u_x)\,\dd x = 0$（边界项消失）。

对其余项分别用 Cauchy-Schwarz 和 Cauchy 不等式（$ab \le \frac{\varepsilon}{2}a^2 + \frac{1}{2\varepsilon}b^2$）：

$\int u_t u_x\,\dd x \le \frac{1}{2}\int(u_t^2 + u_x^2)\,\dd x \le E(t)$；

$-\int u_t^2\,\dd x \le 0$；

$\int u_t f\,\dd x \le \frac{1}{2}\int u_t^2\,\dd x + \frac{1}{2}\int f^2\,\dd x \le E(t) + \frac{1}{2}\|f\|_{L^2}^2$；

$\int u u_t\,\dd x \le \frac{1}{2}\int(u^2 + u_t^2)\,\dd x \le E(t)$。

综上：$E'(t) \le 3E(t) + \frac{1}{2}\|f(\cdot,t)\|_{L^2}^2$。

由 Gronwall 引理（取 $\alpha = 3$，$\beta(t) = \frac{1}{2}\|f\|^2$）：

$$E(t) \le E(0) e^{3t} + \frac{1}{2}\int_0^t \|f(\cdot,\tau)\|_{L^2}^2 e^{3(t-\tau)}\,\dd\tau.$$

取 $C = \max(3, 1/2) = 3$（或其他保守常数），即得题目所需形式。

能量方法的关键步骤总结：

  1. 定义合适的能量（通常包含 $u_t^2$, $u_x^2$, $u^2$）；
  2. 求导并代入方程，将 $u_{tt}$ 用空间导数替换；
  3. 分部积分消去最高阶耦合项；
  4. 用 Cauchy 不等式放缩低阶项；
  5. 得到 $E'(t) \le C E(t) + g(t)$ 形式的微分不等式；
  6. 用 Gronwall 引理完成估计。
能量方法的三种主要应用
应用核心思路典型结论
唯一性 令 $w = u_1 - u_2$，证 $E(0)=0 \implies E(t)\equiv 0$ $w \equiv 0$，即 $u_1 = u_2$
