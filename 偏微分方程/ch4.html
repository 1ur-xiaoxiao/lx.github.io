4.1 为什么要分类？

从具体到抽象

前四章我们分别深入学习了波动方程、热传导方程和Laplace方程。尽管它们都是二阶线性PDE，但它们的数学行为截然不同：

  • 波动方程有有限传播速度，热传导方程有无穷传播速度
  • 波动方程不光滑化初始数据，热传导方程即时光滑化
  • Laplace方程的解在边界取到极值

现在的问题是：这些差异的根源是什么？ 如何不经过求解就直接从方程的形式判断其行为？

答案隐藏在方程主部系数之中——本章的核心发现是：二阶项系数矩阵的判别式符号决定了方程的一切。

4.2 二阶线性方程的一般形式

定义 4.1（二阶线性PDE的一般形式）

两个自变量 $x, y$ 的二阶线性偏微分方程的一般形式为

$$a_{11}u_{xx} + 2a_{12}u_{xy} + a_{22}u_{yy} + b_1 u_x + b_2 u_y + cu = f,$$

其中 $a_{11}, a_{12}, a_{22}, b_1, b_2, c, f$ 均为 $(x, y)$ 的已知函数。

二次偏导数项 $a_{11}u_{xx} + 2a_{12}u_{xy} + a_{22}u_{yy}$ 称为方程的主部（principal part）。

定义 4.2（系数矩阵与判别式）

方程主部的系数矩阵为 $A = \begin{pmatrix} a_{11} & a_{12} \\ a_{12} & a_{22} \end{pmatrix}.$

定义判别式 $\Delta = a_{12}^2 - a_{11}a_{22} = -\det A.$

注意符号约定

若方程写为 $a u_{xx} + 2b u_{xy} + c u_{yy} + \cdots = 0$，则 $\Delta = b^2 - ac$。但若写为 $a u_{xx} + b u_{xy} + c u_{yy} + \cdots = 0$（无因子2），则 $\Delta = (b/2)^2 - ac$。务必看清交叉项的系数约定！

4.3 判别式与方程分类

定义 4.3（方程的分类）

在点 $(x_0, y_0)$ 处，根据判别式 $\Delta = a_{12}^2 - a_{11}a_{22}$：

  • $\Delta > 0$：双曲型（hyperbolic）——"波动家族"
  • $\Delta = 0$：抛物型（parabolic）——"热传导家族"
  • $\Delta < 0$：椭圆型（elliptic）——"Laplace家族"
特征值视角

分类也可通过系数矩阵 $A$ 的特征值判定：

  • $A$ 有两个异号实特征值 → 双曲型
  • $A$ 有一个零特征值（秩为1）→ 抛物型
  • $A$ 有两个同号实特征值（正定或负定）→ 椭圆型

这解释了为什么叫"双曲"、"抛物"、"椭圆"——它们对应二次型 $a_{11}x^2 + 2a_{12}xy + a_{22}y^2$ 的曲线类型！

例题 4.1

判断方程 $u_{xx} + 2u_{xy} + 5u_{yy} = 0$ 的类型，并求系数矩阵的特征值。

▶ 详细解答

$a_{11}=1, a_{12}=1, a_{22}=5$。$\Delta = 1 - 5 = -4 < 0$，方程为椭圆型。

$A = \begin{pmatrix} 1 & 1 \\ 1 & 5 \end{pmatrix}$，特征值 $\lambda = 3 \pm \sqrt{5} > 0$，两个同号正特征值，确认为椭圆型。

例题 4.2

判断 Tricomi 方程 $y u_{xx} + u_{yy} = 0$ 在各区域的类型。

▶ 详细解答

$a_{11}=y, a_{12}=0, a_{22}=1$。$\Delta = 0 - y \cdot 1 = -y$。

$y > 0$：$\Delta < 0$，椭圆型；$y < 0$：$\Delta > 0$，双曲型；$y = 0$：$\Delta = 0$，抛物型。

这是典型的混合型方程，在跨音速空气动力学中有重要应用。

4.4 化标准型

通过自变量变换，可将方程的主部化为标准型，揭示方程的本质结构。

特征方程

特征方程

对二阶线性PDE，其特征方程为

$$a_{11}\left(\frac{\dd y}{\dd x}\right)^2 - 2a_{12}\frac{\dd y}{\dd x} + a_{22} = 0.$$

解此ODE得到的积分曲线族就是特征曲线族。

三类标准型

双曲型标准型（$\Delta > 0$）

特征方程有两个不同实根 → 两族实特征曲线。取 $\xi, \eta$ 为两族特征曲线的不变量：

$$u_{\xi\eta} = \Phi(\xi,\eta,u,u_\xi,u_\eta) \quad \text{（第一标准型）}.$$

再令 $\alpha = \frac{\xi+\eta}{2}, \beta = \frac{\xi-\eta}{2}$ → $u_{\alpha\alpha} - u_{\beta\beta} = \tilde\Phi$（第二标准型）。

抛物型标准型（$\Delta = 0$）

特征方程有重根 → 一族特征曲线。取 $\xi$ 为特征不变量，$\eta$ 为与 $\xi$ 独立的任意函数：

$$u_{\eta\eta} = \Phi(\xi,\eta,u,u_\xi,u_\eta).$$
椭圆型标准型（$\Delta < 0$）

特征方程有共轭复根。取 $\xi, \eta$ 为复特征不变量的实部和虚部：

$$u_{\xi\xi} + u_{\eta\eta} = \Phi(\xi,\eta,u,u_\xi,u_\eta).$$
化标准型的步骤总结
  1. 写出方程主部的系数 $a_{11}, a_{12}, a_{22}$
  2. 求特征方程 $a_{11}\lambda^2 - 2a_{12}\lambda + a_{22} = 0$（其中 $\lambda = \dd y/\dd x$）
  3. 根据根的类型确定方程类型
  4. 解特征方程求特征曲线族，取适当的 $\xi, \eta$
  5. 利用链式法则计算变换后的偏导数，代入原方程得标准型
例题 4.3

将 $u_{xx} + 4u_{xy} + 3u_{yy} = 0$ 化为标准型。

▶ 详细解答

$a_{11}=1, a_{12}=2, a_{22}=3$。$\Delta = 4-3 = 1 > 0$，双曲型。

特征方程 $\lambda^2 - 4\lambda + 3 = 0$ → $\lambda_1=1, \lambda_2=3$。

特征曲线：$\dd y/\dd x = 1$ → $y-x=C_1$；$\dd y/\dd x = 3$ → $y-3x=C_2$。

取 $\xi = y-x, \eta = y-3x$。链式法则计算偏导数，代入后化简得：

$$u_{\xi\eta} = 0.$$

通解：$u = f(y-x) + g(y-3x)$，$f,g$ 为任意函数。

例题 4.4

将 $u_{xx} + 2u_{xy} + 5u_{yy} = 0$ 化为标准型。

▶ 详细解答

$a_{11}=1, a_{12}=1, a_{22}=5$。$\Delta = 1-5 = -4 < 0$，椭圆型。

特征方程 $\lambda^2 - 2\lambda + 5 = 0$ → $\lambda = 1 \pm 2i$。

复特征曲线：$y = (1+2i)x + C$，即 $\zeta = y - (1+2i)x$ 为复不变量。

分离实部虚部：$\zeta = (y-x) - 2ix$ → 取 $\xi = y-x, \eta = 2x$。

链式法则代入，化简得标准型：$u_{\xi\xi} + u_{\eta\eta} = 0$（Laplace方程）。

4.5 叠加原理与Duhamel原理

线性叠加原理

定理 4.1（叠加原理）

设 $L$ 为线性微分算子，满足 $L(au + bv) = aLu + bLv$。则：

  1. 若 $u_1, u_2$ 分别是 $Lu = f_1$ 和 $Lu = f_2$ 的解，则 $u_1+u_2$ 是 $Lu = f_1+f_2$ 的解。
  2. 若 $\{u_n\}$ 是 $Lu = f_n$ 的解，级数 $u = \sum c_n u_n$ 一致收敛且可逐项微分，则 $u$ 是 $Lu = \sum c_n f_n$ 的解。

Duhamel 原理（齐次化原理）

定理 4.2（Duhamel 原理）

将非齐次源项 $f(x,t)$ 视为在时刻 $\tau$ 注入的"瞬时激励"。设 $v(x,t;\tau)$ 是在时刻 $\tau$ 以 $f(x,\tau)$ 为初始值的齐次问题解，则非齐次问题的解为这些瞬时响应的叠加：

$$u(x,t) = \int_0^t v(x,t;\tau)\,\dd\tau.$$
例题 4.5

利用叠加原理分解：$u_{tt} = u_{xx} + \sin(2t)$，$u(x,0)=0, u_t(x,0)=\cos x$。

▶ 详细解答

分解为两个子问题：

子问题1（齐次方程 + 非齐次初值）：$u^{(1)}_{tt} = u^{(1)}_{xx}$, $u^{(1)}(x,0)=0, u^{(1)}_t(x,0)=\cos x$。

子问题2（非齐次方程 + 零初值）：$u^{(2)}_{tt} = u^{(2)}_{xx} + \sin(2t)$, $u^{(2)}(x,0)=0, u^{(2)}_t(x,0)=0$。

子问题1用d'Alembert公式：$u^{(1)} = \cos x \sin t$。

子问题2用Duhamel原理或猜解：$u^{(2)} = \frac{t}{2} - \frac{\sin(2t)}{4}$。

总解：$u = \cos x \sin t + \frac{t}{2} - \frac{\sin(2t)}{4}$。

4.6 Sturm-Liouville 理论

前三章中反复出现的分离变量法，其数学基础是 Sturm-Liouville 理论。本节给出其系统化的数学框架。

定义 4.4（正则 Sturm-Liouville 问题）

在区间 $(a,b)$ 上求非平凡解 $X(x)$ 和常数 $\lambda$，使得

$$\frac{\dd}{\dd x}\left[p(x)\frac{\dd X}{\dd x}\right] + [q(x) + \lambda r(x)]X = 0,$$

满足分离边界条件 $\alpha_1 X(a) + \alpha_2 X'(a) = 0$，$\beta_1 X(b) + \beta_2 X'(b) = 0$。

其中 $p(x) > 0$，$r(x) > 0$ 为权函数。

定理 4.3（S-L问题的基本性质）
  1. 本征值：存在无穷多个实本征值 $\lambda_1 \le \lambda_2 \le \cdots$，$\lambda_n \to +\infty$。
  2. 正交性：不同本征值对应的本征函数关于权 $r(x)$ 正交：$\int_a^b r(x)X_m(x)X_n(x)\dd x = 0$（$m \neq n$）。
  3. 完备性：任意 $f \in L^2_r(a,b)$ 可展开为 Fourier 级数 $f = \sum c_n X_n$（关于 Fourier 级数的正交性与完备性，详见附录 A 节）。
S-L理论的统一视角

前三章中遇到的本征值问题都是S-L问题的特例：

  • 弦振动：$p=1, q=0, r=1$ → 正弦/余弦本征函数
  • Bessel方程：$p(r)=r, q(r)=-m^2/r, r(r)=r$ → Bessel函数
  • Legendre方程：$p(x)=1-x^2, q=0, r=1$ → Legendre多项式

一句话总结

分类不是目的，而是手段。知道了一个二阶线性PDE的类型，你就知道了该用什么方法求解、该期待什么样的解的行为。双曲→特征线法+能量守恒，抛物→积分变换+极值衰减，椭圆→Green函数+调和性质。这个框架是所有后续学习的基础。
