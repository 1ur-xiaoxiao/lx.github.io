6.1 分离变量法

定义 6.1.1 (分离变量法的基本思想)

分离变量法（Fourier方法）是求解线性偏微分方程边值问题的最基本方法。其核心思想是：假设解可以写成各变量的函数之积

$$u(x, t) = X(x) \cdot T(t)$$

代入方程后，利用"变量分离"将偏微分方程化为若干个常微分方程，再利用叠加原理构造一般解。

例题 6.1.1 (两端固定弦的自由振动)

考虑初边值问题

$$\begin{cases} u_{tt} = a^2 u_{xx}, & 0 < x < l, \ t > 0 \\ u(0, t) = u(l, t) = 0, & t \geq 0 \\ u(x, 0) = \varphi(x), \ u_t(x, 0) = \psi(x), & 0 \leq x \leq l \end{cases}$$

第一步：分离变量。

设 $u(x,t) = X(x)T(t)$，代入 $u_{tt} = a^2 u_{xx}$：

$$X(x)T''(t) = a^2 X''(x)T(t)$$

两边除以 $a^2 X(x)T(t)$：

$$\frac{T''(t)}{a^2 T(t)} = \frac{X''(x)}{X(x)} = -\lambda$$

其中 $-\lambda$ 为分离常数（左侧仅为 $t$ 的函数，右侧仅为 $x$ 的函数，故必为常数）。

第二步：求解 $X$ 的常微分方程（本征值问题）。

$$\begin{cases} X''(x) + \lambda X(x) = 0, & 0 < x < l \\ X(0) = X(l) = 0 \end{cases}$$

分三种情况讨论：

情况一：$\lambda < 0$。 设 $\lambda = -\mu^2$（$\mu > 0$），通解 $X = C_1 e^{\mu x} + C_2 e^{-\mu x}$。由 $X(0) = 0$ 得 $C_1 + C_2 = 0$；由 $X(l) = 0$ 得 $C_1(e^{\mu l} - e^{-\mu l}) = 0$。因 $e^{\mu l} \neq e^{-\mu l}$，故 $C_1 = C_2 = 0$，无非平凡解。

情况二：$\lambda = 0$。 通解 $X = C_1 x + C_2$。由 $X(0) = C_2 = 0$，$X(l) = C_1 l = 0$，得 $C_1 = C_2 = 0$，无非平凡解。

情况三：$\lambda > 0$。 通解 $X = C_1 \cos\sqrt{\lambda} x + C_2 \sin\sqrt{\lambda} x$。由 $X(0) = C_1 = 0$，$X(l) = C_2 \sin\sqrt{\lambda} l = 0$。要非平凡解，$C_2 \neq 0$，故 $\sin\sqrt{\lambda} l = 0$，即 $\sqrt{\lambda} l = n\pi$（$n = 1, 2, 3, \dots$）。

因此本征值和本征函数为

$$\lambda_n = \left(\frac{n\pi}{l}\right)^2, \quad X_n(x) = \sin\frac{n\pi x}{l}, \quad n = 1, 2, 3, \dots$$

第三步：求解 $T$ 的常微分方程。

$$T''(t) + a^2 \lambda_n T(t) = T''(t) + \left(\frac{n\pi a}{l}\right)^2 T(t) = 0$$

通解为

$$T_n(t) = A_n \cos\frac{n\pi a t}{l} + B_n \sin\frac{n\pi a t}{l}$$

第四步：叠加原理构造一般解。

$$u(x,t) = \sum_{n=1}^{\infty} \left(A_n \cos\frac{n\pi a t}{l} + B_n \sin\frac{n\pi a t}{l}\right) \sin\frac{n\pi x}{l}$$

第五步：由初始条件确定系数。

由 $u(x,0) = \varphi(x)$：

$$\varphi(x) = \sum_{n=1}^{\infty} A_n \sin\frac{n\pi x}{l}$$

由正弦级数的系数公式：

$$A_n = \frac{2}{l} \int_0^l \varphi(x) \sin\frac{n\pi x}{l} \, \dd x$$

由 $u_t(x,0) = \psi(x)$：

$$\psi(x) = \sum_{n=1}^{\infty} B_n \cdot \frac{n\pi a}{l} \sin\frac{n\pi x}{l}$$ $$B_n = \frac{2}{n\pi a} \int_0^l \psi(x) \sin\frac{n\pi x}{l} \, \dd x$$
注记：分离变量法的适用条件

分离变量法适用于：(1) 线性方程；(2) 区域具有规则的几何形状（矩形、圆、球等）；(3) 边值条件为齐次的（非齐次时需先化为齐次）。其理论基础是 Sturm-Liouville 理论。

6.2 Sturm-Liouville问题

定义 6.2.1 (正则 Sturm-Liouville 问题)

正则S-L问题是如下常微分方程本征值问题：在区间 $(a, b)$ 上求非平凡解 $X(x)$ 和常数 $\lambda$，使得

$$\frac{\dd}{\dd x}\left[p(x) \frac{\dd X}{\dd x}\right] + [q(x) + \lambda r(x)] X = 0, \quad a < x < b$$

满足边值条件

$$\begin{cases} \alpha_1 X(a) + \alpha_2 X'(a) = 0 \\ \beta_1 X(b) + \beta_2 X'(b) = 0 \end{cases}$$

其中 $p(x) > 0$，$r(x) > 0$，$\alpha_1^2 + \alpha_2^2 > 0$，$\beta_1^2 + \beta_2^2 > 0$。

定理 6.2.1 (S-L问题的基本性质)

正则Sturm-Liouville问题具有以下性质：

(1) 本征值的存在性： 存在无穷多个实本征值 $\lambda_1 \leq \lambda_2 \leq \cdots$，且 $\lambda_n \to +\infty$。

(2) 本征值的实性： 所有本征值均为实数。

(3) 正交性： 不同本征值对应的本征函数关于权函数 $r(x)$ 正交：若 $\lambda_m \neq \lambda_n$，则

$$\int_a^b r(x) X_m(x) X_n(x) \, \dd x = 0$$

(4) 完备性： 本征函数系 $\{X_n\}_{n=1}^\infty$ 在 $L^2_r(a,b)$ 中完备，即任意 $f \in L^2_r(a,b)$ 可展开为

$$f(x) = \sum_{n=1}^{\infty} c_n X_n(x), \quad c_n = \frac{\int_a^b r(x) f(x) X_n(x) \, \dd x}{\int_a^b r(x) X_n^2(x) \, \dd x}$$
证明：本征值为实数

设 $\lambda$ 为本征值，$X$ 为对应的本征函数。对方程取复共轭（注意 $p, q, r$ 为实函数）：

$$(pX')' + (q + \bar{\lambda} r)\bar{X} = 0$$

将原方程乘以 $\bar{X}$，共轭方程乘以 $X$，相减并在 $(a,b)$ 上积分：

$$\int_a^b \left[\bar{X}(pX')' - X(p\bar{X}')'\right] \dd x + (\lambda - \bar{\lambda}) \int_a^b r|X|^2 \, \dd x = 0$$

左边第一项经分部积分和边值条件可化为零，故 $(\lambda - \bar{\lambda}) \int_a^b r|X|^2 \, \dd x = 0$。因 $r > 0$，$X$ 不恒为零，故 $\lambda = \bar{\lambda}$，即 $\lambda \in \RR$。

证明：本征函数的正交性

设 $\lambda_m \neq \lambda_n$，对应的本征函数为 $X_m, X_n$，则

$$(pX_m')' + (q + \lambda_m r) X_m = 0 \quad \cdots (1)$$ $$(pX_n')' + (q + \lambda_n r) X_n = 0 \quad \cdots (2)$$

$(1) \times X_n - (2) \times X_m$：

$$X_n(pX_m')' - X_m(pX_n')' + (\lambda_m - \lambda_n) r X_m X_n = 0$$

在 $(a,b)$ 上积分，左边前两项经分部积分和边值条件化为零，得

$$(\lambda_m - \lambda_n) \int_a^b r(x) X_m(x) X_n(x) \, \dd x = 0$$

因 $\lambda_m \neq \lambda_n$，故 $\int_a^b r X_m X_n \, \dd x = 0$。

6.3 常见本征值问题

公式 6.3.1 (弦振动问题)

两端固定弦：$X'' + \lambda X = 0$，$X(0) = X(l) = 0$。

$$\lambda_n = \left(\frac{n\pi}{l}\right)^2, \quad X_n(x) = \sin\frac{n\pi x}{l}, \quad n = 1, 2, 3, \dots$$

正交性：$\int_0^l \sin\frac{m\pi x}{l} \sin\frac{n\pi x}{l} \, \dd x = \frac{l}{2} \delta_{mn}$。

一端固定一端自由：$X'' + \lambda X = 0$，$X(0) = X'(l) = 0$。

$$\lambda_n = \left(\frac{(2n-1)\pi}{2l}\right)^2, \quad X_n(x) = \sin\frac{(2n-1)\pi x}{2l}, \quad n = 1, 2, 3, \dots$$
公式 6.3.2 (圆域上的本征值问题——Bessel方程)

在极坐标下对圆域 $B_R(0)$ 求解 $\lap u = 0$，分离变量后得到Bessel方程：

$$r^2 R'' + rR' + (\lambda r^2 - m^2) R = 0$$

其中 $m = 0, 1, 2, \dots$（由角向周期性决定）。令 $t = \sqrt{\lambda} r$，$R(r) = J_m(t)$，得标准Bessel方程

$$t^2 J_m'' + t J_m' + (t^2 - m^2) J_m = 0$$

本征值由有界性条件 $R(0)$ 有界及边界条件确定：

$$\lambda_{mn} = \left(\frac{j_{m,n}}{R}\right)^2, \quad R_{mn}(r) = J_m\left(\frac{j_{m,n} r}{R}\right)$$

其中 $j_{m,n}$ 为 $m$ 阶Bessel函数 $J_m$ 的第 $n$ 个正零点。

公式 6.3.3 (球域上的本征值问题——Legendre方程)

在球坐标下对球域求解 $\lap u = 0$，分离变量后角向部分得到Legendre方程：

$$(1 - x^2)P'' - 2xP' + l(l+1)P = 0, \quad -1 < x < 1$$

其中 $x = \cos\theta$。当 $l = 0, 1, 2, \dots$ 时，存在有界解——Legendre多项式：

$$P_l(x) = \frac{1}{2^l l!} \frac{\dd^l}{\dd x^l}(x^2 - 1)^l$$

正交性：$\int_{-1}^{1} P_m(x) P_l(x) \, \dd x = \frac{2}{2l+1} \delta_{ml}$。

注记：本征函数系的统一框架

所有上述本征值问题均为Sturm-Liouville问题的特例。弦振动对应 $p=1, q=0, r=1$；Bessel方程可化为S-L标准形，权函数 $r(r) = r$；Legendre方程中 $p(x) = 1-x^2$，权函数 $r(x) = 1$。S-L理论保证了各类本征函数系的正交性和完备性。

6.4 Fourier级数展开求解

例题 6.4.1 (热传导方程的分离变量法)

求解初边值问题

$$\begin{cases} u_t = k u_{xx}, & 0 < x < l, \ t > 0 \\ u(0, t) = u(l, t) = 0 \\ u(x, 0) = f(x) \end{cases}$$

分离变量： 设 $u(x,t) = X(x)T(t)$，代入方程得

$$\frac{T'}{kT} = \frac{X''}{X} = -\lambda$$

本征值问题： $X'' + \lambda X = 0$，$X(0) = X(l) = 0$。由6.3节，

$$\lambda_n = \left(\frac{n\pi}{l}\right)^2, \quad X_n(x) = \sin\frac{n\pi x}{l}$$

时间方程： $T' + k\lambda_n T = 0$，解为

$$T_n(t) = C_n e^{-k\lambda_n t} = C_n e^{-k(n\pi/l)^2 t}$$

叠加：

$$u(x,t) = \sum_{n=1}^{\infty} C_n e^{-k(n\pi/l)^2 t} \sin\frac{n\pi x}{l}$$

确定系数： 由 $u(x,0) = f(x)$：

$$C_n = \frac{2}{l} \int_0^l f(x) \sin\frac{n\pi x}{l} \, \dd x$$

物理意义： 各模态以指数速率 $e^{-k(n\pi/l)^2 t}$ 衰减，高频模态衰减更快（$\lambda_n$ 越大衰减越快），故长时间后温度分布趋于均匀。

例题 6.4.2 (Laplace方程在矩形域上的Dirichlet问题)

求解

$$\begin{cases} \lap u = u_{xx} + u_{yy} = 0, & 0 < x < a, \ 0 < y < b \\ u(x, 0) = 0, \ u(x, b) = f(x) \\ u(0, y) = 0, \ u(a, y) = 0 \end{cases}$$

分离变量： 设 $u(x,y) = X(x)Y(y)$，代入 $u_{xx} + u_{yy} = 0$：

$$\frac{X''}{X} = -\frac{Y''}{Y} = -\lambda$$

$X$ 的本征值问题： $X'' + \lambda X = 0$，$X(0) = X(a) = 0$。

$$\lambda_n = \left(\frac{n\pi}{a}\right)^2, \quad X_n(x) = \sin\frac{n\pi x}{a}, \quad n = 1, 2, 3, \dots$$

$Y$ 的方程： $Y'' - \lambda_n Y = 0$，即 $Y'' - (n\pi/a)^2 Y = 0$。通解为

$$Y_n(y) = A_n \cosh\frac{n\pi y}{a} + B_n \sinh\frac{n\pi y}{a}$$

由 $u(x,0) = 0$ 得 $Y_n(0) = A_n = 0$，故 $Y_n(y) = B_n \sinh\frac{n\pi y}{a}$。

叠加：

$$u(x,y) = \sum_{n=1}^{\infty} B_n \sinh\frac{n\pi y}{a} \sin\frac{n\pi x}{a}$$

确定系数： 由 $u(x,b) = f(x)$：

$$f(x) = \sum_{n=1}^{\infty} B_n \sinh\frac{n\pi b}{a} \sin\frac{n\pi x}{a}$$ $$B_n \sinh\frac{n\pi b}{a} = \frac{2}{a} \int_0^a f(x) \sin\frac{n\pi x}{a} \, \dd x$$ $$B_n = \frac{2}{a \sinh(n\pi b/a)} \int_0^a f(x) \sin\frac{n\pi x}{a} \, \dd x$$

最终解：

$$u(x,y) = \sum_{n=1}^{\infty} \left(\frac{2}{a} \int_0^a f(\xi) \sin\frac{n\pi\xi}{a} \, \dd\xi\right) \frac{\sinh(n\pi y/a)}{\sinh(n\pi b/a)} \sin\frac{n\pi x}{a}$$
例题 6.4.3 (圆域上的Dirichlet问题)

求解 $\lap u = 0$（$r < R$），$u(R, \varphi) = g(\varphi)$。

极坐标下Laplace方程：

$$u_{rr} + \frac{1}{r} u_r + \frac{1}{r^2} u_{\varphi\varphi} = 0$$

分离变量： 设 $u(r,\varphi) = R(r)\Phi(\varphi)$，代入得

$$\frac{r^2 R'' + rR'}{R} = -\frac{\Phi''}{\Phi} = \lambda$$

角向方程： $\Phi'' + \lambda \Phi = 0$，周期条件 $\Phi(\varphi + 2\pi) = \Phi(\varphi)$。

$$\lambda = m^2, \quad \Phi_m(\varphi) = a_m \cos m\varphi + b_m \sin m\varphi, \quad m = 0, 1, 2, \dots$$

径向方程： $r^2 R'' + rR' - m^2 R = 0$（Euler方程）。

$m = 0$ 时：$R_0 = C_0 + D_0 \ln r$。由 $R(0)$ 有界，$D_0 = 0$。

$m \geq 1$ 时：$R_m = C_m r^m + D_m r^{-m}$。由 $R(0)$ 有界，$D_m = 0$。

叠加：

$$u(r,\varphi) = \frac{a_0}{2} + \sum_{m=1}^{\infty} r^m (a_m \cos m\varphi + b_m \sin m\varphi)$$

确定系数： 由 $u(R,\varphi) = g(\varphi)$：

$$a_m = \frac{1}{\pi R^m} \int_0^{2\pi} g(\varphi) \cos m\varphi \, \dd\varphi, \quad b_m = \frac{1}{\pi R^m} \int_0^{2\pi} g(\varphi) \sin m\varphi \, \dd\varphi$$

这与Poisson积分公式给出的结果一致。

可视化 · Fourier 级数逼近

观察 Fourier 级数如何逼近周期函数。增加项数 $N$，观察逼近效果。

蓝线: 原函数　|　红线: Fourier 部分和 $S_N(x) = \frac{a_0}{2} + \sum_{k=1}^{N}(a_k\cos kx + b_k\sin kx)$

6.5 非齐次问题

定义 6.5.1 (本征函数展开法)

对于含有非齐次项的方程，可将解按齐次问题的本征函数系展开，将偏微分方程化为关于展开系数的常微分方程。此方法称为本征函数展开法。

例题 6.5.1 (非齐次热传导方程)

求解

$$\begin{cases} u_t = k u_{xx} + F(x,t), & 0 < x < l, \ t > 0 \\ u(0, t) = u(l, t) = 0 \\ u(x, 0) = \varphi(x) \end{cases}$$

第一步：选取本征函数系。

齐次问题的本征函数为 $X_n(x) = \sin\frac{n\pi x}{l}$（$n = 1, 2, 3, \dots$），对应本征值 $\lambda_n = (n\pi/l)^2$。

第二步：将解和非齐次项按本征函数展开。

$$u(x,t) = \sum_{n=1}^{\infty} T_n(t) \sin\frac{n\pi x}{l}$$ $$F(x,t) = \sum_{n=1}^{\infty} f_n(t) \sin\frac{n\pi x}{l}, \quad f_n(t) = \frac{2}{l} \int_0^l F(x,t) \sin\frac{n\pi x}{l} \, \dd x$$

第三步：代入方程，比较系数。

$$\sum_{n=1}^{\infty} T_n'(t) \sin\frac{n\pi x}{l} = k \sum_{n=1}^{\infty} \left[-\lambda_n T_n(t)\right] \sin\frac{n\pi x}{l} + \sum_{n=1}^{\infty} f_n(t) \sin\frac{n\pi x}{l}$$

由正交性，对每个 $n$：

$$T_n'(t) + k\lambda_n T_n(t) = f_n(t)$$

第四步：求解一阶线性ODE。

初始条件：$T_n(0) = \varphi_n = \frac{2}{l}\int_0^l \varphi(x)\sin\frac{n\pi x}{l}\, \dd x$。由常数变易法：

$$T_n(t) = \varphi_n e^{-k\lambda_n t} + \int_0^t f_n(\tau) e^{-k\lambda_n(t-\tau)} \, \dd\tau$$

最终解：

$$u(x,t) = \sum_{n=1}^{\infty} \left[\varphi_n e^{-k\lambda_n t} + \int_0^t f_n(\tau) e^{-k\lambda_n(t-\tau)} \, \dd\tau\right] \sin\frac{n\pi x}{l}$$
定理 6.5.1 (Duhamel原理)

设 $w(x,t;\tau)$ 为如下齐次方程的解（$\tau$ 为参数）：

$$\begin{cases} w_t = k w_{xx}, & t > \tau \\ w(x, \tau; \tau) = F(x, \tau) \\ w(0, t; \tau) = w(l, t; \tau) = 0 \end{cases}$$

则非齐次方程 $u_t = k u_{xx} + F(x,t)$ 满足零初始条件的解为

$$u(x,t) = \int_0^t w(x, t; \tau) \, \dd\tau$$

Duhamel原理将非齐次问题化为一族齐次问题，其物理意义为：非齐次项 $F(x,t)$ 的效果可视为在各时刻 $\tau$ "注入"的瞬时效应的叠加。

例题 6.5.2 (非齐次波动方程)

求解

$$\begin{cases} u_{tt} = a^2 u_{xx} + f(x,t), & 0 < x < l, \ t > 0 \\ u(0,t) = u(l,t) = 0 \\ u(x,0) = 0, \ u_t(x,0) = 0 \end{cases}$$

本征函数展开： 设 $u(x,t) = \sum_{n=1}^{\infty} T_n(t) \sin\frac{n\pi x}{l}$，$f(x,t) = \sum_{n=1}^{\infty} f_n(t) \sin\frac{n\pi x}{l}$。

代入方程得

$$T_n''(t) + a^2\lambda_n T_n(t) = f_n(t), \quad \lambda_n = \left(\frac{n\pi}{l}\right)^2$$

初始条件：$T_n(0) = 0$，$T_n'(0) = 0$。

由常数变易法（或 Laplace 变换），解为

$$T_n(t) = \frac{1}{n\pi a/l} \int_0^t f_n(\tau) \sin\frac{n\pi a(t-\tau)}{l} \, \dd\tau$$

最终解：

$$u(x,t) = \sum_{n=1}^{\infty} \left[\frac{l}{n\pi a} \int_0^t f_n(\tau) \sin\frac{n\pi a(t-\tau)}{l} \, \dd\tau\right] \sin\frac{n\pi x}{l}$$

这与Duhamel原理给出的结果一致：$w(x,t;\tau)$ 为以 $F(x,\tau)$ 为初始位移的自由振动解，对 $\tau$ 积分即得 $u$。

例题 6.5.3 (非齐次Laplace方程——Poisson方程)

用本征函数展开法求解

$$\begin{cases} \lap u = u_{xx} + u_{yy} = -1, & 0 < x < a, \ 0 < y < b \\ u(0,y) = u(a,y) = 0 \\ u(x,0) = u(x,b) = 0 \end{cases}$$

按 $x$ 方向的本征函数展开：

$$u(x,y) = \sum_{n=1}^{\infty} Y_n(y) \sin\frac{n\pi x}{a}$$

将 $-1$ 展开为正弦级数：

$$-1 = \sum_{n=1}^{\infty} c_n \sin\frac{n\pi x}{a}, \quad c_n = \frac{2}{a}\int_0^a (-1)\sin\frac{n\pi x}{a} \, \dd x = \begin{cases} -\frac{4}{n\pi} & n \text{ 为奇数} \\ 0 & n \text{ 为偶数} \end{cases}$$

代入方程得

$$Y_n'' - \left(\frac{n\pi}{a}\right)^2 Y_n = c_n$$

特解：$Y_n^{(p)} = \frac{c_n}{-(n\pi/a)^2} = -\frac{c_n a^2}{n^2\pi^2}$。齐次解：$A_n \cosh\frac{n\pi y}{a} + B_n \sinh\frac{n\pi y}{a}$。

由 $Y_n(0) = 0$ 和 $Y_n(b) = 0$ 确定常数：

$$A_n = -Y_n^{(p)} = \frac{c_n a^2}{n^2\pi^2}, \quad B_n = Y_n^{(p)} \cdot \frac{1 - \cosh(n\pi b/a)}{\sinh(n\pi b/a)}$$

最终解可写为

$$u(x,y) = \frac{8a^2}{\pi^3} \sum_{k=0}^{\infty} \frac{1}{(2k+1)^3} \left[1 - \frac{\cosh\frac{(2k+1)\pi(y - b/2)}{a}}{\cosh\frac{(2k+1)\pi b}{2a}}\right] \sin\frac{(2k+1)\pi x}{a}$$
注记：分离变量法的总结

分离变量法的核心步骤可归纳为：

(1) 将偏微分方程分离为若干常微分方程；

(2) 求解本征值问题，得到本征值和本征函数系；

(3) 利用叠加原理构造级数形式的解；

(4) 利用本征函数的正交性和完备性确定展开系数；

(5) 验证解的收敛性和满足方程的合法性（正则性）。

对于非齐次问题，本征函数展开法或Duhamel原理是处理非齐次项的两种有效手段。
