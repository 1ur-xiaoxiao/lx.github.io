模拟题一（方法应用与计算）

建议用时：120分钟  |  满分：100分

第一题（15分）— PDE分类与化标准型

考虑二阶方程

$$u_{xx} - 4u_{xy} + 4u_{yy} = 0.$$

(1) 判断该方程的类型（双曲、抛物还是椭圆）。

(2) 将其化为标准型，并写出通解。

第二题（15分）— d'Alembert公式

求解以下 Cauchy 问题：

$$\begin{cases} u_{tt} = 9 u_{xx}, & x \in \RR,\ t > 0, \\ u(x,0) = \cos x,\; u_t(x,0) = 0, & x \in \RR. \end{cases}$$

并求 $u(x, t)$ 在 $t \to \infty$ 时的振幅是否有界。

第三题（15分）— 半无界波动（Dirichlet 延拓）

求解半无界弦的振动问题：

$$\begin{cases} u_{tt} = u_{xx}, & x > 0,\ t > 0, \\ u(0,t) = 0, & t \ge 0, \\ u(x,0) = x^2,\; u_t(x,0) = 0, & x \ge 0. \end{cases}$$

分别写出 $x \ge t$ 和 $0 < x < t$ 时的表达式，并验证 $u(0,t) = 0$。

第四题（15分）— Fourier 变换法解热传导方程

求解全空间热传导方程：

$$\begin{cases} u_t = u_{xx}, & x \in \RR,\ t > 0, \\ u(x,0) = e^{-x^2/2}, & x \in \RR. \end{cases}$$

（提示：$\mathcal{F}[e^{-x^2/2}](\xi) = \sqrt{2\pi}\,e^{-\xi^2/2}$。）

第五题（15分）— 分离变量法（波动方程）

求解有界弦的自由振动：

$$\begin{cases} u_{tt} = u_{xx}, & 0 < x < \pi,\ t > 0, \\ u(0,t) = u(\pi,t) = 0, & t \ge 0, \\ u(x,0) = x(\pi - x),\; u_t(x,0) = 0, & 0 \le x \le \pi. \end{cases}$$

写出解的 Fourier 级数表达式（系数需计算到最简）。

第六题（10分）— 极值原理证唯一性

设 $u(x,t)$ 满足热传导方程 $u_t = u_{xx}$ 在 $0 < x < 1, 0 < t < T$ 内，且满足初始条件 $u(x,0) = \varphi(x)$ 和边界条件 $u(0,t)=u(1,t)=0$。试用极值原理证明：该问题的解是唯一的。

第七题（15分）— Poisson 积分公式（圆域 Dirichlet 问题）

求解单位圆盘上的 Dirichlet 问题：

$$\begin{cases} \lap u = 0, & r < 1,\ 0 \le \theta < 2\pi, \\ u(1,\theta) = \sin^3\theta, & 0 \le \theta < 2\pi. \end{cases}$$

（提示：$\sin^3\theta = \frac{3}{4}\sin\theta - \frac{1}{4}\sin 3\theta$。）

模拟题一 · 参考答案
▶ 第一题解答

(1) 类型判断

$a_{11} = 1$，$2a_{12} = -4 \implies a_{12} = -2$，$a_{22} = 4$。

判别式 $\Delta = a_{12}^2 - a_{11}a_{22} = (-2)^2 - 1 \cdot 4 = 0$，故为抛物型。

(2) 化标准型

特征方程：$a_{11}(\dd y)^2 - 2a_{12}\dd x\dd y + a_{22}(\dd x)^2 = 0$，即 $(\dd y)^2 + 4\dd x\dd y + 4(\dd x)^2 = 0$，$(\dd y + 2\dd x)^2 = 0$。

特征线：$\dd y/\dd x = -2$，$y + 2x = C$。

令 $\xi = y + 2x$（唯一特征族）。选 $\eta = y$ 使 Jacobi $\neq 0$。

计算偏导数代换：

$$u_x = 2u_\xi,\quad u_y = u_\xi + u_\eta,$$ $$u_{xx} = 4u_{\xi\xi},\quad u_{xy} = 2(u_{\xi\xi} + u_{\xi\eta}),\quad u_{yy} = u_{\xi\xi} + 2u_{\xi\eta} + u_{\eta\eta}.$$

代入原方程：

$$4u_{\xi\xi} - 8(u_{\xi\xi} + u_{\xi\eta}) + 4(u_{\xi\xi} + 2u_{\xi\eta} + u_{\eta\eta}) = 4u_{\eta\eta} = 0.$$

标准型：$u_{\eta\eta} = 0$。积分得通解：

$$u = F(\xi) + \eta G(\xi) = F(y+2x) + y\,G(y+2x),$$

其中 $F, G$ 为任意二阶可微函数。

▶ 第二题解答

$a = 3$，由 d'Alembert 公式：

$$u(x,t) = \frac{1}{2}[\varphi(x+3t) + \varphi(x-3t)] + \frac{1}{6}\int_{x-3t}^{x+3t} \psi(s)\,\dd s.$$

$\varphi(x) = \cos x$，$\psi(x) = 0$，故积分项为零：

$$u(x,t) = \frac{1}{2}[\cos(x+3t) + \cos(x-3t)] = \cos x \cos 3t.$$

（用和差化积 $\cos A + \cos B = 2\cos\frac{A+B}{2}\cos\frac{A-B}{2}$）

$u(x,t) = \cos x \cos 3t$。对任意 $x$，$|u(x,t)| \le |\cos 3t| \le 1$，振幅始终有界。

此解为驻波：空间形状为 $\cos x$，振幅以频率 $3$ 随时间振荡。

▶ 第三题解答

$u(0,t)=0$ 是 Dirichlet 条件，用奇延拓。

$\tilde{\varphi}(x) = \begin{cases} x^2, & x \ge 0 \\ -x^2, & x < 0 \end{cases}$（奇延拓），$\tilde{\psi}(x) \equiv 0$。

$a=1$，由 d'Alembert 公式：$u(x,t) = \frac{1}{2}[\tilde{\varphi}(x+t) + \tilde{\varphi}(x-t)]$。

当 $x \ge t$（波前未到边界）：$x-t \ge 0$，$x+t > 0$，两者皆在正半轴。

$$u(x,t) = \frac{1}{2}[(x+t)^2 + (x-t)^2] = \frac{1}{2}[x^2+2xt+t^2 + x^2-2xt+t^2] = x^2 + t^2.$$

当 $0 < x < t$（已反射）：$x-t < 0$，需用奇延拓值 $\tilde{\varphi}(x-t) = -(t-x)^2$。

$$u(x,t) = \frac{1}{2}[(x+t)^2 - (t-x)^2] = \frac{1}{2}[4xt] = 2xt.$$

验证 $u(0,t) = 0$：当 $x=0$ 时 $x

物理上，初位移 $x^2$ 产生的扰动向右传播，到达 $x=0$ 后被反相弹回，形成 $x^2+t^2$ 到 $2xt$ 的切换。

▶ 第四题解答

$a=1$。对 $x$ 做 Fourier 变换：$\hat{u}_t(\xi,t) = -\xi^2 \hat{u}(\xi,t)$。

解得 $\hat{u}(\xi,t) = \hat{\varphi}(\xi) e^{-\xi^2 t}$。

由已知 $\hat{\varphi}(\xi) = \mathcal{F}[e^{-x^2/2}](\xi) = \sqrt{2\pi}\,e^{-\xi^2/2}$。

故 $\hat{u}(\xi,t) = \sqrt{2\pi}\,e^{-\xi^2/2} \cdot e^{-\xi^2 t} = \sqrt{2\pi}\,e^{-\xi^2(1/2 + t)}$。

Fourier 逆变换：$\mathcal{F}^{-1}[\sqrt{2\pi}\,e^{-\xi^2 \sigma^2/2}] = \frac{1}{\sigma}e^{-x^2/(2\sigma^2)}$，其中 $\sigma^2 = 1+2t$。

$$u(x,t) = \frac{1}{\sqrt{1+2t}} e^{-x^2/(2(1+2t))}.$$

检验：$t=0$ 时 $u(x,0) = e^{-x^2/2}$。当 $t \to \infty$ 时解像 $1/\sqrt{t}$ 一样衰减，同时宽度以 $\sqrt{t}$ 增长——这正是扩散过程的特征。

▶ 第五题解答

$a=1, l=\pi$。分离变量得本征值 $\lambda_n = n^2$，$X_n(x) = \sin nx$（$n=1,2,\ldots$）。

通解：$u(x,t) = \sum_{n=1}^{\infty} (A_n \cos nt + B_n \sin nt) \sin nx$。

由 $u_t(x,0) = 0$ 得 $B_n = 0$。

由 $u(x,0) = x(\pi-x)$：

$$A_n = \frac{2}{\pi}\int_0^\pi x(\pi-x)\sin nx\,\dd x.$$

计算积分：$\int_0^\pi x(\pi-x)\sin nx\,\dd x = \pi\int_0^\pi x\sin nx\,\dd x - \int_0^\pi x^2\sin nx\,\dd x$。

用分部积分或查表：

$$\int_0^\pi x\sin nx\,\dd x = \frac{\pi(-1)^{n+1}}{n},$$ $$\int_0^\pi x^2\sin nx\,\dd x = \frac{2\pi(-1)^{n+1}}{n^2} + \frac{2((-1)^n-1)}{n^3}\pi.$$

整理得：

$$A_n = \frac{4}{n^3\pi}(1-(-1)^n) = \begin{cases} \dfrac{8}{n^3\pi}, & n\ \text{为奇数}, \\[8pt] 0, & n\ \text{为偶数}. \end{cases}$$

令 $n=2k-1$（$k=1,2,\ldots$）：

$$u(x,t) = \frac{8}{\pi}\sum_{k=1}^{\infty} \frac{1}{(2k-1)^3}\sin((2k-1)x)\cos((2k-1)t).$$

系数以 $1/n^3$ 衰减，级数快速收敛。基频模态（$k=1$）主导解的形状。

▶ 第六题解答

证：设 $u_1, u_2$ 为两个解，令 $w = u_1 - u_2$。

则 $w$ 满足：

$$\begin{cases} w_t = w_{xx}, & 0 < x < 1,\ 0 < t < T, \\ w(x,0) = 0, & 0 \le x \le 1, \\ w(0,t) = w(1,t) = 0, & 0 \le t \le T. \end{cases}$$

在抛物边界 $\Gamma_T = \{t=0, 0\le x\le 1\} \cup \{x=0, 0\le t\le T\} \cup \{x=1, 0\le t\le T\}$ 上，$w \equiv 0$。

由极值原理，$w$ 在闭区域 $\bar{Q}_T = [0,1] \times [0,T]$ 上的最大值和最小值必在抛物边界上达到。

故 $\max_{\bar{Q}_T} w = \min_{\bar{Q}_T} w = 0$，从而 $w \equiv 0$ 在 $\bar{Q}_T$ 上。

即 $u_1 = u_2$。唯一性得证。$\square$

▶ 第七题解答

利用 Poisson 积分公式：$u(r,\theta) = \frac{1}{2\pi}\int_0^{2\pi} \frac{1-r^2}{1-2r\cos(\theta-\phi)+r^2}\,f(\phi)\,\dd\phi$。

因为 $f(\theta) = \sin^3\theta = \frac{3}{4}\sin\theta - \frac{1}{4}\sin 3\theta$，且 $\sin n\theta$ 是调和函数 $r^n\sin n\theta$ 的边界值，可利用分离变量法的结论：

$$\lap(r^n\sin n\theta) = \frac{1}{r}\frac{\pd}{\pd r}\left(r\cdot nr^{n-1}\sin n\theta\right) + \frac{1}{r^2}(-n^2 r^n\sin n\theta) = (n^2 - n^2)r^{n-2}\sin n\theta = 0.$$

（直接代 Laplace 极坐标公式验证 $r^n\sin n\theta$ 是调和函数。）

由线性叠加原理：

$$u(r,\theta) = \frac{3}{4}\,r\sin\theta - \frac{1}{4}\,r^3\sin 3\theta.$$

验证边界：$r=1$ 时 $u(1,\theta) = \frac{3}{4}\sin\theta - \frac{1}{4}\sin 3\theta = \sin^3\theta$。✓

验证调和性：两项分别为 $r\sin\theta$ 和 $r^3\sin 3\theta$，都是调和函数。✓

模拟题二（性质理解与深入）

建议用时：120分钟  |  满分：100分

第一题（15分）— 含参分类与化标准型

考虑含参数 $k$ 的二阶方程：

$$u_{xx} + 2k u_{xy} + u_{yy} = 0.$$

(1) 根据 $k$ 的取值，讨论该方程的类型。

(2) 当 $k=0$ 时，化为标准型并写出通解。

第二题（15分）— 半无界波动（Neumann 偶延拓）

求解自由端半无界弦的振动：

$$\begin{cases} u_{tt} = u_{xx}, & x > 0,\ t > 0, \\ u_x(0,t) = 0, & t \ge 0, \\ u(x,0) = 0,\; u_t(x,0) = e^{-x}, & x \ge 0. \end{cases}$$

分别写出 $x \ge t$ 和 $0 < x < t$ 时的解，并验证 $u_x(0,t) = 0$。

第三题（15分）— 分离变量法（热传导方程·Neumann 边界）

求解绝热细杆的热传导问题：

$$\begin{cases} u_t = u_{xx}, & 0 < x < \pi,\ t > 0, \\ u_x(0,t) = u_x(\pi,t) = 0, & t \ge 0, \\ u(x,0) = \cos^2 x, & 0 \le x \le \pi. \end{cases}$$

并求 $\lim_{t\to\infty} u(x,t)$，解释其物理含义。

第四题（15分）— 即时光滑化与无限传播速度

考虑热传导方程 $u_t = u_{xx}$（$x \in \RR$）。

(1) 设初值 $\varphi(x) = H(x)$（Heaviside 阶跃函数：$x\ge 0$ 时 $H(x)=1$，$x<0$ 时 $H(x)=0$）。写出 $t>0$ 时解 $u(x,t)$ 的积分表达式（可用误差函数 $\operatorname{erf}(z) = \frac{2}{\sqrt{\pi}}\int_0^z e^{-s^2}\dd s$ 表示）。

(2) 说明为什么初值不连续但 $t>0$ 时解却是 $C^\infty$ 光滑的。这一性质在波动方程中是否成立？为什么？

第五题（15分）— 上半平面 Dirichlet 问题（Poisson 公式）

求解上半平面的 Laplace 方程 Dirichlet 问题：

$$\begin{cases} u_{xx} + u_{yy} = 0, & y > 0, \\ u(x,0) = \dfrac{1}{1+x^2}, & x \in \RR. \end{cases}$$

（提示：可用 Fourier 变换法或 Poisson 积分公式。）

第六题（10分）— 调和函数的 Liouville 定理

设 $u(x,y)$ 是定义在全平面 $\RR^2$ 上的有界调和函数（即存在 $M>0$ 使得 $|u(x,y)| \le M$ 对所有 $(x,y)$ 成立）。证明 $u$ 必为常数。

第七题（15分）— 阻尼波动方程的能量衰减

考虑带阻尼的波动方程：

$$u_{tt} + 2\gamma u_t = u_{xx}, \quad \gamma > 0,\ x \in \RR.$$

定义能量 $E(t) = \frac{1}{2}\displaystyle\int_{-\infty}^{+\infty} (u_t^2 + u_x^2)\,\dd x$。

(1) 证明 $E'(t) = -2\gamma\displaystyle\int_{-\infty}^{+\infty} u_t^2\,\dd x \le 0$，并由此证明解的唯一性。

(2) 从 $E'(t) \le 0$ 能得出 $E(t) \le E(0)$ 吗？从而推出什么稳定性结论？

模拟题二 · 参考答案
▶ 第一题解答

(1) 分类讨论

$a_{11}=1, a_{12}=k, a_{22}=1$。判别式 $\Delta = k^2 - 1$。

  • $|k| > 1$（即 $k < -1$ 或 $k > 1$）：$\Delta > 0$，双曲型。
  • $|k| = 1$（即 $k = \pm 1$）：$\Delta = 0$，抛物型。
  • $|k| < 1$（即 $-1 < k < 1$）：$\Delta < 0$，椭圆型。

(2) $k=0$ 时，方程为 $u_{xx} + u_{yy} = 0$，即二维 Laplace 方程，已是标准型（椭圆型）。

特征方程 $(\dd y)^2 + (\dd x)^2 = 0$，特征线为复曲线 $y \pm ix = C$。

通解（由复变函数论）：$u(x,y) = F(x+iy) + G(x-iy)$，其中 $F, G$ 为任意解析（或反解析）函数。等价地，$u(x,y) = \operatorname{Re}[f(z)]$，其中 $f$ 是解析函数，$z = x+iy$。

▶ 第二题解答

$u_x(0,t)=0$ 是 Neumann 条件，用偶延拓。

$\tilde{\psi}(x) = e^{-|x|}$（注意不加负号），$\tilde{\varphi}(x) \equiv 0$。

$a=1$，由 d'Alembert 公式：$u(x,t) = \frac{1}{2}\int_{x-t}^{x+t} \tilde{\psi}(s)\,\dd s$。

当 $x \ge t$（$x-t \ge 0$，积分区间全在正半轴）：

$$u(x,t) = \frac{1}{2}\int_{x-t}^{x+t} e^{-s}\,\dd s = \frac{1}{2}\Big[-e^{-s}\Big]_{x-t}^{x+t} = \frac{1}{2}(e^{-(x-t)} - e^{-(x+t)}).$$

当 $0 < x < t$（$x-t < 0$，积分区间跨过原点，用偶延拓 $\tilde{\psi}(s)=e^{-|s|}$）：

$$\begin{aligned} u(x,t) &= \frac{1}{2}\left[\int_{x-t}^{0} e^{s}\,\dd s + \int_{0}^{x+t} e^{-s}\,\dd s\right] \\ &= \frac{1}{2}\Big[1 - e^{x-t} + 1 - e^{-(x+t)}\Big] \\ &= 1 - \frac{1}{2}(e^{-(t-x)} + e^{-(t+x)}). \end{aligned}$$

验证 $u_x(0,t)=0$：

当 $x$ 很小时（$x

$u_x(x,t) = -\frac{1}{2}(e^{-(t-x)} - e^{-(t+x)})$。在 $x=0$ 处 $u_x(0,t) = -\frac{1}{2}(e^{-t} - e^{-t}) = 0$。✓

▶ 第三题解答

Neumann 边界 $u_x(0,t)=u_x(\pi,t)=0$。分离变量：$u(x,t)=X(x)T(t)$。

$X'' + \lambda X = 0$，$X'(0)=X'(\pi)=0$。本征值 $\lambda_n = n^2$，$X_n(x) = \cos nx$，$n=0,1,2,\ldots$（注意包含 $n=0$）。

$T' + n^2 T = 0$ → $T_n(t) = e^{-n^2 t}$（$n \ge 0$，其中 $T_0(t) = 1$）。

通解：$u(x,t) = \frac{A_0}{2} + \sum_{n=1}^{\infty} A_n e^{-n^2 t}\cos nx$。

由初值 $\cos^2 x = \frac{1}{2} + \frac{1}{2}\cos 2x$，直接读出系数：

$$A_0 = 1,\quad A_2 = \frac{1}{2},\quad \text{其余 } A_n = 0.$$

（这里 $\frac{A_0}{2} = \frac{1}{2}$，故 $A_0 = 1$。）

$$u(x,t) = \frac{1}{2} + \frac{1}{2} e^{-4t}\cos 2x.$$

极限行为：$\lim_{t\to\infty} u(x,t) = \frac{1}{2}$。

物理含义：两端绝热的细杆最终趋于均匀温度分布，该均匀温度等于初始温度在一个周期上的平均值 $\frac{1}{\pi}\int_0^\pi \cos^2 x\,\dd x = \frac{1}{2}$。热量无法从边界散逸（绝热），只能在杆内重新分布直至均匀。

▶ 第四题解答

(1) 由 Poisson 公式：

$$u(x,t) = \frac{1}{2\sqrt{\pi t}}\int_{-\infty}^{+\infty} e^{-(x-y)^2/(4t)}\,H(y)\,\dd y = \frac{1}{2\sqrt{\pi t}}\int_{0}^{+\infty} e^{-(x-y)^2/(4t)}\,\dd y.$$

令 $s = \frac{y-x}{2\sqrt{t}}$，$\dd y = 2\sqrt{t}\,\dd s$。当 $y=0$ 时 $s = -\frac{x}{2\sqrt{t}}$；当 $y \to +\infty$ 时 $s \to +\infty$。

$$u(x,t) = \frac{1}{\sqrt{\pi}}\int_{-x/(2\sqrt{t})}^{+\infty} e^{-s^2}\,\dd s = \frac{1}{2}\left[1 + \operatorname{erf}\!\left(\frac{x}{2\sqrt{t}}\right)\right].$$

(2) 即时光滑化

Poisson 公式 $u(x,t) = \int \Phi(x-y,t)\varphi(y)\dd y$ 中，热核 $\Phi(x,t) = \frac{1}{2\sqrt{\pi t}}e^{-x^2/(4t)}$ 对 $t>0$ 是 $C^\infty$ 的。由于对 $x$ 的任意阶导数都可以放到 $\Phi$ 上（$\varphi$ 有界可积，积分与求导可交换），$t>0$ 时 $u$ 是 $C^\infty$ 光滑的，无论初值多么不光滑。

波动方程不成立：d'Alembert 公式 $u = \frac{1}{2}[\varphi(x+t) + \varphi(x-t)]$ 中解直接继承初值的光滑性——若初值仅连续但不可导，解也仅连续不可导（奇性沿特征线传播而不消失）。这是双曲型与抛物型方程的本质区别。

▶ 第五题解答

法一：Poisson 积分公式

上半平面 Poisson 公式：

$$u(x,y) = \frac{1}{\pi}\int_{-\infty}^{+\infty} \frac{y}{(x-\xi)^2 + y^2}\,f(\xi)\,\dd\xi.$$

代入 $f(\xi) = 1/(1+\xi^2)$：$u(x,y) = \frac{y}{\pi}\int_{-\infty}^{+\infty} \frac{1}{(x-\xi)^2+y^2}\cdot\frac{1}{1+\xi^2}\,\dd\xi$。

法二：Fourier 变换法（更简洁）

对 $x$ 做 Fourier 变换：$\hat{u}_{yy} - \xi^2\hat{u} = 0$（$y>0$）。

通解 $\hat{u}(\xi,y) = A(\xi)e^{-|\xi|y}$（舍去 $e^{+|\xi|y}$ 因为 $y\to\infty$ 时的有界性）。

$\hat{u}(\xi,0) = \mathcal{F}[\frac{1}{1+x^2}](\xi) = \pi e^{-|\xi|}$，故 $A(\xi) = \pi e^{-|\xi|}$。

$\hat{u}(\xi,y) = \pi e^{-|\xi|} \cdot e^{-|\xi|y} = \pi e^{-|\xi|(1+y)}$。

逆变换：

$$u(x,y) = \mathcal{F}^{-1}[\pi e^{-|\xi|(1+y)}](x) = \frac{1+y}{x^2 + (1+y)^2}.$$

验证：$y=0$ 时 $u(x,0) = 1/(x^2+1) = 1/(1+x^2)$。✓

当 $y\to\infty$ 时 $u(x,y)\to 0$（无穷远处调和函数衰减）。

▶ 第六题解答

证：任取 $(x_0, y_0) \in \RR^2$。对任意 $R > 0$，在以 $(x_0, y_0)$ 为心、$R$ 为半径的圆盘上对 $u$ 应用 Poisson 积分公式。

由平均值定理的 Poisson 积分形式（或直接用圆盘上的 Poisson 公式）：

$$u(x_0, y_0) = \frac{1}{2\pi}\int_0^{2\pi} u(x_0 + R\cos\theta, y_0 + R\sin\theta)\,\dd\theta.$$

（调和函数在圆周上的平均值等于圆心处的值。）

设 $|u| \le M$。任取另一点 $(x_1, y_1)$，考虑半径为 $R$ 的两个大圆盘，它们的对称差区域的面积与圆盘面积之比在 $R\to\infty$ 时趋于零。

更严谨地，由平均值定理：

$$u(x_0,y_0) - u(x_1,y_1) = \frac{1}{\pi R^2}\left(\iint_{B_R(x_0,y_0)} u\,\dd x\dd y - \iint_{B_R(x_1,y_1)} u\,\dd x\dd y\right).$$

两个以不同点为心的半径为 $R$ 的圆盘，当 $R \to \infty$ 时对称差面积 $= O(R)$，而圆盘面积 $= \pi R^2$。

$$|u(x_0,y_0) - u(x_1,y_1)| \le \frac{M \cdot O(R)}{\pi R^2} \to 0 \quad (R \to \infty).$$

故 $u(x_0,y_0) = u(x_1,y_1)$ 对任意两点成立，$u$ 为常数。$\square$

▶ 第七题解答

(1) 计算 $E'(t)$：

$$E'(t) = \int_{-\infty}^{+\infty} (u_t u_{tt} + u_x u_{xt})\,\dd x.$$

代入 $u_{tt} = u_{xx} - 2\gamma u_t$：

$$E'(t) = \int_{-\infty}^{+\infty} [u_t(u_{xx} - 2\gamma u_t) + u_x u_{xt}]\,\dd x.$$

第一项与第三项合并：$\int (u_t u_{xx} + u_x u_{xt})\,\dd x = \int \frac{\pd}{\pd x}(u_t u_x)\,\dd x = [u_t u_x]_{-\infty}^{+\infty} = 0$（无穷远处 $u_t, u_x \to 0$）。

故 $E'(t) = -2\gamma\int_{-\infty}^{+\infty} u_t^2\,\dd x \le 0$。✓

唯一性证明：设 $u_1, u_2$ 为两解，$w = u_1-u_2$ 满足齐次方程 + 零初值。$E_w(0) = 0$，由 $E'(t) \le 0$ 得 $E(t) \le E(0) = 0$，又 $E(t) \ge 0$，故 $E(t) \equiv 0$。从而 $w_t \equiv 0, w_x \equiv 0$，$w \equiv$ 常数。由 $w(x,0) = 0$ 得 $w \equiv 0$，即 $u_1 = u_2$。

(2) 是。由 $E'(t) \le 0$ 积分得 $E(t) \le E(0)$。这意味着$u_t^2$ 和 $u_x^2$ 的积分被初始能量控制。由此推出：

$$\|u_t(\cdot,t)\|_{L^2}^2 + \|u_x(\cdot,t)\|_{L^2}^2 \le \|u_t(\cdot,0)\|_{L^2}^2 + \|u_x(\cdot,0)\|_{L^2}^2,$$

即解连续依赖于初始数据（Lyapunov 稳定性）。结合存在性和唯一性，该问题是适定的。

▶ 第一题解答

非齐次项 $f(x) = \sin x$ 不依赖 $t$。先找稳态特解 $v(x)$ 满足 $v_{xx} + \sin x = 0$（设 $v_{tt}=0$），即 $v''(x) = -\sin x$，$v'(x) = \cos x$，$v(x) = \sin x$。

令 $u(x,t) = v(x) + w(x,t) = \sin x + w(x,t)$。

$w$ 满足齐次方程 $w_{tt} = w_{xx}$ 和初始条件：

$$w(x,0) = u(x,0) - v(x) = -\sin x,\quad w_t(x,0) = 0.$$

由 d'Alembert 公式（$a=1$）：

$$w(x,t) = \frac{1}{2}[-\sin(x+t) - \sin(x-t)] = -\sin x \cos t.$$

（用 $\sin(x+t)+\sin(x-t)=2\sin x\cos t$。）

故原问题的解为：

$$u(x,t) = \sin x - \sin x \cos t = \sin x(1 - \cos t).$$

验证：$u(x,0)=0$。$u_t(x,0)=\sin x \cdot 0 = 0$。$u_{tt} = \sin x \cos t$，$u_{xx} = -\sin x(1-\cos t)$，$u_{tt} - u_{xx} = \sin x\cos t + \sin x - \sin x\cos t = \sin x$。✓

▶ 第二题解答

(1) $a=2$。特征线：$x \pm 2t = \text{const}$。

通解（d'Alembert 形式）：$u(x,t) = F(x+2t) + G(x-2t)$，其中 $F, G$ 为任意二阶可微函数。

(2) 依赖区间：通过 $(3,1)$ 的两条特征线向 $t=0$ 追溯：

$x + 2t = 3+2 = 5$（令 $t=0$ 得 $x=5$），$x - 2t = 3-2 = 1$（令 $t=0$ 得 $x=1$）。

依赖区间为 $[1, 5]$。解 $u(3,1)$ 只取决于初始数据在区间 $[1, 5]$ 上的值。

(3) 初始区间 $[-1, 1]$ 的影响区域：

过 $(-1, 0)$ 的右行特征线：$x - 2t = -1$ → 右边界 $x = 2t-1$。

过 $(1, 0)$ 的左行特征线：$x + 2t = 1$ → 左边界 $x = 1-2t$。

影响区域为 $\{(x,t) : t \ge 0,\ 2t-1 \le x \le 1-2t\}$，即向前特征线围成的三角形区域。当 $t > 0.5$ 时 $2t-1 > 1-2t$，影响区域消失——扰动在 $t=0.5$ 时刻完全"通过"并传播出去。

▶ 第三题解答

Dirichlet 条件 $u(0,t)=0$，用奇延拓。

$\tilde{\varphi}(x) = \operatorname{sgn}(x) = \begin{cases} 1, & x > 0 \\ -1, & x < 0 \end{cases}$

延拓后的解：$u(x,t) = \frac{1}{2\sqrt{\pi t}}\int_{-\infty}^{+\infty} e^{-(x-y)^2/(4t)}\,\tilde{\varphi}(y)\,\dd y$。

利用奇延拓的半无界公式（附录2·C.2节）：

$$u(x,t) = \int_0^{+\infty} [\Phi(x-y,t) - \Phi(x+y,t)] \cdot 1\,\dd y = \int_0^{+\infty} \Phi(x-y,t)\,\dd y - \int_0^{+\infty} \Phi(x+y,t)\,\dd y.$$

第一项：$\int_0^\infty \Phi(x-y,t)\dd y = \frac{1}{2\sqrt{\pi t}}\int_0^\infty e^{-(x-y)^2/(4t)}\dd y$。令 $s = \frac{x-y}{2\sqrt{t}}$，$y=0 \to s=\frac{x}{2\sqrt{t}}$，$y\to\infty \to s\to -\infty$。

$$= \frac{1}{\sqrt{\pi}}\int_{-\infty}^{x/(2\sqrt{t})} e^{-s^2}\dd s = \frac{1}{2}\left[1 + \operatorname{erf}\!\left(\frac{x}{2\sqrt{t}}\right)\right].$$

第二项：$\int_0^\infty \Phi(x+y,t)\dd y = \frac{1}{2\sqrt{\pi t}}\int_0^\infty e^{-(x+y)^2/(4t)}\dd y$。令 $s = \frac{x+y}{2\sqrt{t}}$。

$$= \frac{1}{\sqrt{\pi}}\int_{x/(2\sqrt{t})}^{\infty} e^{-s^2}\dd s = \frac{1}{2}\left[1 - \operatorname{erf}\!\left(\frac{x}{2\sqrt{t}}\right)\right].$$

相减得：

$$u(x,t) = \operatorname{erf}\!\left(\frac{x}{2\sqrt{t}}\right).$$

验证：$x=0$ 时 $u(0,t) = \operatorname{erf}(0) = 0$。$t \to 0^+$：$\frac{x}{2\sqrt{t}} \to +\infty$（$x>0$），$\operatorname{erf}(+\infty) = 1 = u(x,0)$。✓

▶ 第四题解答

将解按本征函数 $\sin nx$ 展开（两端固定弦的本征函数）：

$$u(x,t) = \sum_{n=1}^{\infty} T_n(t)\sin nx.$$

非齐次项 $\sin 2x \sin t$ 已经是 $n=2$ 的本征函数形式。

代入方程：$\sum (T_n'' + n^2 T_n)\sin nx = \sin 2x \sin t$。

比较系数：

  • $n \neq 2$：$T_n'' + n^2 T_n = 0$，$T_n(0)=T_n'(0)=0$ → $T_n(t) \equiv 0$。
  • $n = 2$：$T_2'' + 4T_2 = \sin t$，$T_2(0)=T_2'(0)=0$。

解 $T_2$：齐次通解 $T_2^h = C_1\cos 2t + C_2\sin 2t$。

特解尝试 $T_2^p = A\sin t$（注意 $\sin t$ 不是齐次解，故直接代入）：

$(-A + 4A)\sin t = \sin t$ → $3A = 1$ → $A = \frac{1}{3}$。

$T_2(t) = C_1\cos 2t + C_2\sin 2t + \frac{1}{3}\sin t$。

$T_2(0) = C_1 = 0$ → $C_1 = 0$。

$T_2'(0) = 2C_2 + \frac{1}{3} = 0$ → $C_2 = -\frac{1}{6}$。

故 $T_2(t) = \frac{1}{3}\sin t - \frac{1}{6}\sin 2t$。

$$u(x,t) = \left(\frac{1}{3}\sin t - \frac{1}{6}\sin 2t\right)\sin 2x.$$

物理含义：外力 $\sin 2x\sin t$ 的频率为 $1$，而系统第二固有频率为 $n=2 \to \omega_2 = 2$。外力频率未命中共振频率，故解保持有界。若外力频率等于 $2$（即 $\sin 2x\sin 2t$），则会发生共振。

▶ 第五题解答

(1) 用 Harnack 不等式证明

Harnack 不等式：设 $u \ge 0$ 是 $\RR^n$ 中区域 $\Omega$ 上的调和函数，则对 $\Omega$ 的任意紧子集 $K$，存在常数 $C_K$（只依赖于 $K$ 和 $n$）使得 $\sup_K u \le C_K \inf_K u$。

取 $\Omega = B_R(0)$（半径为 $R$ 的球），$K = \overline{B_{R/2}(0)}$。对非负调和函数 $u$ 应用 Harnack 不等式（常数 $C$ 与 $R$ 无关，由尺度不变性 $C$ 仅取决于维数 $n$）：

$$\sup_{B_{R/2}} u \le C \inf_{B_{R/2}} u.$$

令 $R \to \infty$：$\sup_{\RR^n} u \le C \inf_{\RR^n} u$。因为 $\inf_{\RR^n} u \ge 0$，若 $\inf u = 0$ 则 $u \equiv 0$（常数）；若 $\inf u > 0$，则 $\sup u \le C\inf u$，令 $x_1, x_2$ 任意，考虑以它们为心的球并令 $R\to\infty$：

更标准的方法是：对 $x, y \in \RR^n$ 任意，$|u(x) - u(y)| \le$ ... 实际上用 Harnack 的另一个推论：

$$u(x) \le C u(0),\quad u(y) \le C u(0),\quad \forall x,y.$$

交换 $x,y$ 角色，得 $u(x)/C \le u(y) \le C u(x)$。令 $x,y$ 任意，可得 $\frac{\sup u}{\inf u} \le C$。再考虑 $u_R(x) = u(Rx)$，调和且非负。对任意两点 $x,y$，当 $R$ 足够大时它们都属于 $B_{R/2}$，由尺度不变性 $\frac{u(Rx)}{u(Ry)} \le C$。但 $R$ 可以取任意大，令 $R\to\infty$ 并用调和函数的平移不变性可以推出 $u(x)=u(y)$。详细论证略。

结论：$\RR^n$ 上非负（或有下界）的调和函数必为常数。

(2) 用平均值定理的替代思路

对任意 $x_1, x_2 \in \RR^n$，考虑它们球心处的平均值表示：

$$u(x_1) = \frac{1}{|B_R(x_1)|}\int_{B_R(x_1)} u(y)\,\dd y, \quad u(x_2) = \frac{1}{|B_R(x_2)|}\int_{B_R(x_2)} u(y)\,\dd y.$$

当 $R \to \infty$ 时，$B_R(x_1)$ 和 $B_R(x_2)$ 的对称差与球体积之比趋于零。由于 $u \ge 0$，两球体积分之差受对称差区域控制，令 $R \to \infty$ 即得 $u(x_1) = u(x_2)$。

▶ 第六题解答

情况 $\lambda = 0$：$X(x) = Ax + B$。$X(0)=B=0$，$X'(l)=A=0$，仅有平凡解。

情况 $\lambda < 0$：令 $\lambda = -\mu^2$（$\mu > 0$）。$X(x) = A\cosh\mu x + B\sinh\mu x$。

$X(0) = A = 0$。$X'(l) = \mu B\cosh\mu l = 0$，$\cosh\mu l > 0$ 故 $B=0$。仅有平凡解。

情况 $\lambda > 0$：令 $\lambda = \omega^2$（$\omega > 0$）。$X(x) = A\cos\omega x + B\sin\omega x$。

$X(0) = A = 0$。$X'(l) = \omega B\cos\omega l = 0$。因 $B \neq 0$（否则平凡解），$\cos\omega l = 0$。

$\omega l = \frac{\pi}{2} + k\pi = \frac{(2k+1)\pi}{2}$（$k = 0, 1, 2, \ldots$）。

$\omega_n = \frac{(2n-1)\pi}{2l}$，$n = 1, 2, 3, \ldots$

$$\lambda_n = \left[\frac{(2n-1)\pi}{2l}\right]^2,\quad X_n(x) = \sin\frac{(2n-1)\pi x}{2l},\quad n = 1, 2, 3, \ldots$$

正交性验证：$n \neq m$ 时，

$$\begin{aligned} \int_0^l X_n X_m\,\dd x &= \int_0^l \sin\frac{(2n-1)\pi x}{2l} \sin\frac{(2m-1)\pi x}{2l}\,\dd x \\ &= \frac{1}{2}\int_0^l \left[\cos\frac{(n-m)\pi x}{l} - \cos\frac{(n+m-1)\pi x}{l}\right]\dd x \\ &= \frac{1}{2}\left[\frac{l}{(n-m)\pi}\sin\frac{(n-m)\pi x}{l} - \frac{l}{(n+m-1)\pi}\sin\frac{(n+m-1)\pi x}{l}\right]_0^l = 0. \end{aligned}$$

（两个边界项都为零，因为 $\sin(k\pi)=0$。）正交性得证。$\square$

▶ 第七题解答

计算 $E'(t)$：

$$E'(t) = \int_0^\pi (u_t u_{tt} + u_x u_{xt})\,\dd x.$$

代入 $u_{tt} = u_{xx} - u_t$：

$$E'(t) = \int_0^\pi [u_t(u_{xx} - u_t) + u_x u_{xt}]\,\dd x = \int_0^\pi (u_t u_{xx} + u_x u_{xt})\,\dd x - \int_0^\pi u_t^2\,\dd x.$$

第一项分部积分：$\int_0^\pi (u_t u_{xx} + u_x u_{xt})\,\dd x = \int_0^\pi \frac{\pd}{\pd x}(u_t u_x)\,\dd x = [u_t u_x]_0^\pi$。

由 $u(0,t)=u(\pi,t)=0$ 得 $u_t(0,t)=u_t(\pi,t)=0$，故边界项 $=0$。

$E'(t) = -\int_0^\pi u_t^2\,\dd x \le 0$。

积分得 $E(t) - E(0) = -\int_0^t \int_0^\pi u_t^2\,\dd x\dd\tau \le 0$，故 $E(t) \le E(0)$。

稳定性（连续依赖性）：

设 $u$ 为具有初值 $(\varphi, \psi)$ 的解，$\tilde{u}$ 为具有初值 $(\tilde{\varphi}, \tilde{\psi})$ 的解。令 $w = u - \tilde{u}$。

$w$ 满足相同阻尼波动方程，初值为 $(\varphi-\tilde{\varphi}, \psi-\tilde{\psi})$。由能量不等式：

$$E_w(t) \le E_w(0) = \frac{1}{2}\int_0^\pi [(\psi-\tilde{\psi})^2 + (\varphi'-\tilde{\varphi}')^2]\,\dd x.$$

再由 Poincaré 不等式（因 $w(0,t)=0$）：$\int_0^\pi w^2\,\dd x \le \frac{4}{\pi^2}\int_0^\pi w_x^2\,\dd x \le \frac{8}{\pi^2}E_w(t)$。

故 $\|w(\cdot,t)\|_{L^2} \le C\sqrt{E_w(0)}$，即解的 $L^2$ 偏差被初始数据的偏差控制。这证明了解连续依赖于初始数据（稳定性）。$\square$
