5.1 为什么要单独研究一阶PDE？

本章动机

在前四章中，我们主要研究二阶PDE（波动、热传导、Laplace）。但一阶PDE远非"更简单的二阶PDE"——它们有自己独特而深刻的理论：

  • 特征线法是最自然的几何方法——沿特征线，PDE退化为ODE
  • 一阶PDE的解可以用任意函数而非任意常数来表达
  • 特征线法直接导致了二阶双曲型方程的d'Alembert解法（本质上是特征线法的二阶推广）
  • 一阶双曲型方程组是气体动力学、浅水波等领域的核心模型

5.2 一阶拟线性方程：特征线法

定义 5.1（一阶拟线性方程）

形如 $a(x,y,u)\frac{\pd u}{\pd x} + b(x,y,u)\frac{\pd u}{\pd y} = c(x,y,u)$ 的方程称为一阶拟线性方程。对最高阶导数 $u_x, u_y$ 是线性的，但系数 $a,b,c$ 可依赖于 $u$ 本身。

几何直觉

把解 $u(x,y)$ 想象成 $(x,y,u)$ 空间中的一个曲面。方程 $au_x + bu_y = c$ 表示：曲面在每点的法向量 $(u_x, u_y, -1)$ 与向量 $(a, b, c)$ 垂直。换句话说，向量 $(a,b,c)$ 处处与解曲面相切。

因此，解曲面由处处与向量场 $(a,b,c)$ 相切的曲线（即特征线）织成。沿特征线追踪 $x,y,u$ 的变化，PDE就变成了ODE！

特征线法的核心公式

特征方程组
$$\frac{\dd x}{a(x,y,u)} = \frac{\dd y}{b(x,y,u)} = \frac{\dd u}{c(x,y,u)}.$$

等价地：$\frac{\dd x}{\dd s} = a,\; \frac{\dd y}{\dd s} = b,\; \frac{\dd u}{\dd s} = c$（以 $s$ 为参数）。

定理 5.1（通解结构）

若从特征方程组求得两个独立首次积分 $\Phi_1(x,y,u)=C_1$，$\Phi_2(x,y,u)=C_2$，则通解为 $\Phi_1 = F(\Phi_2)$，其中 $F$ 为任意可微函数。

例题 5.1

求 $u\frac{\pd u}{\pd x} + \frac{\pd u}{\pd y} = 1$ 的通解。

▶ 详细解答

$a=u, b=1, c=1$。特征方程组：$\frac{\dd x}{u} = \frac{\dd y}{1} = \frac{\dd u}{1}$。

由 $\dd y = \dd u$ → $\Phi_1 = u - y = C_1$。

由 $\frac{\dd x}{u} = \frac{\dd u}{1}$ 且 $u = y + C_1$... 更直接：$\dd x = u\,\dd u$ → $x = u^2/2 + C_2$ → $\Phi_2 = x - u^2/2 = C_2$。

通解：$x - u^2/2 = F(u - y)$，$F$ 为任意可微函数。

例题 5.2

求 $x\frac{\pd u}{\pd x} + y\frac{\pd u}{\pd y} = u$ 满足 $u(x,1) = x^2$ 的特解。

▶ 详细解答

特征方程组：$\frac{\dd x}{x} = \frac{\dd y}{y} = \frac{\dd u}{u}$。

首次积分：$\Phi_1 = x/y = C_1$，$\Phi_2 = u/x = C_2$。

通解：$u/x = F(x/y)$ → $u = xF(x/y)$。

代入初始条件 $u(x,1)=x^2$：$xF(x)=x^2$ → $F(x)=x$。

特解：$u = x \cdot (x/y) = x^2/y$。

5.3 Cauchy问题与特征线法

Cauchy问题的提法

对一阶拟线性方程，给定初始曲线 $\Gamma$：$x=x_0(s), y=y_0(s), u=u_0(s)$。求在 $\Gamma$ 上取给定值且满足方程的解 $u(x,y)$。

求解步骤：以 $\Gamma$ 上的点为初始值，沿特征方向积分ODE → 得到参数化的解 → 消去参数得 $u(x,y)$。

例题 5.3

求 $u_x + u_y = u$，$u(x,0) = \cos x$。

▶ 详细解答

特征方程组：$\frac{\dd x}{1} = \frac{\dd y}{1} = \frac{\dd u}{u}$。

设 $\dd x = \dd y = \dd t$，初始值 $x(0)=s, y(0)=0, u(0)=\cos s$。

$x = s+t,\; y = t,\; u = \cos s \cdot e^t$。

消参数：$t=y, s=x-y$ → $u(x,y) = \cos(x-y)e^y$。

定理 5.2（横截条件）

Cauchy问题有唯一解的必要条件：初始曲线 $\Gamma$ 不是方程的特征线，即

$$\frac{\dd x_0}{\dd s} \cdot b - \frac{\dd y_0}{\dd s} \cdot a \neq 0.$$

这等价于 $\Gamma$ 与特征方向横截（不平行）——几何上，特征线从 $\Gamma$ "发出"，织成解曲面。

5.4 一阶完全非线性方程：Charpit方法

定义 5.2（完全非线性一阶方程）
$$F(x, y, u, p, q) = 0, \quad p = \frac{\pd u}{\pd x},\; q = \frac{\pd u}{\pd y},$$

且 $F$ 对 $p, q$ 为非线性。

Charpit 特征方程组
$$\frac{\dd x}{F_p} = \frac{\dd y}{F_q} = \frac{\dd u}{pF_p + qF_q} = \frac{-\dd p}{F_x + pF_u} = \frac{-\dd q}{F_y + qF_u}.$$

求解思路：由 $F=0$ 和特征方程组联立，先求出 $p, q$（含参数 $a$），再积分 $\dd u = p\,\dd x + q\,\dd y$ 得 $u$。

例题 5.4

求 $u_x^2 + u_y^2 = 1$ 的全积分。

▶ 详细解答

$F = p^2 + q^2 - 1 = 0$。$F_p = 2p, F_q = 2q, F_x=F_y=F_u=0$。

由 $\dd p = 0$ 和 $\dd q = 0$ 得 $p = a$（常数）。代入 $F=0$：$q = \pm\sqrt{1-a^2}$（$|a| \le 1$）。

积分 $\dd u = a\,\dd x \pm \sqrt{1-a^2}\,\dd y$ →

$$u = ax \pm \sqrt{1-a^2}\,y + b.$$

这是含两个参数 $a, b$ 的全积分。

5.5 全积分、通解与奇解

定义 5.3（全积分）

含两个独立参数 $a, b$ 的解族 $u = \varphi(x, y, a, b)$，满足 Jacobi 条件 $\frac{\pd(\varphi_a, \varphi_b)}{\pd(x, y)} \neq 0$。

三者关系：全积分→令 $b=F(a)$ 包络→通解（含一个任意函数）；全积分→固定 $a,b$→特解；通解再包络→奇解（不含任意元素）。

5.6 Cauchy-Kovalevskaya 定理

定理 5.3（Cauchy-Kovalevskaya）

若PDE可写成对某个自变量的最高阶导数的显式形式，且所有已知函数和初始数据都是实解析的，则Cauchy问题在初始超平面附近存在唯一的实解析解。

定理的意义与局限
  • 这是PDE的局部存在性定理——仅在初始面附近保证解存在；
  • 要求解析性——比连续可微强得多。Hadamard反例说明仅 $C^\infty$ 不足以保证解存在；
  • 实际中更常用特征线法、分离变量法等构造性方法。

5.7 一阶双曲型方程组

为什么需要方程组？

第一章的波动方程 $u_{tt} = a^2 u_{xx}$ 等价于一阶双曲型方程组：引入 $v = u_t, w = a u_x$，则

$$\begin{cases} v_t - a w_x = 0, \\ w_t - a v_x = 0. \end{cases}$$

写成矩阵形式：$\frac{\pd}{\pd t}\begin{pmatrix}v \\ w\end{pmatrix} + \begin{pmatrix}0 & -a \\ -a & 0\end{pmatrix}\frac{\pd}{\pd x}\begin{pmatrix}v \\ w\end{pmatrix} = 0$。

矩阵 $\begin{pmatrix}0 & -a \\ -a & 0\end{pmatrix}$ 有实特征值 $\pm a$ → 双曲型。这是 d'Alembert 公式中左右行波 $f(x\pm at)$ 的代数根源。

定义 5.4（一阶双曲型方程组）

一阶方程组 $\frac{\pd \mathbf{u}}{\pd t} + A\frac{\pd \mathbf{u}}{\pd x} = 0$ 称为双曲型，如果 $A$ 的所有特征值为实数且有完备的特征向量系。

核心思想

双曲型方程组可以通过对角化化为若干个独立的对流方程（每个沿一个特征方向传播）。这就是第一章特征线法的推广——一阶双曲型方程组描述了多个波模式沿各自的特征速度独立传播。

本章总结

核心方法：特征线法——将一阶PDE化为沿特征方向的ODE（首次积分）。

解法层级：拟线性→特征方程组+首次积分；完全非线性→Charpit方法；方程组→对角化+特征分解。

与二阶方程的联系：一阶双曲型方程组的特征线理论是第一章 d'Alembert 解法的一般化——二阶波动方程等价于一阶双曲型方程组。
