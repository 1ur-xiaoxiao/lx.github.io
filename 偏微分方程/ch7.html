第七章 · 偏微分方程的数值解

当解析解难以求得时——有限差分法、有限元法与谱方法的入门

7.1 为什么需要数值解？

解析解的局限性

前六章我们学习了各种解析方法——d'Alembert公式、Fourier变换、分离变量法、Green函数法。但现实是：

  • 绝大多数实际PDE无法求得解析解——区域不规则、系数变系数、边界条件复杂；
  • 即使解能用级数表示，计算前几项可能也需要数值积分；
  • 工程和科学计算中的PDE基本都依赖数值方法。

本章介绍三种最重要的数值方法，让你理解"计算机是如何求解PDE的"。

7.2 有限差分法

核心思想：用差商代替微商

微积分的基本思想：$\frac{\pd u}{\pd x} \approx \frac{u(x+\Delta x) - u(x)}{\Delta x}$。有限差分法就是把整个偏微分方程中的所有导数都替换为差商近似，在网格点上离散化。

一维热传导方程的例子

考虑 $u_t = a^2 u_{xx}$（$0 < x < L, t > 0$）。

离散化：将空间离散为 $x_j = j\Delta x$（$j = 0, 1, \ldots, N$），时间离散为 $t_n = n\Delta t$。记 $u_j^n \approx u(x_j, t_n)$。

显式差分格式（FTCS）

时间导数：$\frac{\pd u}{\pd t} \approx \frac{u_j^{n+1} - u_j^n}{\Delta t}$（前向差分）

空间导数：$\frac{\pd^2 u}{\pd x^2} \approx \frac{u_{j+1}^n - 2u_j^n + u_{j-1}^n}{(\Delta x)^2}$（中心差分）

代入方程得迭代公式：

$$u_j^{n+1} = u_j^n + r(u_{j+1}^n - 2u_j^n + u_{j-1}^n), \quad r = \frac{a^2\Delta t}{(\Delta x)^2}.$$
稳定性条件

显式格式不是无条件稳定的——必须满足 CFL 条件：$r \le 1/2$。如果违反，数值解会指数爆炸！这是有限差分法中最需要警惕的问题。

隐式格式（如 Crank-Nicolson）无条件稳定，但每步需求解线性方程组——精度和计算代价之间的权衡是数值分析的核心问题。

波动方程：特征线法的离散对应

对 $u_{tt} = a^2 u_{xx}$，离散格式的自然推广是：

$$\frac{u_j^{n+1} - 2u_j^n + u_j^{n-1}}{(\Delta t)^2} = a^2\frac{u_{j+1}^n - 2u_j^n + u_{j-1}^n}{(\Delta x)^2}.$$

CFL条件：$\Delta t \le \Delta x / a$——信息在数值网格上的传播速度不能超过物理波速！这与第一章的特征线理论完美对应。

CFL条件 = 数值特征线

CFL条件 $\Delta t \le \Delta x/a$ 的几何意义：数值依赖区间必须包含物理依赖区间。如果违背，数值方法就无法正确传递信息——这再次印证了特征线理论的深刻性。

7.3 有限元法思想

从弱形式出发

有限元法（FEM）基于第六章的弱形式。以 Poisson 方程 $-\lap u = f$（Dirichlet条件）为例：

弱形式：$\int_\Omega \grad u \cdot \grad v\,\dd x = \int_\Omega f v\,\dd x$，对所有 $v$。

核心步骤：

  1. 将区域 $\Omega$ 划分为有限个"单元"（三角形、四边形等）——网格剖分；
  2. 在每个单元上用简单函数（如分片线性函数）逼近 $u$；
  3. 代入弱形式，将PDE转化为大型稀疏线性方程组；
  4. 求解线性方程组得到节点上的近似值。
为什么FEM如此强大？
  • 适应复杂几何：三角形网格可以拟合任意形状的区域（有限差分需要规则网格）；
  • 自然的误差估计：弱形式提供了误差分析的框架；
  • 处理复杂边界条件：Neumann条件在弱形式中自然出现，无需特殊处理；
  • 数学基础严密：Sobolev空间 + Lax-Milgram定理保证了解的存在唯一性。

7.4 谱方法简介

如果说有限差分法用局部多项式逼近，有限元法用分片多项式逼近，那么谱方法则用全局函数（如Fourier级数或Chebyshev多项式）逼近。

Fourier谱方法的思想

将解表示为 $u(x,t) \approx \sum_{k=-N/2}^{N/2} \hat{u}_k(t) e^{ikx}$。代入PDE后，利用正交性得到关于系数 $\hat{u}_k(t)$ 的ODE系统。

空间导数的计算极为简单：$\frac{\pd}{\pd x}e^{ikx} = ik e^{ikx}$——在谱空间，导数就是乘以 $ik$！

优势：对光滑解，精度随 $N$ 指数收敛（远比有限差分/有限元的代数收敛快）。劣势：要求区域规则（通常是矩形或周期域）、对解的间断敏感（Gibbs现象）。

7.5 方法选择与总结

三种方法的选择指南
方法适用范围精度实现难度
有限差分法规则区域、简单边界中（代数收敛）最简单
有限元法任意区域、复杂边界中（代数收敛）中等
谱方法规则区域、光滑解极高（指数收敛）较复杂

学习建议：从有限差分法开始理解离散化的基本思想（最简单、最直观），然后学习有限元法处理复杂区域，最后了解谱方法的高精度优势。

与前六章的联系：数值方法的每一步都有解析理论的影子——CFL条件是特征线理论的数值对应，FEM的弱形式来自Green公式和变分原理，谱方法本质上就是我们对波动方程和热传导方程所做的Fourier级数求解的数字版本。
