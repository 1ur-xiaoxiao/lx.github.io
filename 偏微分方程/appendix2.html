A. PDE 分类与结构（第零章 + 第四章）

A.1 三类典型方程

三大经典方程的标准形式
方程标准形式类型物理背景
波动方程$u_{tt} = a^2 \lap u$双曲型弦振动、声波、电磁波
热传导方程$u_t = a^2 \lap u$抛物型热扩散、扩散过程
Laplace 方程$\lap u = 0$椭圆型稳态温度、静电场、不可压缩流
Poisson 方程$\lap u = f$椭圆型有源稳态问题

A.2 PDE 分类（按线性性）

二阶PDE一般形式
$$a_{11}u_{xx} + 2a_{12}u_{xy} + a_{22}u_{yy} + F(x,y,u,u_x,u_y) = 0$$
  • 线性：$a_{ij}$ 只依赖于 $(x,y)$，$F = b_1 u_x + b_2 u_y + c u + f$（$u$ 及其导数都是一次的）
  • 半线性：$a_{ij}$ 只依赖于 $(x,y)$，$F$ 可非线性依赖于 $u$（而不依赖于 $u$ 的各阶导数）
  • 拟线性：$a_{ij}$ 可依赖于 $u$ 及其一阶导数，但最高阶导数仍是线性的
  • 完全非线性：最高阶导数以非线性形式出现

A.3 二阶方程判别式分类（第四章核心）

判别式 $\Delta = a_{12}^2 - a_{11}a_{22}$
判别式类型标准型代表性方程
$\Delta > 0$双曲型$u_{\xi\eta}$ 或 $u_{\xi\xi} - u_{\eta\eta}$$u_{tt} = a^2 u_{xx}$
$\Delta = 0$抛物型$u_{\xi\xi}$$u_t = a^2 u_{xx}$
$\Delta < 0$椭圆型$u_{\xi\xi} + u_{\eta\eta}$$u_{xx} + u_{yy} = 0$

A.4 化标准型步骤

两步走
  1. 求特征方程：$a_{11}(\dd y)^2 - 2a_{12}\dd x\dd y + a_{22}(\dd x)^2 = 0$，解得特征曲线 $\varphi(x,y)=C_1, \psi(x,y)=C_2$。
  2. 变量代换：令 $\xi = \varphi(x,y), \eta = \psi(x,y)$，代入原方程化简。
    • 双曲型（$\Delta>0$）：两族实特征线 → 标准型 $u_{\xi\eta} = \cdots$
    • 抛物型（$\Delta=0$）：一族实特征线 → 标准型 $u_{\xi\xi} = \cdots$（再取 $\eta$ 使 Jacobi $\neq 0$）
    • 椭圆型（$\Delta<0$）：复特征线 → 取实部和虚部 → 标准型 $u_{\xi\xi}+u_{\eta\eta} = \cdots$

A.5 定解条件

三类边界条件
类型形式物理含义
Dirichlet（第一类）$u|_{\pd\Omega} = g$边界值已知（固定端、恒温边界）
Neumann（第二类）$\frac{\pd u}{\pd n}\big|_{\pd\Omega} = g$边界法向导数已知（自由端、绝热边界）
Robin（第三类）$\big(\frac{\pd u}{\pd n} + \alpha u\big)\big|_{\pd\Omega} = g$边界热交换（牛顿冷却定律）

A.6 适定性（Hadamard）

一个PDE问题是适定的，当且仅当：

  1. 解存在（Existence）
  2. 解唯一（Uniqueness）
  3. 解连续依赖于定解条件（Stability）

三者缺一不可。反例：Laplace方程的反问题（已知边界法向导数求内部值）是不适定的。

B. 波动方程（第一章）

B.1 一维波动方程 Cauchy 问题

d'Alembert 公式（全空间）
$$\begin{cases} u_{tt} = a^2 u_{xx}, & x \in \RR,\ t > 0, \\ u(x,0) = \varphi(x),\; u_t(x,0) = \psi(x), & x \in \RR. \end{cases}$$ $$u(x,t) = \frac{1}{2}[\varphi(x+at) + \varphi(x-at)] + \frac{1}{2a}\int_{x-at}^{x+at} \psi(s)\,\dd s.$$

解由右行波 $\varphi(x-at)$ 和左行波 $\varphi(x+at)$ 叠加而成。

非齐次一维波动方程 Cauchy 问题（Duhamel 原理 / 齐次化原理）
$$\begin{cases} u_{tt} = a^2 u_{xx} + f(x,t), & x \in \RR,\ t > 0, \\ u(x,0) = \varphi(x),\; u_t(x,0) = \psi(x), & x \in \RR. \end{cases}$$

非齐次 d'Alembert 公式：

$$\boxed{u(x,t) = \frac{1}{2}[\varphi(x+at) + \varphi(x-at)] + \frac{1}{2a}\int_{x-at}^{x+at} \psi(s)\,\dd s + \frac{1}{2a}\int_0^t\!\int_{x-a(t-\tau)}^{x+a(t-\tau)} f(s,\tau)\,\dd s\,\dd\tau}.$$

前两项为齐次解（初值响应），第三项为非齐次特解（源项 $f$ 的累积效应——将 $f$ 在时刻 $\tau$ 的贡献视为瞬时初速度 $f(\cdot,\tau)$，对 $\tau$ 从 $0$ 到 $t$ 积分）。

高维非齐次波动方程（三维）：

$$u(x,t) = \frac{\pd}{\pd t}\Bigl(t M_{at}[\varphi](x)\Bigr) + t M_{at}[\psi](x) + \int_0^t (t-\tau) M_{a(t-\tau)}[f(\cdot,\tau)](x)\,\dd\tau,$$

其中 $M_r[g](x) = \frac{1}{4\pi r^2}\iint_{|y|=r} g(x+y)\,\dd S_y$ 为球面平均值算子。

二维非齐次波动方程（降维法）：

$$u(x,t) = \frac{\pd}{\pd t}\Bigl(\frac{1}{2\pi a}\iint_{|y-x|\le at}\frac{\varphi(y)}{\sqrt{a^2t^2 - |y-x|^2}}\,\dd y\Bigr) + \frac{1}{2\pi a}\iint_{|y-x|\le at}\frac{\psi(y)}{\sqrt{a^2t^2 - |y-x|^2}}\,\dd y + \frac{1}{2\pi a}\int_0^t\!\iint_{|y-x|\le a(t-\tau)}\frac{f(y,\tau)}{\sqrt{a^2(t-\tau)^2 - |y-x|^2}}\,\dd y\,\dd\tau.$$

B.2 半无界问题——延拓法（反射法）

Dirichlet 条件 $u(0,t)=0$（固定端）→ 奇延拓

奇延拓：$\tilde{\varphi}(x) = \begin{cases} \varphi(x), & x \ge 0 \\ -\varphi(-x), & x < 0 \end{cases}$，$\tilde{\psi}$ 同理。

当 $x - at < 0$（已反射）：

$$u(x,t) = \frac{1}{2}[\varphi(x+at) - \varphi(at-x)] + \frac{1}{2a}\left[\int_0^{x+at}\psi(s)\,\dd s - \int_0^{at-x}\psi(s)\,\dd s\right].$$

物理上，波反相反射（波峰变波谷）。

Neumann 条件 $u_x(0,t)=0$（自由端）→ 偶延拓

偶延拓：$\tilde{\varphi}(x) = \begin{cases} \varphi(x), & x \ge 0 \\ \varphi(-x), & x < 0 \end{cases}$（不加负号！），$\tilde{\psi}$ 同理。

当 $x - at < 0$（已反射）：

$$u(x,t) = \frac{1}{2}[\varphi(x+at) + \varphi(at-x)] + \frac{1}{2a}\left[\int_0^{x+at}\psi(s)\,\dd s + \int_0^{at-x}\psi(s)\,\dd s\right].$$

物理上，波同相反射（波峰仍是波峰）。

B.3 有界弦——分离变量法

两端固定弦的自由振动
$$\begin{cases} u_{tt} = a^2 u_{xx}, & 0 < x < l,\ t > 0, \\ u(0,t) = u(l,t) = 0, & t \ge 0, \\ u(x,0) = \varphi(x),\; u_t(x,0) = \psi(x), & 0 \le x \le l. \end{cases}$$

本征值：$\lambda_n = \left(\frac{n\pi}{l}\right)^2$，本征函数：$X_n(x) = \sin\frac{n\pi x}{l}$（$n=1,2,3,\ldots$）

通解：

$$u(x,t) = \sum_{n=1}^{\infty} \left(A_n \cos\frac{n\pi a t}{l} + B_n \sin\frac{n\pi a t}{l}\right) \sin\frac{n\pi x}{l}.$$

Fourier 系数：

$$A_n = \frac{2}{l}\int_0^l \varphi(x)\sin\frac{n\pi x}{l}\,\dd x, \quad B_n = \frac{2}{n\pi a}\int_0^l \psi(x)\sin\frac{n\pi x}{l}\,\dd x.$$
分离变量法五步走
  1. 设解：$u(x,t) = X(x)T(t)$
  2. 分离：$\frac{T''}{a^2 T} = \frac{X''}{X} = -\lambda$
  3. 解 $X$ 的本征值问题：$X'' + \lambda X = 0$ + 齐次边界条件 → $\lambda_n, X_n$
  4. 解 $T$ 的ODE：$T'' + a^2\lambda_n T = 0$ → $T_n(t)$
  5. 叠加 + 初始条件定系数：$u = \sum X_n T_n$，用正交性求系数

B.4 特征线、依赖区间与影响区域

特征线与信息传播
  • 特征线：$x \pm at = \text{const}$，是波前和信息传播的轨迹。
  • 依赖区间：解 $u(x_0, t_0)$ 只依赖于初始数据在 $[x_0-at_0, x_0+at_0]$ 上的值。
  • 决定区域：初始点 $(x_0, 0)$ 的值只影响锥形区域 $|x-x_0| \le a t$ 内的解。
  • 有限传播速度：信息以速度 $a$ 传播——双曲型方程的标志性质。

B.5 能量积分与唯一性

能量积分
$$E(t) = \frac{1}{2}\int (u_t^2 + a^2 u_x^2)\,\dd x.$$

对于齐次波动方程，$E'(t) = 0$（能量守恒）。若有阻尼项 $-\gamma u_t$，则 $E'(t) \le 0$（能量衰减）。

由能量积分可直接证明解的唯一性和稳定性。

C. 热传导方程（第二章）

C.1 全空间 Cauchy 问题——Fourier 变换法

热核（基本解）
$$\begin{cases} u_t = a^2 u_{xx}, & x \in \RR,\ t > 0, \\ u(x,0) = \varphi(x), & x \in \RR. \end{cases}$$

Poisson 公式（热核表示）：

$$u(x,t) = \int_{-\infty}^{+\infty} \Phi(x-y, t)\,\varphi(y)\,\dd y, \quad \Phi(x,t) = \frac{1}{2a\sqrt{\pi t}} e^{-\frac{x^2}{4a^2 t}}.$$

$\Phi(x,t)$ 称为热核（Gauss 核），是热传导方程的基本解。$\Phi(x,t) > 0$ 对所有 $x,t$。

Fourier 变换法步骤
  1. 对 $x$ 做 Fourier 变换：$\hat{u}(\xi,t) = \int_{-\infty}^{+\infty} u(x,t) e^{-i\xi x}\,\dd x$
  2. ODE：$\hat{u}_t = -a^2\xi^2 \hat{u}$ → $\hat{u}(\xi,t) = \hat{\varphi}(\xi) e^{-a^2\xi^2 t}$
  3. Fourier 逆变换：$u(x,t) = \frac{1}{2\pi}\int_{-\infty}^{+\infty} \hat{\varphi}(\xi) e^{-a^2\xi^2 t} e^{i\xi x}\,\dd\xi$
  4. 利用卷积定理即得 Poisson 公式。

C.2 半无界问题——延拓法

半无界热传导的延拓
边界条件延拓方式解公式
$u(0,t)=0$
（Dirichlet）		 奇延拓 $u(x,t) = \int_0^{+\infty} [\Phi(x-y,t) - \Phi(x+y,t)]\,\varphi(y)\,\dd y$
$u_x(0,t)=0$
（Neumann）		 偶延拓 $u(x,t) = \int_0^{+\infty} [\Phi(x-y,t) + \Phi(x+y,t)]\,\varphi(y)\,\dd y$

C.3 有界杆——分离变量法

两端固定温度 $u(0,t)=u(l,t)=0$
$$\begin{cases} u_t = a^2 u_{xx}, & 0 < x < l,\ t > 0, \\ u(0,t) = u(l,t) = 0, & t \ge 0, \\ u(x,0) = \varphi(x), & 0 \le x \le l. \end{cases}$$

本征值：$\lambda_n = \left(\frac{n\pi}{l}\right)^2$，本征函数：$X_n(x) = \sin\frac{n\pi x}{l}$

通解：

$$u(x,t) = \sum_{n=1}^{\infty} A_n e^{-a^2\left(\frac{n\pi}{l}\right)^2 t} \sin\frac{n\pi x}{l}, \quad A_n = \frac{2}{l}\int_0^l \varphi(x)\sin\frac{n\pi x}{l}\,\dd x.$$
两端绝热 $u_x(0,t)=u_x(l,t)=0$

本征函数：$X_n(x) = \cos\frac{n\pi x}{l}$（$n=0,1,2,\ldots$，注意包含 $n=0$）

通解：

$$u(x,t) = \frac{A_0}{2} + \sum_{n=1}^{\infty} A_n e^{-a^2\left(\frac{n\pi}{l}\right)^2 t} \cos\frac{n\pi x}{l}, \quad A_n = \frac{2}{l}\int_0^l \varphi(x)\cos\frac{n\pi x}{l}\,\dd x.$$

当 $t \to \infty$ 时，$u(x,t) \to \frac{A_0}{2} = \frac{1}{l}\int_0^l \varphi(x)\,\dd x$（温度趋于初始平均温度）。

C.4 极值原理

极值原理（热传导方程的核心性质）

弱极值原理：热传导方程的解在区域 $\bar{Q}_T = [0,l] \times [0,T]$ 上的最大值和最小值必在抛物边界 $\{t=0\} \cup \{x=0\} \cup \{x=l\}$ 上达到。

强极值原理：若解在内部点达到最大值，则解为常数。

推论（比较原理）：若 $u, v$ 均为热传导方程的解，且 $u \le v$ 在抛物边界上成立，则在 $\bar{Q}_T$ 上处处有 $u \le v$。

极值原理的用途
  • 唯一性：若 $u_1, u_2$ 满足相同方程和定解条件，则 $w = u_1 - u_2$ 在抛物边界上为零，由极值原理 $w \equiv 0$。
  • 稳定性：初值改变 $\varepsilon$ 导致解改变不超过 $\varepsilon$。
  • 保号性：若初值非负，则解处处非负（$\Phi > 0$ 也保证了这一点）。

C.5 即时光滑化（无限传播速度）

热传导 vs 波动的传播速度对比
  • 热传导：初始扰动瞬间传遍全空间（$\Phi(x,t) > 0$ 对所有 $x$ 成立）。不论初值 $\varphi$ 光滑与否，$t>0$ 时解 $u(x,t)$ 都 $C^\infty$ 光滑。
  • 波动：扰动以有限速度 $a$ 传播（特征线内有影响，特征线外无影响）。初值的光滑性会传递到解中。

D. 调和方程（第三章）

D.1 基本解（各维数完整形式）

Laplace 方程基本解 $\lap\Phi = \delta(x)$（分布意义下）
维数基本解 $\Phi(x)$$|x| \to \infty$ 行为
$n=2$ $\displaystyle \Phi(x) = \frac{1}{2\pi}\ln\frac{1}{\sqrt{x_1^2+x_2^2}} = -\frac{1}{2\pi}\ln r$ $\to +\infty$（对数发散）
$n=3$ $\displaystyle \Phi(x) = \frac{1}{4\pi\sqrt{x_1^2+x_2^2+x_3^2}} = \frac{1}{4\pi r}$ $\to 0$（$O(1/r)$）
$n \ge 3$ $\displaystyle \Phi(x) = \frac{1}{(n-2)\sigma_n}\cdot\frac{1}{|x|^{n-2}}$，其中 $\sigma_n = n\omega_n$ 为单位球面面积 $\to 0$（$O(1/r^{n-2})$）

注：$\omega_n = \frac{\pi^{n/2}}{\Gamma(n/2+1)}$ 为单位球体积，$\sigma_n = \frac{2\pi^{n/2}}{\Gamma(n/2)}$ 为单位球面面积。

在 $\RR^n\setminus\{0\}$ 上 $\lap\Phi = 0$。基本解是 Green 函数法的基石——它代表原点处"点电荷/点热源"产生的稳态场。

D.2 Green 公式（基本积分公式）

Green 第一公式

设 $\Omega \subset \RR^n$ 为有界光滑区域，$u, v \in C^2(\bar{\Omega})$，则

$$\int_\Omega v\lap u\,\dd x = \int_{\pd\Omega} v\frac{\pd u}{\pd n}\,\dd S - \int_\Omega \grad v \cdot \grad u\,\dd x.$$

此为一元分部积分 $\int_a^b v u''\dd x = vu'|_a^b - \int_a^b v'u'\dd x$ 的多元推广。

Green 第二公式（对称形式）

将 Green 第一公式中 $u,v$ 互换后相减：

$$\int_\Omega (v\lap u - u\lap v)\,\dd x = \int_{\pd\Omega} \left(v\frac{\pd u}{\pd n} - u\frac{\pd v}{\pd n}\right)\dd S.$$

这是整个 Green 函数方法的基石——由此导出基本积分公式和 Green 表示公式。

基本积分公式（由 Green 第二公式取 $v = \Phi$）

设 $u \in C^2(\bar{\Omega})$，$\Phi$ 为基本解，则对于 $x_0 \in \Omega$：

$$u(x_0) = \int_{\pd\Omega} \left(\Phi(x-x_0)\frac{\pd u}{\pd n} - u\frac{\pd\Phi(x-x_0)}{\pd n}\right)\dd S_x - \int_\Omega \Phi(x-x_0)\lap u(x)\,\dd x.$$

这个公式说明：区域内部任意点的值由边界上的值和区域内的源项完全决定——调和函数（$\lap u=0$）的内部值完全由边界数据确定。

基本积分公式的二维显式形式

设 $\Omega \subset \RR^2$，$u \in C^2(\bar{\Omega})$，$x_0 = (x_{01}, x_{02}) \in \Omega$。令 $r = |x - x_0| = \sqrt{(x_1-x_{01})^2 + (x_2-x_{02})^2}$：

二维基本解：$\Phi_2(x-x_0) = \dfrac{1}{2\pi}\ln\dfrac{1}{r} = -\dfrac{1}{2\pi}\ln r$。

二维基本积分公式：

$$\begin{aligned} u(x_0) = &\int_{\pd\Omega} \left[\left(-\frac{1}{2\pi}\ln r\right)\frac{\pd u}{\pd n} - u(x)\,\frac{\pd}{\pd n_x}\!\left(-\frac{1}{2\pi}\ln r\right)\right]\dd s_x \\ &- \iint_\Omega \left(-\frac{1}{2\pi}\ln r\right)\lap u(x)\,\dd x_1\dd x_2. \end{aligned}$$

其中 $\frac{\pd}{\pd n_x}\!\left(-\frac{1}{2\pi}\ln r\right) = -\frac{1}{2\pi}\frac{(x-x_0)\cdot n_x}{r^2} = -\frac{1}{2\pi}\frac{\cos(\mathbf{r}, n_x)}{r}$。

基本积分公式的三维显式形式

设 $\Omega \subset \RR^3$，$u \in C^2(\bar{\Omega})$，$r = |x - x_0|$：

三维基本解：$\Phi_3(x-x_0) = \dfrac{1}{4\pi r}$。

三维基本积分公式：

$$\begin{aligned} u(x_0) = &\int_{\pd\Omega} \left[\frac{1}{4\pi r}\,\frac{\pd u}{\pd n} - u(x)\,\frac{\pd}{\pd n_x}\!\left(\frac{1}{4\pi r}\right)\right]\dd S_x \\ &- \iiint_\Omega \frac{1}{4\pi r}\,\lap u(x)\,\dd x_1\dd x_2\dd x_3. \end{aligned}$$

其中 $\frac{\pd}{\pd n_x}\!\left(\frac{1}{4\pi r}\right) = -\frac{1}{4\pi}\frac{(x-x_0)\cdot n_x}{r^3} = -\frac{1}{4\pi}\frac{\cos(\mathbf{r}, n_x)}{r^2}$。

调和函数的边界积分表示（$\lap u = 0$ 时）

当 $u$ 为 $\Omega$ 上的调和函数（$\lap u = 0$），基本积分公式简化为仅含边界积分：

二维：

$$u(x_0) = \frac{1}{2\pi}\int_{\pd\Omega} \left[u(x)\,\frac{\pd}{\pd n_x}(\ln r) - \ln r\,\frac{\pd u}{\pd n}\right]\dd s_x = \frac{1}{2\pi}\int_{\pd\Omega} \left[u(x)\,\frac{\cos(\mathbf{r}, n_x)}{r} - \ln r\,\frac{\pd u}{\pd n}\right]\dd s_x.$$

三维：

$$u(x_0) = \frac{1}{4\pi}\int_{\pd\Omega} \left[\frac{1}{r}\,\frac{\pd u}{\pd n} - u(x)\,\frac{\pd}{\pd n_x}\!\left(\frac{1}{r}\right)\right]\dd S_x = \frac{1}{4\pi}\int_{\pd\Omega} \left[\frac{1}{r}\,\frac{\pd u}{\pd n} + u(x)\,\frac{\cos(\mathbf{r}, n_x)}{r^2}\right]\dd S_x.$$

这里 $\mathbf{r} = x - x_0$，$\cos(\mathbf{r}, n_x) = \frac{\mathbf{r}\cdot n_x}{|\mathbf{r}|}$ 为 $\mathbf{r}$ 与外法向 $n_x$ 夹角的余弦。

D.3 Green 函数与基本积分表示

Green 函数的定义与构造

对于区域 $\Omega$ 上的 Dirichlet 问题，Green 函数定义为：

$$G(x, x_0) = \Phi(x - x_0) + h(x, x_0),$$

其中修正项 $h$ 满足 $\lap_x h = 0$（在 $\Omega$ 内）且 $h|_{\pd\Omega} = -\Phi(x-x_0)|_{\pd\Omega}$。

于是 $G$ 满足：$\lap_x G = \delta(x-x_0)$（分布意义下）且 $G|_{\pd\Omega} = 0$。

Dirichlet 问题的 Green 函数表示公式

Poisson 方程 $\lap u = f$（$\Omega$ 内），Dirichlet 边界 $u|_{\pd\Omega} = \varphi$ 的解：

$$\boxed{u(x_0) = \int_\Omega G(x, x_0)\,f(x)\,\dd x - \int_{\pd\Omega} \varphi(x)\,\frac{\pd G(x, x_0)}{\pd n_x}\,\dd S_x}.$$

特别地，对 Laplace 方程（$f=0$）：$u(x_0) = -\int_{\pd\Omega} \varphi(x)\frac{\pd G}{\pd n_x}\dd S_x$。

球上的 Green 函数（三维，球心在原点）

设 $B_R(0) \subset \RR^3$，$x_0 \in B_R(0)$，$x_0^* = \frac{R^2}{|x_0|^2}x_0$（Kelvin 反演点）：

$$G(x, x_0) = \frac{1}{4\pi|x - x_0|} - \frac{R}{|x_0|}\cdot\frac{1}{4\pi|x - x_0^*|}.$$

对 $n \ge 3$ 的一般情形：$G(x, x_0) = \Phi(x-x_0) - \left(\frac{R}{|x_0|}\right)^{n-2}\Phi(x_0^* - x)$。

反演点性质：$|x_0|\cdot|x_0^*| = R^2$，且 $x_0$ 和 $x_0^*$ 在同一条从原点出发的射线上。对球面上任意点 $x$（$|x|=R$），有 $|x-x_0^*| = \frac{R}{|x_0|}|x-x_0|$。

圆盘上的 Green 函数（二维，圆心在原点）

设 $B_R(0) \subset \RR^2$ 为半径为 $R$ 的圆盘，$x_0 \in B_R(0)$，$x_0^* = \frac{R^2}{|x_0|^2}x_0$ 为反演点：

$$G(x, x_0) = \frac{1}{2\pi}\ln\frac{1}{|x - x_0|} - \frac{1}{2\pi}\ln\frac{R}{|x_0|\cdot|x - x_0^*|}.$$

等价形式：$G(x, x_0) = \dfrac{1}{2\pi}\ln\dfrac{|x_0|\cdot|x - x_0^*|}{R\cdot|x - x_0|}$。

计算边界法向导数（$|x| = R$ 时）：

$$\frac{\pd G}{\pd n_x}\bigg|_{|x|=R} = \frac{\pd G}{\pd r}\bigg|_{r=R} = -\frac{1}{2\pi R}\cdot\frac{R^2 - |x_0|^2}{|x - x_0|^2}.$$

代入 Green 表示公式即得圆域 Poisson 积分公式。

球上 Dirichlet 问题的 Green 表示公式（三维显式）

对 $\lap u = f$（$B_R(0)$ 内），$u|_{\pd B_R} = \varphi$：

$$u(x_0) = \iiint_{B_R(0)} G(x, x_0)\,f(x)\,\dd x + \frac{R^2 - |x_0|^2}{4\pi R}\iint_{\pd B_R(0)} \frac{\varphi(x)}{|x - x_0|^3}\,\dd S_x.$$

其中 $G(x, x_0) = \dfrac{1}{4\pi|x-x_0|} - \dfrac{R}{|x_0|}\cdot\dfrac{1}{4\pi|x-x_0^*|}$。

圆盘上 Dirichlet 问题的 Green 表示公式（二维显式）

对 $\lap u = f$（$B_R(0)$ 内），$u|_{\pd B_R} = \varphi$：

$$u(x_0) = \iint_{B_R(0)} G(x, x_0)\,f(x)\,\dd x + \frac{1}{2\pi R}\int_{\pd B_R(0)} \frac{R^2 - |x_0|^2}{|x - x_0|^2}\,\varphi(x)\,\dd s_x.$$

其中 $G(x, x_0) = \dfrac{1}{2\pi}\ln\dfrac{|x_0|\cdot|x - x_0^*|}{R\cdot|x - x_0|}$。

D.4 镜像源法（静电源像法）

上半空间的 Green 函数（三维）

对 $\RR^3_+ = \{(x,y,z) \mid z > 0\}$，点源位于 $x_0 = (x_0, y_0, z_0)$（$z_0 > 0$），镜像点为 $\tilde{x}_0 = (x_0, y_0, -z_0)$：

$$G(x, x_0) = \frac{1}{4\pi|x - x_0|} - \frac{1}{4\pi|x - \tilde{x}_0|}.$$

当 $x$ 在边界 $z=0$ 上时，$|x-x_0| = |x-\tilde{x}_0|$，故 $G|_{\pd\RR^3_+} = 0$ 自动满足。

上半空间的 Poisson 核与 Poisson 公式（三维）

Poisson 核：$K(x,y; x_0,y_0,z_0) = \dfrac{z_0}{2\pi\big[(x-x_0)^2 + (y-y_0)^2 + z_0^2\big]^{3/2}}$。

Poisson 公式：

$$u(x_0, y_0, z_0) = \frac{z_0}{2\pi}\iint_{\RR^2} \frac{\varphi(x,y)}{\big[(x-x_0)^2 + (y-y_0)^2 + z_0^2\big]^{3/2}}\,\dd x\dd y,$$

其中 $\varphi(x,y) = u(x,y,0)$ 为 Dirichlet 边界数据。

上半空间 Poisson 核的一般 $n$ 维形式

对 $\RR^n_+ = \{(x', x_n) \mid x' \in \RR^{n-1}, x_n > 0\}$：

Green 函数：$G(x, x_0) = \Phi(x - x_0) - \Phi(x - \tilde{x}_0)$，$\tilde{x}_0 = (x_0', -x_{0n})$。

Poisson 核：$K(x', x_0) = \dfrac{2x_{0n}}{n\omega_n}\cdot\dfrac{1}{\big[|x' - x_0'|^2 + x_{0n}^2\big]^{n/2}}$。

Poisson 公式：$u(x_0) = \int_{\RR^{n-1}} K(x', x_0)\,\varphi(x')\,\dd x'$。

镜像法的核心思想

在区域外放置"虚拟点源"（镜像电荷），使边界上的总影响恰好为零。对于球，虚拟电荷放在反演点处；对于平面，虚拟电荷放在镜面对称点处。这一方法在静电学中用来计算导体球或导体平面附近的电位分布。

D.5 Poisson 积分公式

圆域 Dirichlet 问题（二维）
$$\begin{cases} \lap u = 0, & r < R, \\ u(R,\theta) = f(\theta), & 0 \le \theta < 2\pi. \end{cases}$$

Poisson 积分公式：

$$u(r,\theta) = \frac{1}{2\pi}\int_0^{2\pi} \frac{R^2 - r^2}{R^2 - 2Rr\cos(\theta-\phi) + r^2}\,f(\phi)\,\dd\phi.$$

Poisson 核：$P(r,\theta-\phi) = \dfrac{R^2 - r^2}{R^2 - 2Rr\cos(\theta-\phi) + r^2} > 0$（对所有 $r < R$）。

Poisson 核三大性质：(1) $P \ge 0$（非负）；(2) $\frac{1}{2\pi}\int_0^{2\pi} P(r,\theta-\phi)\dd\phi = 1$（归一化）；(3) 当 $r \to R$ 时 $P$ 趋于 Dirac 函数 $\delta(\theta-\phi)$。

球域 Dirichlet 问题（三维）

Poisson 积分公式（三维球 $B_R(0)$）：

$$u(x) = \frac{R^2 - |x|^2}{4\pi R}\iint_{|y|=R} \frac{u(y)}{|x - y|^3}\,\dd S_y.$$

Poisson 核：$K(x,y) = \dfrac{R^2 - |x|^2}{4\pi R|x - y|^3} > 0$，满足 $\iint_{|y|=R} K(x,y)\dd S_y = 1$。

上半平面 Dirichlet 问题（二维）
$$\begin{cases} u_{xx} + u_{yy} = 0, & y > 0, \\ u(x,0) = f(x), & x \in \RR. \end{cases}$$

Poisson 公式：

$$u(x,y) = \frac{1}{\pi}\int_{-\infty}^{+\infty} \frac{y}{(x-\xi)^2 + y^2}\,f(\xi)\,\dd\xi.$$

Poisson 核：$P(x-\xi, y) = \dfrac{1}{\pi}\dfrac{y}{(x-\xi)^2 + y^2}$。

"见到什么题走什么路"——Poisson 公式适用场景
  • 圆域 Dirichlet → 二维 Poisson 积分公式
  • 球域 Dirichlet → 三维 Poisson 积分公式
  • 上半平面 Dirichlet → 上半平面 Poisson 公式
  • 上半空间 Dirichlet → 上半空间 Poisson 公式（D.4 节）

D.6 调和函数的重要性质（定理+推论）

定理 D.6.1（平均值定理 — 调和函数的刻画性特征）

若 $u$ 在 $\Omega$ 内调和，$\overline{B_R(x_0)} \subset \Omega$，则

(1) 球面平均值（一般形式）：

$$u(x_0) = \frac{1}{|\pd B_R(x_0)|}\int_{\pd B_R(x_0)} u(x)\,\dd S_x.$$

二维显式（圆周平均值）：设 $x_0 = (x_{01}, x_{02})$，用极坐标参数化圆周：

$$u(x_{01}, x_{02}) = \frac{1}{2\pi R}\int_0^{2\pi} u(x_{01}+R\cos\theta,\; x_{02}+R\sin\theta)\; R\,\dd\theta = \frac{1}{2\pi}\int_0^{2\pi} u(x_{01}+R\cos\theta,\; x_{02}+R\sin\theta)\,\dd\theta.$$

三维显式（球面平均值）：设 $x_0 = (x_{01}, x_{02}, x_{03})$，用球坐标 $(\theta, \phi)$ 参数化球面：

$$u(x_0) = \frac{1}{4\pi R^2}\iint_{\pd B_R(x_0)} u(x)\,\dd S = \frac{1}{4\pi}\int_0^{2\pi}\!\int_0^{\pi} u(x_0 + R\,\mathbf{e}_r(\theta,\phi))\,\sin\phi\,\dd\phi\,\dd\theta,$$

其中 $\mathbf{e}_r = (\sin\phi\cos\theta,\; \sin\phi\sin\theta,\; \cos\phi)$ 为单位径向向量。

(2) 球体平均值（一般形式）：

$$u(x_0) = \frac{1}{|B_R(x_0)|}\int_{B_R(x_0)} u(x)\,\dd x.$$

二维显式（圆盘平均值）：$|B_R| = \pi R^2$，

$$u(x_{01}, x_{02}) = \frac{1}{\pi R^2}\int_0^R\!\int_0^{2\pi} u(x_{01}+\rho\cos\theta,\; x_{02}+\rho\sin\theta)\,\rho\,\dd\theta\,\dd\rho.$$

三维显式（球体平均值）：$|B_R| = \frac{4}{3}\pi R^3$，

$$u(x_0) = \frac{3}{4\pi R^3}\int_0^R\!\int_0^{2\pi}\!\int_0^{\pi} u(x_0 + \rho\,\mathbf{e}_r(\theta,\phi))\,\rho^2\sin\phi\,\dd\phi\,\dd\theta\,\dd\rho.$$

(3) 逆命题也成立：若 $u \in C(\Omega)$ 且对每个 $x_0 \in \Omega$ 和充分小的 $R$ 满足球面平均值性质，则 $u$ 自动是调和函数（$\lap u = 0$）。平均值性质与"$\lap u = 0$"等价。

定理 D.6.2（极值原理）

弱极值原理：调和函数在闭区域 $\bar{\Omega}$ 上的最大值和最小值必在边界 $\pd\Omega$ 上达到。

强极值原理：若调和函数在 $\Omega$ 内部某点达到最大值（或最小值），则它在整个连通分量上恒为常数。

推论（比较原理）：若 $u, v$ 在 $\Omega$ 内调和，且 $u \le v$ 在 $\pd\Omega$ 上成立，则在 $\bar{\Omega}$ 上处处有 $u \le v$。

定理 D.6.3（Hopf 引理 / 边界点引理）

设 $\Omega$ 满足内球条件，$u \in C^2(\Omega) \cap C^1(\bar{\Omega})$ 在 $\Omega$ 内调和。若 $x_0 \in \pd\Omega$ 是 $u$ 的严格最大值点（即 $u(x_0) > u(x)$ 对所有 $x \in \Omega$），则

$$\frac{\pd u}{\pd n}(x_0) > 0.$$

调和函数在边界严格最大值点处的外法向导数严格大于零。这一定理在证明强极值原理和Neumann 问题唯一性中起关键作用。

定理 D.6.4（Liouville 定理及其推论）

(1) 有界版：在 $\RR^n$ 上有界的调和函数必为常数。

(2) 单侧有界版：在 $\RR^n$ 上有上界（或下界）的调和函数必为常数。

(3) 非负版：在 $\RR^2$ 上非负的调和函数必为常数；但在 $\RR^n$（$n \ge 3$）上不成立——基本解 $\Phi(x) = c/|x|^{n-2}$ 就是非负但非常数的反例！

Liouville 定理的证明核心是平均值性质 + 大球估计。

定理 D.6.5（Harnack 不等式）

一般形式：设 $\Omega$ 为有界连通区域，$K \subset \Omega$ 为紧子集。存在仅依赖于 $n, \Omega, K$ 的常数 $C > 0$，对任何非负调和函数 $u$：

$$\sup_K u \le C \inf_K u.$$

球上的精确形式（$B_R(0)$ 上的非负调和函数）：

$$\frac{R^{n-2}(R-|x|)}{(R+|x|)^{n-1}}\,u(0) \le u(x) \le \frac{R^{n-2}(R+|x|)}{(R-|x|)^{n-1}}\,u(0).$$

二维球上的简化形式：$\displaystyle \frac{R-|x|}{R+|x|}\,u(0) \le u(x) \le \frac{R+|x|}{R-|x|}\,u(0).$

定理 D.6.6（解析性 / 椭圆正则性）

调和函数 $u$（满足 $\lap u = 0$）自动具有：

  • $u \in C^\infty(\Omega)$——无穷次可微；
  • $u$ 是实解析的——在每点的邻域内可展开为收敛的幂级数。

对比：波动方程解可以出现间断，热传导方程解在 $t>0$ 光滑但 $t=0$ 时与初值一样粗糙。唯有椭圆型方程的解在区域内部无条件地光滑和解析。

定理 D.6.7（可去奇点定理）

设 $u$ 在 $\Omega \setminus \{x_0\}$ 上调和。若当 $x \to x_0$ 时：

  • $n \ge 3$：$u(x) = o(|x-x_0|^{2-n})$（增长比基本解慢）；
  • $n = 2$：$u(x) = o(|\ln|x-x_0||)$（增长比对数慢）。

则 $u$ 可延拓为 $\Omega$ 上的调和函数。特别地，若 $u$ 在 $x_0$ 附近有界，则 $x_0$ 必为可去奇点。

定理 D.6.8（Dirichlet 原理 / 变分原理）

Dirichlet 问题 $\lap u = 0$（$\Omega$ 内），$u|_{\pd\Omega} = \varphi$ 的解 $u$ 是 Dirichlet 能量泛函

$$E[v] = \frac{1}{2}\int_\Omega |\grad v|^2\,\dd x$$

在所有满足 $v|_{\pd\Omega} = \varphi$ 的函数中的唯一极小化子。即调和函数使"能量"最小——这是有限元方法的理论基础。

D.7 考试常考推论与知识点速查

Dirichlet 问题解的唯一性

设 $\Omega$ 为有界区域，则 $\lap u = 0$（$\Omega$ 内）+ $u|_{\pd\Omega} = \varphi$ 的解在 $C^2(\Omega) \cap C(\bar{\Omega})$ 中唯一。

证法：两解之差 $w$ 满足 $\lap w = 0$、$w|_{\pd\Omega} = 0$，由极值原理 $w \equiv 0$。

Neumann 问题的可解性必要条件 ★ 选择填空题高频考点

Poisson 方程 Neumann 问题 $\lap u = f$（$\Omega$ 内），$\frac{\pd u}{\pd n}|_{\pd\Omega} = g$ 有解的必要条件：

$$\boxed{\int_\Omega f(x)\,\dd x = \int_{\pd\Omega} g(x)\,\dd S}.$$

对 Laplace 方程（$f=0$）：必须 $\displaystyle \int_{\pd\Omega} g(x)\,\dd S = 0$——净通量为零。

物理意义：稳态下区域内产生的"源"必须等于通过边界流出的"通量"——守恒律。

证法：直接对 $\lap u = \div(\grad u)$ 使用散度定理。

Neumann 问题解的不唯一性

即使相容性条件满足，Laplace 方程 Neumann 问题的解仍相差一个常数：若 $u$ 是解，则 $u + C$ 也是解。

Dirichlet 问题固定了边界的值（"铁丝固定"），解唯一；Neumann 问题只固定了边界的斜率（"只控制导数"），允许整体平移。

外问题的唯一性条件 ★ 选择填空题高频考点

$n=2$ $u(x)$ 在无穷远处有界即可 因二维基本解 $\ln r$ 在无穷远处发散，只需有界

调和函数与解析函数的关系（二维特有）

在 $\RR^2$ 中，若 $f(z) = u(x,y) + iv(x,y)$ 是解析函数，则其实部 $u$ 和虚部 $v$ 都是调和函数（Cauchy-Riemann 方程的推论）。

反过来，任何单连通区域上的调和函数 $u$ 都是某个解析函数的实部。这意味着二维调和函数具有复分析的全部工具。

考试常用例子：$u = x^2-y^2$（$f(z)=z^2$ 的实部）、$u = e^x\sin y$（$f(z)=e^z$ 的虚部）。

调和函数的梯度估计

设 $u$ 在 $B_R(x_0)$ 内调和，则

$$|\grad u(x_0)| \le \frac{n}{R}\max_{\pd B_R(x_0)} |u|.$$

这给出了调和函数梯度的界由函数本身的界控制——另一类重要的先验估计，可由平均值性质对导数求导得到。

方程类型证明唯一性/稳定性的方法
波动方程能量积分法 — 定义 $E(t) = \frac{1}{2}\int(u_t^2 + a^2u_x^2)\dd x$，证明 $E'(t) \le 0$，从而 $E(t) \le E(0)$
热传导方程极值原理 — 两解之差在抛物边界上为零 $\implies$ 内部处处为零

注意：能量积分法对波动方程最自然（能量守恒有明确物理意义），极值原理对抛物/椭圆型方程最自然。不要混淆——热传导方程一般不用能量法，波动方程也不用极值原理。

F. 证明题常见思路速查

每类证明题均附完整简洁的证明过程，点击展开查看。建议先自己尝试证明，再对照检查。

F.1 证明唯一性

标准模板（适用于所有线性PDE）
  1. 设 $u_1, u_2$ 为两个解，令 $w = u_1 - u_2$。
  2. $w$ 满足齐次方程 + 零初边值条件。
  3. 用对应工具证明 $w \equiv 0$。
  4. 故 $u_1 = u_2$，解唯一。
波动方程 Cauchy 问题唯一性（能量积分法）
▶ 展开证明

问题：$u_{tt} = a^2 u_{xx}$，$x \in \RR$，$u(x,0)=\varphi$, $u_t(x,0)=\psi$。

证：设 $u_1, u_2$ 为两解，$w = u_1 - u_2$。则 $w_{tt} = a^2 w_{xx}$，$w(x,0)=0$, $w_t(x,0)=0$。

定义能量 $E(t) = \frac{1}{2}\int_{-\infty}^{+\infty}(w_t^2 + a^2 w_x^2)\dd x$。

$E'(t) = \int(w_t w_{tt} + a^2 w_x w_{xt})\dd x \stackrel{\text{代入}w_{tt}=a^2 w_{xx}}{=} a^2\!\int(w_t w_{xx} + w_x w_{xt})\dd x = a^2\!\int\frac{\pd}{\pd x}(w_t w_x)\dd x = a^2 w_t w_x\big|_{-\infty}^{+\infty} = 0.$

故 $E(t) \equiv E(0) = 0$（因 $w_t(x,0)=w_x(x,0)=0$）。

由 $w_t^2 + a^2 w_x^2 \equiv 0$ 得 $w_t \equiv 0, w_x \equiv 0$。又 $w(x,0)=0$，故 $w \equiv 0$。$\square$

热传导方程初边值问题唯一性（极值原理）
▶ 展开证明

问题：$u_t = a^2 u_{xx}$，$0 < x < l$，$u(0,t)=u(l,t)=0$，$u(x,0)=\varphi(x)$。

证：设 $u_1, u_2$ 为两解，$w = u_1 - u_2$。则 $w_t = a^2 w_{xx}$，$w(0,t)=w(l,t)=0$，$w(x,0)=0$。

$w$ 在抛物边界 $\{t=0\} \cup \{x=0\} \cup \{x=l\}$ 上恒为零。由极值原理：热传导方程的解在区域 $\bar{Q}_T$ 上的最大最小值在抛物边界上达到。

故在 $\bar{Q}_T$ 上 $0 \le w \le 0$，即 $w \equiv 0$。$\square$

调和方程 Dirichlet 问题唯一性（极值原理）
▶ 展开证明

问题：$\lap u = 0$（$\Omega$ 内），$u|_{\pd\Omega} = \varphi$。

证：设 $u_1, u_2$ 为两解，$w = u_1 - u_2$。则 $\lap w = 0$（$\Omega$ 内），$w|_{\pd\Omega} = 0$。

由极值原理：调和函数的最大最小值在边界 $\pd\Omega$ 上达到。因 $w|_{\pd\Omega} = 0$，故在 $\bar{\Omega}$ 上 $0 \le w \le 0$，即 $w \equiv 0$。$\square$

注意：对于 Neumann 问题 $\frac{\pd u}{\pd n}|_{\pd\Omega} = g$，解不唯一（相差常数）。要恢复唯一性需附加条件如 $\int_\Omega u = 0$。

F.2 证明稳定性和连续依赖性

基本思路：证明解的范数被初始/边界数据的范数控制，如 $\|u\| \le C\|\text{数据}\|$。

波动方程稳定性（能量估计 → Gronwall）
▶ 展开证明

设 $u$ 满足 $u_{tt} = a^2 u_{xx} + f$。定义 $E(t) = \frac{1}{2}\int(u_t^2 + a^2 u_x^2 + u^2)\dd x$。

$E'(t) = \int(u_t u_{tt} + a^2 u_x u_{xt} + u u_t)\dd x$。代入 $u_{tt} = a^2 u_{xx} + f$：

$E'(t) = a^2\!\int(u_t u_{xx} + u_x u_{xt})\dd x + \int u_t f\dd x + \int u u_t\dd x$。

第一项 $= a^2\!\int\frac{\pd}{\pd x}(u_t u_x)\dd x = 0$（边界项消失）。

用 Cauchy 不等式：$\int u_t f \le \frac{1}{2}\int(u_t^2 + f^2) \le E + \frac{1}{2}\|f\|_{L^2}^2$，$\int u u_t \le \frac{1}{2}\int(u^2 + u_t^2) \le E$。

故 $E'(t) \le CE(t) + C\|f(\cdot,t)\|_{L^2}^2$。由Gronwall 引理：

$$E(t) \le E(0)e^{Ct} + C\int_0^t \|f(\cdot,\tau)\|_{L^2}^2 e^{C(t-\tau)}\dd\tau.$$

若 $f=0$，则 $E(t) \le E(0)e^{Ct}$——初值的小扰动只导致解的指数级放大，在有限时间区间上"小输入 $\Rightarrow$ 小输出"。$\square$

热传导方程稳定性（极值原理 → 最大模估计）
▶ 展开证明

对 $u_t = a^2 u_{xx}$（$0 < x < l$），由极值原理：

$$\max_{\bar{Q}_T} |u(x,t)| = \max_{\text{抛物边界}} |u(x,t)| = \max\{\max_{[0,l]}|\varphi(x)|,\; \max_{[0,T]}|u(0,t)|,\; \max_{[0,T]}|u(l,t)|\}.$$

若初边值数据的改变量不超过 $\varepsilon$，则解的改变量也不超过 $\varepsilon$。这就是关于最大模的连续依赖性。

对于非齐次方程 $u_t = a^2 u_{xx} + f$：$\max|u| \le \max|\varphi| + \max|u|_{\pd} + \frac{l^2}{2a^2}\max|f|$。$\square$

调和方程稳定性（极值原理 → 最大模估计）
▶ 展开证明

对 $\lap u = 0$（$\Omega$ 内），$u|_{\pd\Omega} = \varphi$。由极值原理：

$$\max_{\bar{\Omega}} |u(x)| = \max_{\pd\Omega} |\varphi(x)|.$$

若边界数据 $\varphi$ 在最大模意义下改变不超过 $\varepsilon$，则解 $u$ 在任何内点的改变也不超过 $\varepsilon$。$\square$

能量估计版本（适用于变系数）：对 $-\lap u = f$，$u|_{\pd\Omega}=0$，有 $\|\grad u\|_{L^2} \le C_P\|f\|_{L^2}$（Poincaré 不等式）。

F.3 证明调和函数的性质

关键工具链

平均值定理 → 极值原理 → (Liouville / Harnack / 梯度估计等)

大部分调和函数的深度性质都从平均值定理出发。记住：平均值定理是调和函数理论的基石。

1. 平均值定理的证明（Green 第二公式法）
▶ 展开证明

设 $u$ 在 $B_R(x_0)$ 内调和。对任意 $0 < \rho < R$，在 $B_\rho(x_0)$ 上对 $u$ 和 $1$ 应用 Green 第二公式：

$$\int_{\pd B_\rho} \left(1\cdot\frac{\pd u}{\pd n} - u\cdot\frac{\pd 1}{\pd n}\right)\dd S = \int_{B_\rho} (1\cdot\lap u - u\cdot\lap 1)\dd x = 0.$$

即 $\int_{\pd B_\rho} \frac{\pd u}{\pd n}\dd S = 0$。在极坐标下 $\frac{\pd u}{\pd n} = \frac{\pd u}{\pd r}$：

$$\int_{\pd B_\rho} \frac{\pd u}{\pd r}\dd S = \rho^{n-1}\frac{\dd}{\dd\rho}\left(\frac{1}{\rho^{n-1}}\int_{\pd B_\rho} u\,\dd S\right) = 0.$$

故 $\frac{1}{\rho^{n-1}}\int_{\pd B_\rho} u\,\dd S = \text{const}$。令 $\rho \to 0$，极限为 $u(x_0)\cdot |S^{n-1}|$（$|S^{n-1}|$ 为单位球面面积），即得球面平均值公式。

球体平均值由球面平均值对 $\rho$ 积分直接得到。$\square$

2. 弱极值原理的证明
▶ 展开证明

引理：若 $\lap u > 0$，则 $u$ 不能在内部取最大值。否则在最大值点 $x_0$，Hessian 矩阵半负定 $\Rightarrow \lap u(x_0) = \text{tr}(D^2u) \le 0$，矛盾。

对调和函数（$\lap u = 0$）：构造 $v_\varepsilon(x) = u(x) + \varepsilon|x|^2$，则 $\lap v_\varepsilon = 2\varepsilon n > 0$。

$v_\varepsilon$ 的最大值在边界 $\pd\Omega$ 上达到。设 $M = \max_{\pd\Omega} u$，$R_0$ 使 $\Omega \subset B_{R_0}$，则对任意 $x \in \Omega$：

$$u(x) = v_\varepsilon(x) - \varepsilon|x|^2 \le \max_{\pd\Omega} v_\varepsilon - \varepsilon|x|^2 \le M + \varepsilon(R_0^2 - |x|^2).$$

令 $\varepsilon \to 0^+$ 得 $u(x) \le M$。对 $-u$ 同理得 $u(x) \ge \min_{\pd\Omega} u$。$\square$

强极值原理：若调和函数在内部达到最值，由平均值定理和连通性可证 $u$ 为常数。

3. Liouville 定理的证明
▶ 展开证明

定理：$\RR^n$ 上有界的调和函数必为常数。

证：设 $|u| \le M$。取 $x_1, x_2 \in \RR^n$，令 $R$ 充分大使得 $B_R(0)$ 包含两点。考虑大球 $B_R(x_1)$ 和 $B_R(x_2)$。

由球体平均值公式：

$$u(x_i) = \frac{1}{|B_R|}\int_{B_R(x_i)} u(y)\dd y, \quad i=1,2.$$

两式相减：

$$|u(x_1) - u(x_2)| = \frac{1}{|B_R|}\Bigl|\int_{B_R(x_1)} u - \int_{B_R(x_2)} u\Bigr| \le \frac{|B_R(x_1) \triangle B_R(x_2)|}{|B_R|} \cdot M,$$

其中 $B_R(x_1) \triangle B_R(x_2)$ 为两个球的对称差。当 $R \to \infty$ 时，$\frac{|B_R(x_1) \triangle B_R(x_2)|}{|B_R|} \to 0$（对称差是 $O(R^{n-1})$ 的柱状区域，而球体积为 $O(R^n)$）。

故 $|u(x_1) - u(x_2)| \to 0$，即 $u(x_1) = u(x_2)$。$\square$

4. Harnack 不等式的证明思路
▶ 展开证明

球上 Harnack 不等式（$B_R(0)$ 上的非负调和函数 $u$）：

由 Poisson 公式：$u(x) = \frac{R^2 - |x|^2}{n\omega_n R}\int_{\pd B_R} \frac{u(y)}{|x-y|^n}\dd S_y$。

注意到对 $|y|=R$：$R - |x| \le |x-y| \le R + |x|$。代入 Poisson 核：

$$\frac{R^2 - |x|^2}{n\omega_n R (R+|x|)^n} \le K(x,y) \le \frac{R^2 - |x|^2}{n\omega_n R (R-|x|)^n}.$$

利用 $\int_{\pd B_R} K = 1$ 和 $u \ge 0$，上下界分别代入即得精确 Harnack 不等式。对一般的紧子集 $K$，用有限覆盖和链式估计得到 Harnack 常数。$\square$

5. 梯度估计的证明
▶ 展开证明

结论：$|\grad u(x_0)| \le \frac{n}{R}\max_{\pd B_R(x_0)} |u|$。

证：由球体平均值公式对 $x_0$ 求导：$\frac{\pd u}{\pd x_i}(x_0) = \frac{1}{|B_R|}\int_{B_R(x_0)} \frac{\pd u}{\pd x_i}(x)\dd x$。

用散度定理转为边界积分：$\frac{\pd u}{\pd x_i}(x_0) = \frac{1}{|B_R|}\int_{\pd B_R(x_0)} u(x)\,n_i\,\dd S_x$，其中 $n_i$ 为外法向分量。

取绝对值：$|\frac{\pd u}{\pd x_i}(x_0)| \le \frac{1}{|B_R|}\int_{\pd B_R} |u||n_i|\dd S \le \frac{|\pd B_R|}{|B_R|}\max_{\pd B_R}|u| = \frac{n}{R}\max_{\pd B_R}|u|.$

（因为 $|\pd B_R| = n\omega_n R^{n-1}$，$|B_R| = \omega_n R^n$，比值为 $n/R$。）对所有方向取上确界即得梯度估计。$\square$

F.4 证明解的非负性 / 保号性

  • 热传导方程：Poisson 公式 $u(x,t) = \int \Phi(x-y,t)\varphi(y)\dd y$ 中热核 $\Phi > 0$，故初值非负 $\Rightarrow$ 解处处非负。或用极值原理：最小值在抛物边界达到，若边界/初值非负则解非负。
  • 调和方程：Poisson 核 $P > 0$（在球或半空间上），故边界值非负 $\Rightarrow$ 内部处处非负。由 Harnack 不等式，若在某点为严格正，则紧子集上有严格正下界。
非负性证明的标准模板（以热传导为例）
▶ 展开证明

方法一（极值原理）：设 $u_t = a^2 u_{xx}$，$u(x,0) = \varphi(x) \ge 0$，$u(0,t) = u(l,t) = 0$。

由弱极值原理：$u$ 在 $\bar{Q}_T$ 上的最小值在抛物边界上达到。抛物边界上 $u \ge 0$，故内部 $u \ge 0$。$\square$

方法二（热核正性）：全空间解 $u(x,t) = \int_{-\infty}^{+\infty} \frac{1}{2a\sqrt{\pi t}}e^{-(x-y)^2/(4a^2t)}\varphi(y)\dd y$。被积函数中热核 $> 0$，若 $\varphi \ge 0$ 则 $u \ge 0$。

若有某点 $\varphi > 0$，则对所有 $x$ 和 $t>0$ 有 $u(x,t) > 0$（无限传播速度的体现）。$\square$

F.5 能量估计的标准推导

能量法推导模板（波动方程完整示例）
▶ 展开完整推导

考虑含阻尼和低阶项的波动方程：$u_{tt} + \gamma u_t = a^2 u_{xx} + b u_x + c u + f$，$\gamma > 0$。

Step 1 定义能量：$E(t) = \frac{1}{2}\int_{-\infty}^{+\infty}(u_t^2 + a^2 u_x^2 + u^2)\dd x$。

Step 2 求导：$E'(t) = \int(u_t u_{tt} + a^2 u_x u_{xt} + u u_t)\dd x$。

Step 3 代入方程：$u_{tt} = a^2 u_{xx} + b u_x + c u - \gamma u_t + f$。

$\begin{aligned} E'(t) &= \int[u_t(a^2 u_{xx} + b u_x + c u - \gamma u_t + f) + a^2 u_x u_{xt} + u u_t]\dd x \\ &= a^2\!\int(u_t u_{xx} + u_x u_{xt})\dd x + \int(b u_t u_x + c u_t u + u u_t - \gamma u_t^2 + u_t f)\dd x. \end{aligned}$

Step 4 分部积分消去最高阶耦合项：$a^2\!\int(u_t u_{xx} + u_x u_{xt})\dd x = a^2\!\int\frac{\pd}{\pd x}(u_t u_x)\dd x = 0$（无穷远边界消失）。

Step 5 Cauchy 不等式放缩低阶项：

$\int b u_t u_x \le |b|_\infty\!\int|u_t u_x| \le \frac{|b|_\infty}{2}\!\int(u_t^2 + u_x^2) \le C_1 E$，

$\int c u_t u \le |c|_\infty\!\int|u_t u| \le \frac{|c|_\infty}{2}\!\int(u_t^2 + u^2) \le C_2 E$，

$\int u u_t \le \frac{1}{2}\!\int(u^2 + u_t^2) \le E$，

$-\gamma\!\int u_t^2 \le 0$（阻尼项贡献负能量），

$\int u_t f \le \frac{1}{2}\!\int(u_t^2 + f^2) \le E + \frac{1}{2}\|f\|_{L^2}^2$。

Step 6 得微分不等式：$E'(t) \le C E(t) + C\|f(\cdot,t)\|_{L^2}^2$，其中 $C$ 依赖于 $a, |b|_\infty, |c|_\infty$。

Step 7 Gronwall 引理：若 $y'(t) \le \alpha y(t) + \beta(t)$，则 $y(t) \le y(0)e^{\alpha t} + \int_0^t \beta(\tau)e^{\alpha(t-\tau)}\dd\tau$。

取 $y=E$，$\alpha=C$，$\beta(t)=C\|f(\cdot,t)\|_{L^2}^2$ 即得最终能量估计。$\square$

热传导方程能量估计（抛物型能量法）
▶ 展开证明

对 $u_t = a^2 u_{xx}$，$u(0,t)=u(l,t)=0$。定义 $E(t) = \frac{1}{2}\int_0^l u^2\dd x$。

$E'(t) = \int_0^l u u_t\dd x = a^2\!\int_0^l u u_{xx}\dd x$。

分部积分（边界项 $uu_x|_0^l = 0$）：$E'(t) = -a^2\!\int_0^l u_x^2\dd x \le 0$。

由 Poincaré 不等式 $\int_0^l u_x^2\dd x \ge \frac{\pi^2}{l^2}\int_0^l u^2\dd x$：$E'(t) \le -\frac{2a^2\pi^2}{l^2}E(t)$。

积分得 $E(t) \le E(0)e^{-2a^2\pi^2 t/l^2}$——能量指数衰减，与分离变量法中基频模态 $e^{-a^2(\pi/l)^2 t}$ 的结论一致。$\square$

椭圆型方程先验估计（Poisson 方程能量法）
▶ 展开证明

对 $-\lap u = f$（$\Omega$ 内），$u|_{\pd\Omega} = 0$。

以 $u$ 乘方程并积分：$-\int_\Omega u\lap u\dd x = \int_\Omega f u\dd x$。

左边用 Green 第一公式：$-\int_\Omega u\lap u = \int_\Omega |\grad u|^2\dd x - \int_{\pd\Omega} u\frac{\pd u}{\pd n}\dd S$。

边界项 $u|_{\pd\Omega}=0$ 得零。故 $\int_\Omega |\grad u|^2\dd x = \int_\Omega f u\dd x$。

Cauchy-Schwarz：$\int f u \le \|f\|_{L^2}\|u\|_{L^2}$。Poincaré 不等式：$\|u\|_{L^2} \le C_P\|\grad u\|_{L^2}$。

故 $\|\grad u\|_{L^2}^2 \le C_P\|f\|_{L^2}\|\grad u\|_{L^2}$，即 $\boxed{\|\grad u\|_{L^2} \le C_P\|f\|_{L^2}}$。$\square$

这是椭圆正则性的基础——源项 $f$ 在 $L^2$ 中控制了解的 $H^1$ 范数。

F.6 Sturm-Liouville 理论与本征函数展开

S-L 问题的标准形式与核心定理
$$-(p(x)y')' + q(x)y = \lambda w(x)y, \quad x \in (a,b), \quad p(x) > 0,\; w(x) > 0.$$

+ 分离边界条件：$\alpha_1 y(a) + \alpha_2 y'(a) = 0$，$\beta_1 y(b) + \beta_2 y'(b) = 0$。

核心结论：

  • 本征值 $\lambda_n$ 是实的，且 $\lambda_1 < \lambda_2 < \cdots \to +\infty$。
  • 本征函数 $\{y_n\}$ 关于权函数 $w(x)$ 正交：$\int_a^b y_m y_n w\,\dd x = 0$（$m \neq n$）。
  • $\{y_n\}$ 在 $L^2_w(a,b)$ 中完备：任意 $f$ 可展开 $f(x) = \sum_{n=1}^{\infty} c_n y_n(x)$，$c_n = \frac{\int_a^b f y_n w\,\dd x}{\int_a^b y_n^2 w\,\dd x}$。
正交性的证明（核心）
▶ 展开证明

设 $\lambda_m \neq \lambda_n$，对应本征函数 $y_m, y_n$。将 S-L 方程写为：

$$-(p y_m')' + q y_m = \lambda_m w y_m, \qquad -(p y_n')' + q y_n = \lambda_n w y_n.$$

第一式乘 $y_n$，第二式乘 $y_m$，相减并在 $[a,b]$ 上积分：

$$\int_a^b [y_n(p y_m')' - y_m(p y_n')']\dd x = (\lambda_n - \lambda_m)\int_a^b y_m y_n w\,\dd x.$$

左边 $= \int_a^b \frac{\dd}{\dd x}[p(y_n y_m' - y_m y_n')]\dd x = [p(y_n y_m' - y_m y_n')]_a^b$。

利用边界条件可证边界项为零（对三类分离边界条件均成立）。故：

$$(\lambda_n - \lambda_m)\int_a^b y_m y_n w\,\dd x = 0.$$

因 $\lambda_m \neq \lambda_n$，必须 $\int_a^b y_m y_n w\,\dd x = 0$。$\square$

推论：本征值是实的（取共轭即可证明），且所有本征函数可归一化构成标准正交系。
