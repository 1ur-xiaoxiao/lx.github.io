3.1 蒙特卡洛方法的核心思想

蒙特卡洛方法（Monte Carlo Method）

蒙特卡洛方法是一类基于随机采样的数值计算方法。其核心思想是：

  1. 把求解的问题转化为某个概率模型
  2. 通过大量随机试验来估计目标值
  3. 用统计量（如频率、均值）作为近似解

它的名称来源于摩纳哥的 Monte Carlo 赌场——因为其核心就是"随机"。也被称为统计模拟方法。

直觉：用投硬币估算概率

投一枚均匀硬币 1000 次，正面出现了 512 次。我们估计 $P(正面) \approx 512/1000 = 0.512$。

这里：概率模型 = 硬币投掷 → 随机试验 = 投 1000 次 → 统计量 = 频率 0.512 ≈ 真实概率 0.5

投得越多，估计越准——这就是大数定律的直觉，也是蒙特卡洛方法的理论基础。

为什么需要蒙特卡洛？很多问题没有解析解（比如高维积分、复杂系统的期望），但可以通过大量随机试验来逼近——虽然"不精确"，但"足够好"，而且实现简单。

3.2 Python 中的随机数

random 模块常用函数
import random

# 基本随机数
x = random.random()          # [0, 1) 均匀分布的随机浮点数
y = random.uniform(1, 10)    # [a, b] 均匀分布
z = random.randint(1, 6)     # [a, b] 随机整数（模拟骰子）

# 序列操作
items = [1, 2, 3, 4, 5]
a = random.choice(items)     # 随机选一个元素
random.shuffle(items)        # 原地打乱列表

# 按指定概率抽样
result = random.choices(
    ['A', 'B', 'C'],
    weights=[0.1, 0.3, 0.6],  # 各自的概率权重
    k=10                       # 抽样 10 次
)

# 设置随机种子（让结果可复现）
random.seed(42)              # 固定种子后，每次运行结果相同
伪随机数

计算机产生的其实是伪随机数——由确定性算法生成，看起来像随机。对于蒙特卡洛模拟来说，高质量的伪随机数（Python 使用 Mersenne Twister 算法）完全够用。设置随机种子 random.seed(42) 可以让每次运行得到相同的"随机"序列，方便调试。

3.3 经典案例：估算圆周率 $\pi$

方法：投点法

在 $[0,1] \times [0,1]$ 的正方形内随机投点。落在一个 1/4 单位圆内的概率 = 扇形面积 / 正方形面积 = $\pi/4$。

所以在正方形内投 $N$ 个点，统计落在 1/4 圆内的比例 $\frac{M}{N}$，则有：

$$\frac{M}{N} \approx \frac{\pi}{4} \quad\Longrightarrow\quad \pi \approx 4 \cdot \frac{M}{N}$$
估算 π 的 Python 实现
import random

def estimate_pi(n):
    """投 n 个点估算 π 值"""
    inside = 0
    for _ in range(n):
        x, y = random.random(), random.random()
        if x * x + y * y <= 1:   # 点在 1/4 单位圆内？
            inside += 1
    return 4 * inside / n

# 不同样本量的精度
for n in [100, 1000, 10000, 100000, 1000000]:
    pi_est = estimate_pi(n)
    error = abs(pi_est - 3.141592653589793)
    print(f"n={n:>8}: π≈{pi_est:.6f}, 误差={error:.6f}")
n= 100: π≈3.120000, 误差=0.021593 n= 1000: π≈3.144000, 误差=0.002407 n= 10000: π≈3.137200, 误差=0.004393 n= 100000: π≈3.142480, 误差=0.000887 n= 1000000: π≈3.141292, 误差=0.000301
收敛速率

蒙特卡洛方法的误差以 $O(1/\sqrt{n})$ 的速率收敛——每多一位有效数字，需要增加约 100 倍的样本量。这不是最快的数值方法，但它极其通用，对高维问题优势明显。

3.4 定积分的蒙特卡洛计算

方法：均值估计

对于定积分 $\int_a^b f(x)\,dx$，在 $[a,b]$ 上均匀随机取 $n$ 个点 $x_i$，则：

$$\int_a^b f(x)\,dx \approx \frac{b-a}{n} \sum_{i=1}^{n} f(x_i)$$

本质：积分 = 区间长度 × 函数均值。我们用随机采样的均值来估计函数的均值。

例 · 计算 $\int_0^1 x^2\,dx$

解析结果为 $1/3 \approx 0.3333$。

import random

def mc_integral(f, a, b, n):
    """蒙特卡洛计算 ∫_a^b f(x) dx"""
    total = 0
    for _ in range(n):
        x = random.uniform(a, b)   # 在 [a,b] 上均匀采样
        total += f(x)
    return (b - a) * total / n     # 区间长度 × 样本均值

f = lambda x: x * x
result = mc_integral(f, 0, 1, 100000)
print(f"∫_0^1 x² dx ≈ {result:.6f}")  # ≈ 0.3333**
例 · 计算 $\int_0^\pi \sin x\,dx$

解析结果为 2。

import math

result = mc_integral(math.sin, 0, math.pi, 100000)
print(f"∫_0^π sin(x) dx ≈ {result:.6f}")  # ≈ 2.000...

3.5 Buffon 投针问题

Buffon 投针（1777 年）

平面上画有间距为 $d$ 的平行线。随机投一根长度为 $\ell$（$\ell < d$）的针，针与任意平行线相交的概率为：

$$P(\text{相交}) = \frac{2\ell}{\pi d}$$

通过大量投针实验统计相交频率，就可以反解出 $\pi$。这是历史上最早用概率方法估算 $\pi$ 的尝试之一。

Buffon 投针模拟
import random, math

def buffon_needle(n, d=1.0, l=0.8):
    """模拟 n 次 Buffon 投针，d=线间距, l=针长"""
    crosses = 0
    for _ in range(n):
        # 针的中心到最近平行线的距离
        y = random.uniform(0, d / 2)
        # 针与平行线的夹角
        theta = random.uniform(0, math.pi / 2)
        # 相交条件：针的垂直投影半长 > 到线的距离
        if y <= (l / 2) * math.sin(theta):
            crosses += 1
    pi_est = (2 * l * n) / (d * crosses) if crosses > 0 else 0
    return crosses / n, pi_est

prob, pi_est = buffon_needle(100000)
print(f"相交频率: {prob:.4f}  (理论值: {2*0.8/math.pi:.4f})")
print(f"π 估计值: {pi_est:.6f}")

3.6 概率分布采样

逆变换采样（Inverse Transform Sampling）

如何生成服从任意分布的随机数？逆变换法：若 $F(x)$ 是目标分布的 CDF，则 $X = F^{-1}(U)$（$U \sim \text{Uniform}(0,1)$）服从该分布。

例 · 生成指数分布的随机数

指数分布 Exp($\lambda$) 的 PDF 为 $f(x)=\lambda e^{-\lambda x}$，CDF 为 $F(x)=1-e^{-\lambda x}$，反函数为 $F^{-1}(u)=-\frac{1}{\lambda}\ln(1-u)$。

import random, math

def exponential_samples(lmbda, n):
    """用逆变换法生成 n 个 Exp(λ) 样本"""
    samples = []
    for _ in range(n):
        u = random.random()
        x = -math.log(1 - u) / lmbda  # F^{-1}(u)
        samples.append(x)
    return samples

samples = exponential_samples(lmbda=2.0, n=10000)
mean = sum(samples) / len(samples)
print(f"样本均值: {mean:.4f}  (理论值: 1/2 = 0.5)")
更多采样方法

除了逆变换法，还有取舍采样（Accept-Reject）、MCMC（马尔可夫链蒙特卡洛）、重要性采样等。在实际的贝叶斯统计和机器学习中，MCMC 是核心工具。

3.7 综合例题

例题1 · 估算 $e$

思路：在 $[0,1]$ 均匀采样 $X_i$，不断累加直到总和超过 1。设 $N$ 为需要的个数，则 $E[N] = e$。

import random

def estimate_e(trials=100000):
    total_n = 0
    for _ in range(trials):
        s, n = 0, 0
        while s <= 1:
            s += random.random()
            n += 1
        total_n += n
    return total_n / trials

print(f"e ≈ {estimate_e():.6f}")  # ≈ 2.718...
例题2 · 计算高维球体的体积

蒙特卡洛方法的真正威力在高维问题中显现。$d$ 维单位超球的体积有解析公式，但我们用模拟来验证。

import random, math

def sphere_volume_mc(d, n=100000):
    """蒙特卡洛估算 d 维单位超球的体积"""
    inside = 0
    for _ in range(n):
        # 在 [-1, 1]^d 的超立方体内随机取点
        point = [random.uniform(-1, 1) for _ in range(d)]
        dist2 = sum(x * x for x in point)  # 到原点的距离平方
        if dist2 <= 1:
            inside += 1
    # 超立方体体积为 2^d，球体积 = 2^d × 在里面 的比例
    return (2 ** d) * inside / n

for d in [2, 3, 5, 10]:
    vol = sphere_volume_mc(d, 200000)
    print(f"d={d}: 超球体积 ≈ {vol:.4f}")
▶ 练习 · 用蒙特卡洛验证生日悖论

生日悖论：一个房间里有 23 人，至少两人生日相同的概率超过 50%。用蒙特卡洛模拟验证。

import random

def birthday_sim(n_people, trials=10000):
    """模拟 n_people 人中至少两人生日相同的概率"""
    same = 0
    for _ in range(trials):
        birthdays = [random.randint(1, 365) for _ in range(n_people)]
        if len(birthdays) != len(set(birthdays)):
            same += 1              # 有重复 → 生日碰撞
    return same / trials

for n in [10, 23, 30, 50, 70]:
    prob = birthday_sim(n)
    print(f"n={n}: P(碰撞) ≈ {prob:.4f}")
# n=10: ≈ 0.117 | n=23: ≈ 0.507 | n=30: ≈ 0.706
# n=50: ≈ 0.970 | n=70: ≈ 0.999
