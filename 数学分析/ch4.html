第四章 · 一元积分学

从求和到反导数：Riemann 积分的严格构造、Newton-Leibniz 公式将微分与积分统一，以及反常积分的审敛。

本章目录

  1. 不定积分
  2. Riemann 积分的定义

  5. 微积分基本定理

4.1 不定积分

定义 · 原函数与不定积分

若 $F'(x) = f(x)$，则称 $F(x)$ 为 $f(x)$ 的一个原函数。$f(x)$ 的全体原函数称为 $f(x)$ 的不定积分，记为

$$\int f(x)\,\dd x = F(x) + C$$
基本积分公式

$\int x^n \dd x = \dfrac{x^{n+1}}{n+1} + C$（$n \neq -1$），$\int \dfrac{1}{x} \dd x = \ln|x| + C$

$\int e^x \dd x = e^x + C$，$\int a^x \dd x = \dfrac{a^x}{\ln a} + C$

$\int \sin x \dd x = -\cos x + C$，$\int \cos x \dd x = \sin x + C$

$\int \sec^2 x \dd x = \tan x + C$，$\int \dfrac{1}{1+x^2} \dd x = \arctan x + C$

$\int \dfrac{1}{\sqrt{1-x^2}} \dd x = \arcsin x + C$

换元积分法

第一类换元法（凑微分）：$\int f(g(x)) g'(x) \dd x = \int f(u) \dd u \Big|_{u=g(x)}$

第二类换元法（变量替换）：令 $x = \varphi(t)$，则 $\int f(x) \dd x = \int f(\varphi(t)) \varphi'(t) \dd t$

分部积分法
$$\int u \dd v = uv - \int v \dd u$$

口诀：反对幂指三——优先选 $u$ 的顺序：反三角 > 对数 > 幂函数 > 指数 > 三角。

▶ 例题 · 分部积分

题目：求 $\int x e^x \dd x$。

解：令 $u = x$，$\dd v = e^x \dd x$，则 $\dd u = \dd x$，$v = e^x$。

$\int x e^x \dd x = xe^x - \int e^x \dd x = xe^x - e^x + C = (x-1)e^x + C$

4.2 Riemann 积分的定义

定义 · Riemann 积分

设 $f$ 在 $[a,b]$ 上有界。对 $[a,b]$ 的任意分划 $T: a = x_0 < x_1 < \cdots < x_n = b$，记 $\Delta x_i = x_i - x_{i-1}$，$\|T\| = \max_i \Delta x_i$。

在每个小区间 $[x_{i-1}, x_i]$ 上任取 $\xi_i$，作 Riemann 和

$$S(T, f) = \sum_{i=1}^n f(\xi_i) \Delta x_i$$

若当 $\|T\| \to 0$ 时，$S(T,f)$ 的极限存在且与分划和 $\xi_i$ 的选取无关，则称 $f$ 在 $[a,b]$ 上 Riemann 可积，此极限值为 $f$ 在 $[a,b]$ 上的定积分，记为

$$\int_a^b f(x)\,\dd x = \lim_{\|T\| \to 0} \sum_{i=1}^n f(\xi_i) \Delta x_i$$

4.3 可积条件

可积的充分条件
  1. $f$ 在 $[a,b]$ 上连续 $\Rightarrow$ $f$ 在 $[a,b]$ 上可积。
  2. $f$ 在 $[a,b]$ 上单调 $\Rightarrow$ $f$ 在 $[a,b]$ 上可积。
  3. $f$ 在 $[a,b]$ 上有界，且只有有限个间断点 $\Rightarrow$ $f$ 在 $[a,b]$ 上可积。
可积 vs 连续

可积不要求连续！有界且间断点"足够少"（如有限个或零测集）的函数也可积。但无界函数一定不可积。

4.4 定积分的性质

定积分的基本性质
  1. 线性：$\int_a^b [\alpha f + \beta g]\dd x = \alpha \int_a^b f\dd x + \beta \int_a^b g\dd x$
  2. 区间可加：$\int_a^b f\dd x = \int_a^c f\dd x + \int_c^b f\dd x$
  3. 保序：若 $f \leq g$，则 $\int_a^b f\dd x \leq \int_a^b g\dd x$
  4. 绝对值不等式：$\left|\int_a^b f\dd x\right| \leq \int_a^b |f|\dd x$
积分中值定理

第一中值定理：若 $f$ 在 $[a,b]$ 上连续，则存在 $\xi \in [a,b]$ 使得

$$\int_a^b f(x)\dd x = f(\xi)(b-a)$$

推广形式：若 $f$ 连续，$g$ 在 $[a,b]$ 上不变号，则存在 $\xi \in [a,b]$ 使得

$$\int_a^b f(x)g(x)\dd x = f(\xi)\int_a^b g(x)\dd x$$

4.5 微积分基本定理

微积分基本定理 I（变限积分求导）

若 $f$ 在 $[a,b]$ 上连续，则变限积分 $\Phi(x) = \int_a^x f(t)\dd t$ 在 $[a,b]$ 上可导，且

$$\Phi'(x) = \frac{\dd}{\dd x}\int_a^x f(t)\dd t = f(x)$$

推广：$\dfrac{\dd}{\dd x}\int_{a(x)}^{b(x)} f(t)\dd t = f(b(x)) \cdot b'(x) - f(a(x)) \cdot a'(x)$

微积分基本定理 II（Newton-Leibniz 公式）

若 $f$ 在 $[a,b]$ 上连续，$F$ 是 $f$ 的一个原函数，则

$$\int_a^b f(x)\dd x = F(b) - F(a) = F(x)\Big|_a^b$$

意义：定积分的计算归结为求原函数——微分与积分的完美统一。

▶ 例题 · 变限积分求导

题目：设 $F(x) = \int_0^{x^2} e^{t^2}\dd t$，求 $F'(x)$。

解：令 $u = x^2$，则 $F(x) = \int_0^u e^{t^2}\dd t$。

由链式法则：$F'(x) = e^{u^2} \cdot u' = e^{x^4} \cdot 2x = 2xe^{x^4}$

4.6 反常积分

定义 · 无穷限反常积分

$\int_a^{+\infty} f(x)\dd x = \lim_{b \to +\infty} \int_a^b f(x)\dd x$。若此极限存在（有限），则称反常积分收敛，否则发散。

定义 · 瑕积分

若 $f$ 在 $x=b$ 附近无界（$b$ 为瑕点），则

$$\int_a^b f(x)\dd x = \lim_{\eps \to 0^+} \int_a^{b-\eps} f(x)\dd x$$
比较审敛法

若 $0 \leq f(x) \leq g(x)$（$x$ 充分大），则：

  • $\int_a^{+\infty} g(x)\dd x$ 收敛 $\Rightarrow$ $\int_a^{+\infty} f(x)\dd x$ 收敛
  • $\int_a^{+\infty} f(x)\dd x$ 发散 $\Rightarrow$ $\int_a^{+\infty} g(x)\dd x$ 发散
$p$-积分的敛散性

$\int_1^{+\infty} \dfrac{1}{x^p}\dd x$：$p > 1$ 收敛，$p \leq 1$ 发散。

$\int_0^1 \dfrac{1}{x^p}\dd x$（$p > 0$，$x=0$ 为瑕点）：$p < 1$ 收敛，$p \geq 1$ 发散。

本章解题技巧总结
  1. 不定积分：先判断类型（有理式、三角、根式），选择合适的换元或分部。
  2. 定积分计算：Newton-Leibniz 公式是首选，奇偶函数在对称区间上的积分可简化。
  3. 变限积分求导：先将被积函数中的 $x$ 提到积分号外或做变量替换。
  4. 反常积分：先判断类型（无穷限/瑕积分），用比较审敛法或 $p$-积分判定。
