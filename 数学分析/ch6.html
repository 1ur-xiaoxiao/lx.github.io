第六章 · 多元函数微分学

从一维到高维的推广：多元函数的极限与连续、偏导数与全微分、隐函数定理与 Lagrange 乘数法。

6.1 多元函数的极限与连续

定义 · 多元函数的极限

设 $f$ 在点 $P_0 = (x_0, y_0)$ 的某去心邻域内有定义，$A \in \RR$。若对任意 $\eps > 0$，存在 $\delta > 0$，当 $0 < \|P - P_0\| < \delta$ 时（即 $0 < \sqrt{(x-x_0)^2+(y-y_0)^2} < \delta$），

$$|f(x,y) - A| < \eps$$

则记 $\lim_{(x,y) \to (x_0,y_0)} f(x,y) = A$。

多元极限的困难

多元极限要求 $(x,y)$ 沿任何路径趋向 $(x_0,y_0)$ 时极限都相同。证明极限存在通常用夹逼定理，证明极限不存在只需找两条不同路径使极限不同。

▶ 例题 · 证明二重极限不存在

题目：证明 $\lim_{(x,y) \to (0,0)} \dfrac{xy}{x^2+y^2}$ 不存在。

证明：沿 $y = kx$ 趋向原点：

$\lim_{x \to 0} \dfrac{x \cdot kx}{x^2 + k^2 x^2} = \dfrac{k}{1+k^2}$

极限值依赖于 $k$（不同路径得到不同极限），故原极限不存在。

定义 · 多元函数的连续

$f$ 在 $(x_0, y_0)$ 连续：$\lim_{(x,y) \to (x_0,y_0)} f(x,y) = f(x_0, y_0)$。

有界闭区域上的连续函数同样具有有界性、最值定理和介值定理。

6.2 偏导数

定义 · 偏导数

$f$ 在 $(x_0, y_0)$ 处对 $x$ 的偏导数：

$$f_x(x_0, y_0) = \lim_{\Delta x \to 0} \frac{f(x_0 + \Delta x, y_0) - f(x_0, y_0)}{\Delta x}$$

即固定 $y = y_0$，对 $x$ 求导。类似定义 $f_y$。

Clairaut 定理（混合偏导数与顺序无关）

若 $f_{xy}$ 和 $f_{yx}$ 在某区域内连续，则 $f_{xy} = f_{yx}$。

偏导数存在 $\not\Rightarrow$ 连续

与一元函数不同，多元函数偏导数存在不能保证连续。例如 $f(x,y) = \begin{cases} \dfrac{xy}{x^2+y^2}, & (x,y) \neq (0,0) \\ 0, & (x,y) = (0,0) \end{cases}$

在原点 $f_x(0,0) = f_y(0,0) = 0$，但 $f$ 在原点不连续。

6.3 全微分

定义 · 全微分

若 $f$ 在 $(x_0, y_0)$ 处的全增量可以写成

$$\Delta z = f(x_0+\Delta x, y_0+\Delta y) - f(x_0, y_0) = A\Delta x + B\Delta y + o(\rho)$$

其中 $\rho = \sqrt{(\Delta x)^2 + (\Delta y)^2}$，$A, B$ 与 $\Delta x, \Delta y$ 无关，则称 $f$ 在 $(x_0,y_0)$ 处可微，$\dd z = A\dd x + B\dd y$ 为全微分。

可微的必要条件与充分条件

必要条件：可微 $\Rightarrow$ 偏导数存在，且 $A = f_x$，$B = f_y$，即 $\dd z = f_x \dd x + f_y \dd y$。

充分条件：偏导数 $f_x, f_y$ 连续 $\Rightarrow$ 可微。

关系链：偏导连续 $\Rightarrow$ 可微 $\Rightarrow$ 偏导存在，可微 $\Rightarrow$ 连续。

6.4 复合函数求导（链式法则）

链式法则

设 $z = f(u,v)$，$u = \varphi(x,y)$，$v = \psi(x,y)$，则

$$\frac{\partial z}{\partial x} = \frac{\partial z}{\partial u}\frac{\partial u}{\partial x} + \frac{\partial z}{\partial v}\frac{\partial v}{\partial x}$$ $$\frac{\partial z}{\partial y} = \frac{\partial z}{\partial u}\frac{\partial u}{\partial y} + \frac{\partial z}{\partial v}\frac{\partial v}{\partial y}$$

记忆：画出变量之间的依赖树，沿每条路径求偏导后相乘，再将所有路径相加。

▶ 例题 · 链式法则

题目：设 $z = e^{u}\sin v$，$u = xy$，$v = x+y$，求 $\dfrac{\partial z}{\partial x}$。

解：$\dfrac{\partial z}{\partial x} = \dfrac{\partial z}{\partial u}\dfrac{\partial u}{\partial x} + \dfrac{\partial z}{\partial v}\dfrac{\partial v}{\partial x}$

$= e^u \sin v \cdot y + e^u \cos v \cdot 1$

$= e^{xy}[y\sin(x+y) + \cos(x+y)]$

6.5 隐函数定理

隐函数存在定理

设 $F(x,y)$ 在 $(x_0,y_0)$ 附近有连续偏导数，$F(x_0,y_0) = 0$，$F_y(x_0,y_0) \neq 0$，

则在 $(x_0,y_0)$ 附近存在唯一连续可微函数 $y = y(x)$ 满足 $F(x, y(x)) = 0$，且

$$\frac{\dd y}{\dd x} = -\frac{F_x}{F_y}$$
隐函数求导的方法

方法一：公式法，直接用 $\dfrac{\dd y}{\dd x} = -\dfrac{F_x}{F_y}$。

方法二：方程两边对 $x$ 求导（注意 $y$ 是 $x$ 的函数），解出 $y'$。

▶ 例题 · 隐函数求导

题目：设 $x^2 + y^2 + z^2 = 1$ 确定 $z = z(x,y)$，求 $\dfrac{\partial z}{\partial x}$ 和 $\dfrac{\partial z}{\partial y}$。

解：令 $F = x^2 + y^2 + z^2 - 1$。

$F_x = 2x$，$F_y = 2y$，$F_z = 2z$

$\dfrac{\partial z}{\partial x} = -\dfrac{F_x}{F_z} = -\dfrac{x}{z}$，$\dfrac{\partial z}{\partial y} = -\dfrac{F_y}{F_z} = -\dfrac{y}{z}$（$z \neq 0$）

6.6 极值与条件极值

无条件极值的必要条件

若 $f(x,y)$ 在 $(x_0,y_0)$ 取极值且偏导存在，则 $f_x(x_0,y_0) = f_y(x_0,y_0) = 0$。

无条件极值的充分条件（Hessian 判别法）

设 $f_x(x_0,y_0) = f_y(x_0,y_0) = 0$，记 $A = f_{xx}$，$B = f_{xy}$，$C = f_{yy}$（均在 $(x_0,y_0)$ 处取值），$\Delta = AC - B^2$：

  • $\Delta > 0$ 且 $A > 0$：极小值
  • $\Delta > 0$ 且 $A < 0$：极大值
  • $\Delta < 0$：非极值（鞍点）
  • $\Delta = 0$：无法判定
Lagrange 乘数法（条件极值）

求 $f(x,y,z)$ 在约束条件 $g(x,y,z) = 0$ 下的极值。

构造 Lagrange 函数：

$$L(x,y,z,\lambda) = f(x,y,z) - \lambda g(x,y,z)$$

令所有偏导为零：

$$L_x = 0, \quad L_y = 0, \quad L_z = 0, \quad L_\lambda = 0 \text{（即 } g=0\text{）}$$

解方程组得到候选极值点。

▶ 例题 · Lagrange 乘数法

题目：求 $f(x,y) = xy$ 在约束 $x + y = 1$ 下的最大值。

解：$L = xy - \lambda(x+y-1)$

$L_x = y - \lambda = 0$，$L_y = x - \lambda = 0$，$L_\lambda = -(x+y-1) = 0$

由前两式：$y = \lambda = x$，代入第三式：$2x = 1$，$x = y = \dfrac{1}{2}$。

$f\left(\dfrac{1}{2}, \dfrac{1}{2}\right) = \dfrac{1}{4}$ 为最大值。

本章解题技巧总结
  1. 二重极限：证明存在用夹逼定理，证明不存在找两条不同路径。
  2. 偏导数计算：固定其他变量，对目标变量求导。
  3. 链式法则：画变量依赖树，沿路径求导后相加。
  4. 隐函数求导：公式法 $y' = -F_x/F_y$ 最简洁。
  5. 条件极值：Lagrange 乘数法是标准方法，解方程组得到候选点。
