第八章 · 欧几里得空间与酉空间

在线性空间中引入"度量"结构：内积使我们可以讨论长度、角度和正交性，这是几何与代数深度融合的篇章。

8.1 欧几里得空间的定义

定义 · 内积与欧几里得空间

设 $V$ 是 $\RR$ 上的线性空间。映射 $(\cdot, \cdot): V \times V \to \RR$ 若满足：

  1. 对称性：$(\alpha, \beta) = (\beta, \alpha)$
  2. 线性性：$(k_1\alpha_1+k_2\alpha_2, \beta) = k_1(\alpha_1,\beta) + k_2(\alpha_2,\beta)$
  3. 正定性：$(\alpha, \alpha) \geq 0$，且 $(\alpha, \alpha) = 0 \iff \alpha = 0$

则称 $(\cdot, \cdot)$ 为 $V$ 上的一个内积，$V$ 连同此内积称为欧几里得空间（内积空间）。

标准内积

在 $\RR^n$ 中，$(\alpha, \beta) = \alpha^T \beta = x_1y_1 + x_2y_2 + \cdots + x_ny_n$。

更一般地，对正定对称矩阵 $A$，$(\alpha, \beta) = \alpha^T A \beta$ 也是一个内积。

内积的基本性质

长度（范数）：$\|\alpha\| = \sqrt{(\alpha, \alpha)}$

Cauchy-Schwarz 不等式：$|(\alpha, \beta)| \leq \|\alpha\| \cdot \|\beta\|$

等号成立 $\iff \alpha$ 与 $\beta$ 线性相关。

三角不等式：$\|\alpha + \beta\| \leq \|\alpha\| + \|\beta\|$

夹角（$\alpha, \beta \neq 0$）：$\cos \theta = \dfrac{(\alpha, \beta)}{\|\alpha\| \|\beta\|}$

▶ 证明 · Cauchy-Schwarz 不等式

证明：

对任意 $t \in \RR$，考虑 $\|\alpha + t\beta\|^2 \geq 0$：

$$0 \leq (\alpha+t\beta,\; \alpha+t\beta) = (\alpha,\alpha) + 2t(\alpha,\beta) + t^2(\beta,\beta) = \|\alpha\|^2 + 2t(\alpha,\beta) + t^2\|\beta\|^2$$

上述关于 $t$ 的二次式对一切 $t$ 均非负，故其判别式 $\Delta \leq 0$：

$$4(\alpha,\beta)^2 - 4\|\alpha\|^2\|\beta\|^2 \leq 0 \quad\Longrightarrow\quad |(\alpha,\beta)| \leq \|\alpha\|\;\|\beta\|$$

等号成立条件：判别式 $= 0$，此时存在某个 $t_0$ 使得 $\|\alpha + t_0\beta\|^2 = 0$，即 $\alpha + t_0\beta = 0$，故 $\alpha$ 与 $\beta$ 线性相关。

∎

8.2 标准正交基

定义 · 正交与标准正交基

若 $(\alpha, \beta) = 0$，称 $\alpha$ 与 $\beta$ 正交，记为 $\alpha \perp \beta$。

一组两两正交的非零向量称为正交组。单位长度的正交组称为标准正交组。

由标准正交组构成的基称为标准正交基。此时 $(\varepsilon_i, \varepsilon_j) = \delta_{ij}$。

Gram-Schmidt 正交化

设 $\alpha_1, \ldots, \alpha_m$ 线性无关，构造正交组 $\beta_1, \ldots, \beta_m$：

$$\beta_1 = \alpha_1, \qquad \beta_k = \alpha_k - \sum_{i=1}^{k-1} \frac{(\alpha_k, \beta_i)}{(\beta_i, \beta_i)} \beta_i \quad (k \geq 2)$$

再单位化：$\eta_i = \beta_i / \|\beta_i\|$，即得标准正交组。

▶ 证明 · Gram-Schmidt 正交化的正确性

证明：

对 $k$ 进行归纳，证明 $\beta_1, \ldots, \beta_k$ 两两正交且 $\span\{\beta_1,\ldots,\beta_k\} = \span\{\alpha_1,\ldots,\alpha_k\}$。

$k=1$：$\beta_1 = \alpha_1$，平凡成立。假设对 $k-1$ 成立。则

$$\beta_k = \alpha_k - \sum_{i=1}^{k-1} \frac{(\alpha_k, \beta_i)}{(\beta_i, \beta_i)} \beta_i$$

对任意 $j < k$，计算：

$$(\beta_k, \beta_j) = (\alpha_k, \beta_j) - \sum_{i=1}^{k-1} \frac{(\alpha_k, \beta_i)}{(\beta_i, \beta_i)} (\beta_i, \beta_j)$$

由归纳假设，$i \neq j$ 时 $(\beta_i, \beta_j) = 0$，故求和式中仅 $i=j$ 一项非零：

$$(\beta_k, \beta_j) = (\alpha_k, \beta_j) - \frac{(\alpha_k, \beta_j)}{(\beta_j, \beta_j)} (\beta_j, \beta_j) = 0$$

因此 $\beta_k \perp \span\{\beta_1, \ldots, \beta_{k-1}\}$。又 $\beta_k \neq 0$，因为 $\alpha_k \notin \span\{\alpha_1,\ldots,\alpha_{k-1}\} = \span\{\beta_1,\ldots,\beta_{k-1}\}$，减去投影不会使其为零。显然 $\beta_k$ 是 $\alpha_k$ 与 $\beta_1,\ldots,\beta_{k-1}$ 的线性组合，故张成的子空间相同。单位化后 $\eta_i = \beta_i / \|\beta_i\|$ 即得标准正交组。

∎

标准正交基的性质
  1. 标准正交基下的内积坐标公式：若 $\alpha, \beta$ 在标准正交基 $\varepsilon_1, \ldots, \varepsilon_n$ 下的坐标分别为 $X, Y$，则 $(\alpha, \beta) = X^T Y$。
  2. 任何线性无关向量组都可以扩充为标准正交基。
  3. 两组标准正交基之间的过渡矩阵是正交矩阵。
▶ 例题 · Gram-Schmidt 正交化

题目：在 $\RR^3$ 中将 $\alpha_1 = (1,1,0)^T$, $\alpha_2 = (1,0,1)^T$, $\alpha_3 = (0,1,1)^T$ 化为标准正交基。

解：

$\beta_1 = \alpha_1 = (1,1,0)^T$

$\beta_2 = \alpha_2 - \frac{(\alpha_2,\beta_1)}{(\beta_1,\beta_1)}\beta_1 = (1,0,1)^T - \frac{1}{2}(1,1,0)^T = (\frac{1}{2}, -\frac{1}{2}, 1)^T$

$\beta_3 = \alpha_3 - \frac{(\alpha_3,\beta_1)}{(\beta_1,\beta_1)}\beta_1 - \frac{(\alpha_3,\beta_2)}{(\beta_2,\beta_2)}\beta_2$

$= (0,1,1)^T - \frac{1}{2}(1,1,0)^T - \frac{1/2}{3/2}(\frac{1}{2}, -\frac{1}{2}, 1)^T = (-\frac{2}{3}, \frac{2}{3}, \frac{2}{3})^T$

单位化后得标准正交基。

8.3 正交补与正交投影

定义 · 正交补

$V$ 的子集 $S$ 的正交补定义为

$$S^\perp = \{\alpha \in V : (\alpha, \beta) = 0 \text{ 对所有 } \beta \in S\}$$

$S^\perp$ 是 $V$ 的子空间，且 $V = W \oplus W^\perp$（对任意子空间 $W$）。

正交补的性质
  1. $(W^\perp)^\perp = W$
  2. $W_1 \subseteq W_2 \Rightarrow W_2^\perp \subseteq W_1^\perp$
  3. $\dim W^\perp = \dim V - \dim W$
  4. $(W_1 + W_2)^\perp = W_1^\perp \cap W_2^\perp$，$(W_1 \cap W_2)^\perp = W_1^\perp + W_2^\perp$
定义 · 正交投影

设 $V = W \oplus W^\perp$，对任意 $\alpha \in V$，有唯一分解 $\alpha = \alpha_1 + \alpha_2$（$\alpha_1 \in W, \alpha_2 \in W^\perp$）。

映射 $P_W: V \to V$，$\alpha \mapsto \alpha_1$ 称为到 $W$ 的正交投影。

若 $\varepsilon_1, \ldots, \varepsilon_k$ 是 $W$ 的标准正交基，则

$$P_W(\alpha) = \sum_{i=1}^k (\alpha, \varepsilon_i) \varepsilon_i$$

8.4 正交变换与正交矩阵

定义 · 正交变换

欧几里得空间 $V$ 上的线性变换 $\mathcal{A}$ 若保持内积：

$$(\mathcal{A}\alpha, \mathcal{A}\beta) = (\alpha, \beta), \qquad \forall \alpha, \beta \in V$$

则称 $\mathcal{A}$ 为正交变换。等价条件：$\|\mathcal{A}\alpha\| = \|\alpha\|$（保持长度）。

正交变换的等价刻画

设 $\mathcal{A}$ 在标准正交基下的矩阵为 $Q$。以下等价：

  1. $\mathcal{A}$ 是正交变换
  2. $Q^T Q = I$（$Q$ 是正交矩阵）
  3. $Q$ 的行（列）向量组是标准正交基
  4. $\mathcal{A}$ 将标准正交基映为标准正交基
  5. $\det Q = \pm 1$（必要条件，非充要）
正交变换的几何意义

$\RR^2$ 中的正交变换只有两类：

  • $\det = +1$：旋转（绕原点旋转某角度 $\theta$）
  • $\det = -1$：反射（关于某条过原点的直线对称）

$\RR^3$ 中的正交变换包括旋转、反射、以及旋转与反射的复合。$\det = +1$ 的正交变换对应三维空间的旋转。

8.5 对称变换与实对称矩阵

定义 · 对称变换（自伴变换）

欧几里得空间 $V$ 上的线性变换 $\mathcal{A}$ 若满足

$$(\mathcal{A}\alpha, \beta) = (\alpha, \mathcal{A}\beta), \qquad \forall \alpha, \beta \in V$$

则称 $\mathcal{A}$ 为对称变换。在标准正交基下，对称变换的矩阵是实对称矩阵（$A^T = A$）。

实对称矩阵的性质
  1. 特征值全为实数（包括计重数）
  2. 不同特征值的特征向量正交
  3. 必可正交对角化：存在正交矩阵 $Q$ 使得 $Q^T A Q = \diag(\lambda_1, \ldots, \lambda_n)$
  4. $A$ 是正定的 $\iff$ 所有特征值 $> 0$
▶ 证明 · 实对称矩阵的性质

证明（特征值全为实数）：

设 $A$ 为实对称矩阵，$\lambda$ 为特征值，$\mathbf{x}$ 为对应的特征向量。考虑 $x^* A x$（共轭转置）：

$$x^* A x = x^*(A x) = x^*(\lambda x) = \lambda\, x^* x = \lambda \|\mathbf{x}\|^2$$

另一方面，利用 $A = A^T = \overline{A}$（$A$ 为实对称）：

$$x^* A x = (x^* A) x = (A x)^* x = (\lambda x)^* x = \overline{\lambda}\, x^* x = \overline{\lambda} \|\mathbf{x}\|^2$$

于是 $\lambda \|\mathbf{x}\|^2 = \overline{\lambda} \|\mathbf{x}\|^2$。由 $\|\mathbf{x}\|^2 \neq 0$（$\mathbf{x}$ 为特征向量），得 $\lambda = \overline{\lambda}$，故 $\lambda \in \RR$。

证明（不同特征值的特征向量正交）：

设 $A\alpha = \lambda\alpha$, $A\beta = \mu\beta$，且 $\lambda \neq \mu$。则：

$$\lambda(\alpha, \beta) = (A\alpha, \beta) = (A\alpha)^T \beta = \alpha^T A^T \beta = \alpha^T (A\beta) = \alpha^T (\mu\beta) = \mu(\alpha, \beta)$$

于是 $(\lambda - \mu)(\alpha, \beta) = 0$。由 $\lambda \neq \mu$，必有 $(\alpha, \beta) = 0$，即 $\alpha \perp \beta$。

∎

正交对角化的证明思路

对维数归纳：取 $A$ 的一个特征值 $\lambda_1$（实数），取单位特征向量 $\xi_1$。将 $\xi_1$ 扩充为 $\RR^n$ 的标准正交基 $\xi_1, \eta_2, \ldots, \eta_n$，构造正交矩阵 $Q_1 = (\xi_1, \eta_2, \ldots, \eta_n)$。则 $Q_1^T A Q_1 = \begin{pmatrix} \lambda_1 & 0 \\ 0 & A_1 \end{pmatrix}$，$A_1$ 仍为对称矩阵。由归纳假设得证。

8.6 酉空间

定义 · 复内积与酉空间

设 $V$ 是 $\CC$ 上的线性空间。映射 $(\cdot, \cdot): V \times V \to \CC$ 若满足：

  1. 共轭对称性：$(\alpha, \beta) = \overline{(\beta, \alpha)}$
  2. 线性性（对第一个变量）：$(k_1\alpha_1+k_2\alpha_2, \beta) = k_1(\alpha_1,\beta) + k_2(\alpha_2,\beta)$
  3. 共轭线性性（对第二个变量）：$(\alpha, k_1\beta_1+k_2\beta_2) = \overline{k_1}(\alpha,\beta_1) + \overline{k_2}(\alpha,\beta_2)$
  4. 正定性：$(\alpha, \alpha) \in \RR_{\geq 0}$，且 $(\alpha, \alpha) = 0 \iff \alpha = 0$

则称 $V$ 为酉空间（复内积空间）。

标准复内积

在 $\CC^n$ 中，$(\alpha, \beta) = \alpha^* \beta = \overline{x_1}y_1 + \cdots + \overline{x_n}y_n$（$\alpha^* = \bar{\alpha}^T$ 为共轭转置）。

定义 · 酉变换与酉矩阵

$V$ 上的线性变换 $\mathcal{A}$ 若保持复内积，称为酉变换。

标准正交基下酉变换的矩阵 $U$ 满足 $U^* U = I$，称为酉矩阵（unitary matrix）。

性质：酉矩阵的特征值的模为 $1$。

定义 · Hermite 变换与 Hermite 矩阵

$V$ 上的线性变换 $\mathcal{A}$ 若满足 $(\mathcal{A}\alpha, \beta) = (\alpha, \mathcal{A}\beta)$，称为Hermite 变换。

标准正交基下 Hermite 变换的矩阵 $H$ 满足 $H^* = H$，称为Hermite 矩阵。

Hermite 矩阵的特征值全为实数（与实对称矩阵类似）。

8.7 正规变换与谱定理

定义 · 正规变换与正规矩阵

设 $V$ 是酉空间。线性变换 $\mathcal{A}$ 若满足 $\mathcal{A}\mathcal{A}^* = \mathcal{A}^*\mathcal{A}$，则称 $\mathcal{A}$ 为正规变换。

其中 $\mathcal{A}^*$ 是 $\mathcal{A}$ 的伴随变换：$(\mathcal{A}\alpha, \beta) = (\alpha, \mathcal{A}^*\beta)$。

矩阵形式：$A A^* = A^* A$，称 $A$ 为正规矩阵。

谱定理（Spectral Theorem）

设 $V$ 是有限维酉空间，$\mathcal{A}$ 是 $V$ 上的线性变换。则

$$\text{$\mathcal{A}$ 可酉对角化} \iff \text{$\mathcal{A}$ 是正规变换}$$

即存在标准正交基使得 $\mathcal{A}$ 的矩阵为对角形 $\iff \mathcal{A}\mathcal{A}^* = \mathcal{A}^*\mathcal{A}$。

▶ 证明 · 谱定理（正规⇔酉对角化）

证明：

($\Leftarrow$) 正规⇐酉对角化：若 $A = U\Lambda U^*$，其中 $U$ 为酉矩阵、$\Lambda$ 为对角阵，则 $A^* = U\overline{\Lambda}U^*$，于是

$$AA^* = U\Lambda\overline{\Lambda}U^* = U\overline{\Lambda}\Lambda U^* = A^*A$$

故 $A$ 为正规矩阵。

($\Rightarrow$) 正规⇒酉对角化：利用 Schur 定理，存在酉矩阵 $U$ 使得 $A = UTU^*$，其中 $T$ 为上三角矩阵。由 $A$ 的正规性：

$$TT^* = U^*AA^*U = U^*A^*AU = T^*T$$

对于上三角矩阵 $T$，比较 $TT^*$ 与 $T^*T$ 的对角元：设 $T = (t_{ij})$（$i > j$ 时 $t_{ij}=0$），则

$$(TT^*)_{ii} = \sum_{k=i}^{n} |t_{ik}|^2, \qquad (T^*T)_{ii} = \sum_{k=1}^{i} |t_{ki}|^2$$

$(TT^*)_{11} = (T^*T)_{11}$ 迫使第一行所有非对角元为 $0$；逐行递推，得 $T$ 必为对角矩阵。故 $A$ 可酉对角化。

∎

本章解题技巧总结
  1. Gram-Schmidt 正交化：从线性无关组出发，逐个减去在之前向量上的投影。
  2. 正交补：$V = W \oplus W^\perp$ 是核心分解，把"几何分解"用于代数问题。
  3. 实对称矩阵对角化：找特征值和特征向量，正交化（不同特征值自动正交），单位化，构造正交矩阵 $Q$。
  4. 酉空间的对角化：Hermite 矩阵和酉矩阵都可以酉对角化（谱定理）。
  5. 正规性检验：$A A^* = A^* A$ 是酉对角化的充要条件（复数情形）。
