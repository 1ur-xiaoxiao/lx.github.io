5.1 域的基本概念与素域

定义 · 域

域是交换除环：$(F, +, \cdot)$ 满足 $(F, +)$ 是 Abel 群，$(F \setminus \{0\}, \cdot)$ 是 Abel 群，乘法对加法满足分配律。

定义 · 素域

不含任何真子域的域称为素域。任何域都包含一个素域作为子域。

素域只有两种（在同构意义下）：

  1. $\QQ$（特征为 0 的素域）
  2. $\ZZ_p$（特征为素数 $p$ 的素域）
定义 · 域的特征

域 $F$ 的特征 $\operatorname{char}(F)$ 是使得 $\underbrace{1 + 1 + \cdots + 1}_{n} = 0$ 的最小正整数 $n$。若不存在这样的 $n$，则特征为 0。

域的特征要么是 0，要么是素数。

5.2 域的扩张

定义 · 域的扩张

若 $F$ 是 $K$ 的子域（$K$ 是 $F$ 的扩域），记为 $K/F$，称 $K$ 是 $F$ 的一个扩张。

$K$ 可以看作 $F$ 上的向量空间，其维数称为扩张次数，记为 $[K : F]$。

扩张次数的乘法公式

设 $F \subseteq L \subseteq K$，则

$$[K : F] = [K : L] \cdot [L : F]$$

特别地，$[K : F]$ 有限 $\iff$ $K$ 中每个元素都是 $F$ 上的代数元。

例1 · 扩张次数
  • $[\QQ(\sqrt{2}) : \QQ] = 2$（基为 $\{1, \sqrt{2}\}$）
  • $[\QQ(\sqrt{2}, \sqrt{3}) : \QQ] = 4$（基为 $\{1, \sqrt{2}, \sqrt{3}, \sqrt{6}\}$）
  • $[\CC : \RR] = 2$（基为 $\{1, i\}$）
  • $[\RR : \QQ] = \infty$（$\pi$ 和 $e$ 都是超越数）

5.3 代数扩张

定义 · 代数元与超越元

设 $K/F$ 是域扩张，$\alpha \in K$。若存在非零多项式 $f(x) \in F[x]$ 使得 $f(\alpha) = 0$，则 $\alpha$ 称为 $F$ 上的代数元，否则称为超越元。

定义 · 最小多项式

代数元 $\alpha$ 在 $F$ 上的最小多项式是满足 $f(\alpha) = 0$ 的最低次数的首一不可约多项式，记为 $\operatorname{irr}(\alpha, F)$。

$[F(\alpha) : F] = \deg \operatorname{irr}(\alpha, F)$

单扩张的结构

若 $\alpha$ 是 $F$ 上的代数元，$\operatorname{irr}(\alpha, F) = x^n + a_{n-1}x^{n-1} + \cdots + a_0$，则

$$F(\alpha) \cong F[x] / (\operatorname{irr}(\alpha, F))$$

$F(\alpha)$ 中每个元素可以唯一表示为 $c_0 + c_1\alpha + \cdots + c_{n-1}\alpha^{n-1}$（$c_i \in F$）。

▶ 例题 · 单扩张

题目：描述 $\QQ(\sqrt[3]{2})$。

解：$\sqrt[3]{2}$ 的最小多项式是 $x^3 - 2$（在 $\QQ$ 上不可约，由 Eisenstein 判别法，$p = 2$）。

$[\QQ(\sqrt[3]{2}) : \QQ] = 3$，基为 $\{1, \sqrt[3]{2}, (\sqrt[3]{2})^2\}$。

$\QQ(\sqrt[3]{2}) = \{a + b\sqrt[3]{2} + c\sqrt[3]{4} : a, b, c \in \QQ\}$

5.4 分裂域

定义 · 分裂域

设 $f(x) \in F[x]$，$E/F$ 是域扩张。若：

  1. $f(x)$ 在 $E[x]$ 中可以分解为一次因式的乘积
  2. $E$ 由 $f(x)$ 的根和 $F$ 生成

则称 $E$ 为 $f(x)$ 在 $F$ 上的分裂域。

分裂域的存在唯一性

任何非常数多项式 $f(x) \in F[x]$ 的分裂域存在且在同构意义下唯一。

例2 · 分裂域
  • $x^2 - 2$ 在 $\QQ$ 上的分裂域是 $\QQ(\sqrt{2})$
  • $x^4 - 2$ 在 $\QQ$ 上的分裂域是 $\QQ(\sqrt[4]{2}, i)$，次数为 8
  • $x^2 + 1$ 在 $\RR$ 上的分裂域是 $\CC$

5.5 有限域

有限域的存在与唯一性

对每个素数幂 $q = p^n$，存在唯一的（同构意义下）$q$ 元域 $\FF_q$（或 $GF(q)$）。

$\FF_q$ 的特征为 $p$，是 $\ZZ_p$ 上的 $n$ 次扩张。

有限域的乘法群

有限域 $\FF_q$ 的乘法群 $\FF_q^* = \FF_q \setminus \{0\}$ 是循环群，阶为 $q - 1$。

生成元称为 $\FF_q$ 的本原元。

有限域的子域

$\FF_{p^n}$ 的子域恰好是 $\FF_{p^d}$，其中 $d$ 整除 $n$。

$\FF_{p^n}$ 中的元素都是 $x^{p^n} - x$ 的根。

例3 · $\FF_4$ 的构造

$\FF_4 = \FF_2[x]/(x^2 + x + 1)$，元素为 $\{0, 1, \alpha, \alpha+1\}$，其中 $\alpha^2 + \alpha + 1 = 0$。

$\FF_4^* = \{1, \alpha, \alpha+1\}$ 是 3 阶循环群，$\alpha$ 是本原元（$\alpha^2 = \alpha + 1$，$\alpha^3 = 1$）。

▶ 例题 · 有限域的计算

题目：在 $\FF_8 = \FF_2[x]/(x^3 + x + 1)$ 中计算 $\alpha^5$，其中 $\alpha$ 是 $x$ 的像。

解：$\alpha^3 = \alpha + 1$（在 $\FF_8$ 中）

$\alpha^4 = \alpha \cdot \alpha^3 = \alpha(\alpha + 1) = \alpha^2 + \alpha$

$\alpha^5 = \alpha \cdot \alpha^4 = \alpha(\alpha^2 + \alpha) = \alpha^3 + \alpha^2 = (\alpha + 1) + \alpha^2 = \alpha^2 + \alpha + 1$

5.6 综合例题

本章核心要点
  1. 素域只有 $\QQ$（特征 0）和 $\ZZ_p$（特征 $p$）
  2. 域的扩张次数满足乘法公式 $[K:F] = [K:L][L:F]$
  3. 分裂域存在且唯一，是 Galois 理论的基础
  4. 有限域 $\FF_q$（$q = p^n$）的乘法群是循环群
  5. $\FF_{p^n}$ 的子域是 $\FF_{p^d}$（$d \mid n$）
