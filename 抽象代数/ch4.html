第四章 · 整环的因子分解

整数分解的推广：Euclid 整环、主理想整环与唯一分解整环，因子分解的三种理想模型。

本章目录

  1. 整除与相伴
  2. Euclid 整环 (ED)
  3. 主理想整环 (PID)
  4. 唯一分解整环 (UFD)
  5. 三种整环的关系
  6. 多项式环的因子分解

4.1 整除与相伴

定义 · 整除与相伴

设 $R$ 是整环，$a, b \in R$。

整除：若 $\exists\, c \in R$ 使得 $b = ac$，称 $a$ 整除 $b$，记为 $a \mid b$。

相伴：若 $a \mid b$ 且 $b \mid a$，称 $a$ 与 $b$ 相伴，记为 $a \sim b$。

$a \sim b \iff \exists\, u \in R^*$（单位）使得 $b = au$。

定义 · 不可约元与素元

设 $R$ 是整环，$a \in R$，$a$ 不是零也不是单位。

不可约元：若 $a = bc$ 则 $b$ 或 $c$ 是单位（不能做"非平凡"分解）。

素元：若 $a \mid bc$ 则 $a \mid b$ 或 $a \mid c$。

素元 vs 不可约元

在一般整环中：素元 $\implies$ 不可约元，但反过来不一定成立。

在 UFD 中两者等价。这个区别是因子分解理论的关键。

4.2 Euclid 整环 (ED)

定义 · Euclid 整环

整环 $R$ 上若存在函数 $\delta: R \setminus \{0\} \to \NN \cup \{0\}$ 使得：

  1. $\forall a, b \in R$，$b \neq 0$，$\exists\, q, r \in R$ 使得 $a = bq + r$，其中 $r = 0$ 或 $\delta(r) < \delta(b)$
  2. $\forall a, b \in R \setminus \{0\}$，$\delta(a) \leq \delta(ab)$

则称 $R$ 为 Euclid 整环。

例1 · 经典的 ED
  • $\ZZ$ 是 ED，$\delta(a) = |a|$（带余除法）
  • $\FF[x]$（域上多项式环）是 ED，$\delta(f) = \deg f$（多项式带余除法）
  • Gauss 整数环 $\ZZ[i] = \{a + bi : a, b \in \ZZ\}$ 是 ED，$\delta(a+bi) = a^2 + b^2$

4.3 主理想整环 (PID)

定义 · 主理想整环

整环 $R$ 中若每个理想都是主理想（由单个元素生成），则称 $R$ 为主理想整环（PID）。

ED 是 PID

设 $R$ 是 Euclid 整环，则 $R$ 是主理想整环。

证明思路：设 $I$ 是 $R$ 的理想，取 $I$ 中 $\delta$ 值最小的非零元素 $d$，证明 $I = (d)$。

PID 中不可约元等价于素元

在 PID 中，不可约元与素元等价。这是 PID 中因子分解唯一性的关键。

4.4 唯一分解整环 (UFD)

定义 · 唯一分解整环

整环 $R$ 若满足：

  1. $R$ 中每个非零非单位元素都可以分解为不可约元的乘积
  2. 分解在相伴意义下唯一（不计次序和单位因子）

则称 $R$ 为唯一分解整环（UFD）。

PID 是 UFD

主理想整环一定是唯一分解整环。

例2 · UFD 但不是 PID

$\ZZ[x]$ 是 UFD 但不是 PID。例如理想 $(2, x) = \{f \in \ZZ[x] : f(0) \text{ 是偶数}\}$ 不是主理想。

4.5 三种整环的关系

核心关系链
$$\text{ED} \subset \text{PID} \subset \text{UFD} \subset \text{整环}$$

每个包含都是严格的（存在反例使反过来不成立）。

例3 · 反例
类型例子说明
PID $\not\subset$ ED$\ZZ\!\left[\frac{1+\sqrt{-19}}{2}\right]$是 PID 但不存在 Euclid 函数
UFD $\not\subset$ PID$\ZZ[x]$ 是 UFD 不是 PID$(2, x)$ 不是主理想
整环 $\not\subset$ UFD$\ZZ[\sqrt{-5}]$ 不是 UFD$6 = 2 \cdot 3 = (1+\sqrt{-5})(1-\sqrt{-5})$

4.6 多项式环的因子分解

Gauss 引理

$\ZZ[x]$ 中的本原多项式在 $\ZZ[x]$ 中可约 $\iff$ 在 $\QQ[x]$ 中可约。

推论：若 $\ZZ[x]$ 中本原多项式在 $\QQ$ 上不可约，则在 $\ZZ$ 上也不可约。

$R[x]$ 是 UFD

若 $R$ 是 UFD，则 $R[x]$ 也是 UFD。

推论：$\ZZ[x_1, x_2, \ldots, x_n]$ 是 UFD，$\FF[x_1, x_2, \ldots, x_n]$（$\FF$ 是域）也是 UFD。

Eisenstein 不可约判别法

设 $f(x) = a_n x^n + \cdots + a_0 \in \ZZ[x]$。若存在素数 $p$ 使得：

  1. $p \nmid a_n$
  2. $p \mid a_i$（$i = 0, 1, \ldots, n-1$）
  3. $p^2 \nmid a_0$

则 $f(x)$ 在 $\QQ[x]$ 中不可约。

▶ 例题 · Eisenstein 判别法

题目：证明 $f(x) = x^5 + 3x^4 - 6x^2 + 9x + 3$ 在 $\QQ$ 上不可约。

证明：取 $p = 3$：

$3 \nmid 1$（首项系数）✓

$3 \mid 3, 3 \mid 0, 3 \mid (-6), 3 \mid 9, 3 \mid 3$ ✓

$9 \nmid 3$（常数项）✓

由 Eisenstein 判别法，$f(x)$ 在 $\QQ$ 上不可约。$\blacksquare$

本章核心要点
  1. ED $\subset$ PID $\subset$ UFD，每层都有经典的判别条件
  2. 在 PID/UFD 中，不可约元等价于素元
  3. $\ZZ[x]$ 和 $\FF[x]$ 都是 UFD 的重要例子
  4. Eisenstein 判别法是判断多项式不可约的有力工具
