第三章 · 初等函数

将实数域中熟悉的初等函数推广到复数域：指数函数获得周期性，对数函数变成多值函数，幂函数展现出全新的分支结构。理解这些推广是掌握复分析工具的关键。

3.1 指数函数

定义 · 复指数函数

对任意 $z = x + iy \in \CC$，定义复指数函数为：

$$e^z = e^{x+iy} = e^x(\cos y + i\sin y)$$

其中 $e^x$ 为实指数函数，$\cos y, \sin y$ 为实三角函数。

等价写法

利用模和辐角的语言，复指数函数可写为：

$$e^z = |e^z| \cdot e^{i\arg(e^z)}, \quad \text{其中 } |e^z| = e^x,\ \arg(e^z) = y$$

也就是说，$e^z$ 的模由实部决定，辐角由虚部决定。这是理解复指数函数映射性质的核心。

定理 · 指数函数的基本性质

复指数函数 $e^z$ 满足以下性质：

  1. 解析性：$e^z$ 在整个复平面 $\CC$ 上解析（整函数），且 $\dfrac{\dd}{\dd z}e^z = e^z$。
  2. 加法定理：对任意 $z_1, z_2 \in \CC$，$e^{z_1+z_2} = e^{z_1} \cdot e^{z_2}$。
  3. 周期性：$e^z$ 以 $2\pi i$ 为周期，即 $e^{z+2\pi i} = e^z$。这是复指数函数与实指数函数最本质的区别。
  4. 无零点：对任意 $z \in \CC$，$e^z \neq 0$。因为 $|e^z| = e^x > 0$。
  5. 值域：$e^z$ 的值域为 $\CC \setminus \{0\}$，即全体非零复数。
证明 · 加法定理

设 $z_1 = x_1 + iy_1$，$z_2 = x_2 + iy_2$，则：

$$e^{z_1} \cdot e^{z_2} = e^{x_1}(\cos y_1 + i\sin y_1) \cdot e^{x_2}(\cos y_2 + i\sin y_2)$$ $$= e^{x_1+x_2}[(\cos y_1\cos y_2 - \sin y_1\sin y_2) + i(\sin y_1\cos y_2 + \cos y_1\sin y_2)]$$ $$= e^{x_1+x_2}[\cos(y_1+y_2) + i\sin(y_1+y_2)] = e^{z_1+z_2}$$

这里用到了实三角函数的和角公式。

Euler 公式

在定义中令 $z = iy$，得到著名的 Euler 公式：

$$e^{iy} = \cos y + i\sin y$$

特别地，令 $y = \pi$：

$$e^{i\pi} + 1 = 0$$

这个等式将五个最重要的数学常数 $e, i, \pi, 1, 0$ 联系在一起。

定理 · 映射性质

$w = e^z$ 将复平面上的水平带域 $y_0 < \Im z < y_0 + 2\pi$ 映为整个 $w$ 平面去掉从原点出发的射线 $\arg w = y_0$。

特别地：

  • 水平线 $y = c$（$c$ 为常数）映为射线 $\arg w = c$；
  • 竖直线 $x = c$ 映为圆周 $|w| = e^c$；
  • 宽度为 $2\pi$ 的水平带域恰好覆盖 $w$ 平面一次。
例1 · 计算具体值

计算 $e^{2+\pi i/3}$。

解：由定义 $e^{2+\pi i/3} = e^2(\cos\frac{\pi}{3} + i\sin\frac{\pi}{3}) = e^2\left(\dfrac{1}{2} + i\dfrac{\sqrt{3}}{2}\right)$。

▶ 例题 · 解方程 $e^z = 1+i$

解：设 $z = x + iy$，则 $e^z = e^x(\cos y + i\sin y) = 1 + i$。

取模：$e^x = |1+i| = \sqrt{2}$，故 $x = \ln\sqrt{2} = \dfrac{1}{2}\ln 2$。

取辐角：$y = \arg(1+i) = \dfrac{\pi}{4} + 2k\pi$，$k \in \ZZ$。

因此 $z = \dfrac{1}{2}\ln 2 + i\left(\dfrac{\pi}{4} + 2k\pi\right)$，$k \in \ZZ$。

注：方程 $e^z = w$（$w \neq 0$）有无穷多个解，这正是因为 $e^z$ 的周期性。

3.2 对数函数

定义 · 对数函数

对 $z \neq 0$，满足 $e^w = z$ 的 $w$ 称为 $z$ 的对数，记为 $w = \log z$。

设 $z = re^{i\theta}$（$r = |z|$，$\theta = \arg z$），则：

$$\log z = \ln|z| + i\arg z = \ln r + i(\theta + 2k\pi), \quad k \in \ZZ$$

因此 $\log z$ 是一个多值函数，有无穷多个值。

定义 · 对数的主值

取 $k = 0$（即辐角取主值 $\Arg z \in (-\pi, \pi]$），定义对数的主值为：

$$\log z = \ln|z| + i\Arg z$$

注意：主值 $\log z$ 是一个单值函数，但它在负实轴上不连续（辐角在负实轴上有跳跃）。

多值性的具体表现

对数函数 $\log z$ 的所有值为：

$$\log z = \log z + 2k\pi i, \quad k \in \ZZ$$

相邻两个值之间相差 $2\pi i$。这也说明 $\log z$ 与 $e^z$ 的关系：$\log z$ 是 $e^z$ 的"反函数"，但因为 $e^z$ 是周期函数，所以反函数是多值的。

分支与支割线

为了从多值函数 $\log z$ 中选出一个单值连续分支，需要：

  1. 支割线（branch cut）：从原点出发沿某方向画一条射线，禁止自变量跨越此线；
  2. 辐角范围：指定 $\arg z$ 的取值范围，如 $(\alpha, \alpha + 2\pi]$。

最常见的选择是沿负实轴做支割线，取 $\Arg z \in (-\pi, \pi]$，得到主值分支 $\log z$。

每个单值连续分支都是解析函数，解析区域为割开的复平面。

定理 · 对数函数的运算性质

对 $z_1, z_2 \neq 0$，对数函数满足：

  1. $\log(z_1 z_2) = \log z_1 + \log z_2$（在多值意义下成立，即两边的值集合相等）
  2. $\log\dfrac{z_1}{z_2} = \log z_1 - \log z_2$（多值意义下）

注意：对主值 $\log$，上述等式不一定成立，可能相差 $2\pi i$ 的整数倍。

例2 · 计算 $\log(-1)$ 和 $\log i$

$\log(-1)$：$|-1| = 1$，$\arg(-1) = \pi + 2k\pi$，所以 $\log(-1) = i(\pi + 2k\pi) = (2k+1)\pi i$。

主值 $\log(-1) = \pi i$。

$\log i$：$|i| = 1$，$\arg(i) = \dfrac{\pi}{2} + 2k\pi$，所以 $\log i = i\left(\dfrac{\pi}{2} + 2k\pi\right)$。

主值 $\log i = \dfrac{\pi}{2}i$。

▶ 例题 · 计算 $\log(1+i)$ 的所有值

解：$1+i = \sqrt{2} \cdot e^{i\pi/4}$，故

$$\log(1+i) = \ln\sqrt{2} + i\left(\frac{\pi}{4} + 2k\pi\right) = \frac{1}{2}\ln 2 + i\left(\frac{\pi}{4} + 2k\pi\right), \quad k \in \ZZ$$

主值为 $\log(1+i) = \dfrac{1}{2}\ln 2 + \dfrac{\pi}{4}i$。

注意：$\ln 2$ 表示实对数，是唯一的实数值。

▶ 例题 · 验证 $\log(-i) = -\dfrac{\pi}{2}i$

解：$|-i| = 1$，$\Arg(-i) = -\dfrac{\pi}{2}$（因为 $-i$ 在负虚轴上，主值辐角取 $-\pi/2$）。

故 $\log(-i) = \ln 1 + i\left(-\dfrac{\pi}{2}\right) = -\dfrac{\pi}{2}i$。

验证：$e^{-\pi i/2} = \cos\left(-\dfrac{\pi}{2}\right) + i\sin\left(-\dfrac{\pi}{2}\right) = 0 + i(-1) = -i$。正确。

3.3 幂函数

定义 · 复幂函数

对 $z \neq 0$ 和任意复数 $a \in \CC$，定义幂函数为：

$$z^a = e^{a\log z}$$

由于 $\log z$ 是多值的，$z^a$ 通常也是多值函数。

幂函数的多值性

将 $\log z = \ln|z| + i(\arg z + 2k\pi)$ 代入：

$$z^a = e^{a[\ln|z| + i\arg z + 2k\pi i]} = |z|^a \cdot e^{ia\arg z} \cdot e^{2k\pi ia}$$

其中 $|z|^a = e^{a\ln|z|}$（实幂函数意义下）。

关键在于因子 $e^{2k\pi ia}$：如果 $a$ 是整数，则此因子为 $1$（$k$ 任意），$z^a$ 是单值的；否则 $z^a$ 有无穷多个值。

定理 · 幂函数的单值性条件

$z^a$ 是单值函数的充要条件是 $a \in \ZZ$（$a$ 为整数）。

更细致地：

  • $a = n \in \ZZ$：$z^n$ 是单值函数。$n>0$ 时为整函数，$n<0$ 时在 $z=0$ 处有极点。
  • $a = p/q$（$p,q$ 互素，$q > 1$）：$z^{p/q}$ 恰好有 $q$ 个不同的值。这是 $q$ 值函数。
  • $a$ 为无理数或虚数：$z^a$ 有无穷多个值。
例3 · 计算 $i^i$

$i^i = e^{i\log i} = e^{i \cdot i(\pi/2 + 2k\pi)} = e^{-\pi/2 - 2k\pi}$，$k \in \ZZ$。

取 $k=0$ 得主值：$i^i = e^{-\pi/2} \approx 0.2079$。

令人惊讶的事实：$i^i$ 的所有值都是正实数！

例4 · 计算 $z^{1/2}$（平方根）

对 $z = re^{i\theta}$：

$$z^{1/2} = e^{\frac{1}{2}\log z} = e^{\frac{1}{2}[\ln r + i(\theta + 2k\pi)]} = \sqrt{r} \cdot e^{i(\theta + 2k\pi)/2}$$

$k=0$：$w_0 = \sqrt{r} \cdot e^{i\theta/2}$

$k=1$：$w_1 = \sqrt{r} \cdot e^{i(\theta/2 + \pi)} = -w_0$

$k=2$：$w_2 = \sqrt{r} \cdot e^{i(\theta/2 + 2\pi)} = w_0$（重复）

所以 $z^{1/2}$ 恰好有两个值 $\pm\sqrt{r} \cdot e^{i\theta/2}$，互为相反数。

根式函数的分支

$z^{1/n}$ 有 $n$ 个不同的值，均匀分布在以原点为圆心、半径为 $|z|^{1/n}$ 的圆周上，相邻两个值之间夹角为 $2\pi/n$。

选取单值连续分支需要：做从原点出发的支割线，并指定 $\arg z$ 的范围，如 $\arg z \in (\alpha, \alpha + 2\pi)$。

▶ 例题 · 计算 $(1+i)^{1+i}$

解：$(1+i)^{1+i} = e^{(1+i)\log(1+i)}$。

先求 $\log(1+i) = \dfrac{1}{2}\ln 2 + i\left(\dfrac{\pi}{4} + 2k\pi\right)$。

取主值 $k=0$：$\log(1+i) = \dfrac{1}{2}\ln 2 + \dfrac{\pi}{4}i$。

则 $(1+i)\log(1+i) = (1+i)\left(\dfrac{1}{2}\ln 2 + \dfrac{\pi}{4}i\right)$

$$= \frac{1}{2}\ln 2 + \frac{\pi}{4}i + \frac{i}{2}\ln 2 + \frac{\pi}{4}i^2 = \left(\frac{1}{2}\ln 2 - \frac{\pi}{4}\right) + i\left(\frac{\pi}{4} + \frac{1}{2}\ln 2\right)$$

所以 $(1+i)^{1+i} = e^{\frac{1}{2}\ln 2 - \frac{\pi}{4}} \cdot e^{i(\frac{\pi}{4} + \frac{1}{2}\ln 2)}$

$$= \sqrt{2} \cdot e^{-\pi/4} \left[\cos\left(\frac{\pi}{4} + \frac{\ln 2}{2}\right) + i\sin\left(\frac{\pi}{4} + \frac{\ln 2}{2}\right)\right]$$

3.4 三角函数与双曲函数

定义 · 复三角函数

由 Euler 公式 $e^{iz} = \cos z + i\sin z$，定义复三角函数为：

$$\cos z = \frac{e^{iz} + e^{-iz}}{2}, \quad \sin z = \frac{e^{iz} - e^{-iz}}{2i}$$ $$\tan z = \frac{\sin z}{\cos z}, \quad \cot z = \frac{\cos z}{\sin z}, \quad \text{等}$$
定理 · 复三角函数的性质

复三角函数保持了实三角函数的基本代数性质：

  1. 周期性：$\cos z$ 和 $\sin z$ 以 $2\pi$ 为周期。
  2. 奇偶性：$\cos(-z) = \cos z$（偶函数），$\sin(-z) = -\sin z$（奇函数）。
  3. Pythagoras 恒等式：$\cos^2 z + \sin^2 z = 1$。
  4. 和角公式：$\cos(z_1+z_2) = \cos z_1\cos z_2 - \sin z_1\sin z_2$，$\sin(z_1+z_2) = \sin z_1\cos z_2 + \cos z_1\sin z_2$。
  5. 解析性：$\cos z$ 和 $\sin z$ 在 $\CC$ 上解析（整函数）。

与实函数的本质区别：

  • $\sin z$ 和 $\cos z$ 不再是有界的！例如 $|\cos(iy)| = \dfrac{e^y + e^{-y}}{2} = \cosh y \to +\infty$（$y \to \pm\infty$）。
  • $\sin z = 0 \Leftrightarrow z = k\pi$（$k \in \ZZ$），$\cos z = 0 \Leftrightarrow z = \dfrac{\pi}{2} + k\pi$（$k \in \ZZ$），零点与实数域相同。
三角函数的实部与虚部

设 $z = x + iy$，则：

$$\cos(x+iy) = \cos x \cosh y - i\sin x \sinh y$$ $$\sin(x+iy) = \sin x \cosh y + i\cos x \sinh y$$

其中 $\cosh y = \dfrac{e^y+e^{-y}}{2}$，$\sinh y = \dfrac{e^y-e^{-y}}{2}$ 为实双曲函数。

定义 · 复双曲函数

类比实双曲函数，定义：

$$\cosh z = \frac{e^z + e^{-z}}{2}, \quad \sinh z = \frac{e^z - e^{-z}}{2}$$ $$\tanh z = \frac{\sinh z}{\cosh z}, \quad \coth z = \frac{\cosh z}{\sinh z}$$
定理 · 三角函数与双曲函数的关系

复三角函数与双曲函数之间有如下基本关系：

$$\cos(iz) = \cosh z, \quad \sin(iz) = i\sinh z$$ $$\cosh(iz) = \cos z, \quad \sinh(iz) = i\sin z$$

由此可得：双曲函数与三角函数本质上通过虚数单位 $i$ 相互转化。

双曲函数的实部与虚部

设 $z = x + iy$，则：

$$\cosh(x+iy) = \cosh x \cos y + i\sinh x \sin y$$ $$\sinh(x+iy) = \sinh x \cos y + i\cosh x \sin y$$
复三角函数的无界性

在实数域中，$|\sin x| \leq 1$，$|\cos x| \leq 1$。但在复数域中，三角函数是无界的。

例如：$\cos(iy) = \cosh y$，当 $y \to \infty$ 时 $\cosh y \to \infty$。

更一般地：$|\cos z|^2 = \cos^2 x + \sinh^2 y$，$|\sin z|^2 = \sin^2 x + \sinh^2 y$，当 $|y| \to \infty$ 时都趋于无穷。

例5 · 计算 $\cos(1+2i)$

$\cos(1+2i) = \cos 1 \cdot \cosh 2 - i\sin 1 \cdot \sinh 2$

其中 $\cosh 2 = \dfrac{e^2+e^{-2}}{2} \approx 3.7622$，$\sinh 2 = \dfrac{e^2-e^{-2}}{2} \approx 3.6269$。

$\cos 1 \approx 0.5403$，$\sin 1 \approx 0.8415$。

故 $\cos(1+2i) \approx 0.5403 \times 3.7622 - i \times 0.8415 \times 3.6269 \approx 2.0327 - 3.0518i$。

▶ 例题 · 求 $\sin z$ 的所有零点

解：$\sin z = 0 \Leftrightarrow \dfrac{e^{iz} - e^{-iz}}{2i} = 0 \Leftrightarrow e^{iz} = e^{-iz} \Leftrightarrow e^{2iz} = 1$。

设 $2iz = 2k\pi i$（$k \in \ZZ$），即 $z = k\pi$，$k \in \ZZ$。

结论：$\sin z$ 的零点与实数域完全相同，都是 $z = k\pi$（$k \in \ZZ$）。这验证了 $\sin z$ 是实函数 $\sin x$ 的真正解析延拓。

▶ 例题 · 证明 $|\sin z|^2 = \sin^2 x + \sinh^2 y$

证明：$\sin(x+iy) = \sin x\cosh y + i\cos x\sinh y$，故

$$|\sin z|^2 = \sin^2 x \cosh^2 y + \cos^2 x \sinh^2 y$$

利用 $\cosh^2 y = 1 + \sinh^2 y$：

$$= \sin^2 x(1 + \sinh^2 y) + \cos^2 x \sinh^2 y = \sin^2 x + \sinh^2 y(\sin^2 x + \cos^2 x) = \sin^2 x + \sinh^2 y$$

由此可以看出：当 $y \neq 0$ 时 $|\sin z| > |\sin x|$，当 $|y| \to \infty$ 时 $|\sin z| \to \infty$。
