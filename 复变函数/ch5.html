第五章 · 洛朗展式与孤立奇点

在奇点附近，Taylor 展开失效——我们需要双边幂级数（洛朗展开）来刻画函数的奇异行为

学完本章，你将能够：
  1. 将函数在环形区域内展开为洛朗级数
  2. 区分三种类型的孤立奇点：可去奇点、极点、本性奇点
  3. 利用洛朗展开提取留数（为下一章做准备）
  4. 理解 Weierstrass 定理和 Picard 定理的结论

本章导读

前一章我们学习了函数在解析点附近的幂级数展开。本章处理函数在奇点附近的行为——这是复分析真正展现出其独特力量的地方。

核心工具是洛朗展开：将函数在环形区域上展开为包含负幂次的双边幂级数。洛朗展开揭示了解析函数在孤立奇点附近的完整结构，由此自然导出可去奇点、极点、本性奇点的三种分类。这个分类直接通向下一章的留数理论——留数正是洛朗展开中 $(z-z_0)^{-1}$ 的系数 $a_{-1}$。

对应钟玉泉《复变函数论》第五章。

5.1 Laurent 展开定理

直觉导航

Taylor 展开只能描述函数在正则点附近的光滑行为——它用非负幂次 $(z-z_0)^n$ 来逼近。但一旦函数有奇点（如 $1/z$ 在 $z=0$），Taylor 展开就失效了——你无法用多项式来逼近一个在有限点趋于无穷的函数。

Laurent 展开的创举在于允许负幂项：$f(z) = \cdots + a_{-2}/(z-z_0)^2 + a_{-1}/(z-z_0) + a_0 + a_1(z-z_0) + \cdots$。负幂项（主要部分）专门用来刻画奇点附近的"爆炸"行为——$(z-z_0)^{-1}$ 项描述一阶奇异性，$(z-z_0)^{-2}$ 描述二阶奇异性，依此类推。正则部分（非负幂）则照常描述远离奇点时的光滑行为。

一个形象的比喻：Taylor 展开像用标准相机拍照——只能拍正常（光滑）的风景。Laurent 展开像用能拍"HDR"的相机——同时捕捉极亮（奇点）和正常亮度（正则）的区域，把它们清晰地分离在两个不同的部分里。

定义 5.4（圆环域）

设 $z_0 \in \CC$，$0 \le r < R \le +\infty$，则集合

$$D = \{z \in \CC : r < |z - z_0| < R\}$$

称为以 $z_0$ 为中心、内半径为 $r$、外半径为 $R$ 的圆环域。

定理 5.10（Laurent 展开定理）

设 $f(z)$ 在圆环域 $D: r < |z - z_0| < R$（$0 \le r < R \le +\infty$）内解析，则 $f(z)$ 在 $D$ 内可唯一地展开为双边级数

$$f(z) = \sum_{n=-\infty}^{+\infty} a_n (z - z_0)^n, \quad r < |z - z_0| < R,$$

其中

$$a_n = \frac{1}{2\pi i} \oint_C \frac{f(\zeta)}{(\zeta - z_0)^{n+1}}\,\dd\zeta, \quad n = 0, \pm 1, \pm 2, \ldots,$$

$C$ 为圆环域内绕 $z_0$ 的任一简单闭曲线。

证明 · Laurent 展开定理

这是复分析中仅次于 Cauchy 积分公式的重要证明。核心思想是：在圆环域内，Cauchy 积分公式需要用到两个圆周（而非一个），两者的几何级数展开分别产生正幂项和负幂项。

第一步：双围道表示。任取 $z$ 满足 $r < |z-z_0| < R$。选取 $r_1, R_1$ 使得 $r < r_1 < |z-z_0| < R_1 < R$。令 $C_1: |\zeta-z_0| = R_1$（外圆周，正向），$C_2: |\zeta-z_0| = r_1$（内圆周，正向）。$f$ 在由 $C_1$ 和 $C_2$ 围成的圆环域及其边界上解析，由 Cauchy 积分公式（多连通域版本）：

$$f(z) = \frac{1}{2\pi i}\oint_{C_1} \frac{f(\zeta)}{\zeta - z}\,\dd\zeta - \frac{1}{2\pi i}\oint_{C_2} \frac{f(\zeta)}{\zeta - z}\,\dd\zeta \qquad (\star)$$

第二步：外圆周 $C_1$ 上的展开（产生正则部分）。在 $C_1$ 上 $|\zeta-z_0| = R_1 > |z-z_0|$，故 $\left|\frac{z-z_0}{\zeta-z_0}\right| < 1$：

$$\frac{1}{\zeta - z} = \frac{1}{\zeta - z_0} \cdot \frac{1}{1 - \frac{z - z_0}{\zeta - z_0}} = \sum_{n=0}^{\infty} \frac{(z - z_0)^n}{(\zeta - z_0)^{n+1}}$$

代入 $(\star)$ 中第一个积分并逐项积分（级数在 $C_1$ 上一致收敛）：

$$\frac{1}{2\pi i}\oint_{C_1} \frac{f(\zeta)}{\zeta - z}\,\dd\zeta = \sum_{n=0}^{\infty} \left(\frac{1}{2\pi i}\oint_{C_1} \frac{f(\zeta)}{(\zeta - z_0)^{n+1}}\,\dd\zeta\right)(z - z_0)^n = \sum_{n=0}^{\infty} a_n (z - z_0)^n$$

第三步：内圆周 $C_2$ 上的展开（产生主要部分）。在 $C_2$ 上 $|\zeta-z_0| = r_1 < |z-z_0|$，故 $\left|\frac{\zeta-z_0}{z-z_0}\right| < 1$。此时展开方向相反：

$$\frac{1}{\zeta - z} = -\frac{1}{z - z_0} \cdot \frac{1}{1 - \frac{\zeta - z_0}{z - z_0}} = -\sum_{m=0}^{\infty} \frac{(\zeta - z_0)^m}{(z - z_0)^{m+1}}$$

代入 $(\star)$ 中第二个积分（注意前面的负号）：

$$-\frac{1}{2\pi i}\oint_{C_2} \frac{f(\zeta)}{\zeta - z}\,\dd\zeta = \sum_{m=0}^{\infty} \left(\frac{1}{2\pi i}\oint_{C_2} f(\zeta)(\zeta - z_0)^m\,\dd\zeta\right)\frac{1}{(z - z_0)^{m+1}}$$

令 $n = -(m+1)$，即 $m = -n-1$，则 $(z-z_0)^{-(m+1)} = (z-z_0)^n$，系数为 $\frac{1}{2\pi i}\oint_{C_2} \frac{f(\zeta)}{(\zeta - z_0)^{n+1}}\,\dd\zeta$。由于被积函数在 $C_1$ 和 $C_2$ 之间的圆环域内解析，积分围道可变形为任意绕 $z_0$ 的圆周 $C$，从而系数统一表示为 $a_n$（$n < 0$）。

$$-\frac{1}{2\pi i}\oint_{C_2} \frac{f(\zeta)}{\zeta - z}\,\dd\zeta = \sum_{n=-1}^{-\infty} a_n (z - z_0)^n$$

第四步：合并。两部分的级数相加即得完整的 Laurent 展开：

$$f(z) = \sum_{n=-\infty}^{+\infty} a_n (z - z_0)^n, \quad a_n = \frac{1}{2\pi i}\oint_C \frac{f(\zeta)}{(\zeta - z_0)^{n+1}}\,\dd\zeta$$

唯一性：若在同一圆环域内有两个 Laurent 展开，其差为一个双边幂级数恒为零，则所有系数必为零（可用围道积分在 $C$ 上逐项积分验证，只有 $n=-1$ 项留下）。$\square$

定义 5.5（正则部分与主要部分）

Laurent 级数可拆分为两部分：

正则部分（解析部分）：

$$\sum_{n=0}^{\infty} a_n (z - z_0)^n = a_0 + a_1(z-z_0) + a_2(z-z_0)^2 + \cdots$$

这是一个在 $|z - z_0| < R$ 内收敛的幂级数。

主要部分：

$$\sum_{n=1}^{\infty} a_{-n} (z - z_0)^{-n} = \frac{a_{-1}}{z - z_0} + \frac{a_{-2}}{(z-z_0)^2} + \cdots$$

这是一个在 $|z - z_0| > r$ 内收敛的级数。

注记：Taylor 展开与 Laurent 展开的关系

Taylor 展开是 Laurent 展开的特殊情形：当 $r = 0$ 且 $f(z)$ 在 $z_0$ 解析时，所有 $a_{-n} = 0$（$n \ge 1$），Laurent 级数退化为 Taylor 级数。

重要区别： Laurent 展开的系数一般不能用 $a_n = f^{(n)}(z_0)/n!$ 计算（因为 $f$ 在 $z_0$ 可能不解析），必须通过围道积分或间接方法求得。

Laurent 展开的常用方法

直接法： 利用系数公式 $a_n = \frac{1}{2\pi i}\oint_C \frac{f(\zeta)}{(\zeta-z_0)^{n+1}}\,\dd\zeta$ 计算（通常较繁琐）。

间接法（常用）：

  1. 利用已知的 Taylor 展开公式；
  2. 利用变量替换、级数乘法、逐项求导/积分等技巧；
  3. 将函数拆分为在不同区域收敛的已知展开之和。
▶ 例 5.5（Laurent 展开的典型计算）

将 $f(z) = \dfrac{1}{(z-1)(z-2)}$ 在以下圆环域内展开为 Laurent 级数：

（i）$1 < |z| < 2$；（ii）$|z| > 2$。

解： 先做部分分式分解 $f(z) = \dfrac{1}{z-2} - \dfrac{1}{z-1}$。

情形（i）$1 < |z| < 2$：

对 $\dfrac{1}{z-2} = -\dfrac{1}{2} \cdot \dfrac{1}{1 - z/2}$，因 $|z| < 2$ 即 $|z/2| < 1$：

$$\frac{1}{z-2} = -\frac{1}{2}\sum_{n=0}^{\infty} \left(\frac{z}{2}\right)^n = -\sum_{n=0}^{\infty} \frac{z^n}{2^{n+1}}.$$

对 $\dfrac{1}{z-1} = \dfrac{1}{z} \cdot \dfrac{1}{1 - 1/z}$，因 $|z| > 1$ 即 $|1/z| < 1$：

$$\frac{1}{z-1} = \frac{1}{z}\sum_{n=0}^{\infty} \frac{1}{z^n} = \sum_{n=0}^{\infty} \frac{1}{z^{n+1}} = \sum_{n=1}^{\infty} \frac{1}{z^n}.$$

因此

$$f(z) = -\sum_{n=0}^{\infty} \frac{z^n}{2^{n+1}} - \sum_{n=1}^{\infty} \frac{1}{z^n}, \quad 1 < |z| < 2.$$

情形（ii）$|z| > 2$：

对 $\dfrac{1}{z-2} = \dfrac{1}{z}\cdot\dfrac{1}{1-2/z}$，因 $|z| > 2$：

$$\frac{1}{z-2} = \frac{1}{z}\sum_{n=0}^{\infty}\frac{2^n}{z^n} = \sum_{n=0}^{\infty}\frac{2^n}{z^{n+1}}.$$

结合 $\dfrac{1}{z-1} = \sum_{n=0}^{\infty}\dfrac{1}{z^{n+1}}$（同上），得

$$f(z) = \sum_{n=0}^{\infty}\frac{2^n - 1}{z^{n+1}} = \frac{1}{z} + \frac{3}{z^2} + \frac{7}{z^3} + \cdots, \quad |z| > 2.$$

▶ 例 5.6（含指数函数的 Laurent 展开）

将 $f(z) = e^{1/z}$ 在 $0 < |z| < +\infty$ 内展开。

解： 由 $e^w = \sum_{n=0}^{\infty} \dfrac{w^n}{n!}$，令 $w = 1/z$：

$$e^{1/z} = \sum_{n=0}^{\infty} \frac{1}{n! \, z^n} = 1 + \frac{1}{z} + \frac{1}{2!\,z^2} + \frac{1}{3!\,z^3} + \cdots.$$

此展开在 $0 < |z| < +\infty$ 内成立。注意正则部分只有常数项 $1$，主要部分有无穷多项，说明 $z = 0$ 是本性奇点。

5.2 孤立奇点的分类

直觉导航

Laurent 展开中负幂项的"个数"决定了奇点的类型——这就像一个三档开关：

  • 可去奇点（0档）：没有负幂项。函数在奇点附近其实是有界的，只是定义上有个"洞"。补上这个洞（重新定义函数在奇点处的值），函数就变回解析的了。典型例子：$\frac{\sin z}{z}$ 在 $z=0$（展开后没有负幂项，补定义 $f(0)=1$ 即可）。
  • $m$ 阶极点（$m$档）：有限个负幂项，最高为 $(z-z_0)^{-m}$。函数在奇点处以 $1/|z-z_0|^m$ 的速度趋于无穷。极点是最"可控"的奇异性——留数公式直接给你答案。典型例子：$1/z^2$ 在 $z=0$ 是二阶极点。
  • 本性奇点（∞档）：无穷多个负幂项。函数在奇点附近的行为极度疯狂——根据 Weierstrass 定理，在本性奇点的任意小邻域内，函数值可以任意逼近任何复数。典型例子：$e^{1/z}$ 在 $z=0$。

理解这三种分类就等于掌握了第六章留数计算的钥匙。

定义 5.6（孤立奇点）

若 $f(z)$ 在 $z_0$ 的去心邻域 $0 < |z - z_0| < \delta$ 内解析，但在 $z_0$ 处不解析（或无定义），则称 $z_0$ 为 $f(z)$ 的 孤立奇点。

定义 5.7（孤立奇点的三种类型）

设 $z_0$ 是 $f(z)$ 的孤立奇点，$f(z)$ 在 $0 < |z - z_0| < \delta$ 内的 Laurent 展开为

$$f(z) = \sum_{n=-\infty}^{+\infty} a_n(z - z_0)^n.$$

（i）可去奇点： 主要部分为零，即 $a_{-n} = 0$（$n \ge 1$）。此时

$$f(z) = a_0 + a_1(z-z_0) + a_2(z-z_0)^2 + \cdots.$$

补充定义 $f(z_0) = a_0$ 后，$f$ 在 $z_0$ 解析。

（ii）$m$ 阶极点： 主要部分只有有限项，且 $a_{-m} \ne 0$，$a_{-n} = 0$（$n > m$）。此时

$$f(z) = \frac{a_{-m}}{(z-z_0)^m} + \cdots + \frac{a_{-1}}{z-z_0} + a_0 + a_1(z-z_0) + \cdots.$$

特别地，$m = 1$ 时称为 简单极点（一阶极点）。

（iii）本性奇点： 主要部分有无穷多项，即有无穷多个 $a_{-n} \ne 0$。

孤立奇点的等价判别法

设 $z_0$ 是 $f(z)$ 的孤立奇点，则：

$z_0$ 为可去奇点 $\Longleftrightarrow$ $\lim_{z\to z_0} f(z)$ 存在（有限） $\Longleftrightarrow$ $f(z)$ 在 $z_0$ 的某去心邻域内有界。

$z_0$ 为 $m$ 阶极点 $\Longleftrightarrow$ $\lim_{z\to z_0}(z-z_0)^m f(z) = c \ne 0$（有限） $\Longleftrightarrow$ $\lim_{z\to z_0} f(z) = \infty$。

$z_0$ 为本性奇点 $\Longleftrightarrow$ $\lim_{z\to z_0} f(z)$ 不存在（不为有限值也不为 $\infty$）。

证明 · 奇点分类的等价条件

这些等价条件将 Laurent 展开的代数分类与极限行为联系起来，是理解奇点的关键桥梁。

可去奇点：

"主要部分为零 $\implies$ $\lim f(z)$ 存在有限"：当主要部分为零时 $f(z) = a_0 + a_1(z-z_0) + \cdots$ 是 $z_0$ 的去心邻域内的幂级数，令 $z \to z_0$ 得极限为 $a_0$。

"$\lim f(z)$ 存在有限 $\implies$ $f$ 有界"：由极限定义，存在 $\delta > 0$ 使得 $0 < |z-z_0| < \delta$ 时 $|f(z)| \le |\lim f| + 1$。

"$f$ 有界 $\implies$ 主要部分为零"：设 $|f(z)| \le M$ 在 $0 < |z-z_0| < \delta$ 内。对 $n \ge 1$，由 Laurent 系数公式取 $C: |z-z_0| = \rho$：

$$|a_{-n}| = \left|\frac{1}{2\pi i}\oint_C f(z)(z-z_0)^{n-1}\,\dd z\right| \le \frac{1}{2\pi} \cdot M \rho^{n-1} \cdot 2\pi\rho = M\rho^n$$

令 $\rho \to 0$ 得 $a_{-n} = 0$ 对所有 $n \ge 1$ 成立。故主要部分为零。这完成了可去奇点的循环等价。

$m$ 阶极点：

"主要部分有限项，最高 $(z-z_0)^{-m}$ $\implies$ $\lim (z-z_0)^m f(z) = a_{-m} \ne 0$"：从 Laurent 展开直接提取：

$$(z-z_0)^m f(z) = a_{-m} + a_{-(m-1)}(z-z_0) + \cdots \to a_{-m} \ne 0$$

"$\lim (z-z_0)^m f(z) = c \ne 0$ $\implies$ $\lim f(z) = \infty$"：显然——分母趋于零而分子趋于非零常数，故 $|f(z)| \sim |c|/|z-z_0|^m \to \infty$。

"$\lim f(z) = \infty$ $\implies$ 主要部分有限项"：若 $\lim_{z\to z_0} f(z) = \infty$，则 $\lim_{z\to z_0} 1/f(z) = 0$，故 $z_0$ 是 $1/f$ 的可去奇点且在该点值为 $0$。设 $z_0$ 是 $1/f$ 的 $m$ 阶零点，则 $f(z) = 1/[(z-z_0)^m g(z)]$ 其中 $g(z_0) \ne 0$，从而 $f$ 的主要部分恰有 $m$ 项。

本性奇点：既不是可去奇点也不是极点的情况即为本性奇点。由前两类奇点的等价条件，这等价于极限既不为有限值也不为 $\infty$，即极限不存在。$\square$

定理 5.11（Weierstrass 定理）

若 $z_0$ 为 $f(z)$ 的本性奇点，则对任意 $w_0 \in \CC \cup \{\infty\}$，存在数列 $\{z_n\}$，$z_n \to z_0$，使得 $f(z_n) \to w_0$。

通俗说法： 在本性奇点的任意小邻域内，$f(z)$ 可以取到任意接近任何复数值的值（Picard 大定理甚至说明 $f(z)$ 取遍所有复数值至多一个例外）。

证明思路 · Casorati-Weierstrass 定理

用反证法。假设结论不成立——即存在某个 $w_0 \in \CC$ 和 $\varepsilon > 0$，使得在 $z_0$ 的任意小去心邻域内，$|f(z) - w_0| \ge \varepsilon$。

考虑函数 $g(z) = \dfrac{1}{f(z) - w_0}$。在上述邻域内，$|g(z)| \le 1/\varepsilon$，即 $g(z)$ 有界。由 Riemann 可去奇点定理（有界解析函数在孤立奇点处可去），$z_0$ 是 $g(z)$ 的可去奇点。

若 $g(z_0) \ne 0$，则 $f(z) = w_0 + 1/g(z)$ 在 $z_0$ 处解析（或可去），与 $z_0$ 是本性奇点矛盾。

若 $g(z_0) = 0$（$m$ 阶零点），则 $1/g(z)$ 以 $z_0$ 为 $m$ 阶极点，从而 $f(z)$ 也以 $z_0$ 为 $m$ 阶极点，同样与本性奇点矛盾。

因此假设不成立，$f(z)$ 在本性奇点的任意小邻域内的值域在 $\CC$ 中稠密（更精确地，其值域在扩充复平面 $\CC \cup \{\infty\}$ 中稠密）。$\square$

例 5.7（判断奇点类型）

判断 $z = 0$ 是以下函数的何种奇点：

（i） $f(z) = \dfrac{\sin z}{z}$：由 $\sin z = z - \dfrac{z^3}{3!} + \cdots$，得

$$f(z) = 1 - \frac{z^2}{3!} + \frac{z^4}{5!} - \cdots,$$

无负幂项，故 $z = 0$ 为 可去奇点。

（ii） $f(z) = \dfrac{1}{z^2(z-1)}$：在 $z = 0$ 的 Laurent 展开中，$\dfrac{1}{z-1} = -\sum_{n=0}^{\infty} z^n$（$|z| < 1$），故

$$f(z) = -\frac{1}{z^2} - \frac{1}{z} - 1 - z - \cdots,$$

主要部分最高阶为 $z^{-2}$，故 $z = 0$ 为 二阶极点。

（iii） $f(z) = e^{1/z}$：由 $e^{1/z} = 1 + \dfrac{1}{z} + \dfrac{1}{2!z^2} + \cdots$，主要部分有无穷多项，故 $z = 0$ 为 本性奇点。

例 5.7b（综合性奇点分类）

判断下列函数的所有有限奇点类型：

（i） $f(z) = \dfrac{z}{(z^2+1)(z-2)^3}$——奇点 $z = \pm i$（一阶极点，因为分子在 $\pm i$ 处不为零且分母单零点），$z = 2$（三阶极点）。

（ii） $f(z) = \dfrac{1-\cos z}{z^2}$——由 $\cos z = 1 - \frac{z^2}{2!} + \frac{z^4}{4!} - \cdots$，$1-\cos z = \frac{z^2}{2!} - \frac{z^4}{4!} + \cdots$，$f(z) = \frac{1}{2!} - \frac{z^2}{4!} + \cdots$（无负幂项），故 $z=0$ 为可去奇点。

（iii） $f(z) = \dfrac{e^z-1}{z^3}$——$e^z-1 = z + \frac{z^2}{2!} + \frac{z^3}{3!} + \cdots$，$f(z) = \frac{1}{z^2} + \frac{1}{2!z} + \frac{1}{3!} + \cdots$，主要部分最高次为 $z^{-2}$，故 $z=0$ 为二阶极点。

关键方法总结：判断奇点类型最直接的方法是求出函数在该点的 Laurent 展开，数一数负幂项的个数：0个→可去；有限m个→m阶极点；无穷多→本性奇点。间接判断法：看 $\lim_{z\to z_0} f(z)$（有限→可去；$\infty$→极点；不存在→本性奇点）。

定义 5.8（无穷远点作为孤立奇点）

令 $w = 1/z$，考察 $g(w) = f(1/w)$ 在 $w = 0$ 的性质。若 $w = 0$ 是 $g(w)$ 的孤立奇点，则称 $z = \infty$ 是 $f(z)$ 的孤立奇点，其类型由 $w = 0$ 的类型确定：

  • $z = \infty$ 为可去奇点 $\Longleftrightarrow$ $f(z)$ 在 $|z| > R$ 内有界；
  • $z = \infty$ 为 $m$ 阶极点 $\Longleftrightarrow$ $\lim_{z\to\infty} f(z) = \infty$ 且 $\lim_{z\to\infty} z^{-m} f(z)$ 为非零有限值；
  • $z = \infty$ 为本性奇点 $\Longleftrightarrow$ $\lim_{z\to\infty} f(z)$ 不存在。

$f(z)$ 在 $z = \infty$ 的 Laurent 展开即 $f(z) = \sum_{n=-\infty}^{+\infty} a_n z^n$ 在 $|z| > R$ 内的展开，其中正幂部分为主要部分，非正幂部分为正则部分（注意与有限奇点相反）。

注记

孤立奇点的分类是留数理论的基础。留数 $\Res[f, z_0] = a_{-1}$ 正是 Laurent 展开中 $(z - z_0)^{-1}$ 项的系数。可去奇点的留数为 $0$；本性奇点的留数一般需要通过展开计算；极点的留数有方便的公式（见第六章）。

例 5.8

求 $f(z) = \dfrac{e^z - 1}{z^2}$ 在 $z = 0$ 的 Laurent 展开，并判断奇点类型。

解：

$$e^z - 1 = z + \frac{z^2}{2!} + \frac{z^3}{3!} + \cdots,$$

$$f(z) = \frac{1}{z} + \frac{1}{2!} + \frac{z}{3!} + \frac{z^2}{4!} + \cdots.$$

主要部分仅含 $a_{-1}/z$ 一项，故 $z = 0$ 为 一阶极点，且 $\Res[f, 0] = a_{-1} = 1$。

常见误区与 FAQ

FAQ：Taylor 展开和 Laurent 展开的根本区别是什么？

Taylor 展开只在正则点（解析点）的邻域内有效，展开式中只有非负幂项。Laurent 展开在圆环域内有效，可以包含负幂项（主要部分），因此允许展开中心是奇点。两者都唯一。实用上说：如果 $f$ 在 $z_0$ 解析，用 Taylor；如果 $z_0$ 是奇点，用 Laurent。

FAQ：收敛半径为什么一定是圆？

因为收敛性取决于 $|z-z_0|$（到展开中心的距离），而与辐角无关。这正是幂级数在复平面上的"刚中带柔"——内部的绝对收敛是刚性的（只要距离小于 $R$ 一定收敛），而边界 $|z|=R$ 上的行为则多种多样。根本原因来自 Abel 定理：如果在某点收敛，则在所有更近的点绝对收敛。

FAQ："可去奇点"到底怎么"去"？

操作上：在 Laurent 展开中，如果主要部分（负幂项）全部为零，那么展开实际上就是一个 Taylor 级数。你只需要在奇点处补充定义 $f(z_0) = a_0$（展开的常数项），函数就在 $z_0$ 处变成解析的了。"可去"意味着奇异性不是"真实的"——它仅仅是定义域上的一个缺口，补上即可。例如 $\frac{\sin z}{z}$ 在 $z=0$ 处，展开为 $1 - z^2/3! + z^4/5! - \cdots$，补充定义 $f(0)=1$ 后函数在整个复平面上解析。

FAQ：本性奇点附近函数到底有多疯狂？

Weierstrass 定理说：在本性奇点的任意小邻域内，函数的值稠密于整个复平面。Picard 大定理更强：函数取遍所有复数值，至多有一个例外。以 $e^{1/z}$ 在 $z=0$ 为例：沿正实轴趋近原点（$z = \varepsilon$），$e^{1/\varepsilon} \to +\infty$；沿负实轴趋近（$z = -\varepsilon$），$e^{-1/\varepsilon} \to 0$；沿虚轴趋近（$z = i\varepsilon$），$e^{-i/\varepsilon}$ 在单位圆上不停旋转。三种路径，三种完全不同、互不协调的极限行为。

本章小结

核心要点回顾

  1. 级数收敛：复数项级数收敛 $\iff$ 实部和虚部级数分别收敛。绝对收敛由 $|z_n|$ 的判别法（比值、根值）处理。
  2. 幂级数：存在收敛半径 $R$（由比值法或根值法确定），收敛圆内绝对收敛、可逐项求导和积分。和函数在收敛圆内解析。
  3. Taylor 展开：解析函数在解析点附近可唯一地展开为幂级数，收敛半径等于该点到最近奇点的距离。常用函数的展开公式要熟记。
  4. Laurent 展开：在圆环域内可展开为双边级数。间接法（利用已知展开+代数技巧）是常用展开方式。
  5. 奇点分类：可去奇点（负幂项=0，极限有限）→ $m$ 阶极点（负幂项最高 $m$ 次，极限 $\infty$）→ 本性奇点（无穷多负幂项，极限不存在）。

承上启下

Laurent 展开中 $(z-z_0)^{-1}$ 的系数 $a_{-1}$ 如此特殊——回顾第四章，它是唯一在围道积分中"幸存"下来的项。这个系数就是留数。第六章将把前五章的所有武器汇聚成留数定理——用这个系数乘以 $2\pi i$ 再求和，就能算出任何闭合围道的积分。更妙的是，留数定理反向帮你计算实分析中极难处理的定积分。前五章的全部理论，都是为了第六章这最后一击。
