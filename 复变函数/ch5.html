5.1 复数项级数

定义 5.1（复数项级数的收敛）

设 $\{z_n\}_{n=1}^{\infty}$ 为复数列，称

$$\sum_{n=1}^{\infty} z_n = z_1 + z_2 + z_3 + \cdots$$

为复数项级数。令部分和 $S_n = \sum_{k=1}^{n} z_k$，若 $\lim_{n\to\infty} S_n = S$ 存在（$S \in \CC$），则称级数 收敛，$S$ 为其和；否则称级数 发散。

定理 5.1（收敛的必要条件）

若 $\sum_{n=1}^{\infty} z_n$ 收敛，则 $\lim_{n\to\infty} z_n = 0$。

证明思路： 设 $S_n \to S$，则 $z_n = S_n - S_{n-1} \to S - S = 0$。

定理 5.2（复数项级数收敛与实部、虚部级数的关系）

设 $z_n = x_n + iy_n$，则 $\sum_{n=1}^{\infty} z_n$ 收敛的充要条件是 $\sum_{n=1}^{\infty} x_n$ 和 $\sum_{n=1}^{\infty} y_n$ 同时收敛，且

$$\sum_{n=1}^{\infty} z_n = \sum_{n=1}^{\infty} x_n + i\sum_{n=1}^{\infty} y_n.$$

定义 5.2（绝对收敛与条件收敛）

若 $\sum_{n=1}^{\infty} |z_n|$ 收敛，则称 $\sum_{n=1}^{\infty} z_n$ 绝对收敛。收敛但不绝对收敛的级数称为 条件收敛。

定理 5.3（绝对收敛蕴含收敛）

若 $\sum_{n=1}^{\infty} |z_n|$ 收敛，则 $\sum_{n=1}^{\infty} z_n$ 收敛。

证明： 由 $|x_n| \le |z_n|$，$|y_n| \le |z_n|$，由比较判别法知 $\sum x_n$，$\sum y_n$ 均绝对收敛从而收敛，再由定理 5.2 得证。

定理 5.4（Cauchy 收敛准则）

$\sum_{n=1}^{\infty} z_n$ 收敛的充要条件是：$\forall\,\varepsilon > 0$，$\exists\, N \in \NN$，当 $n > N$ 时，对任意正整数 $p$，有

$$|z_{n+1} + z_{n+2} + \cdots + z_{n+p}| < \varepsilon.$$

注记

复数项级数的收敛理论本质上可归结为两个实数项级数的收敛问题。但绝对收敛的概念使得我们无需分离实部和虚部即可判断收敛性。常用的正项级数判别法（比较法、比值法、根值法）对复数项级数的绝对收敛性判断同样适用，因为 $|z_n|$ 是非负实数。

5.2 幂级数

定义 5.3（幂级数）

形如

$$\sum_{n=0}^{\infty} a_n (z - z_0)^n = a_0 + a_1(z-z_0) + a_2(z-z_0)^2 + \cdots$$

的级数称为以 $z_0$ 为中心的幂级数，其中 $a_n \in \CC$ 为系数。

定理 5.5（Abel 定理）

若幂级数 $\sum_{n=0}^{\infty} a_n z^n$ 在 $z = z_1 \ne 0$ 处收敛，则它在圆盘 $|z| < |z_1|$ 内绝对收敛且内闭一致收敛。

证明： 因 $\sum a_n z_1^n$ 收敛，故 $a_n z_1^n \to 0$，从而有界：$|a_n z_1^n| \le M$。对 $|z| < |z_1|$，令 $r = |z|/|z_1| < 1$，则

$$|a_n z^n| = |a_n z_1^n| \cdot \left|\frac{z}{z_1}\right|^n \le M r^n,$$

由几何级数 $\sum Mr^n$ 收敛知 $\sum |a_n z^n|$ 收敛。

定理 5.6（收敛半径的存在性）

对幂级数 $\sum_{n=0}^{\infty} a_n z^n$，存在 $R \in [0, +\infty]$（称为收敛半径），使得：

  • 当 $|z| < R$ 时，级数绝对收敛；
  • 当 $|z| > R$ 时，级数发散；
  • 当 $|z| = R$ 时，需要具体判断。

圆盘 $|z| < R$ 称为 收敛圆。

收敛半径的求法

比值法（d'Alembert）： 若 $\lim_{n\to\infty} \left|\dfrac{a_{n+1}}{a_n}\right| = \lambda$，则

$$R = \frac{1}{\lambda} \quad (\lambda = 0 \text{ 时 } R = +\infty,\; \lambda = +\infty \text{ 时 } R = 0).$$

根值法（Cauchy-Hadamard）：

$$R = \frac{1}{\limsup_{n\to\infty} \sqrt[n]{|a_n|}}.$$

（约定 $1/0 = +\infty$，$1/+\infty = 0$。）

例 5.1

求幂级数 $\displaystyle\sum_{n=0}^{\infty} \frac{z^n}{n!}$ 的收敛半径。

解： $a_n = 1/n!$，由比值法

$$\left|\frac{a_{n+1}}{a_n}\right| = \frac{n!}{(n+1)!} = \frac{1}{n+1} \to 0,$$

故 $R = +\infty$，即该级数在全平面收敛。

例 5.2

求幂级数 $\displaystyle\sum_{n=1}^{\infty} n^n z^n$ 的收敛半径。

解： 由根值法

$$\limsup_{n\to\infty} \sqrt[n]{|a_n|} = \lim_{n\to\infty} n = +\infty,$$

故 $R = 0$，级数仅在 $z = 0$ 处收敛。

定理 5.7（幂级数的性质）

设 $\sum a_n z^n$ 的收敛半径为 $R > 0$，则：

  1. 在收敛圆 $|z| < R$ 内，和函数 $f(z) = \sum a_n z^n$ 解析；
  2. 可在收敛圆内逐项求导：$f'(z) = \sum_{n=1}^{\infty} n a_n z^{n-1}$；
  3. 可在收敛圆内逐项积分：$\int_0^z f(\zeta)\,\dd\zeta = \sum_{n=0}^{\infty} \frac{a_n}{n+1} z^{n+1}$；
  4. 逐项求导和逐项积分不改变收敛半径。
注记：Abel 定理的逆否命题

若 $\sum a_n z^n$ 在 $z = z_1$ 处发散，则对所有 $|z| > |z_1|$，级数也发散。这说明收敛圆外的点一定使级数发散。

5.3 Taylor 展开定理

定理 5.8（Taylor 展开定理）

设 $f(z)$ 在圆盘 $D: |z - z_0| < R$ 内解析，则 $f(z)$ 在 $D$ 内可唯一地展开为幂级数

$$f(z) = \sum_{n=0}^{\infty} a_n (z - z_0)^n, \quad |z - z_0| < R,$$

其中

$$a_n = \frac{f^{(n)}(z_0)}{n!} = \frac{1}{2\pi i} \oint_C \frac{f(\zeta)}{(\zeta - z_0)^{n+1}}\,\dd\zeta,$$

$C$ 为圆周 $|\zeta - z_0| = r < R$。

证明思路

取 $r < R$，令 $C: |\zeta - z_0| = r$。对 $|z - z_0| < r$，由 Cauchy 积分公式：

$$f(z) = \frac{1}{2\pi i} \oint_C \frac{f(\zeta)}{\zeta - z}\,\dd\zeta.$$

将 $\dfrac{1}{\zeta - z} = \dfrac{1}{\zeta - z_0} \cdot \dfrac{1}{1 - \frac{z - z_0}{\zeta - z_0}}$ 展开为几何级数（因 $\left|\dfrac{z - z_0}{\zeta - z_0}\right| < 1$），代入后逐项积分即得。唯一性由幂级数系数的唯一性保证。

定理 5.9（Taylor 展开的唯一性）

若 $f(z)$ 在 $|z - z_0| < R$ 内有两种幂级数表示 $f(z) = \sum a_n(z-z_0)^n = \sum b_n(z-z_0)^n$，则 $a_n = b_n$ 对所有 $n$ 成立。

注记：展开的注意事项

Taylor 展开的收敛半径等于展开中心 $z_0$ 到 $f(z)$ 最近奇点的距离。例如 $\dfrac{1}{1+z^2}$ 在 $z_0 = 0$ 处展开的收敛半径为 $1$（因为 $z = \pm i$ 是奇点，距原点距离为 $1$），尽管该函数在实轴上处处光滑。

常用 Taylor 展开公式表

函数 Taylor 展开 收敛域
$e^z$ $\displaystyle\sum_{n=0}^{\infty} \frac{z^n}{n!}$ $|z| < +\infty$
$\sin z$ $\displaystyle\sum_{n=0}^{\infty} \frac{(-1)^n z^{2n+1}}{(2n+1)!}$ $|z| < +\infty$
$\cos z$ $\displaystyle\sum_{n=0}^{\infty} \frac{(-1)^n z^{2n}}{(2n)!}$ $|z| < +\infty$
$\dfrac{1}{1-z}$ $\displaystyle\sum_{n=0}^{\infty} z^n$ $|z| < 1$
$\dfrac{1}{1+z}$ $\displaystyle\sum_{n=0}^{\infty} (-1)^n z^n$ $|z| < 1$
$\log(1+z)$ $\displaystyle\sum_{n=1}^{\infty} \frac{(-1)^{n-1} z^n}{n}$ $|z| < 1$
$(1+z)^\alpha$ $\displaystyle\sum_{n=0}^{\infty} \binom{\alpha}{n} z^n$ $|z| < 1$
$\dfrac{1}{(1-z)^2}$ $\displaystyle\sum_{n=1}^{\infty} n z^{n-1}$ $|z| < 1$
例 5.3

将 $f(z) = \dfrac{1}{z(z-1)}$ 在 $z_0 = 0$ 的邻域内展开。

解： 对 $|z| < 1$，利用 $\dfrac{1}{1-z} = \sum_{n=0}^{\infty} z^n$，得

$$f(z) = \frac{1}{z} \cdot \frac{1}{z-1} = -\frac{1}{z} \cdot \frac{1}{1-z} = -\frac{1}{z} \sum_{n=0}^{\infty} z^n = -\sum_{n=0}^{\infty} z^{n-1} = -\frac{1}{z} - 1 - z - z^2 - \cdots.$$

这不是 Taylor 级数（含负幂项），而是 Laurent 级数，因为 $z = 0$ 是奇点。

▶ 例 5.4（间接展开法）

将 $f(z) = \dfrac{1}{(1-z)^2}$ 在 $z_0 = 0$ 处展开。

解法一（逐项求导）： 由 $\dfrac{1}{1-z} = \sum_{n=0}^{\infty} z^n$，两边对 $z$ 求导：

$$\frac{1}{(1-z)^2} = \sum_{n=1}^{\infty} n z^{n-1} = 1 + 2z + 3z^2 + 4z^3 + \cdots, \quad |z| < 1.$$

解法二（级数乘法）： $\dfrac{1}{(1-z)^2} = \left(\sum_{n=0}^{\infty} z^n\right)^2$，由 Cauchy 乘积公式：

$$c_n = \sum_{k=0}^{n} 1 \cdot 1 = n+1,$$

故 $\dfrac{1}{(1-z)^2} = \sum_{n=0}^{\infty} (n+1)z^n$，收敛半径 $R = 1$。

5.4 Laurent 展开定理

定义 5.4（圆环域）

设 $z_0 \in \CC$，$0 \le r < R \le +\infty$，则集合

$$D = \{z \in \CC : r < |z - z_0| < R\}$$

称为以 $z_0$ 为中心、内半径为 $r$、外半径为 $R$ 的圆环域。

定理 5.10（Laurent 展开定理）

设 $f(z)$ 在圆环域 $D: r < |z - z_0| < R$（$0 \le r < R \le +\infty$）内解析，则 $f(z)$ 在 $D$ 内可唯一地展开为双边级数

$$f(z) = \sum_{n=-\infty}^{+\infty} a_n (z - z_0)^n, \quad r < |z - z_0| < R,$$

其中

$$a_n = \frac{1}{2\pi i} \oint_C \frac{f(\zeta)}{(\zeta - z_0)^{n+1}}\,\dd\zeta, \quad n = 0, \pm 1, \pm 2, \ldots,$$

$C$ 为圆环域内绕 $z_0$ 的任一简单闭曲线。

定义 5.5（正则部分与主要部分）

Laurent 级数可拆分为两部分：

正则部分（解析部分）：

$$\sum_{n=0}^{\infty} a_n (z - z_0)^n = a_0 + a_1(z-z_0) + a_2(z-z_0)^2 + \cdots$$

这是一个在 $|z - z_0| < R$ 内收敛的幂级数。

主要部分：

$$\sum_{n=1}^{\infty} a_{-n} (z - z_0)^{-n} = \frac{a_{-1}}{z - z_0} + \frac{a_{-2}}{(z-z_0)^2} + \cdots$$

这是一个在 $|z - z_0| > r$ 内收敛的级数。

注记：Taylor 展开与 Laurent 展开的关系

Taylor 展开是 Laurent 展开的特殊情形：当 $r = 0$ 且 $f(z)$ 在 $z_0$ 解析时，所有 $a_{-n} = 0$（$n \ge 1$），Laurent 级数退化为 Taylor 级数。

重要区别： Laurent 展开的系数一般不能用 $a_n = f^{(n)}(z_0)/n!$ 计算（因为 $f$ 在 $z_0$ 可能不解析），必须通过围道积分或间接方法求得。

Laurent 展开的常用方法

直接法： 利用系数公式 $a_n = \frac{1}{2\pi i}\oint_C \frac{f(\zeta)}{(\zeta-z_0)^{n+1}}\,\dd\zeta$ 计算（通常较繁琐）。

间接法（常用）：

  1. 利用已知的 Taylor 展开公式；
  2. 利用变量替换、级数乘法、逐项求导/积分等技巧；
  3. 将函数拆分为在不同区域收敛的已知展开之和。
▶ 例 5.5（Laurent 展开的典型计算）

将 $f(z) = \dfrac{1}{(z-1)(z-2)}$ 在以下圆环域内展开为 Laurent 级数：

（i）$1 < |z| < 2$；（ii）$|z| > 2$。

解： 先做部分分式分解 $f(z) = \dfrac{1}{z-2} - \dfrac{1}{z-1}$。

情形（i）$1 < |z| < 2$：

对 $\dfrac{1}{z-2} = -\dfrac{1}{2} \cdot \dfrac{1}{1 - z/2}$，因 $|z| < 2$ 即 $|z/2| < 1$：

$$\frac{1}{z-2} = -\frac{1}{2}\sum_{n=0}^{\infty} \left(\frac{z}{2}\right)^n = -\sum_{n=0}^{\infty} \frac{z^n}{2^{n+1}}.$$

对 $\dfrac{1}{z-1} = \dfrac{1}{z} \cdot \dfrac{1}{1 - 1/z}$，因 $|z| > 1$ 即 $|1/z| < 1$：

$$\frac{1}{z-1} = \frac{1}{z}\sum_{n=0}^{\infty} \frac{1}{z^n} = \sum_{n=0}^{\infty} \frac{1}{z^{n+1}} = \sum_{n=1}^{\infty} \frac{1}{z^n}.$$

因此

$$f(z) = -\sum_{n=0}^{\infty} \frac{z^n}{2^{n+1}} - \sum_{n=1}^{\infty} \frac{1}{z^n}, \quad 1 < |z| < 2.$$

情形（ii）$|z| > 2$：

对 $\dfrac{1}{z-2} = \dfrac{1}{z}\cdot\dfrac{1}{1-2/z}$，因 $|z| > 2$：

$$\frac{1}{z-2} = \frac{1}{z}\sum_{n=0}^{\infty}\frac{2^n}{z^n} = \sum_{n=0}^{\infty}\frac{2^n}{z^{n+1}}.$$

结合 $\dfrac{1}{z-1} = \sum_{n=0}^{\infty}\dfrac{1}{z^{n+1}}$（同上），得

$$f(z) = \sum_{n=0}^{\infty}\frac{2^n - 1}{z^{n+1}} = \frac{1}{z} + \frac{3}{z^2} + \frac{7}{z^3} + \cdots, \quad |z| > 2.$$

▶ 例 5.6（含指数函数的 Laurent 展开）

将 $f(z) = e^{1/z}$ 在 $0 < |z| < +\infty$ 内展开。

解： 由 $e^w = \sum_{n=0}^{\infty} \dfrac{w^n}{n!}$，令 $w = 1/z$：

$$e^{1/z} = \sum_{n=0}^{\infty} \frac{1}{n! \, z^n} = 1 + \frac{1}{z} + \frac{1}{2!\,z^2} + \frac{1}{3!\,z^3} + \cdots.$$

此展开在 $0 < |z| < +\infty$ 内成立。注意正则部分只有常数项 $1$，主要部分有无穷多项，说明 $z = 0$ 是本性奇点。

5.5 孤立奇点的分类

定义 5.6（孤立奇点）

若 $f(z)$ 在 $z_0$ 的去心邻域 $0 < |z - z_0| < \delta$ 内解析，但在 $z_0$ 处不解析（或无定义），则称 $z_0$ 为 $f(z)$ 的 孤立奇点。

定义 5.7（孤立奇点的三种类型）

设 $z_0$ 是 $f(z)$ 的孤立奇点，$f(z)$ 在 $0 < |z - z_0| < \delta$ 内的 Laurent 展开为

$$f(z) = \sum_{n=-\infty}^{+\infty} a_n(z - z_0)^n.$$

（i）可去奇点： 主要部分为零，即 $a_{-n} = 0$（$n \ge 1$）。此时

$$f(z) = a_0 + a_1(z-z_0) + a_2(z-z_0)^2 + \cdots.$$

补充定义 $f(z_0) = a_0$ 后，$f$ 在 $z_0$ 解析。

（ii）$m$ 阶极点： 主要部分只有有限项，且 $a_{-m} \ne 0$，$a_{-n} = 0$（$n > m$）。此时

$$f(z) = \frac{a_{-m}}{(z-z_0)^m} + \cdots + \frac{a_{-1}}{z-z_0} + a_0 + a_1(z-z_0) + \cdots.$$

特别地，$m = 1$ 时称为 简单极点（一阶极点）。

（iii）本性奇点： 主要部分有无穷多项，即有无穷多个 $a_{-n} \ne 0$。

孤立奇点的等价判别法

设 $z_0$ 是 $f(z)$ 的孤立奇点，则：

$z_0$ 为可去奇点 $\Longleftrightarrow$ $\lim_{z\to z_0} f(z)$ 存在（有限） $\Longleftrightarrow$ $f(z)$ 在 $z_0$ 的某去心邻域内有界。

$z_0$ 为 $m$ 阶极点 $\Longleftrightarrow$ $\lim_{z\to z_0}(z-z_0)^m f(z) = c \ne 0$（有限） $\Longleftrightarrow$ $\lim_{z\to z_0} f(z) = \infty$。

$z_0$ 为本性奇点 $\Longleftrightarrow$ $\lim_{z\to z_0} f(z)$ 不存在（不为有限值也不为 $\infty$）。

定理 5.11（Weierstrass 定理）

若 $z_0$ 为 $f(z)$ 的本性奇点，则对任意 $w_0 \in \CC \cup \{\infty\}$，存在数列 $\{z_n\}$，$z_n \to z_0$，使得 $f(z_n) \to w_0$。

通俗说法： 在本性奇点的任意小邻域内，$f(z)$ 可以取到任意接近任何复数值的值（Picard 大定理甚至说明 $f(z)$ 取遍所有复数值至多一个例外）。

例 5.7（判断奇点类型）

判断 $z = 0$ 是以下函数的何种奇点：

（i） $f(z) = \dfrac{\sin z}{z}$：由 $\sin z = z - \dfrac{z^3}{3!} + \cdots$，得

$$f(z) = 1 - \frac{z^2}{3!} + \frac{z^4}{5!} - \cdots,$$

无负幂项，故 $z = 0$ 为 可去奇点。

（ii） $f(z) = \dfrac{1}{z^2(z-1)}$：在 $z = 0$ 的 Laurent 展开中，$\dfrac{1}{z-1} = -\sum_{n=0}^{\infty} z^n$（$|z| < 1$），故

$$f(z) = -\frac{1}{z^2} - \frac{1}{z} - 1 - z - \cdots,$$

主要部分最高阶为 $z^{-2}$，故 $z = 0$ 为 二阶极点。

（iii） $f(z) = e^{1/z}$：由 $e^{1/z} = 1 + \dfrac{1}{z} + \dfrac{1}{2!z^2} + \cdots$，主要部分有无穷多项，故 $z = 0$ 为 本性奇点。

定义 5.8（无穷远点作为孤立奇点）

令 $w = 1/z$，考察 $g(w) = f(1/w)$ 在 $w = 0$ 的性质。若 $w = 0$ 是 $g(w)$ 的孤立奇点，则称 $z = \infty$ 是 $f(z)$ 的孤立奇点，其类型由 $w = 0$ 的类型确定：

  • $z = \infty$ 为可去奇点 $\Longleftrightarrow$ $f(z)$ 在 $|z| > R$ 内有界；
  • $z = \infty$ 为 $m$ 阶极点 $\Longleftrightarrow$ $\lim_{z\to\infty} f(z) = \infty$ 且 $\lim_{z\to\infty} z^{-m} f(z)$ 为非零有限值；
  • $z = \infty$ 为本性奇点 $\Longleftrightarrow$ $\lim_{z\to\infty} f(z)$ 不存在。

$f(z)$ 在 $z = \infty$ 的 Laurent 展开即 $f(z) = \sum_{n=-\infty}^{+\infty} a_n z^n$ 在 $|z| > R$ 内的展开，其中正幂部分为主要部分，非正幂部分为正则部分（注意与有限奇点相反）。

注记

孤立奇点的分类是留数理论的基础。留数 $\Res[f, z_0] = a_{-1}$ 正是 Laurent 展开中 $(z - z_0)^{-1}$ 项的系数。可去奇点的留数为 $0$；本性奇点的留数一般需要通过展开计算；极点的留数有方便的公式（见第六章）。

例 5.8

求 $f(z) = \dfrac{e^z - 1}{z^2}$ 在 $z = 0$ 的 Laurent 展开，并判断奇点类型。

解：

$$e^z - 1 = z + \frac{z^2}{2!} + \frac{z^3}{3!} + \cdots,$$

$$f(z) = \frac{1}{z} + \frac{1}{2!} + \frac{z}{3!} + \frac{z^2}{4!} + \cdots.$$

主要部分仅含 $a_{-1}/z$ 一项，故 $z = 0$ 为 一阶极点，且 $\Res[f, 0] = a_{-1} = 1$。
