第一章 · 复数与复变函数

1.1 复数及其代数运算

定义 1.1（复数）

设 $x, y \in \RR$，称 $z = x + iy$ 为一个复数，其中 $i$ 为虚数单位，满足 $i^2 = -1$。$x = \Re(z)$ 称为 $z$ 的实部，$y = \Im(z)$ 称为 $z$ 的虚部。

  • 当 $y = 0$ 时，$z = x$ 为实数。
  • 当 $x = 0,\, y \ne 0$ 时，$z = iy$ 为纯虚数。
  • $z = 0$ 当且仅当 $x = 0$ 且 $y = 0$。
定义 1.2（共轭复数与模）

设 $z = x + iy$，定义：

  • 共轭复数：$\bar{z} = x - iy$。
  • 模（绝对值）：$|z| = \sqrt{x^2 + y^2}$。

四则运算

设 $z_1 = x_1 + iy_1$，$z_2 = x_2 + iy_2$，则：

$$z_1 + z_2 = (x_1 + x_2) + i(y_1 + y_2)$$ $$z_1 - z_2 = (x_1 - x_2) + i(y_1 - y_2)$$ $$z_1 z_2 = (x_1 x_2 - y_1 y_2) + i(x_1 y_2 + x_2 y_1)$$ $$\frac{z_1}{z_2} = \frac{x_1 x_2 + y_1 y_2}{x_2^2 + y_2^2} + i\frac{x_2 y_1 - x_1 y_2}{x_2^2 + y_2^2} \quad (z_2 \ne 0)$$
定理 1.1（模与共轭的基本性质）

设 $z, z_1, z_2 \in \CC$，则：

  1. $\overline{\bar{z}} = z$
  2. $|z|^2 = z\bar{z}$
  3. $\overline{z_1 + z_2} = \bar{z}_1 + \bar{z}_2$，$\overline{z_1 z_2} = \bar{z}_1 \bar{z}_2$
  4. $|z_1 z_2| = |z_1|\,|z_2|$
  5. 三角不等式：$|z_1 + z_2| \le |z_1| + |z_2|$，$\bigl||z_1| - |z_2|\bigr| \le |z_1 - z_2|$
例题 1.1

设 $z_1 = 3 + 4i$，$z_2 = 1 - 2i$，计算 $z_1 z_2$、$|z_1/z_2|$ 和 $\overline{z_1 z_2}$。

▶ 展开解答

计算乘积：

$$z_1 z_2 = (3+4i)(1-2i) = 3 - 6i + 4i - 8i^2 = 3 - 2i + 8 = 11 - 2i$$

计算模的商：

$$|z_1| = \sqrt{9+16} = 5, \quad |z_2| = \sqrt{1+4} = \sqrt{5}$$ $$\left|\frac{z_1}{z_2}\right| = \frac{|z_1|}{|z_2|} = \frac{5}{\sqrt{5}} = \sqrt{5}$$

共轭：$\overline{z_1 z_2} = \overline{11 - 2i} = 11 + 2i$。

验证：$\bar{z}_1 \bar{z}_2 = (3-4i)(1+2i) = 3 + 6i - 4i - 8i^2 = 11 + 2i$。一致。

例题 1.2

设 $z = \dfrac{(3+4i)(2-i)}{(1+i)^2}$，求 $|z|$ 和 $z + \bar{z}$。

▶ 展开解答

利用模的性质计算 $|z|$：

$$|3+4i| = 5, \quad |2-i| = \sqrt{5}, \quad |1+i| = \sqrt{2}$$ $$|z| = \frac{|3+4i| \cdot |2-i|}{|1+i|^2} = \frac{5\sqrt{5}}{2}$$

计算 $z + \bar{z} = 2\Re(z)$：

$$(3+4i)(2-i) = 6 - 3i + 8i - 4i^2 = 10 + 5i$$ $$(1+i)^2 = 1 + 2i + i^2 = 2i$$ $$z = \frac{10+5i}{2i} = \frac{(10+5i)(-2i)}{(2i)(-2i)} = \frac{-20i - 10i^2}{4} = \frac{10 - 20i}{4} = \frac{5}{2} - 5i$$ $$z + \bar{z} = 2\Re(z) = 2 \cdot \frac{5}{2} = 5$$

1.2 复数的几何表示

定义 1.3（复平面）

建立直角坐标系：横轴为 实轴（$\Re$），纵轴为 虚轴（$\Im$）。复数 $z = x + iy$ 对应平面上的点 $(x, y)$，此平面称为复平面（$\CC$ 平面 / Argand 平面）。复数也可视为从原点到 $(x,y)$ 的向量。

定义 1.4（辐角）

设 $z \ne 0$，从正实轴到向量 $Oz$ 的有向角 $\theta$ 称为 $z$ 的辐角。满足 $z = |z|(\cos\theta + i\sin\theta)$ 的所有 $\theta$ 值构成辐角集 $\arg(z)$。

辐角的主值 $\Arg(z)$ 通常取 $(-\pi, \pi]$ 中的唯一值，即：

$$\Arg(z) = \begin{cases} \arctan\dfrac{y}{x}, & x > 0 \\[6pt] \arctan\dfrac{y}{x} + \pi, & x < 0,\, y \ge 0 \\[6pt] \arctan\dfrac{y}{x} - \pi, & x < 0,\, y < 0 \\[6pt] \dfrac{\pi}{2}, & x = 0,\, y > 0 \\[6pt] -\dfrac{\pi}{2}, & x = 0,\, y < 0 \end{cases}$$

辐角的任意值满足 $\arg(z) = \Arg(z) + 2k\pi$，$k \in \mathbb{Z}$。

公式：三角形式与指数形式

设 $r = |z|$，$\theta = \arg(z)$，则：

三角形式：$z = r(\cos\theta + i\sin\theta)$

指数形式（利用 Euler 公式 $e^{i\theta} = \cos\theta + i\sin\theta$）：

$$z = re^{i\theta}$$
注记：复数乘法的几何意义

设 $z_1 = r_1 e^{i\theta_1}$，$z_2 = r_2 e^{i\theta_2}$，则：

$$z_1 z_2 = r_1 r_2 \, e^{i(\theta_1 + \theta_2)}$$

即复数相乘时模相乘、辐角相加。几何上，$z_1 z_2$ 相当于将 $z_1$ 伸缩 $|z_2|$ 倍再旋转 $\arg(z_2)$ 弧度。

例题 1.3

将 $z = -1 + \sqrt{3}\,i$ 化为指数形式，并计算 $z^6$。

▶ 展开解答

求模和辐角：

$$r = |z| = \sqrt{1 + 3} = 2$$

$z$ 在第二象限，$\tan\theta = \dfrac{\sqrt{3}}{-1} = -\sqrt{3}$，故 $\theta = \pi - \dfrac{\pi}{3} = \dfrac{2\pi}{3}$。

$$z = 2e^{i \cdot 2\pi/3}$$

计算 $z^6$：

$$z^6 = 2^6 \cdot e^{i \cdot 6 \cdot 2\pi/3} = 64 \, e^{i \cdot 4\pi} = 64$$

也可用棣莫弗公式：$z^6 = 2^6[\cos(4\pi) + i\sin(4\pi)] = 64$。

例题 1.4

设 $z_1 = 1 + i$，$z_2 = \sqrt{3} - i$，利用指数形式计算 $\dfrac{z_1}{z_2}$ 的三角形式。

▶ 展开解答

化为指数形式：

$$z_1 = \sqrt{2}\,e^{i\pi/4}, \quad z_2 = 2\,e^{-i\pi/6}$$ $$\frac{z_1}{z_2} = \frac{\sqrt{2}}{2}\,e^{i(\pi/4 + \pi/6)} = \frac{\sqrt{2}}{2}\,e^{i \cdot 5\pi/12}$$

三角形式：

$$\frac{z_1}{z_2} = \frac{\sqrt{2}}{2}\left(\cos\frac{5\pi}{12} + i\sin\frac{5\pi}{12}\right)$$

1.3 棣莫弗公式与复数方根

定理 1.2（棣莫弗公式，De Moivre）

对任意正整数 $n$，有：

$$(\cos\theta + i\sin\theta)^n = \cos n\theta + i\sin n\theta$$

即 $(e^{i\theta})^n = e^{in\theta}$。利用二项式展开可推导 $\cos n\theta$ 和 $\sin n\theta$ 用 $\cos\theta, \sin\theta$ 表示的公式。

例题 1.5（用棣莫弗公式展开 $\cos 3\theta$）
▶ 展开解答

由棣莫弗公式：

$$\cos 3\theta + i\sin 3\theta = (\cos\theta + i\sin\theta)^3$$

展开右边：

$$(\cos\theta + i\sin\theta)^3 = \cos^3\theta + 3i\cos^2\theta\sin\theta - 3\cos\theta\sin^2\theta - i\sin^3\theta$$

比较实部：

$$\cos 3\theta = \cos^3\theta - 3\cos\theta\sin^2\theta = \cos^3\theta - 3\cos\theta(1-\cos^2\theta) = 4\cos^3\theta - 3\cos\theta$$
定理 1.3（$n$ 次方根）

设 $w \ne 0$，方程 $z^n = w$（$n \ge 2$ 为正整数）恰有 $n$ 个不同的解。设 $w = \rho e^{i\varphi}$，则：

$$z_k = \rho^{1/n} \, e^{i(\varphi + 2k\pi)/n}, \quad k = 0, 1, 2, \ldots, n-1$$

几何意义：$n$ 个根均匀分布在以原点为圆心、$\rho^{1/n}$ 为半径的圆周上，相邻两根的辐角差为 $\dfrac{2\pi}{n}$。

例题 1.6

求 $z^3 = 8$ 的全部根，并在复平面上标出。

▶ 展开解答

将 $8$ 写为指数形式：$8 = 8e^{i \cdot 0}$。由公式：

$$z_k = 8^{1/3} e^{i(0 + 2k\pi)/3} = 2\,e^{i \cdot 2k\pi/3}, \quad k = 0, 1, 2$$

三个根为：

  • $z_0 = 2\,e^{0} = 2$
  • $z_1 = 2\,e^{i2\pi/3} = 2\left(-\dfrac{1}{2} + i\dfrac{\sqrt{3}}{2}\right) = -1 + \sqrt{3}\,i$
  • $z_2 = 2\,e^{i4\pi/3} = 2\left(-\dfrac{1}{2} - i\dfrac{\sqrt{3}}{2}\right) = -1 - \sqrt{3}\,i$

三个根均匀分布在半径为 $2$ 的圆上，间隔 $120°$。

例题 1.7

求 $(1+i)$ 的全部四次方根。

▶ 展开解答

先将 $1+i$ 化为指数形式：

$$1 + i = \sqrt{2}\,e^{i\pi/4}$$

四次方根：

$$z_k = 2^{1/4} \, e^{i(\pi/4 + 2k\pi)/4} = 2^{1/4}\,e^{i(\pi/16 + k\pi/2)}, \quad k = 0,1,2,3$$

即四个根的辐角分别为 $\dfrac{\pi}{16}$、$\dfrac{\pi}{16} + \dfrac{\pi}{2}$、$\dfrac{\pi}{16} + \pi$、$\dfrac{\pi}{16} + \dfrac{3\pi}{2}$，模均为 $2^{1/4}$。

1.4 复平面上的点集

定义 1.5（邻域）

设 $z_0 \in \CC$，$\delta > 0$，称集合

$$U(z_0, \delta) = \{z \in \CC : |z - z_0| < \delta\}$$

为 $z_0$ 的 $\delta$-邻域。称 $U^\circ(z_0, \delta) = U(z_0, \delta) \setminus \{z_0\}$ 为 $z_0$ 的 去心 $\delta$-邻域。

定义 1.6（开集、闭集）

设 $E \subset \CC$，

  • 若 $\forall\, z_0 \in E$，$\exists\,\delta > 0$ 使得 $U(z_0, \delta) \subset E$，则 $E$ 为开集。
  • 若 $E$ 的余集 $\CC \setminus E$ 为开集，则 $E$ 为闭集。
  • $E$ 的闭包 $\bar{E} = E \cup E'$，其中 $E'$ 为 $E$ 的所有聚点。
定义 1.7（区域与闭区域）

复平面上满足以下两个条件的点集 $D$ 称为区域（域 / 开域）：

  1. $D$ 是开集。
  2. $D$ 是连通的，即 $D$ 中任意两点可用全部在 $D$ 中的折线连接。

区域 $D$ 与其边界 $\partial D$ 的并集 $\bar{D} = D \cup \partial D$ 称为闭区域。

定义 1.8（单连通域与多连通域）

设 $D$ 为区域，

  • 若 $D$ 内任意简单闭曲线所围的区域全部属于 $D$，则 $D$ 为单连通域。直观上，$D$ 内无"洞"。
  • 否则 $D$ 为多连通域（复连通域）。
注记

例如 $\{z : |z| < 1\}$ 是单连通域；$\{z : 0 < |z| < 2\}$ 是多连通域（因原点处有"洞"，内部有不包含在区域中的点）。有界开集不一定是区域——还需满足连通性。

例题 1.8

判断下列集合是区域还是闭区域，是否单连通：

  1. $D_1 = \{z : 1 < |z| < 3\}$
  2. $D_2 = \{z : \Re(z) > 0,\, \Im(z) > 0\}$（第一象限）
  3. $D_3 = \{z : |z| \le 1\}$
▶ 展开解答
  1. $D_1$ 是区域（开集且连通）。它是多连通域——圆环内部有"洞"（圆 $|z| \le 1$ 不在 $D_1$ 中）。
  2. $D_2$ 是区域（开的半平面交集，连通）。它是单连通域——内部无洞。
  3. $D_3$ 是闭区域（闭集，边界为 $|z|=1$）。注意它不是区域，因为它不是开集。
例题 1.9

设 $E = \left\{z = x + iy : x > 0,\, 0 < y < \dfrac{1}{x}\right\}$，证明 $E$ 是区域但不是有界集。

▶ 展开解答

开集：$E$ 是两个开半平面的交再与开集 $\{y > 0\}$ 取交，是开集。

连通性：$E$ 中任意两点可用光滑曲线（如 $y = c/x$ 的一段）在 $E$ 内连接，故 $E$ 连通。因此 $E$ 是区域。

无界：取 $x_n = n$，$y_n = \dfrac{1}{2n}$，则 $z_n = n + \dfrac{i}{2n} \in E$，但 $|z_n| \to \infty$。故 $E$ 无界。

1.5 复变函数的概念

定义 1.9（复变函数）

设 $D$ 为复平面上的点集。若存在对应法则 $f$，使得 $D$ 中每一个 $z$ 对应唯一的 $w \in \CC$，则称 $f$ 为定义在 $D$ 上的复变函数，记作 $w = f(z)$。$D$ 为定义域，值域为 $f(D)$。

将 $z = x + iy$，$w = u + iv$ 分离实虚部，$f$ 可写为：

$$w = f(z) = u(x,y) + iv(x,y)$$

其中 $u(x,y)$ 和 $v(x,y)$ 是关于 $x, y$ 的二元实函数，分别称为 $f$ 的实部和虚部。

定义 1.10（极限）

设 $f(z)$ 在 $z_0$ 的去心邻域内有定义。若存在 $w_0 \in \CC$，使得 $\forall\,\varepsilon > 0$，$\exists\,\delta > 0$，当 $0 < |z - z_0| < \delta$ 时有 $|f(z) - w_0| < \varepsilon$，则称 $f(z)$ 当 $z \to z_0$ 时以 $w_0$ 为极限，记作 $\lim_{z \to z_0} f(z) = w_0$。

等价条件（化为二元实函数）：

$$\lim_{z \to z_0} f(z) = w_0 \iff \begin{cases} \lim_{(x,y)\to(x_0,y_0)} u(x,y) = u_0 \\ \lim_{(x,y)\to(x_0,y_0)} v(x,y) = v_0 \end{cases}$$

其中 $z_0 = x_0 + iy_0$，$w_0 = u_0 + iv_0$。

定义 1.11（连续性）

$f(z)$ 在 $z_0$ 连续 $\iff$ $\lim_{z \to z_0} f(z) = f(z_0)$。

等价地，$f(z) = u(x,y) + iv(x,y)$ 在 $z_0$ 连续 $\iff$ $u, v$ 在 $(x_0, y_0)$ 处均连续。

注记：映射的观点

复变函数 $w = f(z)$ 本质上是从 $\CC$ 的子集 $D$ 到 $\CC$ 的映射。它将 $z$ 平面上的点集映射为 $w$ 平面上的点集。例如 $w = z^2$ 将 $z$ 平面上的圆 $|z| = r$ 映射为 $w$ 平面上的圆 $|w| = r^2$；将射线 $\arg(z) = \theta$ 映射为射线 $\arg(w) = 2\theta$。

例题 1.10

设 $f(z) = z^2 + 2z$，求其实部 $u(x,y)$ 和虚部 $v(x,y)$。

▶ 展开解答

令 $z = x + iy$：

$$z^2 = (x+iy)^2 = x^2 - y^2 + 2ixy$$ $$f(z) = x^2 - y^2 + 2ixy + 2x + 2iy = (x^2 - y^2 + 2x) + i(2xy + 2y)$$

故 $u(x,y) = x^2 - y^2 + 2x$，$v(x,y) = 2xy + 2y$。

例题 1.11

设 $f(z) = \dfrac{\bar{z}}{z}$（$z \ne 0$），求 $u(x,y)$、$v(x,y)$，并证明 $\lim_{z \to 0} f(z)$ 不存在。

▶ 展开解答

分离实虚部：

$$\frac{\bar{z}}{z} = \frac{x - iy}{x + iy} = \frac{(x-iy)^2}{x^2 + y^2} = \frac{x^2 - y^2 - 2ixy}{x^2 + y^2}$$ $$u(x,y) = \frac{x^2 - y^2}{x^2 + y^2}, \quad v(x,y) = \frac{-2xy}{x^2 + y^2}$$

证明极限不存在：沿不同路径趋近原点得到不同极限值。

  • 沿实轴（$y = 0$）：$f(x) = \dfrac{x^2}{x^2} = 1 \to 1$
  • 沿虚轴（$x = 0$）：$f(iy) = \dfrac{-y^2}{y^2} = -1 \to -1$

两路径极限不同，故 $\lim_{z \to 0} f(z)$ 不存在。
