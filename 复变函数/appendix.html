第一部分：核心公式速查

以下仅列考试中出现频率最高、需要熟练记忆和使用的公式。推导过程见对应章节正文。

第一章 · 复数与复变函数

公式说明
$|z|^2 = z\bar{z}$模与共轭的核心恒等式（去分母、证不等式的基础）
$|z_1 z_2| = |z_1|\,|z_2|$模的乘积性质
$|z_1 + z_2| \le |z_1| + |z_2|$三角不等式（证明中频繁使用）
$z = re^{i\theta} = r(\cos\theta + i\sin\theta)$指数形式 / 三角形式
$\Arg(z) = \begin{cases} \arctan\frac{y}{x}, & x>0 \\ \arctan\frac{y}{x}+\pi, & x<0, y\ge 0 \\ \arctan\frac{y}{x}-\pi, & x<0, y<0 \end{cases}$辐角主值（$x=0$时直接几何判断）
$(\cos\theta + i\sin\theta)^n = \cos n\theta + i\sin n\theta$棣莫弗公式（$n\in\ZZ$）
$z_k = \rho^{1/n} e^{i(\varphi + 2k\pi)/n},\; k=0,1,\ldots,n-1$$n$次方根：$n$个根均匀分布
$\lim_{z\to z_0}f(z)=w_0 \iff \begin{cases} u(x,y)\to u_0 \\ v(x,y)\to v_0 \end{cases}$极限归结为实部、虚部的二元极限

第二章 · 解析函数

公式说明
$\displaystyle f'(z_0) = \lim_{\Delta z\to 0}\frac{f(z_0+\Delta z)-f(z_0)}{\Delta z}$复导数定义：所有方向极限一致
$\dfrac{\partial u}{\partial x} = \dfrac{\partial v}{\partial y},\quad \dfrac{\partial u}{\partial y} = -\dfrac{\partial v}{\partial x}$★ Cauchy-Riemann 方程（全书最重要的方程）
$f'(z) = u_x + iv_x = v_y - iu_y$C-R条件满足时的导数公式
可导 $\xLeftarrow{\text{C-R+可微}}$ 解析在一点可导 ≠ 在一点解析（后者需邻域内处处可导）
$\Delta u = u_{xx} + u_{yy} = 0$Laplace方程（解析函数的实部、虚部均为调和函数）
$v(x,y) = \int u_x\,dy + g(x),\quad g'(x) = -u_y - \frac{\partial}{\partial x}\!\int u_x\,dy$由实部 $u$ 求共轭调和函数 $v$ 的方法

第二章（续）· 初等函数

公式说明
$e^z = e^x(\cos y + i\sin y)$复指数函数：模由实部定，辐角由虚部定
$e^{z+2\pi i} = e^z$$2\pi i$ 是基本周期（最关键的性质）
$\log z = \ln|z| + i\arg z$多值对数；$\Log z = \ln|z| + i\Arg z$（主值，$\Arg z\in(-\pi,\pi]$）
$z^a = e^{a\log z}$幂函数：整数$a$→单值；有理数$a=p/q$→$q$值；其他→无穷多值
$\sin z = \dfrac{e^{iz}-e^{-iz}}{2i},\quad \cos z = \dfrac{e^{iz}+e^{-iz}}{2}$复三角函数（定义）
$\sin(x+iy) = \sin x\cosh y + i\cos x\sinh y$正弦的实虚部分离（$\cos$同理）
$\cosh z = \dfrac{e^z+e^{-z}}{2},\quad \sinh z = \dfrac{e^z-e^{-z}}{2}$双曲函数
$\sin z = 0 \Leftrightarrow z = k\pi$复三角函数零点与实数域完全相同

第三章 · 复变函数的积分 ★★★（分值最重）

公式说明
$\displaystyle\int_C f(z)dz = \int_\alpha^\beta f(z(t))z'(t)dt$参数化计算公式
$\displaystyle\oint_{|z-a|=r} (z-a)^n dz = \begin{cases} 2\pi i, & n=-1 \\ 0, & n\neq -1 \end{cases}$★ 基础公式：整个留数理论的出发点
$\left|\int_C f(z)dz\right| \le M\cdot L$积分模的估计（$M$为$|f|$上界，$L$为弧长）
若$f$在单连通域$D$内解析, 则$\oint_C f(z)dz = 0$★ Cauchy 积分定理
$\displaystyle f(a) = \frac{1}{2\pi i}\oint_C \frac{f(z)}{z-a}dz$★ Cauchy 积分公式（边界值决定内部值）
$\displaystyle f^{(n)}(a) = \frac{n!}{2\pi i}\oint_C \frac{f(z)}{(z-a)^{n+1}}dz$★ 高阶导数公式
$|f^{(n)}(a)| \le \dfrac{n! M}{R^n}$Cauchy 估计（推导 Liouville 定理）
有界整函数必为常数Liouville 定理
$n$次多项式恰有$n$个复根（计重数）代数基本定理
解析函数$|f|$的最大值在边界上取到最大模原理

第四章 · 幂级数 & 第五章 · 洛朗展式与孤立奇点

公式说明
$\displaystyle R = \lim_{n\to\infty}\left|\frac{a_n}{a_{n+1}}\right| = \frac{1}{\limsup\sqrt[n]{|a_n|}}$幂级数收敛半径
$\displaystyle f(z) = \sum_{n=0}^\infty \frac{f^{(n)}(z_0)}{n!}(z-z_0)^n$Taylor 展开（收敛半径 = 到最近奇点的距离）
$\displaystyle f(z) = \sum_{n=-\infty}^\infty a_n(z-z_0)^n$★ Laurent 展开（圆环域内，含负幂项）
$\displaystyle a_n = \frac{1}{2\pi i}\oint_C \frac{f(\zeta)}{(\zeta-z_0)^{n+1}}d\zeta$Laurent 系数公式（$n$可为负）
常用展开：$\frac{1}{1-z}=\sum_{n=0}^\infty z^n\;(|z|<1)$几何级数——所有间接展开的基础
可去奇点：无负幂项，$\lim f(z)$有限孤立奇点三分法（根据 Laurent 展开的主要部分）
$m$阶极点：最高负幂为$(z-z_0)^{-m}$，$\lim f(z)=\infty$
本性奇点：无穷多负幂项，$\lim f(z)$不存在

第六章 · 留数 ★★★（计算积分的主力章节）

公式说明
$\Res[f, z_0] = a_{-1}$（Laurent 展开中 $(z-z_0)^{-1}$ 的系数）留数定义
一阶极点：$\Res[f, z_0] = \lim_{z\to z_0}(z-z_0)f(z)$一阶极点留数公式
$f=\frac{p}{q}$且$p(z_0)\neq0, q(z_0)=0, q'(z_0)\neq0$：$\Res=\frac{p(z_0)}{q'(z_0)}$有理函数一阶极点留数（最常用）
$m$阶极点：$\Res = \frac{1}{(m-1)!}\lim_{z\to z_0}\frac{d^{m-1}}{dz^{m-1}}[(z-z_0)^m f(z)]$$m$阶极点留数公式
$\displaystyle\oint_C f(z)dz = 2\pi i\sum_{k=1}^n \Res[f, z_k]$★ 留数定理
类型一：$\int_0^{2\pi}R(\cos\theta,\sin\theta)d\theta$令 $z=e^{i\theta}$，化为单位圆周积分
类型二：$\int_{-\infty}^\infty \frac{p(x)}{q(x)}dx$（$\deg q \ge \deg p + 2$）上半平面半圆围道，$q$在实轴上无零点
类型三：$\int_{-\infty}^\infty f(x)e^{iax}dx$（$a>0$）Jordan引理 + 上半平面围道
$\frac{1}{2\pi i}\oint_C\frac{f'(z)}{f(z)}dz = N - P$辐角原理（零点数 $-$ 极点数）
若在 $C$ 上 $|f(z)-g(z)| < |g(z)|$，则 $f$ 与 $g$ 在 $C$ 内零点数相同Rouché 定理

第七章 · 共形映射（§7.1 – §7.3）

公式说明
伸缩率 $|f'(z_0)|$，旋转角 $\arg f'(z_0)$导数的几何意义
若 $f'(z_0)\neq 0$ 且 $f$ 为单射，则 $f$ 是共形映射共形映射的充分条件
$\displaystyle w = \frac{az+b}{cz+d},\; ad-bc\neq 0$分式线性变换（Möbius变换）
保圆性、保对称点性、交比不变Möbius变换的三大核心性质
$\displaystyle (z_1,z_2;z_3,z_4) = \frac{(z_3-z_1)(z_4-z_2)}{(z_3-z_2)(z_4-z_1)}$交比定义（Möbius变换下不变）
$w = z^n$：将角域 $\{0<\arg z<\alpha\}$ 映为 $\{0<\arg w幂函数映射（展开/压缩角度）
$w = e^z$：将水平带形映为角域；$|w|=e^x$，$\arg w = y$指数函数映射
$w = \frac{1}{2}(z+\frac{1}{z})$：将 $|z|>1$ 映为 $\CC\setminus[-1,1]$儒科夫斯基变换（机翼剖面）
上半平面 → 单位圆：$w = e^{i\theta}\frac{z-\alpha}{z-\bar{\alpha}}$（$\Im\alpha>0$）最重要的标准映射之一

第二部分：9天学习计划

每天建议 4–6 小时有效学习时间。标注 ★ 的为重点日。

计划设计原则
  • 前三章是基础工具，需要熟练掌握运算方法（而非死记硬背）。
  • 第三章（复积分）是理论核心和考试重点，分配两天。
  • 第五、六章是计算工具（留数计算实积分是必考题），各分配一天。
  • 第七章（共形映射）只考前3节，分配一天。第八章（解析延拓）浏览即可。
  • 最后两天用于综合复习和真题/模拟训练。

第1天：第一章 复数与复变函数

学习目标
  • 熟练掌握复数的四则运算、模和共轭的性质
  • 能在直角坐标和指数形式之间灵活转换
  • 掌握棣莫弗公式和$n$次方根的计算
  • 理解区域、闭区域、单连通/多连通的概念

重点练习：用指数形式做乘除和幂运算；利用 $|z|^2=z\bar{z}$ 简化表达式；三角不等式的应用；判断点集的类型和连通性。

自查：能否在30秒内将 $1+\sqrt{3}i$ 化为指数形式？能否求出 $(1+i)^8$ 而不展开二项式？

★ 第2天：第二章 解析函数（含初等函数）

学习目标
  • 理解复可导与实可导的本质区别（所有方向极限一致）
  • 牢记 C-R 方程并熟练用于判断解析性
  • 掌握由实部求虚部的方法（C-R方程积分）
  • 理解调和函数和共轭调和函数的概念

重点练习：用C-R方程判断函数是否解析；给定$u$求$v$（和反过来）；验证调和性。

易错点：可导$\neq$解析（解析需要邻域内处处可导，$|z|^2$在$z=0$可导但不解析是经典反例）。C-R方程只是必要条件，需配合可微性才是充分条件。

第3天：第三章 复变函数的积分（上）

学习目标
  • 掌握参数化计算复积分的方法
  • 牢记 $\oint (z-a)^n dz$ 的结果
  • 理解 Cauchy 积分定理的条件和结论
  • 熟练运用 Cauchy 积分公式计算围道积分

重点练习：参数化法计算简单积分；用 Cauchy 积分公式计算 $\oint \frac{f(z)}{z-a}dz$ 型积分。

易错点：可导≠解析（解析需要邻域内处处可导）；C-R 方程只是必要条件，需配合可微性才是充分条件。

★★ 第4天：第三章 复变函数的积分（下）—— 高阶导数与深刻推论

学习目标
  • 掌握参数化计算复积分的方法
  • 牢记 $\oint (z-a)^n dz$ 的结果（$n=-1$是唯一例外）
  • 理解 Cauchy 估计 → Liouville 定理 → 代数基本定理的逻辑链
  • 掌握奇点的等价判别法（极限判别）

重点练习：参数化法计算简单积分；用几何级数 $\frac{1}{1-z}=\sum z^n$ 做间接展开；求幂级数的收敛半径。

自我检测：$\oint_{|z|=3}\frac{\cos z}{(z-\pi)^3}dz = ?$（答案：$\pi i \sin\pi = 0$）

★ 第5天：第四章 解析函数的幂级数表示

学习目标
  • 掌握奇点的等价判别法（极限判别）
  • 理解 Weierstrass 定理（本性奇点的稠密性）（可去/极点/本性）
  • 理解收敛半径 = 到最近奇点的距离

重点练习：用几何级数 $\frac{1}{1-z}=\sum z^n$ 做间接展开；求幂级数的收敛半径。

自我检测：$\sum_{n=1}^{\infty} \frac{n}{2^n} z^n$ 的收敛半径是多少？（答案：$R=2$）

★ 第6天：第五章 洛朗展式与孤立奇点

学习目标
  • 掌握奇点的等价判别法（极限判别）
  • 理解 Weierstrass 定理（本性奇点的稠密性）（可去/极点/本性）
  • 掌握奇点的等价判别法（极限判别）
  • 理解 Weierstrass 定理（本性奇点的稠密性）

重点练习：用几何级数 $\frac{1}{1-z}=\sum z^n$ 做间接展开；在不同圆环域内展开同一函数；判断奇点类型（数负幂项个数是最直接的方法）。

易错点：Laurent展开依赖圆环域——同一个函数在不同区域展开结果不同。展开前先明确$z_0$和收敛域。

★★ 第7天：第六章 留数

学习目标
  • 熟练计算各类奇点的留数（一阶极点、$m$阶极点、本性奇点）
  • 用留数定理计算围道积分
  • （必考）用留数定理计算三类实积分
  • 理解辐角原理和 Rouché 定理的思想

重点练习：一阶极点留数的快速计算（有理函数优先用 $p/q'$ 公式）；三类实积分各做至少2道完整例题；总结围道选择策略。

实积分围道速查：$\int_0^{2\pi} R(\cos\theta,\sin\theta)d\theta$ → 单位圆；$\int_{-\infty}^\infty$ 有理函数 → 上半平面半圆；$\int_{-\infty}^\infty f(x)e^{iax}dx$ → 同上 + Jordan引理。

第8天：第七章（§7.1–§7.3）

学习目标
  • 理解导数的几何意义（伸缩率 + 旋转角）
  • 掌握分式线性变换的三大性质和交比公式
  • 能用初等函数（$z^n$, $e^z$, $\frac{1}{2}(z+1/z)$）构造简单共形映射

重点练习：用交比公式求Möbius变换；用幂函数/指数函数构造将给定区域映为单位圆或上半平面的映射。

不必花大力气：Riemann映射定理（了解结论即可）；Schwarz-Christoffel公式（掌握思路，具体积分一般不考）。

★★ 第9天：综合复习与模拟训练

学习目标
  • 用公式速查表快速过一遍全课程公式
  • 重做各章经典例题（特别是积分和级数计算题）
  • 做1–2套往年真题或模拟题，严格计时
  • 整理错题和易混淆概念

推荐复习顺序：先过公式表 → 各章FAQ/误区 → 翻看自己做错的题 → 再掐时间做一套完整的模拟卷。

第三部分：备考建议

一、考试题型与分值分布（典型估计）

题型分值主要覆盖章节
填空/选择题20–30%Ch1–Ch3 基础概念和计算
计算题40–50%Ch4（积分计算）、Ch6（留数 + 实积分）
证明题15–25%Ch2（C-R方程）、Ch4（Cauchy相关推论）
综合/应用题10–15%Ch5（展开与奇点分类）、Ch7（映射构造）

二、各章节的考试权重与策略

按考试重要性排序
  1. Ch4 + Ch6（★★★★★）：积分计算是绝对核心。Cauchy积分公式、高阶导数公式、留数定理、三类实积分——这些是计算题的"常规武器"。两道大题至少有一道来自这里。
  2. Ch5（★★★★）：Taylor/Laurent展开和奇点分类既是独立考点也是Ch6留数计算的前提。间接展开法（利用已知展开）是最实用的技能。
  3. Ch2（★★★★）：C-R方程是证明题的高频考点。典型考法：给定$u$或$v$，用C-R求另一个并构造$f(z)$；或判断函数在何处解析。
  4. Ch3（★★★）：初等函数的计算和性质（周期性、多值性）出现在填空/选择中概率高。$e^z$和$\Log z$是最常考的两个。
  5. Ch1（★★）：基础运算。不会单独出大题，但所有题目都依赖这些技能。
  6. Ch7 §§1-3（★★）：Möbius变换和初等函数映射。通常出一道小题或大题的起始步骤。

三、考场策略

时间分配建议（以120分钟为例）
  • 0–25分钟：快速完成填空/选择题。不会的果断跳过，不要在一道小题上纠结超过3分钟。
  • 25–85分钟：主攻计算题（通常2–3道）。留数计算实积分是最"稳"的得分点——只要你正确识别类型、选对围道、算对留数，就能拿满分。
  • 85–110分钟：完成证明题和综合题。C-R相关证明优先做（套路固定）。
  • 110–120分钟：回头检查。重点检查：留数计算有无漏奇点、实积分的围道选择是否正确、分式线性变换的条件$ad-bc\neq0$。

四、高频易错点清单

考前必查
  1. C-R方程是必要非充分——偏导数连续 + C-R 才是充要条件。
  2. 可导 ≠ 解析——解析要求在邻域内处处可导。
  3. $\Log(z_1z_2) \neq \Log z_1 + \Log z_2$——主值的运算可能差 $2\pi i$。
  4. Cauchy积分定理要求被积函数在围道内处处解析——内部有奇点时定理不适用，需要用Cauchy积分公式或留数定理。
  5. 高阶导数公式的分母是 $(z-a)^{n+1}$，系数是 $n!/2\pi i$——注意 $n$ 和指数 $n+1$ 的对应关系。
  6. Laurent展开依赖圆环域——同一个函数在不同区域展开结果不同。
  7. 留数计算时别忘了因子 $2\pi i$——围道积分 = $2\pi i\times\sum$留数（不是简单的留数之和）。
  8. 实积分围道：确认上半平面内的所有奇点——只计入围道内部的奇点，围道外的奇点不参与求和。
  9. 判断奇点类型时注意"可去"和"极点"的区别——$\sin z/z$ 在 $z=0$ 是可去（不是极点！），$\lim_{z\to 0}\sin z/z = 1$。
  10. Möbius变换：$ad-bc \neq 0$ 是必须的条件——如果忘了验证这一点可能被判错。

五、最后叮嘱

考前24小时做什么
  1. 不要再学新内容。把本附录的公式速查表过一遍。
  2. 重做之前做错的2–3道题（而不是大量刷新题）。
  3. 确认自己能用留数定理从零开始完整计算 $\int_{-\infty}^\infty \frac{dx}{x^2+1} = \pi$（这道题基本涵盖了留数计算实积分的全套流程）。
  4. 检查计算器、铅笔、准考证等物品。
  5. 保证充足睡眠——复变函数考试需要的清醒头脑比多记一个公式更重要。
