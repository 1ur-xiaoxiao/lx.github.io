C. 分布间的重要关系

分布的可加性
  • 二项分布：$X \sim B(n,p)$，$Y \sim B(m,p)$，$X,Y$ 独立 $\Rightarrow$ $X+Y \sim B(n+m, p)$
  • 泊松分布：$X \sim P(\lambda_1)$，$Y \sim P(\lambda_2)$，$X,Y$ 独立 $\Rightarrow$ $X+Y \sim P(\lambda_1+\lambda_2)$
  • 正态分布：$X \sim N(\mu_1,\sigma_1^2)$，$Y \sim N(\mu_2,\sigma_2^2)$，$X,Y$ 独立 $\Rightarrow$ $X+Y \sim N(\mu_1+\mu_2,\sigma_1^2+\sigma_2^2)$
  • 指数分布：$X \sim Exp(\lambda)$，$Y \sim Exp(\lambda)$，$X,Y$ 独立 $\Rightarrow$ $X+Y \sim Gamma(2,\lambda)$（不再是指数分布）
近似关系
  • 泊松近似：当 $n$ 大（$\geq 20$）、$p$ 小（$\leq 0.05$）、$np$ 适中时，$B(n,p) \approx P(np)$
  • 正态近似：当 $n$ 大时，$B(n,p) \approx N(np, np(1-p))$（中心极限定理）
  • 超几何→二项：当 $N$ 很大、$n$ 较小时，$H(N,M,n) \approx B(n, M/N)$
正态分布的标准化
$$X \sim N(\mu,\sigma^2) \implies Z = \frac{X-\mu}{\sigma} \sim N(0,1)$$ $$P(a \lt X \lt b) = \Phi\!\left(\frac{b-\mu}{\sigma}\right) - \Phi\!\left(\frac{a-\mu}{\sigma}\right)$$
正态分布的线性变换
$$X \sim N(\mu,\sigma^2) \implies aX+b \sim N(a\mu+b,\; a^2\sigma^2)$$

D. 核心公式速查

第一章 · 概率

概率公式
  • 加法公式：$P(A\cup B) = P(A)+P(B)-P(AB)$
  • 条件概率：$P(B|A) = \dfrac{P(AB)}{P(A)}$
  • 乘法公式：$P(AB) = P(A)P(B|A)$
  • 全概率公式：$P(A) = \displaystyle\sum_{i=1}^n P(B_i)P(A|B_i)$
  • 贝叶斯公式：$P(B_j|A) = \dfrac{P(B_j)P(A|B_j)}{\displaystyle\sum_{i=1}^n P(B_i)P(A|B_i)}$
  • 德摩根律：$\overline{A\cup B}=\bar{A}\cap\bar{B}$，$\overline{A\cap B}=\bar{A}\cup\bar{B}$

第三章 · 多维分布

多维公式
  • 边缘密度：$f_X(x) = \displaystyle\int_{-\infty}^{+\infty} f(x,y)\, \dd y$
  • 条件密度：$f_{X|Y}(x|y) = \dfrac{f(x,y)}{f_Y(y)}$
  • 独立性：$f(x,y)=f_X(x)f_Y(y)$
  • 卷积公式：$f_{X+Y}(z) = \displaystyle\int_{-\infty}^{+\infty} f_X(x)f_Y(z-x)\, \dd x$
  • 极值分布（独立）：$F_{\max}(z)=F_X(z)F_Y(z)$，$F_{\min}(z)=1-[1-F_X(z)][1-F_Y(z)]$

第四章 · 大数定律与中心极限定理

特征函数
  • 定义：$\varphi_X(t) = \EE(e^{itX})$（总是存在）
  • 独立和：$X,Y$ 独立 $\Rightarrow$ $\varphi_{X+Y}(t) = \varphi_X(t)\varphi_Y(t)$
  • 矩公式：$\varphi_X^{(k)}(0) = i^k\EE(X^k)$
  • 唯一性：特征函数唯一确定分布
  • 列维连续性定理：$X_n \xrightarrow{d} X$ $\Longleftrightarrow$ $\varphi_{X_n}(t) \to \varphi_X(t)$ 逐点收敛
  • 正态 $N(\mu,\sigma^2)$：$\varphi(t) = \exp\{i\mu t - \frac{1}{2}\sigma^2 t^2\}$
  • 标准正态 $N(0,1)$：$\varphi(t) = e^{-t^2/2}$
  • 二项 $B(n,p)$：$\varphi(t) = (1 - p + p e^{it})^n$
  • 泊松 $P(\lambda)$：$\varphi(t) = \exp\{\lambda(e^{it} - 1)\}$
大数定律与中心极限定理
  • 切比雪夫不等式：$P(|X-\mu|\geq\varepsilon) \leq \dfrac{\sigma^2}{\varepsilon^2}$
  • 辛钦大数定律：i.i.d.、$E(X_i)=\mu$ 存在 $\Rightarrow$ $\bar{X}_n \xrightarrow{P} \mu$
  • 伯努利大数定律：频率 $\dfrac{n_A}{n} \xrightarrow{P}$ 概率 $p$
  • 中心极限定理：i.i.d.、$E(X_i)=\mu$，$\Var(X_i)=\sigma^2$ $\Rightarrow$ $\dfrac{\bar{X}_n - \mu}{\sigma/\sqrt{n}} \xrightarrow{d} N(0,1)$
  • 棣莫弗-拉普拉斯：$X_n \sim B(n,p)$ $\Rightarrow$ $\dfrac{X_n - np}{\sqrt{np(1-p)}} \xrightarrow{d} N(0,1)$

数字特征公式（贯穿各章）

期望、方差与协方差
  • 期望线性：$E(aX+bY+c) = aE(X)+bE(Y)+c$
  • 方差公式：$\Var(X) = E(X^2) - [E(X)]^2$
  • 方差加法：$\Var(X+Y) = \Var(X)+\Var(Y)+2\Cov(X,Y)$
  • 独立时方差加法：$\Var(X+Y) = \Var(X)+\Var(Y)$
  • 协方差：$\Cov(X,Y) = E(XY)-E(X)E(Y)$
  • 相关系数：$\rho_{XY} = \dfrac{\Cov(X,Y)}{\sqrt{\Var(X)\Var(Y)}}$

E. 标准正态分布表使用说明

$\Phi(x)$ 的含义
$$\Phi(x) = P(Z \leq x) = \int_{-\infty}^{x} \frac{1}{\sqrt{2\pi}} e^{-t^2/2}\, \dd t$$

即标准正态分布的分布函数在 $x$ 处的值。

查表技巧
  • 对称性：$\Phi(-x) = 1 - \Phi(x)$
  • 特殊值：$\Phi(0) = 0.5$
  • 求 $P(a\lt Z\lt b)$：$P(a\lt Z\lt b) = \Phi(b) - \Phi(a)$
  • 求 $P(|Z|>c)$：$P(|Z|>c) = 2[1-\Phi(c)]$
  • 求 $P(|Z|\lt c)$：$P(|Z|\lt c) = 2\Phi(c)-1$
一般正态 → 标准正态

若 $X \sim N(\mu, \sigma^2)$，标准化步骤：

  1. 令 $Z = \dfrac{X-\mu}{\sigma}$
  2. $P(a \lt X \lt b) = P\!\left(\dfrac{a-\mu}{\sigma} \lt Z \lt \dfrac{b-\mu}{\sigma}\right) = \Phi\!\left(\dfrac{b-\mu}{\sigma}\right) - \Phi\!\left(\dfrac{a-\mu}{\sigma}\right)$
常用 $\Phi$ 值
$x$ 0.00 0.674 1.00 1.645 1.960 2.576 3.00
$\Phi(x)$ 0.5000 0.7500 0.8413 0.9500 0.9750 0.9950 0.9987

这些值对应常用置信水平，记住可以加速解题。

F. 各章解题方法总结

第一章 · 概率计算
  • "至少"问题 → 对立事件
  • "恰好"问题 → 精确列举
  • "分情况"问题 → 全概率公式
  • "由结果推原因" → 贝叶斯公式
  • 独立性 → 验证 $P(AB)=P(A)P(B)$
第二章 · 随机变量
  • 离散型 → 列分布律，验证 $\sum p_k=1$
  • 连续型 → 先求分布函数，再求导得密度
  • 正态分布 → 第一步标准化 $Z=(X-\mu)/\sigma$
  • 函数的分布 → 单调用公式法，非单调用分布函数法
第三章 · 多维分布
  • 求常数 → 归一性 $\iint f(x,y)\, \dd x\, \dd y=1$
  • 求边缘 → "积掉"另一个变量
  • 判断独立 → 验证 $f(x,y)=f_X(x)f_Y(y)$
  • $Z=X+Y$ → 卷积公式
  • 极值分布 → $F_{\max}=F_XF_Y$，$F_{\min}=1-(1-F_X)(1-F_Y)$
第四章 · 大数定律与中心极限定理
  • 求独立和的分布 → 特征函数乘积 + 唯一性定理
  • 证明依分布收敛 → 特征函数 + 列维连续性定理
  • 样本均值估计 → 辛钦大数定律
  • 频率估计概率 → 伯努利大数定律
  • 大量独立变量之和 → 中心极限定理
  • 二项分布正态近似 → 连续性修正

G. 易错点与重要结论汇总

本节精炼总结概率论中最易混淆的概念和考试高频结论，帮助考前快速梳理薄弱环节。

G1. 边际分布与独立性

联合 → 边际（单向）

由高维联合分布可以唯一确定低维边际分布，但反之不成立。不同的联合分布可以有完全相同的边际分布。

经典反例：两个不同的二维正态分布（相关系数不同）可以有相同的一维正态边际。这说明边际分布不能唯一确定联合分布，除非额外假定独立性。

特殊分布的边际
  • 多项分布：边际分布仍为低维的多项分布或二项分布
  • 二维正态：边际分布为一维正态；但两个一维正态不一定构成二维正态——还需 $X,Y$ 的任意线性组合服从正态（或 $X,Y$ 相互独立）
  • 多维正态的等价刻画：$(X,Y)$ 服从二维正态 $\iff$ 对任意 $a,b \in \RR$，$aX+bY$ 服从一维正态
独立性的等价条件（$X,Y$ 独立 $\Longleftrightarrow$）
  1. 联合分布函数拆分：$F(x,y) = F_X(x) \cdot F_Y(y)$（最基础）
  2. 联合密度/分布律拆分（必考）：$f(x,y) = f_X(x) \cdot f_Y(y)$ 或 $P(X=x_i, Y=y_j) = P(X=x_i)P(Y=y_j)$
  3. 可分离变量判据（秒杀法）：若 $f(x,y)$ 能写成 $g(x)h(y)$ 的形式（不含交叉项 $xy$、变量彻底分家），且定义域可分离（为矩形区域），则 $X,Y$ 独立
  4. 概率直接拆分：$P(X \leq a, Y \leq b) = P(X \leq a)P(Y \leq b)$，以及 $P(X>a, Y>b) = P(X>a)P(Y>b)$ 等——不需要算二重积分
  5. 期望拆分：$\EE(XY) = \EE(X)\EE(Y)$
  6. 方差简化：$\Var(X \pm Y) = \Var(X) + \Var(Y)$（协方差 $=0$，相关系数 $=0$）
  7. 事件独立性：对任意事件 $\{X \in A\}$ 与 $\{Y \in B\}$ 独立

G2. 可分离变量判据的细节（极易出错）

仅函数可分离不够——定义域也必须可分离！

充分条件：$f(x,y) = g(x)h(y)$ 且定义域为矩形 $D = \{a \lt x \lt b, c \lt y \lt d\}$ $\Rightarrow$ $X,Y$ 独立。

反例：$f(x,y) = 2$，定义域为 $0 \lt x \lt y \lt 1$（三角形区域）。虽然密度函数是常数（可写成 $g(x)h(y)$ 形式），但定义域不是矩形（$x$ 和 $y$ 的范围相互约束），因此 $X,Y$ 不独立。

判断口诀：函数能分家且区域是矩形，则独立；函数分不开或区域非矩形（三角/圆/菱形等），则不独立。

G3. 分布的可加性（在独立条件下）

分布族可加条件和分布注意事项
二项分布 $X \sim B(n_1,p)$，$Y \sim B(n_2,p)$，独立 $X+Y \sim B(n_1+n_2, p)$ 必须同 $p$，$p$ 不同不能直接加
泊松分布 $X \sim P(\lambda_1)$，$Y \sim P(\lambda_2)$，独立 $X+Y \sim P(\lambda_1+\lambda_2)$ 无条件，参数直接相加
正态分布 $X \sim N(\mu_1,\sigma_1^2)$，$Y \sim N(\mu_2,\sigma_2^2)$，独立 $X+Y \sim N(\mu_1+\mu_2, \sigma_1^2+\sigma_2^2)$ 即使不独立，线性组合仍正态
指数分布 $X \sim Exp(\lambda)$，$Y \sim Exp(\lambda)$，独立 $X+Y \sim Gamma(2,\lambda)$ 不再是指数分布！常见陷阱
$\chi^2$ 分布 $X \sim \chi^2(n_1)$，$Y \sim \chi^2(n_2)$，独立 $X+Y \sim \chi^2(n_1+n_2)$ 自由度直接相加

G4. 独立与不相关（最易混淆的概念对）

核心结论
关系结论理由
独立 ⇒ 不相关一定成立独立 $\Rightarrow$ $\EE(XY)=\EE(X)\EE(Y)$ $\Rightarrow$ $\Cov=0$
不相关 ⇒ 独立一般不成立不相关只保证无线性关系，可能存在非线性关联
相关 ⇒ 不独立一定成立逆否命题——相关系数不为零必有线性依赖
什么情况下不相关 ⇔ 独立？
  • 二维正态分布：不相关与独立等价（这是正态分布的特殊性质，其他分布不满足）
  • 退化分布：二者完全等价
  • 两点分布：二者完全等价
  • 一般连续分布：不相关 $\neq$ 独立（经典反例：$X \sim N(0,1)$，$Y = X^2$，则 $\Cov(X,Y)=0$ 但显然不独立）
协方差的本质

协方差是一种期望：$\Cov(X,Y) = \EE[(X-\EE X)(Y-\EE Y)] = \EE(XY) - \EE(X)\EE(Y)$。

$\Cov=0$ 只说明没有线性关联，不能排除非线性关联（如 $Y=X^2$）。

$\Cov>0$ $\iff$ 正线性关联；$\Cov\lt 0$ $\iff$ 负线性关联。

G5. 方差与协方差计算易错点

方差公式对比（极易用混）
场景公式备注
一般情况$\Var(X \pm Y) = \Var(X) + \Var(Y) \pm 2\Cov(X,Y)$含协方差项
$X,Y$ 独立$\Var(X \pm Y) = \Var(X) + \Var(Y)$协方差=0，加号不变
线性组合$\Var(aX+bY) = a^2\Var(X) + b^2\Var(Y) + 2ab\Cov(X,Y)$系数平方！
$X,Y$ 独立，线性组合$\Var(aX+bY) = a^2\Var(X) + b^2\Var(Y)$$a^2$ 和 $b^2$ 不是 $a$ 和 $b$！

G6. 特征函数核心性质

特征函数 vs 矩母函数
对比特征函数 $\varphi(t)$矩母函数 $M(t)$
定义$\EE(e^{itX})$$\EE(e^{tX})$
存在性总是存在（$|e^{itX}|=1$）不一定存在（如柯西分布）
独立和乘积 $\varphi_X\varphi_Y$乘积 $M_X M_Y$
求矩$\varphi^{(k)}(0) = i^k \EE(X^k)$$M^{(k)}(0) = \EE(X^k)$
一致连续性在 $\RR$ 上一致连续不一定
特征函数的三大核心结论
  1. 唯一性定理：分布函数由其特征函数唯一确定——特征函数是分布的"指纹"
  2. 逆转公式：由特征函数可反演出密度函数（傅里叶逆变换），保证了一一对应
  3. 列维连续性定理：依分布收敛 $\Longleftrightarrow$ 特征函数逐点收敛——证明一切极限定理的枢纽

G7. 概率计算中的易错点

事件关系与概率——不能互相推出
关于事件的判断能否推出概率关系反例/说明
$A \subset B$$\Rightarrow$ $P(A) \leq P(B)$单调性，成立
$P(A) \leq P(B)$$\nRightarrow$ $A \subset B$$A$ 和 $B$ 可能是不同的事件，概率大小不反映包含关系
$A \cap B = \varnothing$$\Rightarrow$ $P(AB)=0$互斥则交概率为零
$P(AB)=0$$\nRightarrow$ $A \cap B = \varnothing$概率为 0 不代表不可能事件（连续型单点概率为 0）
$P(A)=1$$\nRightarrow$ $A = \Omega$概率为 1 不代表必然事件（"几乎必然"≠"必然"）
$P(A)=0$$\nRightarrow$ $A = \varnothing$概率为 0 不代表不可能事件
条件概率中的常见误区
  • 误区一：$P(A|B) = P(B|A)$。❌ 除非 $P(A)=P(B)$，一般不成立。正确关系是贝叶斯公式。
  • 误区二：$P(A|B) + P(\bar{A}|B) = 1$。✅ 成立（给定 $B$ 的条件下，$A$ 和 $\bar{A}$ 仍为对立事件）
  • 误区三：$P(A|B) + P(A|\bar{B}) = 1$。❌ 一般不成立！除非 $A$ 与 $B$ 独立且 $P(A)=0.5$。
  • 误区四：条件概率 $P(\cdot|B)$ 本身是一个合法的概率测度，满足概率的三条公理。
全概率公式的使用条件——完备事件组

全概率公式 $P(A) = \sum_i P(B_i)P(A|B_i)$ 要求 $\{B_i\}$ 构成样本空间的一个划分（完备事件组）：

  • $B_i \cap B_j = \varnothing$（两两互斥）
  • $\bigcup_i B_i = \Omega$（覆盖全部样本空间）
  • $P(B_i) > 0$（条件概率有意义）

常见错误：遗漏某个 $B_i$ 导致 $\sum P(B_i) \neq 1$，或 $B_i$ 之间有重叠。

G8. 分布函数与密度函数的易混淆点

分布函数 $F(x)$ 的性质（必考）
  • $F(x)$ 单调不减
  • $F(-\infty)=0$，$F(+\infty)=1$
  • $F(x)$ 右连续：$F(x^+) = F(x)$
  • $P(a \lt X \leq b) = F(b) - F(a)$（注意左端是 $\lt$，右端是 $\leq$）
  • $P(X = a) = F(a) - F(a^-)$（分布函数在 $a$ 处的跳跃高度等于该点的概率）
密度函数 $f(x)$ 的两个常见误解
  • 误解一：$f(x) \leq 1$。❌ 密度可以大于 1！（如 $U(0, 0.1)$ 的密度 $f(x)=10$）
  • 误解二：$f(x) = P(X=x)$。❌ 对于连续型，$P(X=x)=0$ 恒成立。密度本身不是概率，密度乘以 $\dd x$ 近似小区间的概率。
  • 正确理解：$f(x)$ 是概率的"浓度"——$P(x \lt X \lt x+\dd x) \approx f(x)\,\dd x$。

G9. 常见分布的特殊性质汇总

分布特殊性质考试用途
几何分布无记忆性：$P(X>m+n|X>m)=P(X>n)$条件概率简化计算
指数分布无记忆性（连续型中唯一）可靠性理论、排队论
正态分布线性变换仍为正态；独立+不相关等价标准化是最核心的操作
泊松分布$E(X)=\Var(X)=\lambda$（均=方差）快速验证是否为泊松分布
均匀分布 $U(0,1)$若 $X \sim F$ 连续，则 $F(X) \sim U(0,1)$概率积分变换（模拟基础）
二项分布$n=1$ 退化为 0-1 分布和可加性使用时要求 $p$ 相同

G10. 收敛性与极限定理的易混淆点

四种收敛的关系
  • 几乎处处收敛 $\Rightarrow$ 依概率收敛 $\Rightarrow$ 依分布收敛
  • 依概率收敛 + 一致可积 $\Rightarrow$ $L^p$ 收敛
  • 依分布收敛于常数 $\Longleftrightarrow$ 依概率收敛于该常数
  • 大数定律给出的是依概率收敛
  • 中心极限定理给出的是依分布收敛
大数定律中 $n$ 的理解
  • 大数定律说的是 $n \to \infty$ 时的极限行为——不能用于有限样本的精确概率计算
  • "频率趋近概率"是定性结论，不能给出 $n=100$ 时频率与概率的差距
  • 要算具体概率或确定样本量 → 用中心极限定理
  • 柯西分布是 LLN 不成立的经典反例——因为期望不存在
CLT 使用注意事项
  • CLT 要求方差存在且非零——方差为 0（退化分布）或方差不存在（柯西分布）时 CLT 不适用
  • $n \geq 30$ 是经验规则，对于严重偏态的分布可能需要更大的 $n$
  • 棣莫弗-拉普拉斯定理要求连续性修正来提高近似精度，不做修正会扣分
  • 仅有独立还不够，CLT 还需要同分布（林德伯格-列维）或验证李雅普诺夫条件（不同分布时）

G11. 计算技巧与秒杀方法

期望计算的实用技巧
  • 非负随机变量的期望：$\EE(X) = \int_0^\infty [1-F(x)]\,\dd x = \int_0^\infty P(X>x)\,\dd x$（连续非负）
  • 离散非负：$\EE(X) = \sum_{k=0}^\infty P(X>k)$
  • 利用对称性：若分布关于 $\mu$ 对称，则 $\EE(X)=\mu$，且所有奇数阶中心矩为 0
  • 条件期望递推：$\EE(X) = \EE[\EE(X|Y)]$（重期望公式 / 塔性质）
相关系数的几何意义
  • $-1 \leq \rho \leq 1$
  • $\rho = 1$ $\iff$ $P(Y=aX+b)=1$（$a>0$，完全正线性相关）
  • $\rho = -1$ $\iff$ $P(Y=aX+b)=1$（$a\lt 0$，完全负线性相关）
  • $\rho = 0$ $\iff$ 不相关（无线性关联）
  • $|\rho|$ 越接近 1，线性关联越强
  • 注意：$\rho=0$ 只排除线性关系，$Y=X^2$ 的 $\rho$ 可以为零但显然存在确定性关系
