短期备考策略建议

备考时间分配（假设 3~5 天冲刺）
  • 第 1 天（40%）：第一章 + 第二章前半（至常见分布）。概率计算是全部基础，二项/泊松/正态是核心分布。
  • 第 2 天（30%）：第二章后半（随机变量函数的分布）+ 第三章前半（边缘分布与条件分布）。
  • 第 3 天（20%）：第三章后半（卷积、极值、独立性）+ 第四章（大数定律与 CLT）。
  • 考前半天（10%）：附录速查 + 模拟题自测 + 错题回顾。

知识重要性分级

等级 知识点 对应章节 常考题型 说 明
必考 全概率公式 + 贝叶斯公式 §1.3~1.4 大题第 1~2 道 全概率"由因求果"，贝叶斯"由果推因"。每年必出一道 8~10 分大题。
必考 正态分布标准化 + 概率计算 §2.4 填空 / 大题 标准化 $Z=\frac{X-\mu}{\sigma}$ 是核心操作，必须查表熟练。
必考 边缘分布 + 条件分布 + 独立性判定 §3.1~3.3 大题 给定联合密度求边缘、条件密度，判断独立性，是第三章最高频考题。
必考 中心极限定理应用 §4.3 填空 / 大题 大量独立同分布变量之和近似正态。常考"至少需要多少样本量"类问题。
高频 常见分布（二项/泊松/均匀/指数） §2.2~2.3 填空 / 选择 记住分布律/密度 + 期望 + 方差。指数分布无记忆性是高频考点。
高频 随机变量函数的分布 §2.5 计算题 单调函数用公式法，非单调用分布函数法。$Y=X^2$ 是经典套路。
高频 二维正态分布 §3.4 填空 不相关 ⇔ 独立（这是二维正态特有的性质）。记住五个参数的含义。
高频 协方差 + 相关系数 §3.3 计算题 $\Cov(X,Y)=E(XY)-E(X)E(Y)$，$\rho=0$ ⇔ 不相关。常与独立性联合考查。
常规 古典概型 + 几何概型 §1.2 填空 / 判断 排列组合基本功。几何概型本质是面积/长度比。
常规 事件独立性 §1.5 判断 / 填空 $P(AB)=P(A)P(B)$。注意：两两独立 ≠ 相互独立。
常规 卷积公式 + 极值分布 §3.5 计算题 $Z=X+Y$ 用卷积，$M=\max(X,Y)$、$N=\min(X,Y)$ 用分布函数公式。
常规 切比雪夫不等式 + 大数定律 §4.1~4.2 填空 / 判断 不等式用于估计概率界。辛钦大数定律是依概率收敛，不是几乎处处收敛。

答题技巧速记

各题型解题要点
  • 判断题：反例法是王道。遇到"一定""必然"等绝对化表述应高度警惕。
  • 填空题：先写公式再代数。二项分布忘记公式时可用 $n$ 次伯努利手推。
  • 全概率/贝叶斯大题：第一步明确划分 $\{B_i\}$（互斥完备），第二步画概率树。用表格法代替公式不易错。
  • 边缘/条件密度：积分上下限看支撑域！画积分区域图是保分关键。
  • 正态概率：先标准化 $Z=\frac{X-\mu}{\sigma}$，再查表。符号方向审清楚。
  • 中心极限定理：先写 $\frac{\bar{X}-\mu}{\sigma/\sqrt{n}}\xrightarrow{d}N(0,1)$，再代入求解。
考前 2 小时最佳行动清单
  1. 用附录速查表过一遍所有分布（离散 5 个 + 连续 3 个），确认记住记号、分布律/密度、期望、方差
  2. 背一遍贝叶斯公式的结构：分子 = 所求那一支的概率，分母 = 全概率
  3. 做 3 道题找回手感：1 道全概率 + 1 道边缘密度 + 1 道正态概率
  4. 默写：边缘密度公式、条件密度公式、卷积公式、协方差公式
  5. 检查计算器电池，确认带正态分布表
