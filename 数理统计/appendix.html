A.0 Gamma 分布与 Beta 分布速查

$\Gamma$ 函数

$$\Gamma(\alpha) = \int_0^\infty x^{\alpha-1} e^{-x} \,\dd x, \quad \alpha > 0$$

核心性质：$\Gamma(\alpha+1) = \alpha\Gamma(\alpha)$，$\Gamma(n) = (n-1)!$，$\Gamma(1/2) = \sqrt{\pi}$

Gamma 分布 $\text{Gamma}(\alpha, \lambda)$

密度：$f(x) = \frac{\lambda^\alpha}{\Gamma(\alpha)} x^{\alpha-1} e^{-\lambda x},\; x > 0$

$E[X] = \frac{\alpha}{\lambda}$，$\Var(X) = \frac{\alpha}{\lambda^2}$

  • $\text{Exp}(\lambda) = \text{Gamma}(1, \lambda)$
  • $\chi^2(n) = \text{Gamma}\!\left(\frac{n}{2}, \frac{1}{2}\right)$
  • 可加性：独立 $\text{Gamma}(\alpha_i, \lambda)$ 之和 $\sim \text{Gamma}(\sum\alpha_i, \lambda)$

$B$ 函数

$$B(\alpha, \beta) = \int_0^1 x^{\alpha-1}(1-x)^{\beta-1}\dd x = \frac{\Gamma(\alpha)\Gamma(\beta)}{\Gamma(\alpha+\beta)}$$

Beta 分布 $\text{Beta}(\alpha, \beta)$

密度：$f(x) = \frac{1}{B(\alpha,\beta)} x^{\alpha-1}(1-x)^{\beta-1},\; 0 < x < 1$

$E[X] = \frac{\alpha}{\alpha+\beta}$，$\Var(X) = \frac{\alpha\beta}{(\alpha+\beta)^2(\alpha+\beta+1)}$

  • $U(0,1) = \text{Beta}(1,1)$
  • $U(0,1)$ 样本的第 $k$ 次序统计量 $\sim \text{Beta}(k, n-k+1)$

A.1 三大抽样分布速查

分布 定义 期望 方差 关键性质
$\chi^2(n)$ $\sum_{i=1}^n Z_i^2$，$Z_i \overset{\text{iid}}{\sim} N(0,1)$ $n$ $2n$ 可加性：$X+Y \sim \chi^2(m+n)$
$t(n)$ $\frac{Z}{\sqrt{V/n}}$，$Z \sim N(0,1)$，$V \sim \chi^2(n)$，独立 $0\;(n>1)$ $\frac{n}{n-2}\;(n>2)$ 对称，$n\to\infty$ 趋于 $N(0,1)$
$F(m,n)$ $\frac{U/m}{V/n}$，$U \sim \chi^2(m)$，$V \sim \chi^2(n)$，独立 $\frac{n}{n-2}\;(n>2)$ （略） $1/F(m,n) \sim F(n,m)$
分位数关系

$t_{1-\alpha}(n) = -t_\alpha(n)$（对称性）

$F_{1-\alpha}(m,n) = \dfrac{1}{F_\alpha(n,m)}$（倒数性质）

$t_\alpha^2(n) = F_\alpha(1, n)$

A.2 正态总体抽样分布定理

设 $X_1, \dots, X_n \overset{\text{iid}}{\sim} N(\mu, \sigma^2)$，$\bar{X} = \frac{1}{n}\sum X_i$，$S^2 = \frac{1}{n-1}\sum(X_i - \bar{X})^2$。

编号 结论 条件
定理一 $\bar{X} \sim N\!\left(\mu, \dfrac{\sigma^2}{n}\right)$ —
定理二 $\dfrac{(n-1)S^2}{\sigma^2} \sim \chi^2(n-1)$ $\bar{X}$ 与 $S^2$ 独立
定理三 $\dfrac{\bar{X} - \mu}{S / \sqrt{n}} \sim t(n-1)$ 结合定理一、二
定理四（两总体） $\dfrac{(\bar{X}-\bar{Y})-(\mu_1-\mu_2)}{S_w\sqrt{\frac{1}{n_1}+\frac{1}{n_2}}} \sim t(n_1+n_2-2)$ $\sigma_1^2=\sigma_2^2$，$S_w^2 = \frac{(n_1-1)S_1^2+(n_2-1)S_2^2}{n_1+n_2-2}$

A.3 参数估计公式

MLE 求解步骤

  1. 写出似然函数：$L(\theta) = \prod_{i=1}^n f(x_i; \theta)$
  2. 取对数：$\ell(\theta) = \ln L(\theta) = \sum_{i=1}^n \ln f(x_i; \theta)$
  3. 求导令零：$\frac{\dd\ell}{\dd\theta} = 0$，解出 $\hat{\theta}$
  4. 验证二阶导 $\frac{\dd^2\ell}{\dd\theta^2} < 0$（确保是最大值）

常见分布的 MLE 结果

分布 参数 MLE
$N(\mu, \sigma^2)$ $\mu$ $\hat{\mu} = \bar{X}$
$N(\mu, \sigma^2)$ $\sigma^2$ $\hat{\sigma}^2 = \frac{1}{n}\sum(X_i - \bar{X})^2 = \frac{n-1}{n}S^2$
$\text{Exp}(\lambda)$ $\lambda$ $\hat{\lambda} = \frac{1}{\bar{X}}$
$\text{Poisson}(\lambda)$ $\lambda$ $\hat{\lambda} = \bar{X}$
$U(0, \theta)$ $\theta$ $\hat{\theta} = X_{(n)} = \max(X_1, \dots, X_n)$

正态总体置信区间速查

参数 枢轴量 置信区间 条件
$\mu$（$\sigma^2$ 已知） $Z = \frac{\bar{X}-\mu}{\sigma/\sqrt{n}}$ $\bar{X} \pm z_{\alpha/2}\frac{\sigma}{\sqrt{n}}$ —
$\mu$（$\sigma^2$ 未知） $t = \frac{\bar{X}-\mu}{S/\sqrt{n}}$ $\bar{X} \pm t_{\alpha/2}(n-1)\frac{S}{\sqrt{n}}$ —
$\sigma^2$ $\chi^2 = \frac{(n-1)S^2}{\sigma^2}$ $\left[\frac{(n-1)S^2}{\chi^2_{\alpha/2}(n-1)},\; \frac{(n-1)S^2}{\chi^2_{1-\alpha/2}(n-1)}\right]$ $\mu$ 未知
$\mu_1-\mu_2$ $t$（定理四） $(\bar{X}-\bar{Y}) \pm t_{\alpha/2}(n_1+n_2-2)\,S_w\sqrt{\tfrac{1}{n_1}+\tfrac{1}{n_2}}$ $\sigma_1^2=\sigma_2^2$
$\sigma_1^2/\sigma_2^2$ $F = S_1^2/S_2^2$ $\left[\frac{S_1^2/S_2^2}{F_{\alpha/2}(n_1-1,n_2-1)},\; \frac{S_1^2/S_2^2}{F_{1-\alpha/2}(n_1-1,n_2-1)}\right]$ $\mu_1, \mu_2$ 未知

A.4 假设检验速查表

正态总体四种检验

检验 $H_0$ 统计量 分布 拒绝域（双侧）
$Z$ 检验 $\mu = \mu_0$（$\sigma^2$ 已知） $Z = \frac{\bar{X}-\mu_0}{\sigma/\sqrt{n}}$ $N(0,1)$ $|Z| \geq z_{\alpha/2}$
$t$ 检验 $\mu = \mu_0$（$\sigma^2$ 未知） $t = \frac{\bar{X}-\mu_0}{S/\sqrt{n}}$ $t(n-1)$ $|t| \geq t_{\alpha/2}(n-1)$
$\chi^2$ 检验 $\sigma^2 = \sigma_0^2$ $\chi^2 = \frac{(n-1)S^2}{\sigma_0^2}$ $\chi^2(n-1)$ $\chi^2 \leq \chi^2_{1-\alpha/2}$ 或 $\chi^2 \geq \chi^2_{\alpha/2}$
$F$ 检验 $\sigma_1^2 = \sigma_2^2$ $F = \frac{S_1^2}{S_2^2}$ $F(n_1-1, n_2-1)$ $F \leq F_{1-\alpha/2}$ 或 $F \geq F_{\alpha/2}$
$p$ 值公式

以右侧检验为例（$H_1: \theta > \theta_0$），检验统计量观测值为 $T_{\text{obs}}$：

$$p = \PP(T \geq T_{\text{obs}} \mid H_0)$$

双侧检验：$p = 2 \cdot \PP(T \geq |T_{\text{obs}}| \mid H_0)$

拒绝 $H_0$ 当且仅当 $p \leq \alpha$。

A.5 回归分析公式

一元线性回归 $Y = \beta_0 + \beta_1 x + \varepsilon$

$$\hat{\beta}_1 = \frac{\sum_{i=1}^n (x_i - \bar{x})(y_i - \bar{y})}{\sum_{i=1}^n (x_i - \bar{x})^2} = \frac{S_{xy}}{S_{xx}}, \qquad \hat{\beta}_0 = \bar{y} - \hat{\beta}_1 \bar{x}$$
回归方程的显著性检验（$F$ 检验）

$H_0: \beta_1 = 0$（线性关系不显著）

$$F = \frac{S_{\text{回}}}{S_{\text{残}}/(n-2)} = \frac{\hat{\beta}_1^2 S_{xx}}{S_{\text{残}}/(n-2)} \sim F(1, n-2)$$

其中 $S_{\text{回}} = \hat{\beta}_1^2 S_{xx} = \hat{\beta}_1 S_{xy}$，$S_{\text{残}} = S_{yy} - \hat{\beta}_1 S_{xy}$。

决定系数
$$R^2 = \frac{S_{\text{回}}}{S_{\text{总}}} = 1 - \frac{S_{\text{残}}}{S_{\text{总}}} = \frac{\hat{\beta}_1 S_{xy}}{S_{yy}}$$

$R^2 \in [0, 1]$，越接近 1 表示回归方程的拟合效果越好。

A.6 方差分析

单因素方差分析

$a$ 个水平，第 $i$ 个水平 $n_i$ 次重复，$n = \sum n_i$，$y_{ij}$ 为第 $i$ 组第 $j$ 个观测。

$$S_T = S_A + S_E$$
平方和 公式 自由度
总平方和 $S_T$ $\sum_{i=1}^a \sum_{j=1}^{n_i} (y_{ij} - \bar{y})^2$ $n - 1$
组间平方和 $S_A$ $\sum_{i=1}^a n_i (\bar{y}_{i\cdot} - \bar{y})^2$ $a - 1$
组内平方和 $S_E$ $\sum_{i=1}^a \sum_{j=1}^{n_i} (y_{ij} - \bar{y}_{i\cdot})^2$ $n - a$
$F$ 统计量

$H_0: \mu_1 = \mu_2 = \cdots = \mu_a$（各组均值相等）

$$F = \frac{S_A/(a-1)}{S_E/(n-a)} = \frac{MS_A}{MS_E} \sim F(a-1,\; n-a)$$

拒绝域：$F \geq F_\alpha(a-1, n-a)$

A.7 贝叶斯公式

$$\pi(\theta \mid x) = \frac{L(\theta \mid x)\,\pi(\theta)}{\displaystyle\int_\Theta L(\theta \mid x)\,\pi(\theta)\,\dd\theta} \propto L(\theta \mid x)\,\pi(\theta)$$

常见共轭先验表

似然 共轭先验 后验参数更新
$N(\theta, \sigma^2)$ $N(\mu_0, \tau^2)$ $\frac{1}{\sigma_n^2} = \frac{n}{\sigma^2} + \frac{1}{\tau^2}$，$\mu_n = \sigma_n^2\!\left(\frac{n\bar{x}}{\sigma^2}+\frac{\mu_0}{\tau^2}\right)$
$\text{Bin}(n, \theta)$ $\text{Beta}(\alpha, \beta)$ $\alpha' = \alpha + x$，$\beta' = \beta + n - x$
$\text{Poisson}(\lambda)$ $\text{Gamma}(\alpha, \beta)$ $\alpha' = \alpha + \sum x_i$，$\beta' = \beta + n$
$\text{Exp}(\lambda)$ $\text{Gamma}(\alpha, \beta)$ $\alpha' = \alpha + n$，$\beta' = \beta + \sum x_i$
贝叶斯估计（平方损失）
$$\hat{\theta}_{\text{Bayes}} = \EE[\theta \mid x] = \int_\Theta \theta\,\pi(\theta \mid x)\,\dd\theta$$

MAP 估计：$\hat{\theta}_{\text{MAP}} = \arg\max_\theta\,\pi(\theta \mid x)$
