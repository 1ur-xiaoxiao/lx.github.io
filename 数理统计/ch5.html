第五章 · 贝叶斯统计简介

将先验知识与样本数据结合，用后验分布描述对参数的认知更新

5.1 频率学派 vs 贝叶斯学派

定义 · 两种统计学派

频率学派（Frequentist）：参数 $\theta$ 是固定的未知常数，不存在概率分布。样本 $X_1, \dots, X_n$ 是随机的，统计推断基于重复抽样的频率性质（如无偏性、置信水平）。

贝叶斯学派（Bayesian）：参数 $\theta$ 是随机变量，具有先验分布 $\pi(\theta)$，反映在观测数据之前对 $\theta$ 的认知。通过贝叶斯公式将先验信息与样本信息结合，得到后验分布。

核心区别对比

比较维度 频率学派 贝叶斯学派
参数 $\theta$ 固定未知常数 随机变量，有分布
先验信息 不使用 用先验分布 $\pi(\theta)$ 表达
推断依据 似然函数 $L(\theta \mid x)$ 后验分布 $\pi(\theta \mid x)$
区间估计 置信区间（频率覆盖概率） 可信区间（后验概率）
评价标准 无偏性、均方误差等 后验风险最小化
直觉理解

频率学派问："如果重复实验无数次，这个方法的表现如何？"

贝叶斯学派问："在看到这些数据之后，我对参数的认知更新了多少？"

当样本量 $n \to \infty$ 时，先验的影响逐渐消失，两种方法的结果趋于一致（Bernstein-von Mises 定理）。

5.2 贝叶斯公式（连续形式）

贝叶斯公式的三大要素
  • 先验分布 $\pi(\theta)$：观测数据之前对 $\theta$ 的认知
  • 似然函数 $L(\theta \mid x) = f(x \mid \theta)$：给定 $\theta$ 时观测到数据 $x$ 的概率密度
  • 后验分布 $\pi(\theta \mid x)$：结合先验与数据之后对 $\theta$ 的更新认知
$$\pi(\theta \mid x) = \frac{L(\theta \mid x)\,\pi(\theta)}{\displaystyle\int_\Theta L(\theta \mid x)\,\pi(\theta)\,\dd\theta} \propto L(\theta \mid x)\,\pi(\theta)$$
直觉解释

后验 $\propto$ 先验 $\times$ 似然

先验分布编码了"参数最可能是什么"的信念；似然函数编码了"数据支持哪些参数值"。二者相乘得到后验分布——综合了先验知识与数据证据的最优推断。

分母 $\int L(\theta \mid x)\pi(\theta)\,\dd\theta$ 只是归一化常数，确保后验是一个合法的概率密度函数。

例 5.1 · 正态均值的贝叶斯推断

设 $X_1, \dots, X_n \overset{\text{iid}}{\sim} N(\theta, \sigma^2)$，$\sigma^2$ 已知，先验 $\theta \sim N(\mu_0, \tau^2)$。

似然函数：

$$L(\theta \mid x) \propto \exp\!\left\{-\frac{1}{2\sigma^2}\sum_{i=1}^n (x_i - \theta)^2\right\} \propto \exp\!\left\{-\frac{n}{2\sigma^2}(\bar{x} - \theta)^2\right\}$$

后验：

$$\theta \mid x \;\sim\; N\!\left(\frac{\frac{n}{\sigma^2}\bar{x} + \frac{1}{\tau^2}\mu_0}{\frac{n}{\sigma^2} + \frac{1}{\tau^2}},\; \frac{1}{\frac{n}{\sigma^2} + \frac{1}{\tau^2}}\right)$$

后验均值是样本均值 $\bar{x}$ 与先验均值 $\mu_0$ 的加权平均，权重与各自的精度（方差的倒数）成正比。

5.3 共轭先验族

定义 · 共轭先验

如果先验分布 $\pi(\theta)$ 属于某个分布族 $\mathcal{F}$，且后验分布 $\pi(\theta \mid x)$ 也属于同一分布族 $\mathcal{F}$，则称 $\mathcal{F}$ 为该似然函数的共轭先验族。

共轭先验的计算优势：后验参数可以通过简单公式从先验参数和数据统计量中更新得到。

5.3.1 正态均值的共轭先验

定理 5.1 · 正态-正态共轭

设 $X_1, \dots, X_n \overset{\text{iid}}{\sim} N(\theta, \sigma^2)$，$\sigma^2$ 已知。先验 $\theta \sim N(\mu_0, \tau^2)$，则后验为：

$$\theta \mid x \;\sim\; N(\mu_n, \sigma_n^2)$$

其中：

$$\frac{1}{\sigma_n^2} = \frac{n}{\sigma^2} + \frac{1}{\tau^2}, \qquad \mu_n = \sigma_n^2\!\left(\frac{n\bar{x}}{\sigma^2} + \frac{\mu_0}{\tau^2}\right)$$

解读：后验精度 = 数据精度 + 先验精度；后验均值按精度加权平均。

5.3.2 Bernoulli / Binomial 的共轭先验：Beta 分布

定理 5.2 · Beta-Binomial 共轭

设 $X \mid \theta \sim \text{Bin}(n, \theta)$，先验 $\theta \sim \text{Beta}(\alpha, \beta)$，则后验：

$$\theta \mid x \;\sim\; \text{Beta}(\alpha + x,\; \beta + n - x)$$

后验参数 = 先验参数 + 观测数据：$\alpha$ 增加成功次数 $x$，$\beta$ 增加失败次数 $n-x$。

例 5.2 · Beta-Binomial 后验计算

某硬币先验为 $\theta \sim \text{Beta}(2, 2)$（对称，集中在 0.5 附近）。投掷 10 次出现 7 次正面，求后验。

解：后验为 $\theta \mid x \sim \text{Beta}(2+7,\; 2+3) = \text{Beta}(9, 5)$。

后验均值：$\EE[\theta \mid x] = \frac{9}{9+5} = \frac{9}{14} \approx 0.643$。

注意：先验均值为 0.5，MLE 为 0.7，后验均值 0.643 介于二者之间。

5.3.3 Poisson 的共轭先验：Gamma 分布

定理 5.3 · Gamma-Poisson 共轭

设 $X_1, \dots, X_n \overset{\text{iid}}{\sim} \text{Poisson}(\lambda)$，先验 $\lambda \sim \text{Gamma}(\alpha, \beta)$（密度 $f(\lambda) \propto \lambda^{\alpha-1}e^{-\beta\lambda}$），则后验：

$$\lambda \mid x \;\sim\; \text{Gamma}\!\left(\alpha + \sum_{i=1}^n x_i,\; \beta + n\right)$$

后验参数更新：$\alpha \to \alpha + \sum x_i$（加上总事件数），$\beta \to \beta + n$（加上总观测时间）。

例 5.3 · Gamma-Poisson 后验计算

某网站每小时点击数服从 Poisson$(\lambda)$，先验 $\lambda \sim \text{Gamma}(3, 1)$。观测 5 小时，总点击数为 20，求后验。

解：$\lambda \mid x \sim \text{Gamma}(3+20,\; 1+5) = \text{Gamma}(23, 6)$。

后验均值：$\EE[\lambda \mid x] = \frac{23}{6} \approx 3.83$。先验均值为 3，MLE 为 $\bar{x} = 4$。

共轭先验汇总

似然 共轭先验 后验
$N(\theta, \sigma^2)$（$\sigma^2$ 已知） $N(\mu_0, \tau^2)$ $N(\mu_n, \sigma_n^2)$
$\text{Bin}(n, \theta)$ $\text{Beta}(\alpha, \beta)$ $\text{Beta}(\alpha+x, \beta+n-x)$
$\text{Poisson}(\lambda)$ $\text{Gamma}(\alpha, \beta)$ $\text{Gamma}(\alpha+\sum x_i, \beta+n)$
$\text{Exp}(\lambda)$ $\text{Gamma}(\alpha, \beta)$ $\text{Gamma}(\alpha+n, \beta+\sum x_i)$

5.4 贝叶斯估计

三种贝叶斯点估计

给定后验分布 $\pi(\theta \mid x)$，根据不同损失函数得到不同的最优估计：

  • 后验均值（平方损失 $L(\theta, a) = (\theta - a)^2$ 下的最优估计）： $$\hat{\theta}_{\text{Bayes}} = \EE[\theta \mid x] = \int_\Theta \theta\,\pi(\theta \mid x)\,\dd\theta$$
  • 后验中位数（绝对损失 $L(\theta, a) = |\theta - a|$ 下的最优估计）
  • 后验众数（0-1 损失下的最优估计），也称为 MAP（Maximum A Posteriori）估计： $$\hat{\theta}_{\text{MAP}} = \arg\max_\theta \pi(\theta \mid x) = \arg\max_\theta [L(\theta \mid x)\,\pi(\theta)]$$
MAP 与 MLE 的关系

当先验分布 $\pi(\theta)$ 为均匀分布（即 $\pi(\theta) \propto$ 常数）时：

$$\hat{\theta}_{\text{MAP}} = \arg\max_\theta L(\theta \mid x) = \hat{\theta}_{\text{MLE}}$$

即 MLE 是先验为均匀分布时的 MAP 估计。这说明频率学派的 MLE 可视为"无信息先验"下的贝叶斯估计。

例 5.4 · 正态模型的贝叶斯估计

$X_1, \dots, X_n \overset{\text{iid}}{\sim} N(\theta, \sigma^2)$，$\sigma^2$ 已知，先验 $\theta \sim N(\mu_0, \tau^2)$。

后验 $\theta \mid x \sim N(\mu_n, \sigma_n^2)$，故：

  • 贝叶斯估计（后验均值）：$\hat{\theta} = \mu_n = \frac{\frac{n}{\sigma^2}\bar{x} + \frac{1}{\tau^2}\mu_0}{\frac{n}{\sigma^2} + \frac{1}{\tau^2}}$
  • MAP 估计：后验众数 = 后验均值（正态分布的对称性）= $\mu_n$
  • 后验中位数 = 后验均值 = $\mu_n$（正态分布三者重合）

当 $n \to \infty$ 时，$\mu_n \to \bar{x} = \hat{\theta}_{\text{MLE}}$，先验的影响消失。

例 5.5 · Beta-Binomial 的贝叶斯估计

$X \mid \theta \sim \text{Bin}(n, \theta)$，先验 $\theta \sim \text{Beta}(\alpha, \beta)$，后验 $\theta \mid x \sim \text{Beta}(\alpha+x, \beta+n-x)$。

  • 后验均值：$\hat{\theta} = \frac{\alpha + x}{\alpha + \beta + n}$
  • MAP 估计：$\hat{\theta}_{\text{MAP}} = \frac{\alpha + x - 1}{\alpha + \beta + n - 2}$（当 $\alpha, \beta > 1$ 时）
  • MLE：$\hat{\theta}_{\text{MLE}} = \frac{x}{n}$

5.5 贝叶斯决策简介

贝叶斯决策框架

损失函数 $L(\theta, a)$：当参数真值为 $\theta$、采取行动 $a$ 时的损失。

后验风险：给定后验分布 $\pi(\theta \mid x)$，行动 $a$ 的后验风险为：

$$R(a \mid x) = \EE_{\theta \mid x}[L(\theta, a)] = \int_\Theta L(\theta, a)\,\pi(\theta \mid x)\,\dd\theta$$

贝叶斯决策规则：选择使后验风险最小的行动：

$$a^* = \arg\min_a R(a \mid x)$$
常见损失函数下的贝叶斯估计
损失函数 $L(\theta, a)$ 贝叶斯估计 $a^*$
平方损失 $(\theta - a)^2$ 后验均值 $\EE[\theta \mid x]$
绝对损失 $|\theta - a|$ 后验中位数
0-1 损失（离散） $\mathbb{1}(\theta \neq a)$ 后验众数（MAP）
例 5.6 · 正态模型下的贝叶斯决策

设 $X_1, \dots, X_n \overset{\text{iid}}{\sim} N(\theta, \sigma^2)$，$\sigma^2$ 已知，先验 $\theta \sim N(\mu_0, \tau^2)$。

若采用平方损失函数 $L(\theta, a) = (\theta - a)^2$，则贝叶斯估计为后验均值：

$$a^* = \EE[\theta \mid x] = \frac{\frac{n}{\sigma^2}\bar{x} + \frac{1}{\tau^2}\mu_0}{\frac{n}{\sigma^2} + \frac{1}{\tau^2}}$$

后验风险（即后验方差）：

$$R(a^* \mid x) = \Var(\theta \mid x) = \frac{1}{\frac{n}{\sigma^2} + \frac{1}{\tau^2}}$$

随着 $n$ 增大，后验方差趋于 0，估计越来越精确。

贝叶斯方法的优势与局限

优势：

  • 自然地融合先验知识与数据信息
  • 提供完整的后验分布，而非单个点估计
  • 小样本时先验可提供有效正则化
  • 贝叶斯可信区间有更直观的概率解释

局限：

  • 先验的选择具有主观性，不同先验可能导致不同结论
  • 复杂模型的后验计算通常需要 MCMC 等数值方法
  • 共轭先验仅适用于简单的标准模型
