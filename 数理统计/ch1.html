第一章 · 数理统计基本概念

统计推断的理论基础：总体、样本与统计量。三大抽样分布（$\chi^2$、$t$、$F$）是后续所有统计方法的基石，正态总体的抽样分布定理贯穿整个课程。

1.1 总体与样本

定义 · 总体与个体

研究对象的全体称为总体，组成总体的每个基本单位称为个体。

总体按其包含的个体数目可分为：

  • 有限总体：个体数目有限，如一批产品共 $N$ 件
  • 无限总体：个体数目无限或可视为无限，如某工厂生产的所有产品（含未来产品）
定义 · 样本与简单随机抽样

从总体中按一定规则抽取 $n$ 个个体，得到 $X_1, X_2, \ldots, X_n$，称为容量为 $n$ 的样本。

若样本 $X_1, X_2, \ldots, X_n$ 满足：

  1. 代表性：每个 $X_i$ 都与总体同分布；
  2. 独立性：$X_1, X_2, \ldots, X_n$ 相互独立。

则称 $X_1, X_2, \ldots, X_n$ 为简单随机样本，简称样本。

样本的联合分布

设总体 $X$ 的分布函数为 $F(x;\theta)$（其中 $\theta$ 为未知参数），则简单随机样本 $X_1, \ldots, X_n$ 的联合分布函数为：

$$F(x_1, x_2, \ldots, x_n; \theta) = \prod_{i=1}^n F(x_i; \theta)$$

若总体为连续型，密度为 $f(x;\theta)$，则联合密度为：

$$f(x_1, x_2, \ldots, x_n; \theta) = \prod_{i=1}^n f(x_i; \theta)$$

若总体为离散型，分布律为 $P(X=x) = p(x;\theta)$，则联合分布律为：

$$P(X_1=x_1, \ldots, X_n=x_n; \theta) = \prod_{i=1}^n p(x_i; \theta)$$
样本的二重性

样本 $X_1, \ldots, X_n$ 具有二重性：

  • 抽样前：$X_1, \ldots, X_n$ 是 $n$ 个随机变量，其联合分布由总体分布决定
  • 抽样后：$x_1, \ldots, x_n$ 是一组具体的观测值（数据），称为样本观测值

统计推断就是根据样本观测值 $x_1, \ldots, x_n$（已知）来推断总体分布或参数 $\theta$（未知）。

1.2 统计量

定义 · 统计量

设 $X_1, X_2, \ldots, X_n$ 是来自总体 $X$ 的样本，$T = T(X_1, X_2, \ldots, X_n)$ 是样本的函数。若 $T$ 中不含任何未知参数，则称 $T$ 为统计量。

统计量的关键要求

"不含未知参数"是统计量的核心要求。例如：

  • $\bar{X} = \frac{1}{n}\sum_{i=1}^n X_i$ 是统计量（不含未知参数）
  • $\frac{\bar{X}-\mu}{\sigma/\sqrt{n}}$ 不是统计量（若 $\mu$ 或 $\sigma$ 未知）

统计量是"由数据可以算出来的量"，是统计推断的基本工具。

常用统计量

常用统计量一览
名称定义含义
样本均值$\bar{X} = \dfrac{1}{n}\displaystyle\sum_{i=1}^n X_i$样本的"重心"
样本方差$S^2 = \dfrac{1}{n-1}\displaystyle\sum_{i=1}^n (X_i - \bar{X})^2$样本的分散程度
样本标准差$S = \sqrt{S^2}$与原数据同单位的离散度量
样本 $k$ 阶原点矩$A_k = \dfrac{1}{n}\displaystyle\sum_{i=1}^n X_i^k$总体 $k$ 阶原点矩的估计
样本 $k$ 阶中心矩$B_k = \dfrac{1}{n}\displaystyle\sum_{i=1}^n (X_i - \bar{X})^k$总体 $k$ 阶中心矩的估计
样本中位数$\tilde{X} = X_{((n+1)/2)}$（按序排列后居中者）对异常值稳健的位置度量
样本极差$R = X_{(n)} - X_{(1)}$样本的最大波动幅度

注意：样本方差的分母为 $n-1$ 而非 $n$，这保证了 $S^2$ 是 $\sigma^2$ 的无偏估计（详见第二章）。

样本方差的计算公式

设 $\bar{X}$ 为样本均值，则：

$$\sum_{i=1}^n (X_i - \bar{X})^2 = \sum_{i=1}^n X_i^2 - n\bar{X}^2 = \sum_{i=1}^n X_i^2 - \frac{1}{n}\left(\sum_{i=1}^n X_i\right)^2$$

证明：展开平方项：

$$\sum_{i=1}^n (X_i - \bar{X})^2 = \sum_{i=1}^n X_i^2 - 2\bar{X}\sum_{i=1}^n X_i + n\bar{X}^2 = \sum_{i=1}^n X_i^2 - 2n\bar{X}^2 + n\bar{X}^2 = \sum_{i=1}^n X_i^2 - n\bar{X}^2$$
▶ 例题 · 统计量的判断

题目：设 $X_1, \ldots, X_n$ 是来自 $N(\mu, \sigma^2)$ 的样本，其中 $\mu$ 已知、$\sigma^2$ 未知。判断以下哪些是统计量：

(a) $\bar{X}$；(b) $S^2$；(c) $\sum_{i=1}^n \frac{(X_i-\mu)^2}{\sigma^2}$；(d) $\max X_i - \min X_i$

解：

(a) $\bar{X}$：不含未知参数，是统计量。

(b) $S^2$：不含未知参数，是统计量。

(c) $\sum_{i=1}^n \frac{(X_i-\mu)^2}{\sigma^2}$：含未知参数 $\sigma^2$，不是统计量。

(d) $\max X_i - \min X_i$（极差）：不含未知参数，是统计量。

1.3 三大抽样分布

统计量作为随机变量，有其自身的分布，称为抽样分布。以下三个分布在统计推断中占据核心地位。

$\chi^2$ 分布

定义 · $\chi^2$ 分布

设 $X_1, X_2, \ldots, X_n$ 相互独立，且 $X_i \sim N(0,1)$，则称

$$\chi^2 = X_1^2 + X_2^2 + \cdots + X_n^2$$

服从自由度为 $n$ 的 $\chi^2$ 分布，记为 $\chi^2 \sim \chi^2(n)$。

$\chi^2(n)$ 的密度函数

$\chi^2(n)$ 分布的密度函数为：

$$f(x; n) = \frac{1}{2^{n/2}\Gamma(n/2)} x^{n/2-1} e^{-x/2}, \quad x > 0$$

推导思路：利用 Gamma 分布。若 $Y \sim \text{Gamma}(\alpha, \lambda)$，密度为 $\frac{\lambda^\alpha}{\Gamma(\alpha)} x^{\alpha-1} e^{-\lambda x}$。

$X_i \sim N(0,1)$，则 $X_i^2 \sim \chi^2(1)$。利用 $N(0,1)$ 密度的对称性和变量替换可证 $X_i^2$ 的密度为：

$$f_{X_i^2}(x) = \frac{1}{\sqrt{2\pi}} x^{-1/2} e^{-x/2}, \quad x > 0$$

这恰好是 $\text{Gamma}(1/2, 1/2)$ 分布。由 Gamma 分布的可加性，$n$ 个独立 $\text{Gamma}(1/2, 1/2)$ 之和为 $\text{Gamma}(n/2, 1/2)$，即 $\chi^2(n)$。

$\chi^2$ 分布的性质
  1. 可加性：若 $X \sim \chi^2(m)$，$Y \sim \chi^2(n)$，$X$ 与 $Y$ 独立，则 $X+Y \sim \chi^2(m+n)$
  2. 期望：$E[\chi^2(n)] = n$
  3. 方差：$\Var(\chi^2(n)) = 2n$

证明（期望与方差）：$\chi^2(n) = \sum_{i=1}^n X_i^2$，其中 $X_i \sim N(0,1)$ 独立。

$E[X_i^2] = \Var(X_i) + (E[X_i])^2 = 1+0 = 1$，所以 $E[\chi^2(n)] = n \cdot 1 = n$。

$\Var(X_i^2) = E[X_i^4] - (E[X_i^2])^2 = 3 - 1 = 2$（标准正态的四阶矩为 $3$），所以 $\Var(\chi^2(n)) = n \cdot 2 = 2n$。

$t$ 分布

定义 · $t$ 分布

设 $Z \sim N(0,1)$，$V \sim \chi^2(n)$，$Z$ 与 $V$ 独立，则称

$$T = \frac{Z}{\sqrt{V/n}}$$

服从自由度为 $n$ 的 $t$ 分布，记为 $T \sim t(n)$。

$t(n)$ 的密度函数
$$f(t; n) = \frac{\Gamma\!\left(\frac{n+1}{2}\right)}{\sqrt{n\pi}\;\Gamma\!\left(\frac{n}{2}\right)} \left(1 + \frac{t^2}{n}\right)^{-(n+1)/2}, \quad -\infty < t < +\infty$$
$t$ 分布的性质
  1. 对称性：$f(t)$ 关于 $t=0$ 对称，即 $f(t) = f(-t)$
  2. 期望与方差：$E[T] = 0$（$n>1$），$\Var(T) = \dfrac{n}{n-2}$（$n>2$）
  3. 与正态的关系：当 $n \to \infty$ 时，$t(n)$ 分布趋近于 $N(0,1)$。实际应用中，$n \geq 30$ 时两者已非常接近
$t$ 分布的直观理解

$t$ 分布可以看作"加了噪声的正态分布"。当用样本标准差 $S$ 代替已知的 $\sigma$ 时，额外的随机性使得分布尾部变厚（相对于标准正态），自由度越小尾部越厚。

$t$ 分布的密度曲线与 $N(0,1)$ 类似，但"更矮更胖"，自由度增大后逐渐逼近标准正态。

$F$ 分布

定义 · $F$ 分布

设 $U \sim \chi^2(m)$，$V \sim \chi^2(n)$，$U$ 与 $V$ 独立，则称

$$F = \frac{U/m}{V/n}$$

服从自由度为 $(m,n)$ 的 $F$ 分布，记为 $F \sim F(m,n)$，其中 $m$ 称为第一自由度（分子自由度），$n$ 称为第二自由度（分母自由度）。

$F(m,n)$ 的密度函数
$$f(x; m,n) = \frac{\Gamma\!\left(\frac{m+n}{2}\right)}{\Gamma\!\left(\frac{m}{2}\right)\Gamma\!\left(\frac{n}{2}\right)} \left(\frac{m}{n}\right)^{m/2} x^{m/2-1} \left(1+\frac{m}{n}x\right)^{-(m+n)/2}, \quad x > 0$$
$F$ 分布的性质
  1. 倒数性质：若 $F \sim F(m,n)$，则 $\dfrac{1}{F} \sim F(n,m)$
  2. 若 $T \sim t(n)$，则 $T^2 \sim F(1,n)$
  3. 期望：$E[F(m,n)] = \dfrac{n}{n-2}$（$n>2$）

证明（倒数性质）：$F = \frac{U/m}{V/n}$，则 $\frac{1}{F} = \frac{V/n}{U/m}$，由 $F$ 分布定义，$\frac{1}{F} \sim F(n,m)$。

证明（$T^2 \sim F(1,n)$）：$T = \frac{Z}{\sqrt{V/n}}$，其中 $Z \sim N(0,1)$，$V \sim \chi^2(n)$，独立。$T^2 = \frac{Z^2}{V/n} = \frac{Z^2/1}{V/n}$。由于 $Z^2 \sim \chi^2(1)$，由 $F$ 分布定义，$T^2 \sim F(1,n)$。

三大分布之间的关系
关系公式
$\chi^2$ 与 $t$$T = \frac{Z}{\sqrt{\chi^2(n)/n}} \sim t(n)$
$t$ 与 $F$$T^2 \sim F(1,n)$

1.4 正态总体的抽样分布定理

以下四个定理是数理统计的核心工具，后续的参数估计和假设检验都直接建立在这些定理之上。

定理 1：样本均值的分布

定理 1 · 样本均值的精确分布

设 $X_1, X_2, \ldots, X_n$ 是来自 $N(\mu, \sigma^2)$ 的样本，则样本均值

$$\bar{X} \sim N\!\left(\mu, \frac{\sigma^2}{n}\right)$$

等价地：$\dfrac{\bar{X}-\mu}{\sigma/\sqrt{n}} \sim N(0,1)$。

证明

由独立正态变量的线性组合仍为正态分布。

$E[\bar{X}] = \frac{1}{n}\sum_{i=1}^n E[X_i] = \frac{1}{n} \cdot n\mu = \mu$

$\Var(\bar{X}) = \frac{1}{n^2}\sum_{i=1}^n \Var(X_i) = \frac{1}{n^2} \cdot n\sigma^2 = \frac{\sigma^2}{n}$

由于 $X_i$ 独立且服从正态分布，$\bar{X}$ 也服从正态分布，故 $\bar{X} \sim N(\mu, \sigma^2/n)$。

定理 1 的意义

$\bar{X}$ 的方差是单个观测方差 $\sigma^2$ 的 $1/n$，即样本量越大，$\bar{X}$ 越集中在 $\mu$ 附近。这就是大数定律在正态总体下的精确体现。

定理 2：样本方差的分布（Fisher 定理）

定理 2 · Fisher 定理

设 $X_1, X_2, \ldots, X_n$ 是来自 $N(\mu, \sigma^2)$ 的样本，$\bar{X}$ 和 $S^2$ 分别为样本均值和样本方差，则：

  1. $\dfrac{(n-1)S^2}{\sigma^2} \sim \chi^2(n-1)$
  2. $\bar{X}$ 与 $S^2$ 相互独立
证明思路

关键步骤：将 $X_i$ 标准化后利用正交变换。

令 $Z_i = \frac{X_i - \mu}{\sigma}$，则 $Z_1, \ldots, Z_n$ i.i.d. $\sim N(0,1)$。

构造正交矩阵 $A$（第一行为 $(1/\sqrt{n}, \ldots, 1/\sqrt{n})$），令 $Y = AZ$，其中 $Y = (Y_1, \ldots, Y_n)^T$。

由正交变换的性质：

  1. $Y_1, \ldots, Y_n$ 相互独立，且 $Y_i \sim N(0,1)$（正交变换保持独立标准正态性）
  2. $Y_1 = \frac{1}{\sqrt{n}}\sum_{i=1}^n Z_i = \sqrt{n}\bar{Z} = \frac{\sqrt{n}(\bar{X}-\mu)}{\sigma}$
  3. $\sum_{i=1}^n Y_i^2 = \sum_{i=1}^n Z_i^2 = \sum_{i=1}^n \frac{(X_i-\mu)^2}{\sigma^2}$

因此 $\sum_{i=2}^n Y_i^2 = \sum_{i=1}^n Y_i^2 - Y_1^2 = \sum_{i=1}^n \frac{(X_i-\mu)^2}{\sigma^2} - \frac{n(\bar{X}-\mu)^2}{\sigma^2} = \frac{(n-1)S^2}{\sigma^2}$。

由于 $Y_2, \ldots, Y_n$ 独立于 $Y_1$（而 $Y_1$ 的函数决定了 $\bar{X}$），故 $\bar{X}$ 与 $S^2$ 独立。

又 $Y_2, \ldots, Y_n$ i.i.d. $\sim N(0,1)$，故 $\sum_{i=2}^n Y_i^2 \sim \chi^2(n-1)$，即 $\frac{(n-1)S^2}{\sigma^2} \sim \chi^2(n-1)$。

定理 3：$t$ 分布的构造

定理 3

设 $X_1, \ldots, X_n$ 是来自 $N(\mu, \sigma^2)$ 的样本，则

$$\frac{\bar{X}-\mu}{S/\sqrt{n}} \sim t(n-1)$$
证明

由定理 1：$Z = \frac{\bar{X}-\mu}{\sigma/\sqrt{n}} \sim N(0,1)$

由定理 2：$V = \frac{(n-1)S^2}{\sigma^2} \sim \chi^2(n-1)$，且 $Z$ 与 $V$ 独立。

由 $t$ 分布的定义：

$$\frac{Z}{\sqrt{V/(n-1)}} = \frac{\frac{\bar{X}-\mu}{\sigma/\sqrt{n}}}{\sqrt{\frac{(n-1)S^2}{\sigma^2(n-1)}}} = \frac{\bar{X}-\mu}{S/\sqrt{n}} \sim t(n-1)$$
定理 3 的应用场景

当总体方差 $\sigma^2$ 未知时，用样本标准差 $S$ 代替 $\sigma$，统计量从标准正态变为 $t(n-1)$。这是实际中最常用的情形——因为真实场景中 $\sigma$ 通常是未知的。

定理 4：两个正态总体的 $F$ 分布

定理 4

设 $X_1, \ldots, X_m$ 是来自 $N(\mu_1, \sigma_1^2)$ 的样本，$Y_1, \ldots, Y_n$ 是来自 $N(\mu_2, \sigma_2^2)$ 的样本，两组样本相互独立。$\bar{X}, S_1^2$ 和 $\bar{Y}, S_2^2$ 分别为两组的样本均值和方差，则：

$$\frac{S_1^2/\sigma_1^2}{S_2^2/\sigma_2^2} \sim F(m-1, n-1)$$
证明

由定理 2：

$\frac{(m-1)S_1^2}{\sigma_1^2} \sim \chi^2(m-1)$，$\frac{(n-1)S_2^2}{\sigma_2^2} \sim \chi^2(n-1)$

两组样本独立，故这两个 $\chi^2$ 变量独立。

由 $F$ 分布定义：

$$\frac{\frac{(m-1)S_1^2}{\sigma_1^2}\bigg/(m-1)}{\frac{(n-1)S_2^2}{\sigma_2^2}\bigg/(n-1)} = \frac{S_1^2/\sigma_1^2}{S_2^2/\sigma_2^2} \sim F(m-1, n-1)$$
特别地：检验方差是否相等

当 $\sigma_1^2 = \sigma_2^2$ 时，$\frac{S_1^2}{S_2^2} \sim F(m-1, n-1)$。这是检验两总体方差是否相等（$H_0: \sigma_1^2 = \sigma_2^2$）的基础。

▶ 例题 · 抽样分布定理的综合应用

题目：设 $X_1, \ldots, X_{16}$ 是来自 $N(5, 4)$ 的样本，求 $P(\bar{X} > 5.5)$。

解：$\bar{X} \sim N(5, 4/16) = N(5, 0.25)$，$\sigma_{\bar{X}} = 0.5$。

$P(\bar{X} > 5.5) = P\!\left(\frac{\bar{X}-5}{0.5} > \frac{5.5-5}{0.5}\right) = P(Z > 1) = 1 - \Phi(1) \approx 0.1587$

▶ 例题 · 求样本方差的概率

题目：设 $X_1, \ldots, X_{10}$ 是来自 $N(0, \sigma^2)$ 的样本，求 $P\!\left(\frac{S^2}{\sigma^2} \leq 2.09\right)$。

解：由定理 2，$\frac{(n-1)S^2}{\sigma^2} \sim \chi^2(9)$，即 $\frac{9S^2}{\sigma^2} \sim \chi^2(9)$。

$P\!\left(\frac{S^2}{\sigma^2} \leq 2.09\right) = P\!\left(\frac{9S^2}{\sigma^2} \leq 9 \times 2.09\right) = P(\chi^2(9) \leq 18.81)$

查 $\chi^2$ 分布表，$\chi^2_{0.05}(9) = 16.919$，$\chi^2_{0.025}(9) = 19.023$。$18.81$ 介于两者之间，故 $P \approx 0.97$ 左右。
