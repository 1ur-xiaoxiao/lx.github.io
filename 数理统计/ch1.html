第一章 · 数理统计基本概念

统计推断的理论基础：总体、样本与统计量。三大抽样分布（$\chi^2$、$t$、$F$）是后续所有统计方法的基石，正态总体的抽样分布定理贯穿整个课程。

本章目录

  1. 总体与样本
  2. 统计量
  3. 三大抽样分布
  4. 正态总体的抽样分布定理

1.1 总体与样本

定义 · 总体与个体

研究对象的全体称为总体，组成总体的每个基本单位称为个体。

总体按其包含的个体数目可分为：

  • 有限总体：个体数目有限，如一批产品共 $N$ 件
  • 无限总体：个体数目无限或可视为无限，如某工厂生产的所有产品（含未来产品）
定义 · 样本与简单随机抽样

从总体中按一定规则抽取 $n$ 个个体，得到 $X_1, X_2, \ldots, X_n$，称为容量为 $n$ 的样本。

若样本 $X_1, X_2, \ldots, X_n$ 满足：

  1. 代表性：每个 $X_i$ 都与总体同分布；
  2. 独立性：$X_1, X_2, \ldots, X_n$ 相互独立。

则称 $X_1, X_2, \ldots, X_n$ 为简单随机样本，简称样本。

样本的联合分布

设总体 $X$ 的分布函数为 $F(x;\theta)$（其中 $\theta$ 为未知参数），则简单随机样本 $X_1, \ldots, X_n$ 的联合分布函数为：

$$F(x_1, x_2, \ldots, x_n; \theta) = \prod_{i=1}^n F(x_i; \theta)$$

若总体为连续型，密度为 $f(x;\theta)$，则联合密度为：

$$f(x_1, x_2, \ldots, x_n; \theta) = \prod_{i=1}^n f(x_i; \theta)$$

若总体为离散型，分布律为 $P(X=x) = p(x;\theta)$，则联合分布律为：

$$P(X_1=x_1, \ldots, X_n=x_n; \theta) = \prod_{i=1}^n p(x_i; \theta)$$
样本的二重性

样本 $X_1, \ldots, X_n$ 具有二重性：

  • 抽样前：$X_1, \ldots, X_n$ 是 $n$ 个随机变量，其联合分布由总体分布决定
  • 抽样后：$x_1, \ldots, x_n$ 是一组具体的观测值（数据），称为样本观测值

统计推断就是根据样本观测值 $x_1, \ldots, x_n$（已知）来推断总体分布或参数 $\theta$（未知）。

1.2 统计量

定义 · 统计量

设 $X_1, X_2, \ldots, X_n$ 是来自总体 $X$ 的样本，$T = T(X_1, X_2, \ldots, X_n)$ 是样本的函数。若 $T$ 中不含任何未知参数，则称 $T$ 为统计量。

统计量的关键要求

"不含未知参数"是统计量的核心要求。例如：

  • $\bar{X} = \frac{1}{n}\sum_{i=1}^n X_i$ 是统计量（不含未知参数）
  • $\frac{\bar{X}-\mu}{\sigma/\sqrt{n}}$ 不是统计量（若 $\mu$ 或 $\sigma$ 未知）

统计量是"由数据可以算出来的量"，是统计推断的基本工具。

常用统计量

常用统计量一览
名称定义含义
样本均值$\bar{X} = \dfrac{1}{n}\displaystyle\sum_{i=1}^n X_i$样本的"重心"
样本方差$S^2 = \dfrac{1}{n-1}\displaystyle\sum_{i=1}^n (X_i - \bar{X})^2$样本的分散程度
样本标准差$S = \sqrt{S^2}$与原数据同单位的离散度量
样本 $k$ 阶原点矩$A_k = \dfrac{1}{n}\displaystyle\sum_{i=1}^n X_i^k$总体 $k$ 阶原点矩的估计
样本 $k$ 阶中心矩$B_k = \dfrac{1}{n}\displaystyle\sum_{i=1}^n (X_i - \bar{X})^k$总体 $k$ 阶中心矩的估计
样本中位数$\tilde{X} = X_{((n+1)/2)}$（按序排列后居中者）对异常值稳健的位置度量
样本极差$R = X_{(n)} - X_{(1)}$样本的最大波动幅度

注意：样本方差的分母为 $n-1$ 而非 $n$，这保证了 $S^2$ 是 $\sigma^2$ 的无偏估计（详见第二章）。

样本方差的计算公式

设 $\bar{X}$ 为样本均值，则：

$$\sum_{i=1}^n (X_i - \bar{X})^2 = \sum_{i=1}^n X_i^2 - n\bar{X}^2 = \sum_{i=1}^n X_i^2 - \frac{1}{n}\left(\sum_{i=1}^n X_i\right)^2$$

证明：展开平方项：

$$\sum_{i=1}^n (X_i - \bar{X})^2 = \sum_{i=1}^n X_i^2 - 2\bar{X}\sum_{i=1}^n X_i + n\bar{X}^2 = \sum_{i=1}^n X_i^2 - 2n\bar{X}^2 + n\bar{X}^2 = \sum_{i=1}^n X_i^2 - n\bar{X}^2$$
▶ 例题 · 统计量的判断

题目：设 $X_1, \ldots, X_n$ 是来自 $N(\mu, \sigma^2)$ 的样本，其中 $\mu$ 已知、$\sigma^2$ 未知。判断以下哪些是统计量：

(a) $\bar{X}$；(b) $S^2$；(c) $\sum_{i=1}^n \frac{(X_i-\mu)^2}{\sigma^2}$；(d) $\max X_i - \min X_i$

解：

(a) $\bar{X}$：不含未知参数，是统计量。

(b) $S^2$：不含未知参数，是统计量。

(c) $\sum_{i=1}^n \frac{(X_i-\mu)^2}{\sigma^2}$：含未知参数 $\sigma^2$，不是统计量。

(d) $\max X_i - \min X_i$（极差）：不含未知参数，是统计量。

1.3 三大抽样分布

统计量作为随机变量，有其自身的分布，称为抽样分布。以下三个分布在统计推断中占据核心地位。

前置知识

以下推导中出现的 $\Gamma$ 函数、$\text{Gamma}$ 分布等概念，详见第零章 · Gamma分布与Beta分布基础。如果你对 $\Gamma(\alpha)$ 和 $\text{Gamma}(\alpha, \lambda)$ 还不熟悉，建议先阅读第零章再回来看本节。

$\chi^2$ 分布

定义 · $\chi^2$ 分布

设 $Z_1, Z_2, \ldots, Z_n$ 相互独立，且 $Z_i \sim N(0,1)$，则称

$$\chi^2 = Z_1^2 + Z_2^2 + \cdots + Z_n^2$$

服从自由度为 $n$ 的 $\chi^2$ 分布，记为 $\chi^2 \sim \chi^2(n)$。

直观含义：$n$ 个独立标准正态变量的平方和。自由度 $n$ 就是参与求和的标准正态个数——也是约束为零的独立信息的条数。

$\chi^2(n)$ 的密度函数
$$f(x; n) = \frac{1}{2^{n/2}\,\Gamma(n/2)}\,x^{n/2-1}\,e^{-x/2}, \quad x > 0$$

推导思路：先证 $Z_i^2 \sim \text{Gamma}(1/2,\, 1/2) = \chi^2(1)$（利用密度变换公式和 $\Gamma(1/2)=\sqrt{\pi}$），再用 Gamma 分布的可加性得到 $n$ 个独立 $\text{Gamma}(1/2, 1/2)$ 之和 $\sim \text{Gamma}(n/2, 1/2) = \chi^2(n)$。详细推导见第零章 §0.2。

性质（一）· 基本数字特征
特征值推导
期望$E[\chi^2(n)] = n$$E[Z_i^2] = \Var(Z_i) + (E[Z_i])^2 = 1$，求和得 $n$
方差$\Var(\chi^2(n)) = 2n$$\Var(Z_i^2) = E[Z_i^4] - 1^2 = 3 - 1 = 2$（正态四阶矩 = 3），求和得 $2n$
众数（mode）$n-2$（当 $n > 2$）对密度求导令零；$n \leq 2$ 时密度单调递减，众数在 $x=0$
偏度$\sqrt{8/n}$随 $n$ 增大趋近于 0（趋于对称）
性质（二）· 分布关系与可加性
  1. 与 Gamma 分布的关系：$\chi^2(n) = \text{Gamma}\!\left(\dfrac{n}{2},\, \dfrac{1}{2}\right)$。这是理解 $\chi^2$ 分布最简洁的方式——所有性质都可以从 Gamma 分布直接读出。
  2. 可加性：若 $X \sim \chi^2(m)$，$Y \sim \chi^2(n)$，且 $X \perp\!\!\!\perp Y$，则 $X + Y \sim \chi^2(m+n)$。这等价于 $\text{Gamma}(m/2,\,1/2) + \text{Gamma}(n/2,\,1/2) = \text{Gamma}((m+n)/2,\,1/2)$。
  3. 可加性的推论：若 $X_1, \ldots, X_k$ 独立且 $X_i \sim \chi^2(n_i)$，则 $\sum_{i=1}^k X_i \sim \chi^2(\sum_{i=1}^k n_i)$。
性质（三）· 极限行为

当 $n \to \infty$ 时，$\chi^2(n)$ 近似服从 $N(n,\, 2n)$。即：

$$\frac{\chi^2(n) - n}{\sqrt{2n}} \xrightarrow{d} N(0, 1)$$

这是中心极限定理的直接推论：$\chi^2(n) = \sum_{i=1}^n Z_i^2$，而 $Z_i^2$ 独立同分布，均值为 1、方差为 2。实际中 $n > 30$ 时正态近似已较好。

常用推论：当 $n$ 较大时，$\sqrt{2\chi^2} - \sqrt{2n-1}$ 更接近 $N(0,1)$（Fisher 近似），精度高于直接的正态近似。

$t$ 分布

定义 · $t$ 分布

设 $Z \sim N(0,1)$，$V \sim \chi^2(n)$，且 $Z \perp\!\!\!\perp V$，则称

$$T = \frac{Z}{\sqrt{V/n}}$$

服从自由度为 $n$ 的 $t$ 分布，记为 $T \sim t(n)$。

命名来源：W.S. Gosset 于 1908 年以笔名「Student」发表此分布，故亦称学生氏分布（Student's $t$-distribution）。

$t(n)$ 的密度函数
$$f(t; n) = \frac{\Gamma\!\left(\frac{n+1}{2}\right)}{\sqrt{n\pi}\;\Gamma\!\left(\frac{n}{2}\right)} \left(1 + \frac{t^2}{n}\right)^{-(n+1)/2}, \quad -\infty < t < +\infty$$

密度关于 $t=0$ 对称，形状类似标准正态但尾部更厚。注意系数中含有 $\Gamma$ 函数——这来自 $\chi^2$ 密度的 Gamma 结构。

性质（一）· 矩的存在性与数值

$t(n)$ 分布的 $k$ 阶矩 $E[T^k]$ 存在当且仅当 $k < n$。这是 $t$ 分布最本质的性质——自由度越小，能存在的矩越少。

矩存在条件值
$E[T]$$n > 1$$0$（由对称性）
$\Var(T)$$n > 2$$\dfrac{n}{n-2}$
偏度$n > 3$$0$（对称分布偏度恒为零）
峰度$n > 4$$3 + \dfrac{6}{n-4}$（$> 3$，即比正态更厚尾）

关键结论：$t(n)$ 的方差恒大于 1（标准正态的方差），因为分母中的随机变量 $V/n$ 引入了额外的波动。当 $n \to \infty$ 时 $\Var \to 1$。

性质（二）· 重要特例与推论
  1. $t(1)$ = 标准 Cauchy 分布（柯西分布）：当 $n = 1$ 时， $$f(t; 1) = \frac{1}{\pi(1 + t^2)}, \quad -\infty < t < +\infty$$ 这就是标准的 Cauchy 分布 $\text{Cauchy}(0, 1)$。

    Cauchy 分布的奇特性质：

    • 均值不存在：积分 $\int_{-\infty}^\infty t \cdot \frac{1}{\pi(1+t^2)}\dd t$ 不绝对收敛（$\int_0^\infty \frac{t}{1+t^2}\dd t = \infty$）。虽然由对称性 Cauchy 主值积分为 0，但期望按定义不存在。
    • 方差不存在：均值都不存在，高阶矩更不必说。
    • 没有矩母函数：$E[e^{tX}] = \infty$ 对所有 $t \neq 0$。
    • 样本均值的分布不变：若 $X_1, \ldots, X_n$ i.i.d. $\sim \text{Cauchy}(0,1)$，则 $\bar{X} \sim \text{Cauchy}(0,1)$——均值不会因样本量增大而更集中！这就是没有大数定律的例子。

    为什么在 $t$ 分布中会出现 Cauchy？回顾 $t$ 分布的定义：分子和分母必须独立。设分子中的标准正态为 $Z_1 \sim N(0,1)$，分母中的 $\chi^2(1)$ 来自另一个独立的标准正态 $Z_2$，即 $\chi^2(1) = Z_2^2$。则

    $$t(1) = \frac{Z_1}{\sqrt{Z_2^2 / 1}} = \frac{Z_1}{|Z_2|}$$

    由于 $Z_2$ 关于 0 对称，$Z_2$ 取正和取负的概率各为 $\frac{1}{2}$，且 $|Z_2|$ 与 $\text{sign}(Z_2)$ 独立。因此 $Z_1/|Z_2|$ 的分布与 $Z_1/Z_2$ 相同。而两个独立标准正态的比值 $Z_1/Z_2$ 服从标准 Cauchy 分布——这是概率论中的经典结论（下文推论给出简洁证明）。

  2. $t(2)$：均值存在（$=0$），但方差不存在。$\Var(T) = n/(n-2)$ 在 $n=2$ 时分母为零。
  3. $t(3)$：均值 $=0$，方差 $=3$。这是 $t$ 分布中方差最小的整数自由度情形。
性质（三）· 极限行为与正态近似
  1. 收敛到标准正态：当 $n \to \infty$ 时，$t(n)$ 依分布收敛到 $N(0,1)$。这是因为 $\sqrt{V/n} \xrightarrow{P} 1$（由大数定律，$\chi^2(n)/n \to 1$），所以 $T = Z/\sqrt{V/n} \to Z \sim N(0,1)$。
  2. 实际近似准则：$n \geq 30$ 时，$t(n)$ 分布与 $N(0,1)$ 的差异在多数应用中可以忽略。但若 $n$ 较小且需要精确的尾部概率（如 99% 置信区间），务必使用 $t$ 分布表。
  3. 尾部比较：对任意有限的 $n$，$t(n)$ 的尾部始终比 $N(0,1)$ 厚——$P(|t(n)| > c) > P(|N(0,1)| > c)$ 对所有 $c > 0$ 成立。这就是为什么用 $S$ 替代 $\sigma$ 后，置信区间必须变宽。
推论 · 两个独立正态之比与 Cauchy 分布

若 $X, Y$ 独立且均服从 $N(0,1)$，则

$$\frac{X}{Y} \sim \text{Cauchy}(0, 1)$$

证明：$t(1) = \frac{Z_1}{\sqrt{Z_2^2/1}} = \frac{Z_1}{|Z_2|}$。由于 $Z_2 \sim N(0,1)$ 是对称的，$\frac{Z_1}{|Z_2|}$ 和 $\frac{Z_1}{Z_2}$ 有相同的分布（$Z_2$ 的正负各半，绝对值不影响）。而 $t(1)$ 的密度为 $\frac{1}{\pi(1+t^2)}$，正是标准 Cauchy 分布。所以两个独立标准正态的比值服从 Cauchy 分布——一个在概率论中经典的结论，在这里通过 $t$ 分布自然地统一起来。

$t$ 分布小结
项目内容

$E$ 存在条件$n > 1$，值为 $0$
$\Var$ 存在条件$n > 2$，值为 $n/(n-2)$
$k$ 阶矩存在条件$k < n$

$F$ 分布

定义 · $F$ 分布

设 $U \sim \chi^2(m)$，$V \sim \chi^2(n)$，且 $U \perp\!\!\!\perp V$，则称

$$F = \frac{U/m}{V/n}$$

服从自由度为 $(m,n)$ 的 $F$ 分布，记为 $F \sim F(m,n)$。

  • $m$：第一自由度（分子自由度）
  • $n$：第二自由度（分母自由度）

直观含义：两个独立的、各自除以自由度的 $\chi^2$ 变量之比值。$F$ 分布天然用于比较两个方差（或两个变异来源的大小）。

命名来源：为纪念 R.A. Fisher（他在方差分析中大量使用此分布）而命名为 $F$ 分布。

$F(m,n)$ 的密度函数
$$f(x; m,n) = \frac{\Gamma\!\left(\frac{m+n}{2}\right)}{\Gamma\!\left(\frac{m}{2}\right)\Gamma\!\left(\frac{n}{2}\right)} \left(\frac{m}{n}\right)^{m/2} x^{m/2-1} \left(1+\frac{m}{n}x\right)^{-(m+n)/2}, \quad x > 0$$

注意到密度中 $x$ 的幂次为 $m/2-1$（与分子的 $\chi^2(m)$ 对应），而 $(1 + \frac{m}{n}x)^{-(m+n)/2}$ 同时涉及两个自由度。系数 $\frac{\Gamma((m+n)/2)}{\Gamma(m/2)\Gamma(n/2)} = \frac{1}{B(m/2,\, n/2)}$ 中包含 Beta 函数（见第零章 §0.3）。

性质（一）· 基本数字特征
特征存在条件值
$E[F]$$n > 2$$\dfrac{n}{n-2} \;(> 1)$
$\Var(F)$$n > 4$$\dfrac{2n^2(m+n-2)}{m(n-2)^2(n-4)}$
众数$m > 2$$\dfrac{m-2}{m} \cdot \dfrac{n}{n+2} \;(< 1)$

注意：$F$ 分布期望的存在条件由分母自由度 $n$ 决定，与分子自由度 $m$ 无关。这是因为 $F$ 取值由分母 $\chi^2(n)$ 的行为主导（分母接近 0 时 $F$ 爆炸）。

性质（二）· 分布关系
  1. 倒数性质：若 $F \sim F(m,n)$，则 $\dfrac{1}{F} \sim F(n,m)$。

    证明：$\frac{1}{F} = \frac{V/n}{U/m}$，由定义即得。这个性质在实际查表时极其实用——$F$ 分布表通常只给出右侧分位数，左侧分位数通过倒数性质转化为右侧。

  2. 与 $t$ 分布的关系：若 $T \sim t(n)$，则 $T^2 \sim F(1, n)$。

    证明：$T = \frac{Z}{\sqrt{V/n}}$，$T^2 = \frac{Z^2}{V/n} = \frac{Z^2/1}{V/n}$。由于 $Z^2 \sim \chi^2(1)$，由 $F$ 分布定义 $T^2 \sim F(1, n)$。

    实用推论：双侧 $t$ 检验的拒绝域 $\{|T| > t_{\alpha/2}(n)\}$ 等价于 $F$ 检验的拒绝域 $\{T^2 > F_\alpha(1,n)\}$。事实上 $t_{\alpha/2}(n) = \sqrt{F_\alpha(1,n)}$。

  3. 与 Beta 分布的关系：若 $F \sim F(m,n)$，则 $$\frac{mF/n}{1 + mF/n} \sim \text{Beta}\!\left(\frac{m}{2},\, \frac{n}{2}\right)$$ 这个关系在推导 $F$ 分布的分位数和构造 $F$ 检验的置信区间时有用。它也说明 $F$ 分布本质上是 $(0,1)$ 上 Beta 分布的一个单调变换。
性质（三）· 极限行为
  1. $n \to \infty$（分母自由度增大）：$V/n \xrightarrow{P} 1$，因此 $mF = \frac{U}{V/n} \xrightarrow{d} U \sim \chi^2(m)$。即 $m \cdot F(m,n) \xrightarrow{d} \chi^2(m)$ 当 $n \to \infty$。
  2. $m \to \infty$（分子自由度增大）：$n/F(m,n) = \frac{V}{U/m} \xrightarrow{d} V \sim \chi^2(n)$。
  3. 当 $m$ 和 $n$ 都很大时，$F(m,n) \approx 1$，检验失去鉴别力——这也是为什么 $F$ 检验在超大样本下需要谨慎解读。
$F$ 分布的分位数与查表技巧

$F$ 分布有两个自由度，分位数表通常只给出右尾概率。对于左尾分位数 $F_{1-\alpha}(m,n)$（即 $P(F \leq F_{1-\alpha}) = 1-\alpha$），利用倒数性质：

$$F_{1-\alpha}(m,n) = \frac{1}{F_\alpha(n,m)}$$

注意等号右边分子分母的自由度交换了位置。这是 $F$ 分布查表中最常见的知识点，也是考试的高频考点。

三大分布的整体关系图

一张图看透三大分布

所有关系都源自标准正态和独立性的组合：

$N(0,1)^2 \;\longrightarrow\; \chi^2(1)$

$n$ 个独立 $N(0,1)^2$ 之和 $\;\longrightarrow\; \chi^2(n)$

$\chi^2(n)$ = $\text{Gamma}(n/2,\, 1/2)$ $\;\longleftrightarrow\;$ Gamma 可加性 = $\chi^2$ 可加性

$\dfrac{N(0,1)}{\sqrt{\chi^2(n)/n}} \;\longrightarrow\; t(n)$

$t(n)^2 \;\longrightarrow\; F(1,n)$

$\dfrac{\chi^2(m)/m}{\chi^2(n)/n} \;\longrightarrow\; F(m,n)$

$n=1$：$t(1) = \text{Cauchy}(0,1)$；$m=1$：$F(1,n) = t(n)^2$

从这个关系图可以看出：三大分布本质上都是独立正态变量的函数。$\chi^2$ 是「平方和」、$t$ 是「标准化比」、$F$ 是「方差比」。理解了正态分布和独立性，就理解了三大分布。

1.4 正态总体的抽样分布定理

以下四个定理是数理统计的核心工具，后续的参数估计和假设检验都直接建立在这些定理之上。

定理 1：样本均值的分布

定理 1 · 样本均值的精确分布

设 $X_1, X_2, \ldots, X_n$ 是来自 $N(\mu, \sigma^2)$ 的样本，则样本均值

$$\bar{X} \sim N\!\left(\mu, \frac{\sigma^2}{n}\right)$$

等价地：$\dfrac{\bar{X}-\mu}{\sigma/\sqrt{n}} \sim N(0,1)$。

证明

由独立正态变量的线性组合仍为正态分布。

$E[\bar{X}] = \frac{1}{n}\sum_{i=1}^n E[X_i] = \frac{1}{n} \cdot n\mu = \mu$

$\Var(\bar{X}) = \frac{1}{n^2}\sum_{i=1}^n \Var(X_i) = \frac{1}{n^2} \cdot n\sigma^2 = \frac{\sigma^2}{n}$

由于 $X_i$ 独立且服从正态分布，$\bar{X}$ 也服从正态分布，故 $\bar{X} \sim N(\mu, \sigma^2/n)$。

定理 1 的意义

$\bar{X}$ 的方差是单个观测方差 $\sigma^2$ 的 $1/n$，即样本量越大，$\bar{X}$ 越集中在 $\mu$ 附近。这就是大数定律在正态总体下的精确体现。

定理 2：样本方差的分布（Fisher 定理）

定理 2 · Fisher 定理

设 $X_1, X_2, \ldots, X_n$ 是来自 $N(\mu, \sigma^2)$ 的样本，$\bar{X}$ 和 $S^2$ 分别为样本均值和样本方差，则：

  1. $\dfrac{(n-1)S^2}{\sigma^2} \sim \chi^2(n-1)$
  2. $\bar{X}$ 与 $S^2$ 相互独立
证明的整体思路

Fisher 定理的证明核心是一句话：把 $n$ 个相关的偏差 $(X_i - \bar{X})$ 变成 $n-1$ 个独立的标准正态变量。

我们想做的是：

  1. 把 $X_i$ 标准化为 $Z_i \sim N(0,1)$（独立）
  2. 用「旋转」把 $Z_1, \ldots, Z_n$ 变成一组新的独立标准正态变量 $Y_1, \ldots, Y_n$，使得 $Y_1$ 只包含 $\bar{X}$ 的信息，$Y_2, \ldots, Y_n$ 只包含 $S^2$ 的信息
  3. 这样 $S^2$ 就变成了 $n-1$ 个独立标准正态的平方和，自然服从 $\chi^2(n-1)$

下面先用一个简单的例子建立直觉，再给出严格的矩阵证明。

预备知识：需要用到的三个事实

事实 1 · 线性组合保持正态性

若 $Z_1, \ldots, Z_n$ 相互独立且 $Z_i \sim N(0,1)$，则它们的任意线性组合仍服从正态分布。特别地，如果系数满足某种条件（见事实 3），这些线性组合不仅各自正态，而且相互独立。

示例：$0.6 Z_1 + 0.8 Z_2$ 和 $-0.8 Z_1 + 0.6 Z_2$ 是两个独立的标准正态变量（因为系数向量 $(0.6, 0.8)$ 和 $(-0.8, 0.6)$ 正交且长度均为 1）。

事实 2 · 平方和不变（勾股定理的高维推广）

如果从一个向量 $Z = (Z_1, \ldots, Z_n)$ 通过旋转/反射得到另一个向量 $Y = (Y_1, \ldots, Y_n)$（即 $Y = AZ$，$A$ 是正交矩阵），那么向量的长度平方不变：

$$\sum_{i=1}^n Y_i^2 = \sum_{i=1}^n Z_i^2$$

这和平面几何中「旋转不改变点到原点的距离」是同一个道理。$n$ 维空间中的正交变换就是高维的旋转（或旋转加反射）。

事实 3 · 正交变换保持独立标准正态性（Fisher 引理）

若 $Z_1, \ldots, Z_n$ i.i.d. $\sim N(0,1)$，$A$ 是 $n \times n$ 正交矩阵（即 $A^T A = I$，等价于各行是单位向量且两两正交），令 $Y = AZ$，则：

  1. $Y_1, \ldots, Y_n$ 仍然是 i.i.d. $\sim N(0,1)$
  2. 特别地，$\sum Y_i^2 = \sum Z_i^2$（由事实 2）

为什么成立？（给高代不熟的同学）

  • $Y_i = \sum_{j=1}^n a_{ij} Z_j$，这是独立正态的线性组合，所以 $Y_i$ 服从正态分布。
  • $E[Y_i] = \sum a_{ij} \cdot 0 = 0$。
  • $\Var(Y_i) = \sum a_{ij}^2 \cdot 1$（独立变量线性组合的方差 = 系数平方和）。因为 $A$ 的每一行长度都是 1，所以 $\sum a_{ij}^2 = 1$，$\Var(Y_i) = 1$。
  • $\Cov(Y_i, Y_k) = \sum a_{ij} a_{kj}$（独立变量线性组合的协方差 = 对应系数乘积之和）。因为 $A$ 的不同行正交，所以 $\sum a_{ij} a_{kj} = 0$，$\Cov(Y_i, Y_k) = 0$。
  • 对于联合正态分布，协方差为零 ⇔ 独立。所以 $Y_1, \ldots, Y_n$ 相互独立。

核心要点：正交矩阵的行构成标准正交基——每行长度=1（保证方差为1），不同行内积=0（保证协方差为0，从而独立）。

$n=2$ 时的直觉：把证明「看见」

简单情形 · $n=2$ 的手工验证

取 $n=2$，$X_1, X_2$ i.i.d. $\sim N(\mu, \sigma^2)$。标准化：$Z_i = (X_i - \mu)/\sigma \sim N(0,1)$。

构造一个 $2 \times 2$ 的正交矩阵，第一行取 $(\frac{1}{\sqrt{2}}, \frac{1}{\sqrt{2}})$，第二行需要与之正交且长度=1，自然取 $(\frac{1}{\sqrt{2}}, -\frac{1}{\sqrt{2}})$：

$$A = \begin{pmatrix} \frac{1}{\sqrt{2}} & \frac{1}{\sqrt{2}} \\[4pt] \frac{1}{\sqrt{2}} & -\frac{1}{\sqrt{2}} \end{pmatrix}$$

令 $Y = AZ$，即：

$$Y_1 = \frac{Z_1 + Z_2}{\sqrt{2}}, \qquad Y_2 = \frac{Z_1 - Z_2}{\sqrt{2}}$$

现在检验关键事实：

  • $E[Y_1] = 0$，$\Var(Y_1) = \frac{1}{2} + \frac{1}{2} = 1$ ✓
  • $E[Y_2] = 0$，$\Var(Y_2) = \frac{1}{2} + \frac{1}{2} = 1$ ✓
  • $\Cov(Y_1, Y_2) = \frac{1}{2}\cdot\frac{1}{\sqrt{2}}\cdot\frac{1}{\sqrt{2}} + \frac{1}{2}\cdot\frac{1}{\sqrt{2}}\cdot(-\frac{1}{\sqrt{2}}) = \frac{1}{4} - \frac{1}{4} = 0$，所以 $Y_1 \perp\!\!\!\perp Y_2$ ✓

$Y_1$ 与 $\bar{X}$ 的关系：

$$Y_1 = \frac{Z_1 + Z_2}{\sqrt{2}} = \frac{1}{\sqrt{2}} \cdot \frac{(X_1-\mu)+(X_2-\mu)}{\sigma} = \frac{\sqrt{2}(\bar{X}-\mu)}{\sigma}$$

即 $Y_1$ 只和 $\bar{X}$ 有关。再看平方和：

$$Y_1^2 + Y_2^2 = Z_1^2 + Z_2^2 \quad \text{（勾股定理 / 旋转保长度）}$$

因此：

$$Y_2^2 = (Z_1^2 + Z_2^2) - Y_1^2 = \sum_{i=1}^2 \frac{(X_i-\mu)^2}{\sigma^2} - \frac{2(\bar{X}-\mu)^2}{\sigma^2}$$

回忆恒等式 $\sum (X_i - \mu)^2 = \sum (X_i - \bar{X})^2 + n(\bar{X} - \mu)^2$，当 $n=2$ 时：

$$Y_2^2 = \frac{\sum (X_i - \bar{X})^2}{\sigma^2} = \frac{(2-1)S^2}{\sigma^2} = \frac{S^2}{\sigma^2}$$

而 $Y_2 \sim N(0,1)$（由事实 3），故 $Y_2^2 \sim \chi^2(1)$，即 $\frac{S^2}{\sigma^2} \sim \chi^2(1) = \chi^2(2-1)$。

同时 $Y_1$（即 $\bar{X}$）与 $Y_2$（即 $S^2$）独立。$\quad\blacksquare$（$n=2$ 情形）

一般 $n$ 的严格证明

Fisher 定理的完整证明

第 1 步：标准化。令

$$Z_i = \frac{X_i - \mu}{\sigma}, \quad i = 1, 2, \ldots, n$$

由于 $X_i$ 独立且 $X_i \sim N(\mu, \sigma^2)$，所以 $Z_1, \ldots, Z_n$ i.i.d. $\sim N(0,1)$。

第 2 步：构造正交矩阵。找一个 $n \times n$ 的正交矩阵 $A$，使其第一行为

$$\left(\frac{1}{\sqrt{n}},\; \frac{1}{\sqrt{n}},\; \ldots,\; \frac{1}{\sqrt{n}}\right)$$

这样的矩阵一定存在吗？是的。第一行已经是单位向量（长度 = $\sqrt{n \cdot (1/n)} = 1$）。根据线性代数中的 Gram-Schmidt 正交化过程，我们可以从这个向量出发，构造出 $n-1$ 个与之正交的单位向量，填满矩阵的其余 $n-1$ 行。所得矩阵就是正交矩阵。

（你不需要写出 $A$ 的具体形式——只需知道它存在。这就像你知道平面旋转 45° 的矩阵是 $\frac{1}{\sqrt{2}}\begin{pmatrix}1&1\\-1&1\end{pmatrix}$，高维情形同理。）

我们将 $A$ 的行记为 $a_1, a_2, \ldots, a_n$，其中 $a_1 = (\frac{1}{\sqrt{n}}, \ldots, \frac{1}{\sqrt{n}})$。由于 $A$ 是正交矩阵，满足：

  • 每行长度为 1：$\sum_{j=1}^n a_{ij}^2 = 1$，对所有 $i$
  • 不同行正交：$\sum_{j=1}^n a_{ij} a_{kj} = 0$，对所有 $i \neq k$

第 3 步：做正交变换。令 $Y = AZ$，即

$$Y_i = \sum_{j=1}^n a_{ij} Z_j, \quad i = 1, 2, \ldots, n$$

由事实 3，$Y_1, Y_2, \ldots, Y_n$ 仍然是 i.i.d. $\sim N(0,1)$。特别地，$Y_1, \ldots, Y_n$ 相互独立。

第 4 步：识别 $Y_1$。$Y_1$ 是第一行 $a_1$ 和 $Z$ 的内积：

$$Y_1 = \sum_{j=1}^n \frac{1}{\sqrt{n}} Z_j = \frac{1}{\sqrt{n}} \sum_{j=1}^n \frac{X_j - \mu}{\sigma} = \frac{\sqrt{n}(\bar{X} - \mu)}{\sigma}$$

所以 $Y_1$ 完全由 $\bar{X}$ 决定——它就是标准化后的样本均值。

第 5 步：平方和恒等式。

由事实 2（正交变换保长度），$\sum_{i=1}^n Y_i^2 = \sum_{i=1}^n Z_i^2$。

因此：

\begin{align*} \sum_{i=2}^n Y_i^2 &= \sum_{i=1}^n Y_i^2 - Y_1^2 \\ &= \sum_{i=1}^n Z_i^2 - \left(\frac{\sqrt{n}(\bar{X} - \mu)}{\sigma}\right)^2 \\ &= \sum_{i=1}^n \frac{(X_i - \mu)^2}{\sigma^2} - \frac{n(\bar{X} - \mu)^2}{\sigma^2} \end{align*}

这里用到一个关键的代数恒等式（展开即证）：

$$\sum_{i=1}^n (X_i - \mu)^2 = \sum_{i=1}^n (X_i - \bar{X})^2 + n(\bar{X} - \mu)^2$$

验证：$\sum (X_i - \mu)^2 = \sum [(X_i - \bar{X}) + (\bar{X} - \mu)]^2$，展开后交叉项 $\sum 2(X_i - \bar{X})(\bar{X} - \mu) = 2(\bar{X} - \mu)\sum (X_i - \bar{X}) = 2(\bar{X} - \mu) \cdot 0 = 0$。

将恒等式代入上式：

\begin{align*} \sum_{i=2}^n Y_i^2 &= \frac{\sum_{i=1}^n (X_i - \bar{X})^2 + n(\bar{X} - \mu)^2}{\sigma^2} - \frac{n(\bar{X} - \mu)^2}{\sigma^2} \\ &= \frac{\sum_{i=1}^n (X_i - \bar{X})^2}{\sigma^2} = \frac{(n-1)S^2}{\sigma^2} \end{align*}

第 6 步：得出结论。

  • 由于 $Y_2, \ldots, Y_n$ i.i.d. $\sim N(0,1)$，由 $\chi^2$ 分布的定义： $$\sum_{i=2}^n Y_i^2 \sim \chi^2(n-1) \quad \Longrightarrow \quad \frac{(n-1)S^2}{\sigma^2} \sim \chi^2(n-1)$$
  • 由于 $Y_1, Y_2, \ldots, Y_n$ 相互独立，$Y_1$（只含 $\bar{X}$）与 $Y_2, \ldots, Y_n$（决定 $S^2$）独立，因此 $\bar{X}$ 与 $S^2$ 相互独立。

证毕。$\quad\blacksquare$

为什么自由度是 $n-1$？——两种理解方式

方式一（代数）：$n$ 个偏差 $(X_i - \bar{X})$ 满足一个线性约束 $\sum_{i=1}^n (X_i - \bar{X}) = 0$，所以实际上只有 $n-1$ 个「自由变化的量」。在上面证明中，这体现在 $Y_2, \ldots, Y_n$ 正好有 $n-1$ 个独立的标准正态变量。

方式二（几何）：向量 $(X_1 - \bar{X}, \ldots, X_n - \bar{X})$ 与向量 $(1, 1, \ldots, 1)$ 正交（内积为零）。在 $n$ 维空间中，与给定方向正交的子空间是 $n-1$ 维的。正交变换 $Y = AZ$ 本质上把 $Z$ 投影到了由 $A$ 的行张成的坐标轴上——$Y_1$ 对应 $(1,1,\ldots,1)$ 方向（$\bar{X}$ 的方向），而 $Y_2, \ldots, Y_n$ 张成与之正交的 $n-1$ 维子空间（偏差的方向）。

定理 3：$t$ 分布的构造

定理 3

设 $X_1, \ldots, X_n$ 是来自 $N(\mu, \sigma^2)$ 的样本，则

$$\frac{\bar{X}-\mu}{S/\sqrt{n}} \sim t(n-1)$$
证明

由定理 1：$Z = \frac{\bar{X}-\mu}{\sigma/\sqrt{n}} \sim N(0,1)$

由定理 2：$V = \frac{(n-1)S^2}{\sigma^2} \sim \chi^2(n-1)$，且 $Z$ 与 $V$ 独立。

由 $t$ 分布的定义：

$$\frac{Z}{\sqrt{V/(n-1)}} = \frac{\frac{\bar{X}-\mu}{\sigma/\sqrt{n}}}{\sqrt{\frac{(n-1)S^2}{\sigma^2(n-1)}}} = \frac{\bar{X}-\mu}{S/\sqrt{n}} \sim t(n-1)$$
定理 3 的应用场景

当总体方差 $\sigma^2$ 未知时，用样本标准差 $S$ 代替 $\sigma$，统计量从标准正态变为 $t(n-1)$。这是实际中最常用的情形——因为真实场景中 $\sigma$ 通常是未知的。

定理 4：两个正态总体的 $F$ 分布

定理 4

设 $X_1, \ldots, X_m$ 是来自 $N(\mu_1, \sigma_1^2)$ 的样本，$Y_1, \ldots, Y_n$ 是来自 $N(\mu_2, \sigma_2^2)$ 的样本，两组样本相互独立。$\bar{X}, S_1^2$ 和 $\bar{Y}, S_2^2$ 分别为两组的样本均值和方差，则：

$$\frac{S_1^2/\sigma_1^2}{S_2^2/\sigma_2^2} \sim F(m-1, n-1)$$
证明

由定理 2：

$\frac{(m-1)S_1^2}{\sigma_1^2} \sim \chi^2(m-1)$，$\frac{(n-1)S_2^2}{\sigma_2^2} \sim \chi^2(n-1)$

两组样本独立，故这两个 $\chi^2$ 变量独立。

由 $F$ 分布定义：

$$\frac{\frac{(m-1)S_1^2}{\sigma_1^2}\bigg/(m-1)}{\frac{(n-1)S_2^2}{\sigma_2^2}\bigg/(n-1)} = \frac{S_1^2/\sigma_1^2}{S_2^2/\sigma_2^2} \sim F(m-1, n-1)$$
特别地：检验方差是否相等

当 $\sigma_1^2 = \sigma_2^2$ 时，$\frac{S_1^2}{S_2^2} \sim F(m-1, n-1)$。这是检验两总体方差是否相等（$H_0: \sigma_1^2 = \sigma_2^2$）的基础。

总结：四个定理的对比

▶ 例题 · 抽样分布定理的综合应用

题目：设 $X_1, \ldots, X_{16}$ 是来自 $N(5, 4)$ 的样本，求 $P(\bar{X} > 5.5)$。

解：$\bar{X} \sim N(5, 4/16) = N(5, 0.25)$，$\sigma_{\bar{X}} = 0.5$。

$P(\bar{X} > 5.5) = P\!\left(\frac{\bar{X}-5}{0.5} > \frac{5.5-5}{0.5}\right) = P(Z > 1) = 1 - \Phi(1) \approx 0.1587$

▶ 例题 · 求样本方差的概率

题目：设 $X_1, \ldots, X_{10}$ 是来自 $N(0, \sigma^2)$ 的样本，求 $P\!\left(\frac{S^2}{\sigma^2} \leq 2.09\right)$。

解：由定理 2，$\frac{(n-1)S^2}{\sigma^2} \sim \chi^2(9)$，即 $\frac{9S^2}{\sigma^2} \sim \chi^2(9)$。

$P\!\left(\frac{S^2}{\sigma^2} \leq 2.09\right) = P\!\left(\frac{9S^2}{\sigma^2} \leq 9 \times 2.09\right) = P(\chi^2(9) \leq 18.81)$

查 $\chi^2$ 分布表，$\chi^2_{0.05}(9) = 16.919$，$\chi^2_{0.025}(9) = 19.023$。$18.81$ 介于两者之间，故 $P \approx 0.97$ 左右。
